# మార్పుల చరిత్ర: MCP ప్రారంభికుల కోసం పాఠ్యక్రమం

ఈ డాక్యుమెంట్ మోడల్ కాంటెక్స్ట్ ప్రోటోకాల్ (MCP) ప్రారంభికుల కోసం పాఠ్యక్రమంలో చేసిన అన్ని ముఖ్యమైన మార్పుల రికార్డ్‌గా పనిచేస్తుంది. మార్పులు రివర్స్ క్రోనాలాజికల్ ఆర్డర్‌లో డాక్యుమెంట్ చేయబడ్డాయి (తాజా మార్పులు ముందుగా).

## అక్టోబర్ 6, 2025

### ప్రారంభం సెక్షన్ విస్తరణ – అధునాతన సర్వర్ ఉపయోగం & సులభమైన ధృవీకరణ

#### అధునాతన సర్వర్ ఉపయోగం (03-GettingStarted/10-advanced)
- **కొత్త అధ్యాయం జోడించబడింది**: సాధారణ మరియు లో-లెవల్ సర్వర్ ఆర్కిటెక్చర్లను కవర్ చేసే అధునాతన MCP సర్వర్ ఉపయోగానికి సమగ్ర గైడ్ పరిచయం చేయబడింది.
  - **సాధారణ vs. లో-లెవల్ సర్వర్**: రెండు విధానాల కోసం Python మరియు TypeScript లో వివరమైన తులన మరియు కోడ్ ఉదాహరణలు.
  - **హ్యాండ్లర్-ఆధారిత డిజైన్**: స్కేలబుల్, ఫ్లెక్సిబుల్ సర్వర్ అమలులకు హ్యాండ్లర్-ఆధారిత టూల్/వనరు/ప్రాంప్ట్ నిర్వహణ వివరణ.
  - **ప్రాక్టికల్ ప్యాటర్న్లు**: అధునాతన ఫీచర్లు మరియు ఆర్కిటెక్చర్ కోసం లో-లెవల్ సర్వర్ ప్యాటర్న్లు ఉపయోగపడే వాస్తవ ప్రపంచ పరిస్థితులు.

#### సులభమైన ధృవీకరణ (03-GettingStarted/11-simple-auth)
- **కొత్త అధ్యాయం జోడించబడింది**: MCP సర్వర్లలో సులభమైన ధృవీకరణను అమలు చేయడానికి దశల వారీ గైడ్.
  - **ధృవీకరణ భావనలు**: ధృవీకరణ మరియు అనుమతుల మధ్య స్పష్టమైన వివరణ, మరియు క్రెడెన్షియల్ నిర్వహణ.
  - **బేసిక్ ఆథ్ అమలు**: Python (Starlette) మరియు TypeScript (Express) లో మిడిల్వేర్-ఆధారిత ధృవీకరణ ప్యాటర్న్లు, కోడ్ నమూనాలతో.
  - **అధునాతన భద్రతకు ప్రగతి**: సులభమైన ఆథ్ తో ప్రారంభించి OAuth 2.1 మరియు RBAC కి ఎలా ముందుకు పోవాలో మార్గదర్శనం, అధునాతన భద్రతా మాడ్యూల్స్ కు సూచనలు.

ఈ జోడింపులు మరింత బలమైన, భద్రతగల, మరియు ఫ్లెక్సిబుల్ MCP సర్వర్ అమలులకు ప్రాక్టికల్, హ్యాండ్స్-ఆన్ మార్గదర్శకాన్ని అందిస్తాయి, ప్రాథమిక భావనలను అధునాతన ఉత్పత్తి ప్యాటర్న్లతో కలుపుతూ.

## సెప్టెంబర్ 29, 2025

### MCP సర్వర్ డేటాబేస్ ఇంటిగ్రేషన్ ల్యాబ్స్ - సమగ్ర హ్యాండ్స్-ఆన్ నేర్చుకునే మార్గం

#### 11-MCPServerHandsOnLabs - కొత్త పూర్తి డేటాబేస్ ఇంటిగ్రేషన్ పాఠ్యక్రమం
- **పూర్తి 13-ల్యాబ్ నేర్చుకునే మార్గం**: PostgreSQL డేటాబేస్ ఇంటిగ్రేషన్ తో ప్రొడక్షన్-రెడీ MCP సర్వర్లు నిర్మించడానికి సమగ్ర హ్యాండ్స్-ఆన్ పాఠ్యక్రమం జోడించబడింది
  - **వాస్తవ ప్రపంచ అమలు**: ఎంటర్‌ప్రైజ్-గ్రేడ్ ప్యాటర్న్లను చూపించే Zava రిటైల్ అనలిటిక్స్ ఉపయోగ కేసు
  - **సంఘటిత నేర్చుకునే ప్రగతి**:
    - **ల్యాబ్స్ 00-03: పునాది** - పరిచయం, కోర్ ఆర్కిటెక్చర్, భద్రత & మల్టీ-టెనెన్సీ, వాతావరణ సెటప్
    - **ల్యాబ్స్ 04-06: MCP సర్వర్ నిర్మాణం** - డేటాబేస్ డిజైన్ & స్కీమా, MCP సర్వర్ అమలు, టూల్ అభివృద్ధి  
    - **ల్యాబ్స్ 07-09: అధునాతన ఫీచర్లు** - సెమాంటిక్ సెర్చ్ ఇంటిగ్రేషన్, టెస్టింగ్ & డీబగ్గింగ్, VS కోడ్ ఇంటిగ్రేషన్
    - **ల్యాబ్స్ 10-12: ఉత్పత్తి & ఉత్తమ ఆచారాలు** - డిప్లాయ్‌మెంట్ వ్యూహాలు, మానిటరింగ్ & ఆబ్జర్వబిలిటీ, ఉత్తమ ఆచారాలు & ఆప్టిమైజేషన్
  - **ఎంటర్‌ప్రైజ్ సాంకేతికతలు**: FastMCP ఫ్రేమ్‌వర్క్, pgvector తో PostgreSQL, Azure OpenAI ఎంబెడ్డింగ్స్, Azure కంటైనర్ యాప్స్, అప్లికేషన్ ఇన్సైట్స్
  - **అధునాతన ఫీచర్లు**: రో లెవల్ సెక్యూరిటీ (RLS), సెమాంటిక్ సెర్చ్, మల్టీ-టెనెంట్ డేటా యాక్సెస్, వెక్టర్ ఎంబెడ్డింగ్స్, రియల్-టైమ్ మానిటరింగ్

#### పదజాలం ప్రమాణీకరణ - మాడ్యూల్ నుండి ల్యాబ్ మార్పిడి
- **సమగ్ర డాక్యుమెంటేషన్ నవీకరణ**: 11-MCPServerHandsOnLabs లోని అన్ని README ఫైళ్లను "Module" పదం స్థానంలో "Lab" పదాన్ని ఉపయోగించేలా సిస్టమాటిక్‌గా నవీకరించబడింది
  - **సెక్షన్ హెడ్డర్లు**: అన్ని 13 ల్యాబ్స్ లో "What This Module Covers" ను "What This Lab Covers" గా మార్చారు
  - **కంటెంట్ వివరణ**: "This module provides..." ను "This lab provides..." గా మార్చారు
  - **నైపుణ్య లక్ష్యాలు**: "By the end of this module..." ను "By the end of this lab..." గా నవీకరించారు
  - **నావిగేషన్ లింకులు**: అన్ని "Module XX:" సూచనలను "Lab XX:" గా మార్చారు
  - **పూర్తి ట్రాకింగ్**: "After completing this module..." ను "After completing this lab..." గా మార్చారు
  - **సాంకేతిక సూచనలు నిలుపుదల**: కాన్ఫిగరేషన్ ఫైళ్లలో Python మాడ్యూల్ సూచనలు (ఉదా: `"module": "mcp_server.main"`) నిలుపుకున్నారు

#### MCP కస్టమ్ ట్రాన్స్‌పోర్ట్స్ (05-AdvancedTopics/mcp-transport/) - కొత్త అధునాతన అమలు గైడ్

  - **Azure ఈవెంట్ గ్రిడ్ ట్రాన్స్‌పోర్ట్**: సమగ్ర సర్వర్‌లెస్ ఈవెంట్-డ్రివెన్ ట్రాన్స్‌పోర్ట్ అమలు
    - C#, TypeScript, మరియు Python ఉదాహరణలు Azure Functions ఇంటిగ్రేషన్ తో
    - స్కేలబుల్ MCP పరిష్కారాల కోసం ఈవెంట్-డ్రివెన్ ఆర్కిటెక్చర్ ప్యాటర్న్లు
    - వెబ్‌హుక్ రిసీవర్స్ మరియు పుష్-ఆధారిత సందేశ నిర్వహణ
  - **Azure ఈవెంట్ హబ్ ట్రాన్స్‌పోర్ట్**: హై-త్రూపుట్ స్ట్రీమింగ్ ట్రాన్స్‌పోర్ట్ అమలు
    - తక్కువ లేటెన్సీ పరిస్థితుల కోసం రియల్-టైమ్ స్ట్రీమింగ్ సామర్థ్యాలు
    - పార్టిషనింగ్ వ్యూహాలు మరియు చెక్‌పాయింట్ నిర్వహణ
    - సందేశ బ్యాచింగ్ మరియు పనితీరు ఆప్టిమైజేషన్

    - బహుళ Azure Functions మధ్య పంపిణీ MCP ప్రాసెసింగ్
    - బహుళ ట్రాన్స్‌పోర్ట్ రకాల కలయికతో హైబ్రిడ్ ట్రాన్స్‌పోర్ట్ ఆర్కిటెక్చర్లు

    - అప్లికేషన్ ఇన్సైట్స్ టెలిమెట్రీ మరియు పనితీరు మానిటరింగ్
    - సర్క్యూట్ బ్రేకర్స్ మరియు లోప సహన ప్యాటర్న్లు

  - క్లయింట్ ప్రిమిటివ్‌లు: శాంప్లింగ్ (LLM పూర్తి), ఎలిసిటేషన్ (వినియోగదారు ఇన్‌పుట్), లాగింగ్ (డీబగ్గింగ్/మానిటరింగ్)
  - ప్రస్తుత డిస్కవరీ (`*/list`), రిట్రీవల్ (`*/get`), మరియు ఎగ్జిక్యూషన్ (`*/call`) పద్ధతుల నమూనాలతో నవీకరించబడింది
- **ప్రోటోకాల్ ఆర్కిటెక్చర్**: రెండు-పట్టాల ఆర్కిటెక్చర్ మోడల్ ప్రవేశపెట్టబడింది
  - డేటా లేయర్: JSON-RPC 2.0 ఆధారం, లైఫ్‌సైకిల్ నిర్వహణ మరియు ప్రిమిటివ్‌లు
  - ట్రాన్స్‌పోర్ట్ లేయర్: STDIO (స్థానిక) మరియు స్ట్రీమబుల్ HTTP SSE (రిమోట్) ట్రాన్స్‌పోర్ట్ మెకానిజమ్స్
- **సెక్యూరిటీ ఫ్రేమ్‌వర్క్**: స్పష్టమైన వినియోగదారు అనుమతి, డేటా గోప్యత రక్షణ, టూల్ ఎగ్జిక్యూషన్ భద్రత, మరియు ట్రాన్స్‌పోర్ట్ లేయర్ సెక్యూరిటీ సహా సమగ్ర సెక్యూరిటీ సూత్రాలు
- **కమ్యూనికేషన్ నమూనాలు**: ప్రారంభం, డిస్కవరీ, ఎగ్జిక్యూషన్, మరియు నోటిఫికేషన్ ప్రవాహాలను చూపించే ప్రోటోకాల్ సందేశాలు నవీకరించబడ్డాయి
- **కోడ్ ఉదాహరణలు**: ప్రస్తుత MCP SDK నమూనాలను ప్రతిబింబించే బహుభాషా ఉదాహరణలు (.NET, Java, Python, JavaScript) రిఫ్రెష్ చేయబడ్డాయి

#### సెక్యూరిటీ (02-Security/) - సమగ్ర సెక్యూరిటీ పునఃరూపకల్పన  
- **ప్రమాణాల సరిపోలిక**: MCP స్పెసిఫికేషన్ 2025-06-18 సెక్యూరిటీ అవసరాలతో పూర్తి సరిపోలిక
- **ధృవీకరణ పరిణామం**: కస్టమ్ OAuth సర్వర్ల నుండి బాహ్య ఐడెంటిటీ ప్రొవైడర్ డెలిగేషన్ (Microsoft Entra ID) వరకు పరిణామాన్ని డాక్యుమెంట్ చేయడం
- **AI-స్పెసిఫిక్ ముప్పు విశ్లేషణ**: ఆధునిక AI దాడి వెక్టర్ల మెరుగైన కవర్
  - వాస్తవ ప్రపంచ ఉదాహరణలతో వివరమైన ప్రాంప్ట్ ఇంజెక్షన్ దాడి సన్నివేశాలు
  - టూల్ విషపూరణ మెకానిజమ్స్ మరియు "రగ్ పుల్" దాడి నమూనాలు
  - కాంటెక్స్ట్ విండో విషపూరణ మరియు మోడల్ గందరగోళ దాడులు
- **Microsoft AI సెక్యూరిటీ పరిష్కారాలు**: Microsoft సెక్యూరిటీ ఎకోసిస్టమ్ సమగ్ర కవర్
  - AI Prompt Shields అధునాతన గుర్తింపు, స్పాట్‌లైటింగ్, మరియు డెలిమిటర్ సాంకేతికతలతో
  - Azure Content Safety ఇంటిగ్రేషన్ నమూనాలు
  - సరఫరా గొలుసు రక్షణ కోసం GitHub Advanced Security
- **అధునాతన ముప్పు నివారణ**: వివరమైన సెక్యూరిటీ నియంత్రణలు
  - MCP-స్పెసిఫిక్ దాడి సన్నివేశాలు మరియు క్రిప్టోగ్రాఫిక్ సెషన్ ID అవసరాలతో సెషన్ హైజాకింగ్
  - MCP ప్రాక్సీ సన్నివేశాలలో Confused Deputy సమస్యలు స్పష్టమైన అనుమతి అవసరాలతో
  - తప్పనిసరి ధృవీకరణ నియంత్రణలతో టోకెన్ పాస్స్త్రూ లోపాలు
- **సరఫరా గొలుసు సెక్యూరిటీ**: ఫౌండేషన్ మోడల్స్, ఎంబెడ్డింగ్స్ సర్వీసులు, కాంటెక్స్ట్ ప్రొవైడర్లు, మరియు మూడవ పక్ష API లతో విస్తృత AI సరఫరా గొలుసు కవర్
- **ఫౌండేషన్ సెక్యూరిటీ**: జీరో ట్రస్ట్ ఆర్కిటెక్చర్ మరియు Microsoft సెక్యూరిటీ ఎకోసిస్టమ్ సహా ఎంటర్ప్రైజ్ సెక్యూరిటీ నమూనాలతో మెరుగైన ఇంటిగ్రేషన్
- **వనరు సంస్థీకరణ**: రకరకాల వనరు లింకులను (అధికారిక డాక్స్, ప్రమాణాలు, పరిశోధన, Microsoft పరిష్కారాలు, అమలు మార్గదర్శకాలు) వర్గీకరించడం
