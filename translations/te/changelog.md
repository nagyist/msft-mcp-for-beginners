ఈ డాక్యుమెంట్ మోడల్ కాంటెక్స్ట్ ప్రోటోకాల్ (MCP) ప్రారంభకుల కోర్సు లో చేసిన ముఖ్యమైన మార్పులను రికార్డుగా భద్రం చేస్తుంది. మార్పులు కాలాతీత క్రమంలో (ఇటీవల వివరాలు మొదట) డాక్యుమెంట్ చేయబడ్డాయి.

## జనవరి 28, 2026

### MCP స్పెసిఫికేషన్ 2025-11-25 అనుగుణత సమీక్ష

#### కోర్ కాన్సెప్ట్స్ అభివృద్ధి (01-CoreConcepts/)
- **కొత్త క్లయింట్ ప్రిమిటివ్ - Roots**: రూట్ క్లయింట్ ప్రిమిటివ్ పై సమగ్ర డాక్యుమెంటేషన్ జత చేయబడింది; సర్వర్లు ఫైల్ సిస్టమ్ సరిహద్దులు మరియు అనుమతులు అర్థం చేసుకోవడానికి వీలు
- **టూల్ అనోటేషన్లు**: `readOnlyHint`, `destructiveHint` వంటి టూల్ ప్రవర్తన అనోటేషన్ల డాక్యుమెంటేషన్ జత చేయబడింది, టూల్ అమలులో మెరుగైన నిర్ణయాలకు సహాయపడేందుకు
- **సాంప్లింగ్‌లో టూల్ కాలింగ్**: సాంప్లింగ్ డాక్యుమెంటేషన్ ను `tools` మరియు `toolChoice` పారామీటర్లను కలిపి నవీకరించబడింది, మోడల్ ఆధారిత టూల్ పిలవడం కోసం
- **URL మోడ్ ఎలిసిటేషన్**: సర్వర్ ప్రారంభించిన బాహ్య వెబ్ ఇంటరాక్షన్ల కోసం URL ఆధారిత ఎలిసిటేషన్ డాక్యుమెంటేషన్ జత చేయబడింది
- **టాస్కులు (ప్రయోగాత్మక)**: నిరంతరం అమలుకు ర్యాపర్స్ మరియు వాయిదా విడివిడిగా ఫలితం పొందేందుకు ప్రయోగాత్మక టాస్కుల ఫీచర్ డాక్యుమెంటేషన్ జత చేయబడింది
- **ఐకాన్ల మద్దతు**: టూల్స్, వనరులు, వనరు టెంప్లేట్లు, ప్రాంప్ట్లు ఇప్పుడు అదనపు మెటాడేటాగా ఐకాన్లను కలిగి ఉండవచ్చు

#### డాక్యుమెంటేషన్ నవీకరణలు
- **README.md**: MCP స్పెసిఫికేషన్ 2025-11-25 వెర్షన్ సూచన మరియు తేదీ ఆధారిత వెర్షనింగ్ వివరణ జత చేయబడింది
- **study_guide.md**: కొత్త టాస్కులు మరియు టూల్ అనోటేషన్స్ ని కోర్ కాన్సెప్ట్స్ విభాగంలో చేర్చుతూ కోర్సు మ్యాప్ నవీకరించబడింది; డాక్యుమెంట్ టైమ్‌స్టాంప్ నవీకరణ

#### స్పెసిఫికేషన్ అనుగుణత ధృవీకరణ
- **ప్రోటోకాల్ వెర్షన్**: అన్ని డాక్యుమెంట్స్ ప్రస్తుత MCP స్పెసిఫికేషన్ 2025-11-25 ను సూచిస్తున్నాయని నిర్ధారించబడింది
- **ఆర్కిటెక్చర్ సరళి**: రెండు-స్థాయి నిర్మాణం (డేటా లేయర్ + ట్రాన్స్‌పోర్ట్ లేయర్) డాక్యుమెంటేషన్ ఖచ్చితంగా సరిపోతుందనే ధృవీకరణ
- **ప్రిమిటివ్స్ డాక్యుమెంటేషన్**: సర్వర్ ప్రిమిటివ్స్ (వనరులు, ప్రాంప్ట్లు, టూల్స్) మరియు క్లయింట్ ప్రిమిటివ్స్ (సాంప్లింగ్, ఎలిసిటేషన్, లాగింగ్, రూట్స్) సరైన డాక్యుమెంటేషన్
- **ట్రాన్స్‌పోర్ట్ మెకానిజమ్స్**: STDIO మరియు స్ట్రీమబుల్ HTTP ట్రాన్స్‌పోర్ట్ డాక్యుమెంటేషన్ ఖచ్చిత పరిశీలన
- **భద్రత మార్గదర్శకత**: ప్రస్తుత MCP భద్రత ఉత్తమ పద్ధతుల డాక్యుమెంటేషన్ సరిపోలిక నిర్ధారింపు

#### ప్రధాన MCP 2025-11-25 ఫీచర్లు డాక్యుమెంటేషన్
- **OpenID కనెక్ట్ డిస్కవరీ**: OIDC ద్వారా అథర్ సర్వర్ గుర్తింపు
- **OAuth క్లయింట్ ID మెటాడేటా డాక్యుమెంట్లు**: క్లయింట్ రిజిస్ట్రేషన్ యంత్రాంగం యొక్క సిఫార్సు
- **JSON స్కీమా 2020-12**: MCP స్కీమా నిర్వచనాలకు డిఫాల్ట్ డైలెక్ట్
- **SDK టియరింగ్ సిస్టమ్**: SDK ఫీచర్ మద్దతు మరియు నిర్వహణ అవసరాల ఫార్మల్ డిఫినిషన్

- **SDK డాక్యుమెంటేషన్**: అధికారిక SDK జాబితాలో Go SDK జత చేయబడింది; అన్ని SDK సూచనలు MCP స్పెసిఫికేషన్ 2025-11-25 కి సరిపోల్చి నవీకరించబడ్డాయి
- **ట్రాన్స్‌పోర్ట్ స్పష్టత**: STDIO మరియు HTTP స్ట్రీమింగ్ ట్రాన్స్‌పోర్ట్ వివరణలను స్పష్టమైన స్పెక్ సూచనలతో నవీకరించబడింది

#### మాడ్యూల్ 04 - ప్రాక్టికల్ అమలు
- **SDK నవీకరణలు**: Go SDK జత చేయబడింది; SDK జాబితా స్పెసిఫికేషన్ వెర్షన్ సూచనతో నవీకరించబడింది
- **అధికారం స్పెక్**: MCP అధికారం స్పెసిఫికేషన్ లింక్ ప్రస్తుత 2025-11-25 వెర్షన్ కు నవీకరించబడింది

#### మాడ్యూల్ 05 - అడ్వాన్స్డ్ టాపిక్స్
- **కొత్త ఫీచర్లు**: MCP స్పెసిఫికేషన్ 2025-11-25 లో కొత్త ఫీచర్లు (టాస్కులు, టూల్ అనోటేషన్లు, URL మోడ్ ఎలిసిటేషన్, రూట్స్) గురించిన నోట్సు జత చేయబడింది
- **భద్రత వనరులు**: OWASP MCP టాప్ 10 మరియు షెర్పా వర్క్‌షాప్ లింకులు అదనపు రిఫరెన్స్‌లకు జత చేయబడ్డాయి

#### మాడ్యూల్ 06 - కమ్యూనిటీ కాంట్రిబ్యూషన్స్

#### మాడ్యూల్ 07 - ప్రారంభ అంగీకారంలో పాఠాలు

  - **Auth కాన్సెప్ట్స్**: ప్రమాణీకరణ, అధికారం మరియు క్రెడెన్షియల్ నిర్వహణ స్పష్టమైన వివరణ  
  - **ప్రాథమిక Auth అమలు**: Python (Starlette) మరియు TypeScript (Express) లో మిడిల్‌వేర్ ఆధారిత ప్రమాణీకరణ ప్యాటర్న్లు, కోడ్ నమూనెలతో  
  - **అధునాతన భద్రతకు ప్రగతి**: సరళమైన auth నుంచి OAuth 2.1 మరియు RBAC వైపు మారగలిగే మార్గదర్శకత, అడ్వాన్స్డ్ భద్రతా మాడ్యూల్స్ సూచనలతో

ఈ జోడింపులు మరింత స్థిరమైన, భద్రతా పరమైన, మరియు అనువైన MCP సర్వర్ అమలులకు ప్రాక్టికల్ మరియు హ్యాండ్స్-ఆన్ మార్గదర్శకతను అందిస్తాయి, ప్రాథమిక కాన్సెప్ట్లు తో అడ్వాన్స్డ్ ఉత్పత్తి నమూనాలను కలిపి.

    - AI-స్పెసిఫిక్ సెక్యూరిటీ నియంత్రణలు, ప్రాంప్ట్ ఇంజెక్షన్ & టూల్ పాయిజనింగ్ నివారణ  
    - Confused Deputy దాడి నివారణ OAuth ప్రాక్సీ సెక్యూరిటీతో  
    - టూల్ ఎగ్జిక్యూషన్ సెక్యూరిటీ, సాండ్‌బాక్సింగ్ & ఐసోలేషన్  
    - సప్లై చైన్ సెక్యూరిటీ నియంత్రణలు డిపెండెన్సీ వెరిఫికేషన్‌తో  
    - మానిటరింగ్ & డిటెక్షన్ నియంత్రణలు SIEM ఇంటిగ్రేషన్‌తో  
    - ఘటన ప్రతిస్పందన & పునరుద్ధరణ స్వయంచాలకతతో  
  - **అమలీకరణ ఉదాహరణలు**: సమగ్ర YAML కాన్ఫిగరేషన్ బ్లాక్స్ మరియు కోడ్ ఉదాహరణలు చేర్చనివి  
  - **Microsoft పరిష్కారాల ఇంటిగ్రేషన్**: Azure సెక్యూరిటీ సేవలు, GitHub Advanced Security, ఎంటర్ప్రైజ్ ఐడెంటిటీ మేనేజ్‌మెంట్ కవరేజ్  

#### ఆధునిక అంశాల సెక్యూరిటీ (05-AdvancedTopics/mcp-security/) - ప్రొడక్షన్-రెడీ అమలు  
- **README.md**: ఎంటర్ప్రైజ్ సెక్యూరిటీ అమలుకోసం పూర్తి పునఃరచన  
  - **ప్రస్తుతం స్పెసిఫికేషన్ అనుగుణత**: MCP Specification 2025-06-18 లోని మినహాయింపు అవసరాలకు సరి  
  - **వృద్ధిచేసిన ఆథెంటికేషన్**: Microsoft Entra ID ఇంటిగ్రేషన్ .NET & Java Spring Security ఉదాహరణలతో  
  - **AI సెక్యూరిటీ ఇంటిగ్రేషన్**: Microsoft Prompt Shields, Azure Content Safety పై Python లో వివరమైన అమలు  
  - **ఆధునిక థ్రెట్ నివారణ**:  
    - PKCE & యూజర్ కన్సెంట్ ధృవీకరణతో Confused Deputy దాడి నివారణ  
    - ఆడియెన్స్ ధృవీకరణ మరియు సురక్షిత టోకెన్ నిర్వహణతో టోకెన్ పాస్‌త్రూ నివారణ  
    - క్రిప్టోగ్రాఫిక్ బైండింగ్ మరియు ప్రవర్తనా విశ్లేషణతో సెషన్ హైజాకింగ్ నివారణ  
  - **ఎంటర్ప్రైజ్ సెక్యూరిటీ ఇంటిగ్రేషన్**: Azure అప్లికేషన్ ఇన్సైట్స్‌మానిటరింగ్, ముప్పు గుర్తింపు పైప్లైన్లు, సప్లై చైన్ సెక్యూరిటీ  
  - **అమలీకరణ చెక్లిస్ట్**: స్పష్టమైన మినహాయింపులు vs. సిఫార్సు చేసే నియంత్రణలు Microsoft సెక్యూరిటీ ఎకోసిస్టమ్ లాభాలతో  

### డాక్యుమెంటేషన్ నాణ్యత & ప్రమాణాల అనుగుణత  
- **స్పెసిఫికేషన్ సూచనలు**: MCP Specification 2025-06-18 కి అన్ని రిఫరెన్స్‌లను నవీకరించడం  
- **Microsoft సెక్యూరిటీ ఎకోసిస్టమ్**: అన్ని సెక్యూరిటీ డాక్యుమెంటేషన్ విదులలో ఇంటిగ్రేషన్ మార్గదర్శకాలు మెరుగుపరచడం  
- **ప్రాయోగిక అమలు**: .NET, Java, Python లో సమగ్ర కోడ్ ఉదాహరణలతో ఎంటర్ప్రైజ్ నమూనాలు  
- **వనరు వ్యవస్థీకరణ**: అధికారిక డాక్యుమెంటేషన్, సెక్యూరిటీ ప్రమాణాలు, అమలీకరణ గైడ్‌ల విభజన  
- **దృశ్య సూచనలు**: తప్పనిసరి అవసరాలు vs. సిఫార్సు విధానాల స్పష్టమైన గుర్తింపులు  

#### కోర్ కాన్సెప్ట్స్ (01-CoreConcepts/) - పూర్తి ఆధునీకరణ  
- **ప్రోటోకాల్ వర్షన్ అప్డేట్**: MCP Specification 2025-06-18 ఆధారంగా తేదీ ఆధారిత వర్షన్ (YYYY-MM-DD ఫార్మాట్)  
- **ఆర్కిటెక్చర్ మెరుగుదల**: హోస్ట్స్, క్లయింట్లు, సర్వర్ల ప్రస్తుత MCP నిర్మాణ నమూనాలతో వివరాలను పెంపొందించడం
  - హోస్ట్స్ ఇప్పుడు స్పష్టంగా అనేక MCP క్లయింట్ కనెక్షన్స్‌ను సమన్వయ పరచే AI అప్లికేషన్‌లుగా నిర్వచించబడ్డాయి
  - క్లయింట్లు ఒకటో ఒకటో సర్వర్ సంబంధాలను నిర్వహించే ప్రోటోకాల్ కనెక్టర్‌లుగా वर्णించబడ్డాయి
  - సర్వర్లు స్థానిక మరియు దూరం (remote) అమలు సందర్భాలతో మెరుగుపరచబడ్డాయి
- **ప్రాథమిక పునర్గఠనం**: సర్వర్ మరియు క్లయింట్ ప్రాథమిక అంశాల పూర్తిపూర్వక మళ్లింపు
  - సర్వర్ ప్రాథమికాలు: వనరులు (డేటా మూలాలు), ప్రాంప్ట్స్ (టెంప్లెట్లు), టూల్స్ (నిర్వహించదగిన ఫంక్షన్లు) వివరణలు మరియు ఉదాహరణలతో
  - క్లయింట్ ప్రాథమికాలు: శాంప్లింగ్ (LLM పూర్తికరణ), elicitation (వినియోగదారు ఇన్‌పుట్), లాగింగ్ (డీబగ్గింగ్/మానిటరింగ్)
  - ప్రస్తుత అవగాహన (`*/list`), పొందిక (`*/get`), మరియు అమలుపరచడం (`*/call`) పద్ధతులు నవీకరించబడ్డాయి
- **ప్రోటోకాల్ ఆర్కిటెక్చర్**: రెండు-స్థాయి ఆర్కిటెక్చర్ మోడల్ ప్రవేశపెట్టబడింది
  - డేటా లేయర్: JSON-RPC 2.0 ప్రాతిపదికతో లైఫ్‌సైకిల్ నిర్వహణ మరియు ప్రాథమికాలతో
  - ట్రాన్స్‌పోర్ట్ లేయర్: STDIO (స్థానిక) మరియు స్ట్రీమబుల్ HTTP తో SSE (దూర) ట్రాన్స్‌పోర్ట్ యంత్రాలతో
- **సెక్యురిటీ ఫ్రేమ్‌వర్క్**: స్పష్టమైన వినియోగదారు అనుమతి, డేటా గోప్యత రక్షణ, టూల్ అమలు భద్రత, మరియు ట్రాన్స్‌పోర్ట్ లేయర్ భద్రత సహా సమగ్ర భద్రతా సూత్రాలు
- **కమ్యూనికేషన్ నమూనాలు**: ఆరంభ, అవగాహన, అమలు, మరియు నోటిఫికేషన్ ప్రవాహాలను చూపించే ప్రోటోకాల్ సందేశాలు నవీకరించబడ్డాయి
- **కోడ్ ఉదాహరణలు**: ప్రస్తుత MCP SDK నమూనాలను ప్రతిబింబించే బహుభాషా (.NET, జావా, పython, జావాస్క్రిప్ట్) ఉదాహరణలు పునరుద్ధరించబడ్డాయి

#### సెక్యురిటీ (02-Security/) - సమగ్ర భద్రతా పునఃసంరచన  
- **స్టాండర్డ్స్ అనుసరణ**: MCP స్పెసిఫికేషన్ 2025-06-18 భద్రతా అవసరాలకు పూర్తిస్థాయి అనుసరణ
- **ఆథెంటికేషన్ అభివృద్ధి**: కస్టమ్ OAuth సర్వర్‌ల నుండి బాహ్య ఐడెంటిటీ ప్రొవైడర్ డెలిగేషన్ (Microsoft Entra ID) వరకు అభివృద్ధి డాక్యుమెంట్ చేయబడింది
- **AI-స్పెసిఫిక్ ముప్పుల విశ్లేషణ**: ఆధునిక AI దాడి వక్తలు విస్తృతంగా కవర్ చేయబడ్డాయి
  - వాస్తవ ప్రపంచ ఉదాహరణలతో ప్రాంప్ట్ ఇంజెక్షన్ దాడి సందర్భాలు వివరించబడ్డాయి
  - టూల్ విషకరణ మరియు "రగ్ పుల్" దాడి నమూనాలు
  - కాంటెక్స్ట్ విండో విషకరణ మరియు మోడల్ గందరగోళ దాడులు
- **Microsoft AI భద్రతా పరిష్కారాలు**: Microsoft భద్రతా వ్యవస్థ యొక్క సమగ్ర కవరేజ్
  - AI ప్రాంప్ట్ షీల్డ్స్ అధునాతన గుర్తింపు, స్పాట్‌లైటింగ్, మరియు డెలిమిటర్ పద్ధతులతో
  - Azure కంటెంట్ సేఫ్టీ సమీకరణ నమూనాలు
  - GitHub అడ్వాన్స్డ్ సెక్యురిటీ ద్వారా సరఫరా చైన్ రక్షణ
- **అధునాతన ముప్పు నిరోధనం**:  
  - MCP-ప్రత్యేక దాడి దృశ్యాలతో సెషన్ హైజాకింగ్ మరియు క్రిప్టోగ్రఫిక్ సెషన్ ID అవసరాలు
  - MCP ప్రాక్సీ సందర్భాలలో గందరగోళ పరిపాలక సమస్యలు స్పష్టమైన అనుమతి అవశ్యకతలతో
  - టోకెన్ పాస్త్రో ద్వంద్వాలు తప్పనిసరి ధృవీకరణ నియంత్రణలతో
- **సరఫరా చైన్ భద్రత**: ఫౌండేషన్ మోడల్స్, ఎంబెడ్డింగ్స్ సర్వీసులు, కాంటెక్స్ట్ ప్రొవైడర్స్, మూడవ-పార్టీ APIలతో విస్తృత AI సరఫరా చైన్ కవర్పాటు
- **ఫౌండేషన్ సెక్యురిటీ**: జీరో ట్రస్టు ఆర్కిటెక్చర్ మరియు Microsoft భద్రతా వ్యవస్థతో ఎంటర్‌ప్రైజ్ భద్రతా నమూనాలకు మెరుగైన సమీకరణ
- **వనరు సంస్థ**: అధికారిక డాక్స్, స్టాండర్డ్స్, పరిశోధన, Microsoft పరిష్కారాలు, అమలు మార్గదర్శకాలు వర్గీకరించిన సమగ్ర వనరు లింకులు

### డాక్యుమెంటేషన్ నాణ్యత మెరుగుదలలు
- **సంగ్రహిత లెర్నింగ్ లక్ష్యాలు**: ప్రత్యేక, అమలులో ఉన్న ఫలితాలతో మెరుగైన అభ్యాస లక్ష్యాలు
- **ఆంతర సంబంధాలు**: సంబంధిత భద్రతా మరియు మూల సూత్ర అంశాల మధ్య లింకులు జోడించబడ్డాయి
- **ప్రస్తుత సమాచారం**: అన్ని తేదీ సూచనలు మరియు స్పెసిఫికేషన్ లింకులను ప్రస్తుత ప్రమాణాలకు నవీకరించారు
- **అమలు మార్గదర్శకం**: రెండు విభాగాల్లో ప్రత్యేక, అమలులో ఉన్న మార్గదర్శకాలు జోడించబడ్డాయి

- కార్డ్-ఆధారిత, టాబ్-రికమైన, మరియు అకార్డియన్ శైలీ నావిగేషన్ ఉదాహరణలు జోడించబడ్డాయి
- రిపొ జిటరీ నిర్మాణం విభాగాన్ని తాజా ఫైళ్ళతో నవీకరించారు
- "ఈ పాఠ్యాంశాన్ని ఎలాగు ఉపయోగించాలి" విభాగంలో స్పష్టమైన సూచనలను చేర్చారు
- MCP స్పెసిఫికేషన్ లింకులను సరైన URLలకు మార్చారు
- పాఠ్యాంశ నిర్మాణంలో Context Engineering విభాగం (5.14) జోడించబడింది

### అధ్యయన గైడ్ నవీకరింపులు
- ప్రస్తుత రిపొజిటరీ నిర్మాణంతో అనుగుణంగా అధ్యయన గైడ్‌ను పూర్తిగా పునఃరచించారు
- MCP క్లయింట్లు మరియు టూల్స్, మరియు ప్రజాదరణ పొందిన MCP సర్వర్లు కోసం కొత్త విభాగాల్ని చేర్చారు
- విజువల్ పాఠ్యాంశ మ్యాప్‌ను అన్ని అంశాలను నిర్దిష్టంగా ప్రతిబింబించేలా నవీకరించారు
- అత్యుత్తమ విషయాల వివరణను అన్ని ప్రత్యేక విభాగాలలో విస్తరించారు
- కేస్ స్టడీస్ విభాగాన్ని వాస్తవ ఉదాహరణలతో నవీకరించారు
- ఈ సమగ్ర చేంజ్‌లాగ్‌ను జోడించారు

### కమ్యూనిటీ సహకారాలు (06-CommunityContributions/)
- చిత్రం సృష్టి కోసం MCP సర్వర్లపై విస్తృత సమాచారం జోడించారు
- VSCodeలో Claude ఉపయోగించడం పై సమగ్ర విభాగం చేర్చారు
- Cline టర్మినల్ క్లయింట్ సెటప్ మరియు ఉపయోగం సూచనలు జోడించారు
- MCP క్లయింట్ విభాగాన్ని అన్ని ప్రజాదరణ పొందిన క్లయింట్ ఎంపికలతో నవీకరించారు
- సహకార ఉదాహరణలను మరింత ఖచ్చితమైన కోడ్ నమూనాల తో మెరుగుపరిచారు

### ముందడుగు విషయాలు (05-AdvancedTopics/)
- అన్ని ప్రత్యేక అంశం ఫోల్డర్లను సమన్వయ పేరుతో క్రమపద్దతిలో ఏర్పాటు చేశారు
- కాంటెక్స్ట్ ఇంజనీరింగ్ పదార్థాలు మరియు ఉదాహరణలు చేర్చారు
- Foundry ఏజెంట్ ఇంటిగ్రేషన్ డాక్యుమెంటేషన్ జత చేశారు
- Entra ID భద్రతా ఇంటిగ్రేషన్ డాక్యుమెంటేషన్ మెరుగుపరచబడింది
