# ఈ నమూనాను నడపడం

> [!NOTE]
> ఈ రస్ట్ క్లయింట్ ఈ ప్రాజెక్ట్‌కు సోదర డైరెక్టరీ అయిన [server](../../../../../../03-GettingStarted/02-client/solution/server) డైరెక్టరీలో నమూనా MCP సర్వర్ నడపడానికి dotnet 8.0 SDK ఇన్‌స్టాల్ చేసినట్లు భావిస్తుంది.

## -1- డిపెండెన్సీలను ఇన్‌స్టాల్ చేసి ప్రాజెక్ట్‌ను నిర్మించండి

```bash
cargo build
```

## -2- నమూనాను నడపండి

```bash
cargo run
```
