# MCP సర్వర్లను మొబతికూడడం

మీ MCP సర్వర్‌ను మొబతికూడడం ద్వారా మీ స్థానిక వాతావరణానికి బయట ఇతరులు దాని సాధనాలు మరియు వనరులను ప్రాప్తి చేయగలుగుతారు. మీ స్కేలబిలిటీ, విశ్వసనీయత మరియు నిర్వహణ సులువుని పరిగణనలోకి తీసుకుని అనేక మొబతికూడటం విధానాలు ఉన్నాయి. క్రింద మీరు MCP సర్వర్లను స్థానికంగా, కంటైనర్లలో మరియు క్లౌడ్‌లో ఎలా మొబతికూడాలో సూచనలు పొందగలుగుతారు.

## అవలోకనం

ఈ పాఠం మీ MCP సర్వర్ యాప్‌ను ఎలా మొబతికూడాలో వివరిస్తుంది.

## నేర్చుకునే లక్ష్యాలు

ఈ పాఠం ముగిసిన తర్వాత, మీరు చేయగలుగుతారు:

- విభిన్న మొబతికూడటం దృక్పథాలను మూల్యాంకనం చేయగలుగుతారు.
- మీ యాప్‌ను మొబతికూడగలుగుతారు.

## స్థానిక అభివృద్ధి మరియు మొబతికూడటం

మీ సర్వర్ వినియోగదారుల యంత్రంపై నడుస్తూ వినియోగించబడాలని ఉద్దేశించబడితే, మీరు ఈ క్రింది దశలను అనుసరించవచ్చు:

1. **సర్వర్‌ను డౌన్లోడ్ చేసుకోండి**. మీరు సర్వర్‌ను రాయకపోతే, ముందుగా దానిని మీ యంత్రంలో డౌన్లోడ్ చేసుకోండి.  
1. **సర్వర్ ప్రక్రియను ప్రారంభించండి**: మీ MCP సర్వర్ అనువర్తనాన్ని నడపండి

SSE కోసం (stdio టైప్ సర్వర్‌కు అవసరం లేదు)

1. **నెట్‌వర్కింగ్‌ను అమర్చండి**: సర్వర్ కనుక ఆశించిన పోర్టులో ప్రాప్తিস্থాయినట్లు చూడండి  
1. **క్లయింట్లను সংযোগించండి**: `http://localhost:3000` వంటి స్థానిక కనెక్షన్ URLs ను ఉపయోగించండి  

## క్లౌడ్ మొబతికూడటం

MCP సర్వర్లు వివిధ క్లౌడ్ ప్లాట్‌ఫామ్లకు మొబతికూడబడవచ్చు:

- **సర్వర్‌లెస్ ఫంక్షన్లు**: తేలికపాటి MCP సర్వర్లను సర్వర్‌లెస్ ఫంక్షన్లుగా మొబతికూడండి  
- **కంటైనర్ సేవలు**: Azure Container Apps, AWS ECS, లేదా Google Cloud Run వంటి సేవలను ఉపయోగించండి  
- **కుబెర్నిటిస్**: MCP సర్వర్లను హై ఆవైలబిలిటీ కోసం కుబెర్నిటిస్ క్లస్టర్లలో మొబతికూడండి మరియు నిర్వహించండి  

### ఉదాహరణ: Azure Container Apps

Azure Container Apps MCP సర్వర్లను మొబతికూడటానికి మద్దతు ఇస్తుంది. ఇది ఇంకా అభివృద్ధి స్ధాయిలో ఉంది మరియు ప్రస్తుతం SSE సర్వర్లను మద్దతు ఇస్తోంది.

ఇది ఎలా చేయాలో ఇక్కడ ఉంది:

1. ఒక రిపోను క్లోన్ చేసుకోండి:

  ```sh
  git clone https://github.com/anthonychu/azure-container-apps-mcp-sample.git
  ```

1. దానిని స్థానికంగా నడిపి పరీక్ష చేయండి:

  ```sh
  uv venv
  uv sync

  # లినక్స్/మ్యాక్OS
  export API_KEYS=<AN_API_KEY>
  # విండోస్
  set API_KEYS=<AN_API_KEY>

  uv run fastapi dev main.py
  ```

1. స్థానికంగా ప్రయత్నించడానికి, *.vscode* డైరెక్టరీలో *mcp.json* ఫైల్ సృష్టించి క్రింది కంటెంట్‌ని జత చేయండి:

  ```json
  {
      "inputs": [
          {
              "type": "promptString",
              "id": "weather-api-key",
              "description": "Weather API Key",
              "password": true
          }
      ],
      "servers": {
          "weather-sse": {
              "type": "sse",
              "url": "http://localhost:8000/sse",
              "headers": {
                  "x-api-key": "${input:weather-api-key}"
              }
          }
      }
  }
  ```

  SSE సర్వర్ ప్రారంభించిన తర్వాత, మీరు JSON ఫైల్‌లో ప్లే ఐకాన్‌పై క్లిక్ చేయవచ్చు, ఇప్పుడు మీరు GitHub Copilot ద్వారా సర్వర్ పై సాధనాలు పట్టుకున్నట్టు చూడగలుగుతారు, Tool ఐకాన్ చూడండి.

1. మొబతికూడటానికి, క్రింది ఆదేశాన్ని నడపండి:

  ```sh
  az containerapp up -g <RESOURCE_GROUP_NAME> -n weather-mcp --environment mcp -l westus --env-vars API_KEYS=<AN_API_KEY> --source .
  ```

ఇలా మీరు స్థానికంగా మొబతికూడగలరు, Azure కు కూడా ఈ దశల ద్వారా మొబతికూడగలరు.
