# నమూనా నడపండి

## ఆధారాలు ఇన్‌స్టాల్ చేయండి

```bash
npm install
```

## నిర్మించండి

```bash
npm run build
```

## సర్వర్ ప్రారంభించండి

```bash
npm start
```

ఇది సర్వర్‌ను ప్రారంభించాలి

### క్లయింట్ ప్రారంభించండి

వేరే టెర్మినల్‌లో, నడపండి:

```bash
node run client
```
