# నమూనా నడపండి

## వర్చువల్ ఎన్విరాన్‌మెంట్ సెట్ చేయండి

```sh
python -m venv venv
source ./venv/bin/activate
```

## ఆధారాలు ఇన్‌స్టాల్ చేయండి

```sh
pip install "mcp[cli]"
```

## కోడ్ నడపండి

```sh
python client.py
```
