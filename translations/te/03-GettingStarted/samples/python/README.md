# MCP క్యాల్క్యులేటర్ సర్వర్ (Python)



సాధారణ మోడల్ కాంటెక్స్ట్ ప్రోటోకాల్ (MCP) సర్వర్ అమలు Python లో, ఇది ప్రాథమిక క్యాల్క్యులేటర్ ఫంక్షనాలిటీని అందిస్తుంది.


## ఇన్‌స్టాలేషన్

అవసరమైన డిపెండెన్సీలను ఇన్‌స్టాల్ చేయండి:

```bash
pip install -r requirements.txt
```

లేదా MCP Python SDK ని నేరుగా ఇన్‌స్టాల్ చేయండి:

```bash
pip install mcp>=1.18.0
```

## ఉపయోగం

### సర్వర్ నడపడం

సర్వర్ MCP క్లయింట్లు (ఉదాహరణకు Claude Desktop) ఉపయోగించడానికి రూపొందించబడింది. సర్వర్ ప్రారంభించడానికి:

```bash
python mcp_calculator_server.py
```

**గమనిక**: టెర్మినల్‌లో నేరుగా నడిపినప్పుడు, మీరు JSON-RPC ధృవీకరణ లోపాలను చూడవచ్చు. ఇది సాధారణ ప్రవర్తన - సర్వర్ సరైన ఫార్మాట్‌లో ఉన్న MCP క్లయింట్ సందేశాలను ఎదురుచూస్తోంది.

### ఫంక్షన్లను పరీక్షించడం

క్యాల్క్యులేటర్ ఫంక్షన్లు సరిగ్గా పనిచేస్తున్నాయా అని పరీక్షించడానికి:

```bash
python test_calculator.py
```

## సమస్య పరిష్కారం

### ఇంపోర్ట్ లోపాలు

`ModuleNotFoundError: No module named 'mcp'` అనే లోపం వస్తే, MCP Python SDK ని ఇన్‌స్టాల్ చేయండి:

```bash
pip install mcp>=1.18.0
```

### నేరుగా నడిపినప్పుడు JSON-RPC లోపాలు

సర్వర్ నేరుగా నడిపినప్పుడు "Invalid JSON: EOF while parsing a value" వంటి లోపాలు రావడం సాధారణం. సర్వర్‌కు MCP క్లయింట్ సందేశాలు అవసరం, నేరుగా టెర్మినల్ ఇన్‌పుట్ కాదు.
