# ప్రాథమిక క్యాల్క్యులేటర్ MCP సర్వీస్

ఈ సర్వీస్ మోడల్ కాంటెక్స్ట్ ప్రోటోకాల్ (MCP) ద్వారా ప్రాథమిక క్యాల్క్యులేటర్ ఆపరేషన్లను అందిస్తుంది. ఇది MCP అమలుల గురించి నేర్చుకుంటున్న ప్రారంభకులకు ఒక సాదారణ ఉదాహరణగా రూపొందించబడింది.

మరింత సమాచారం కోసం, చూడండి [C# SDK](https://github.com/modelcontextprotocol/csharp-sdk)

## లక్షణాలు

ఈ క్యాల్క్యులేటర్ సర్వీస్ క్రింది సామర్థ్యాలను అందిస్తుంది:

1. **ప్రాథమిక గణిత ఆపరేషన్లు**:
   - రెండు సంఖ్యల జోడింపు
   - ఒక సంఖ్యను మరొకటి నుండి తీసివేయడం
   - రెండు సంఖ్యల గుణకం
   - ఒక సంఖ్యను మరొకదితో భాగించడం (శూన్య భాగింపు తనిఖీతో)

## `stdio` రకం ఉపయోగించడం
  
## కాన్ఫిగరేషన్

1. **MCP సర్వర్లను కాన్ఫిగర్ చేయండి**:
   - మీ వర్క్‌స్పేస్‌ను VS కోడ్‌లో తెరవండి.
   - MCP సర్వర్లను కాన్ఫిగర్ చేయడానికి మీ వర్క్‌స్పేస్ ఫోల్డర్‌లో `.vscode/mcp.json` ఫైల్ సృష్టించండి. ఉదాహరణ కాన్ఫిగరేషన్:

     ```jsonc
     {
       "inputs": [
         {
           "type": "promptString",
           "id": "repository-root",
           "description": "The absolute path to the repository root"
         }
       ],
       "servers": {
         "calculator-mcp-dotnet": {
           "type": "stdio",
           "command": "dotnet",
           "args": [
             "run",
             "--project",
             "${input:repository-root}/03-GettingStarted/samples/csharp/src/calculator.csproj"
           ]
         }
       }
     }
     ```

   - మీరు GitHub రిపాజిటరీ రూట్‌ను నమోదు చేయమని అడుగుతారు, ఇది `git rev-parse --show-toplevel` కమాండ్ ద్వారా పొందవచ్చు.

## సర్వీస్ ఉపయోగించడం

సర్వీస్ MCP ప్రోటోకాల్ ద్వారా క్రింది API ఎండ్‌పాయింట్లను అందిస్తుంది:

- `add(a, b)`: రెండు సంఖ్యలను కలిపి జోడించండి
- `subtract(a, b)`: రెండవ సంఖ్యను మొదటి నుండి తీసివేయండి
- `multiply(a, b)`: రెండు సంఖ్యలను గుణించండి
- `divide(a, b)`: మొదటి సంఖ్యను రెండవదితో భాగించండి (శూన్య తనిఖీతో)
- isPrime(n): ఒక సంఖ్య ప్రైమ్ కాదా అని తనిఖీ చేయండి

## VS కోడ్‌లో Github Copilot చాట్‌తో పరీక్షించండి

1. MCP ప్రోటోకాల్ ఉపయోగించి సర్వీస్‌కు అభ్యర్థన చేయండి. ఉదాహరణకు, మీరు అడగవచ్చు:
   - "5 మరియు 3 జోడించండి"
   - "4 నుండి 10 తీసివేయండి"
   - "6 మరియు 7 గుణించండి"
   - "8 ను 2 తో భాగించండి"
   - "37854 ప్రైమ్ కాదా?"
   - "4242 ముందు తర్వాత 3 ప్రైమ్ సంఖ్యలు ఏమిటి?"
2. ఇది టూల్స్ ఉపయోగిస్తున్నదని నిర్ధారించడానికి ప్రాంప్ట్‌లో #MyCalculator జోడించండి. ఉదాహరణకు:
   - "5 మరియు 3 జోడించండి #MyCalculator"
   - "4 నుండి 10 తీసివేయండి #MyCalculator"

## కంటైనరైజ్డ్ వెర్షన్

మీ వద్ద .NET SDK ఇన్‌స్టాల్ చేసి, అన్ని డిపెండెన్సీలు సిద్ధంగా ఉన్నప్పుడు పూర్వపు పరిష్కారం బాగుంది. అయితే, మీరు పరిష్కారాన్ని పంచుకోవాలనుకుంటే లేదా వేరే వాతావరణంలో నడపాలనుకుంటే, కంటైనరైజ్డ్ వెర్షన్ ఉపయోగించవచ్చు.

1. Docker ప్రారంభించి అది నడుస్తున్నదని నిర్ధారించండి.
1. టెర్మినల్ నుండి `03-GettingStarted\samples\csharp\src` ఫోల్డర్‌లోకి వెళ్లండి
1. క్యాల్క్యులేటర్ సర్వీస్ కోసం Docker ఇమేజ్‌ను నిర్మించడానికి క్రింది కమాండ్‌ను అమలు చేయండి (`<YOUR-DOCKER-USERNAME>` స్థానంలో మీ Docker Hub యూజర్‌నేమ్ ఉంచండి):
   ```bash
   docker build -t <YOUR-DOCKER-USERNAME>/mcp-calculator .
   ``` 
1. ఇమేజ్ నిర్మించిన తర్వాత, దాన్ని Docker Hub కు అప్లోడ్ చేద్దాం. క్రింది కమాండ్‌ను అమలు చేయండి:
   ```bash
    docker push <YOUR-DOCKER-USERNAME>/mcp-calculator
  ```

## Use the Dockerized Version

1. In the `.vscode/mcp.json` file, replace the server configuration by the following:
   ```json
    "mcp-calc": {
      "command": "docker",
      "args": [
        "run",
        "--rm",
        "-i",
        "<YOUR-DOCKER-USERNAME>/mcp-calc"
      ],
      "envFile": "",
      "env": {}
    }
   ```
   కాన్ఫిగరేషన్ చూస్తే, కమాండ్ `docker` మరియు ఆర్గ్స్ `run --rm -i <YOUR-DOCKER-USERNAME>/mcp-calc` అని ఉన్నది. `--rm` ఫ్లాగ్ కంటైనర్ ఆపిన తర్వాత దాన్ని తొలగిస్తుంది, మరియు `-i` ఫ్లాగ్ కంటైనర్ యొక్క స్టాండర్డ్ ఇన్‌పుట్‌తో ఇంటరాక్ట్ చేయడానికి అనుమతిస్తుంది. చివరి ఆర్గ్యుమెంట్ మనం నిర్మించి Docker Hub కు పుష్ చేసిన ఇమేజ్ పేరు.

## Dockerైజ్డ్ వెర్షన్‌ను పరీక్షించండి

`"mcp-calc": {` పై ఉన్న చిన్న స్టార్ట్ బటన్‌ను క్లిక్ చేసి MCP సర్వర్‌ను ప్రారంభించండి, మరియు మునుపటి విధంగా క్యాల్క్యులేటర్ సర్వీస్‌ను మీరు గణితం చేయమని అడగవచ్చు.
