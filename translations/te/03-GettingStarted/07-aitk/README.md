# Visual Studio Code కోసం AI టూల్‌కిట్ విస్తరణ నుండి సర్వర్‌ను వినియోగించడం

మీరు AI ఏజెంట్‌ను నిర్మిస్తున్నప్పుడు, అది కేవలం తెలివైన ప్రతిస్పందనలు సృష్టించడం మాత్రమే కాదు; మీ ఏజెంట్‌కు చర్య తీసుకునే సామర్థ్యాన్ని కూడా ఇవ్వడం ముఖ్యం. అప్పుడు Model Context Protocol (MCP) ఉపయోగపడుతుంది. MCP ఏజెంట్లకు బాహ్య టూల్స్ మరియు సేవలను సుస్పష్టమైన విధంగా యాక్సెస్ చేయడం సులభం చేస్తుంది. దీన్ని మీ ఏజెంట్‌ను నిజంగా ఉపయోగించగల టూల్‌బాక్స్‌లో ప్లగ్ చేయడం లాగా భావించండి.

మీరు ఒక ఏజెంట్‌ను మీ క్యాల్క్యులేటర్ MCP సర్వర్‌కు కనెక్ట్ చేస్తే, అకస్మాత్తుగా, మీ ఏజెంట్ “47 గుణించు 89 ఎంత?” వంటి ప్రాంప్ట్‌ను అందుకోవడం ద్వారా గణిత చర్యలు చేయగలదు—లాజిక్‌ను హార్డ్‌కోడ్ చేయాల్సిన అవసరం లేకుండా లేదా కస్టమ్ APIలను నిర్మించాల్సిన అవసరం లేకుండా.

## అవలోకనం

ఈ పాఠం Visual Studio Codeలో [AI Toolkit](https://aka.ms/AIToolkit) విస్తరణతో క్యాల్క్యులేటర్ MCP సర్వర్‌ను ఏజెంట్‌కు ఎలా కనెక్ట్ చేయాలో, మరియు మీ ఏజెంట్‌ను సహజ భాష ద్వారా జోడింపు, తీసివేత, గుణకం, భాగాకారం వంటి గణిత చర్యలు చేయగలిగేలా ఎలా చేయాలో వివరించబడింది.

AI Toolkit Visual Studio Code కోసం శక్తివంతమైన విస్తరణ, ఇది ఏజెంట్ అభివృద్ధిని సులభతరం చేస్తుంది. AI ఇంజనీర్లు స్థానికంగా లేదా క్లౌడ్‌లో జనరేటివ్ AI మోడల్స్‌ను అభివృద్ధి చేసి పరీక్షించవచ్చు. ఈ విస్తరణ ఈ రోజున అందుబాటులో ఉన్న ప్రధాన జనరేటివ్ మోడల్స్‌ను మద్దతు ఇస్తుంది.

*గమనిక*: AI Toolkit ప్రస్తుతం Python మరియు TypeScript మద్దతు ఇస్తుంది.

## నేర్చుకునే లక్ష్యాలు

ఈ పాఠం ముగిసిన తర్వాత, మీరు చేయగలుగుతారు:

- AI Toolkit ద్వారా MCP సర్వర్‌ను వినియోగించడం.
- ఏజెంట్ కాన్ఫిగరేషన్‌ను సెట్ చేసి MCP సర్వర్ అందించే టూల్స్‌ను కనుగొని ఉపయోగించగలిగేలా చేయడం.
- సహజ భాష ద్వారా MCP టూల్స్‌ను ఉపయోగించడం.

## దృష్టికోణం

మనం ఈ విధంగా ఉన్నత స్థాయిలో దృష్టి పెట్టాలి:

- ఒక ఏజెంట్ సృష్టించి దాని సిస్టమ్ ప్రాంప్ట్‌ను నిర్వచించండి.
- క్యాల్క్యులేటర్ టూల్స్‌తో MCP సర్వర్ సృష్టించండి.
- ఏజెంట్ బిల్డర్‌ను MCP సర్వర్‌కు కనెక్ట్ చేయండి.
- సహజ భాష ద్వారా ఏజెంట్ టూల్ పిలుపును పరీక్షించండి.

అద్భుతం, ఇప్పుడు ప్రవాహాన్ని అర్థం చేసుకున్నాము, MCP ద్వారా బాహ్య టూల్స్‌ను ఉపయోగించడానికి AI ఏజెంట్‌ను సెట్ చేద్దాం, దాని సామర్థ్యాలను పెంచుదాం!

### -2- ఏజెంట్ కోసం సిస్టమ్ ప్రాంప్ట్ సృష్టించండి

ఏజెంట్ నిర్మాణం పూర్తయిన తర్వాత, దాని వ్యక్తిత్వం మరియు ఉద్దేశ్యాన్ని నిర్వచించాలి. ఈ విభాగంలో, మీరు **Generate system prompt** ఫీచర్ ఉపయోగించి ఏజెంట్ యొక్క ప్రవర్తనను వివరించండి—ఈ సందర్భంలో, క్యాల్క్యులేటర్ ఏజెంట్—మరియు మోడల్ మీకు సిస్టమ్ ప్రాంప్ట్ రాయనిస్తుంది.

![Visual Studio Code కోసం AI Toolkitలో "Calculator Agent" ఇంటర్‌ఫేస్ స్క్రీన్‌షాట్, "Generate a prompt" అనే మోడల్ విండో తెరుచుకున్నది. విండోలో ప్రాంప్ట్ టెంప్లేట్‌ను సృష్టించడానికి ప్రాథమిక వివరాలు పంచుకోవాలని చెప్పబడింది. టెక్స్ట్ బాక్స్‌లో నమూనా సిస్టమ్ ప్రాంప్ట్ ఉంది: "You are a helpful and efficient math assistant. When given a problem involving basic arithmetic, you respond with the correct result." దిగువన "Close" మరియు "Generate" బటన్లు ఉన్నాయి. నేపథ్యంగా ఏజెంట్ కాన్ఫిగరేషన్ భాగం కనిపిస్తుంది, ఎంచుకున్న మోడల్ "OpenAI GPT-4o (via GitHub)" మరియు సిస్టమ్, యూజర్ ప్రాంప్ట్ ఫీల్డ్స్ ఉన్నాయి.](../../../../translated_images/te/aitk-generate-prompt.ba9e69d3d2bbe2a2.webp)

1. **Prompts** విభాగంలో **Generate system prompt** బటన్ క్లిక్ చేయండి. ఇది ఏజెంట్ కోసం సిస్టమ్ ప్రాంప్ట్ సృష్టించడానికి AI ఉపయోగించే ప్రాంప్ట్ బిల్డర్ తెరుస్తుంది.
1. **Generate a prompt** విండోలో ఈ వాక్యం టైప్ చేయండి: `You are a helpful and efficient math assistant. When given a problem involving basic arithmetic, you respond with the correct result.`
1. **Generate** బటన్ క్లిక్ చేయండి. కుడి దిగువ మూలలో ఒక నోటిఫికేషన్ వస్తుంది, సిస్టమ్ ప్రాంప్ట్ సృష్టించబడుతోందని తెలియజేస్తుంది. సృష్టి పూర్తయిన తర్వాత, ప్రాంప్ట్ **Agent (Prompt) Builder**లో **System prompt** ఫీల్డ్‌లో కనిపిస్తుంది.
1. **System prompt**ని సమీక్షించి అవసరమైతే మార్చండి.

### -3- MCP సర్వర్ సృష్టించండి

మీ ఏజెంట్ యొక్క సిస్టమ్ ప్రాంప్ట్‌ను నిర్వచించిన తర్వాత—దాని ప్రవర్తన మరియు ప్రతిస్పందనలను మార్గనిర్దేశం చేస్తుంది—ఏజెంట్‌కు ప్రాక్టికల్ సామర్థ్యాలు ఇవ్వాల్సి ఉంది. ఈ విభాగంలో, మీరు జోడింపు, తీసివేత, గుణకం, భాగాకారం లాంటి గణిత లెక్కింపులను నిర్వహించే టూల్స్‌తో క్యాల్క్యులేటర్ MCP సర్వర్ సృష్టిస్తారు. ఈ సర్వర్ సహజ భాష ప్రాంప్ట్‌లకు ప్రత్యక్ష గణిత చర్యలు చేయడానికి ఏజెంట్‌కు వీలు కల్పిస్తుంది.

!["Calculator Agent" ఇంటర్‌ఫేస్ దిగువ భాగం స్క్రీన్‌షాట్, AI Toolkit విస్తరణలో. ఇది “Tools” మరియు “Structure output” విస్తరించదగిన మెనూలను చూపిస్తుంది, “Choose output format” అనే డ్రాప్‌డౌన్ మెనూ “text”గా సెట్ చేయబడింది. కుడి వైపున "+ MCP Server" అనే బటన్ ఉంది, ఇది Model Context Protocol సర్వర్ జోడించడానికి. టూల్స్ విభాగం పై భాగంలో ఇమేజ్ ఐకాన్ ప్లేస్‌హోల్డర్ ఉంది.](../../../../translated_images/te/aitk-add-mcp-server.9742cfddfe808353.webp)

AI Toolkit మీ స్వంత MCP సర్వర్ సృష్టించడాన్ని సులభతరం చేసే టెంప్లేట్లతో వస్తుంది. క్యాల్క్యులేటర్ MCP సర్వర్ సృష్టించడానికి Python టెంప్లేట్‌ను ఉపయోగిస్తాము.

*గమనిక*: AI Toolkit ప్రస్తుతం Python మరియు TypeScript మద్దతు ఇస్తుంది.

1. **Agent (Prompt) Builder**లో **Tools** విభాగంలో **+ MCP Server** బటన్ క్లిక్ చేయండి. విస్తరణ **Command Palette** ద్వారా సెటప్ విజార్డ్ ప్రారంభిస్తుంది.
1. **+ Add Server** ఎంచుకోండి.
1. **Create a New MCP Server** ఎంచుకోండి.
1. టెంప్లేట్‌గా **python-weather** ఎంచుకోండి.
1. MCP సర్వర్ టెంప్లేట్ సేవ్ చేయడానికి **Default folder** ఎంచుకోండి.
1. సర్వర్‌కు ఈ పేరు ఇవ్వండి: **Calculator**
1. కొత్త Visual Studio Code విండో తెరుస్తుంది. **Yes, I trust the authors** ఎంచుకోండి.
1. టెర్మినల్‌లో (**Terminal** > **New Terminal**) వర్చువల్ ఎన్విరాన్‌మెంట్ సృష్టించండి: `python -m venv .venv`
1. టెర్మినల్‌లో వర్చువల్ ఎన్విరాన్‌మెంట్ యాక్టివేట్ చేయండి:
    1. Windows - `.venv\Scripts\activate`
    1. macOS/Linux - `source .venv/bin/activate`
1. టెర్మినల్‌లో డిపెండెన్సీలు ఇన్‌స్టాల్ చేయండి: `pip install -e .[dev]`
1. **Activity Bar**లో **Explorer** వీక్షణలో **src** డైరెక్టరీని విస్తరించి **server.py** ఫైల్ ఎడిటర్‌లో తెరవండి.
1. **server.py** ఫైల్‌లోని కోడ్‌ను క్రింది కోడ్‌తో మార్చి సేవ్ చేయండి:

    ```python
    """
    Sample MCP Calculator Server implementation in Python.

    
    This module demonstrates how to create a simple MCP server with calculator tools
    that can perform basic arithmetic operations (add, subtract, multiply, divide).
    """
    
    from mcp.server.fastmcp import FastMCP
    
    server = FastMCP("calculator")
    
    @server.tool()
    def add(a: float, b: float) -> float:
        """Add two numbers together and return the result."""
        return a + b
    
    @server.tool()
    def subtract(a: float, b: float) -> float:
        """Subtract b from a and return the result."""
        return a - b
    
    @server.tool()
    def multiply(a: float, b: float) -> float:
        """Multiply two numbers together and return the result."""
        return a * b
    
    @server.tool()
    def divide(a: float, b: float) -> float:
        """
        Divide a by b and return the result.
        
        Raises:
            ValueError: If b is zero
        """
        if b == 0:
            raise ValueError("Cannot divide by zero")
        return a / b
    ```

### -4- క్యాల్క్యులేటర్ MCP సర్వర్‌తో ఏజెంట్‌ను నడపండి

ఇప్పుడు మీ ఏజెంట్‌కు టూల్స్ ఉన్నాయి, వాటిని ఉపయోగించే సమయం వచ్చింది! ఈ విభాగంలో, మీరు ఏజెంట్‌కు ప్రాంప్ట్‌లు సమర్పించి ఏజెంట్ సరైన టూల్‌ను క్యాల్క్యులేటర్ MCP సర్వర్ నుండి ఉపయోగిస్తున్నదో లేదో పరీక్షిస్తారు.

![Visual Studio Code కోసం AI Toolkit విస్తరణలో Calculator Agent ఇంటర్‌ఫేస్ స్క్రీన్‌షాట్. ఎడమ ప్యానెల్‌లో “Tools” కింద local-server-calculator_server అనే MCP సర్వర్ జోడించబడింది, అందులో నాలుగు టూల్స్ ఉన్నాయి: add, subtract, multiply, divide. నాలుగు టూల్స్ యాక్టివ్ ఉన్నట్లు బ్యాడ్జ్ చూపిస్తుంది. దిగువన “Structure output” విభాగం కుదించబడింది, మరియు నీలం “Run” బటన్ ఉంది. కుడి ప్యానెల్‌లో “Model Response” కింద ఏజెంట్ multiply మరియు subtract టూల్స్‌ను {"a": 3, "b": 25} మరియు {"a": 75, "b": 20} ఇన్‌పుట్స్‌తో పిలుస్తోంది. తుది “Tool Response” 75.0గా ఉంది. దిగువన “View Code” బటన్ ఉంది.](../../../../translated_images/te/aitk-agent-response-with-tools.e7c781869dc8041a.webp)

మీరు మీ స్థానిక డెవ్ మెషీన్‌లో MCP క్లయింట్‌గా **Agent Builder** ద్వారా క్యాల్క్యులేటర్ MCP సర్వర్‌ను నడిపిస్తారు.

1. MCP సర్వర్ డీబగ్గింగ్ ప్రారంభించడానికి `F5` నొక్కండి. **Agent (Prompt) Builder** కొత్త ఎడిటర్ ట్యాబ్‌లో తెరుస్తుంది. సర్వర్ స్థితి టెర్మినల్‌లో కనిపిస్తుంది.
1. **Agent (Prompt) Builder**లో **User prompt** ఫీల్డ్‌లో ఈ ప్రాంప్ట్ టైప్ చేయండి: `I bought 3 items priced at $25 each, and then used a $20 discount. How much did I pay?`
1. ఏజెంట్ ప్రతిస్పందన సృష్టించడానికి **Run** బటన్ క్లిక్ చేయండి.
1. ఏజెంట్ అవుట్‌పుట్‌ను సమీక్షించండి. మోడల్ మీరు **$55** చెల్లించారని తేల్చుకోవాలి.
1. ఈ క్రింది ప్రక్రియ జరుగుతుంది:
    - ఏజెంట్ లెక్కింపుకు సహాయంగా **multiply** మరియు **subtract** టూల్స్ ఎంచుకుంటుంది.
    - **multiply** టూల్‌కు `a` మరియు `b` విలువలు కేటాయించబడతాయి.
    - **subtract** టూల్‌కు `a` మరియు `b` విలువలు కేటాయించబడతాయి.
    - ప్రతి టూల్ నుండి వచ్చిన ప్రతిస్పందన **Tool Response**లో అందించబడుతుంది.
    - మోడల్ నుండి తుది అవుట్‌పుట్ **Model Response**లో అందించబడుతుంది.
1. ఏజెంట్‌ను మరింత పరీక్షించడానికి అదనపు ప్రాంప్ట్‌లు సమర్పించండి. **User prompt** ఫీల్డ్‌లో ప్రస్తుత ప్రాంప్ట్‌ను మార్చవచ్చు.
1. పరీక్షలు పూర్తయిన తర్వాత, టెర్మినల్‌లో **CTRL/CMD+C** నొక్కి సర్వర్‌ను ఆపండి.

## అసైన్‌మెంట్

మీ **server.py** ఫైల్‌లో అదనపు టూల్ ఎంట్రీని జోడించండి (ఉదా: సంఖ్య యొక్క వర్గమూలం తిరిగి ఇవ్వడం). ఏజెంట్ మీ కొత్త టూల్ (లేదా ఉన్న టూల్స్) ఉపయోగించాల్సిన అదనపు ప్రాంప్ట్‌లు సమర్పించండి. కొత్త టూల్స్ లోడ్ కావడానికి సర్వర్‌ను మళ్లీ ప్రారంభించండి.
