# ఈ నమూనాను నడపడం

## -1- Streamable HTTP సర్వర్‌ను నడపండి

ఈ ప్రాజెక్ట్ యొక్క రూట్ నుండి ఒక టెర్మినల్ తెరవండి మరియు క్రింది కమాండ్లను నడపండి:

```bash
cd ../calculator-httpserver
cargo run
```

క్లయింట్ కనెక్ట్ అయ్యే HTTP సర్వర్ నడుస్తుండటానికి ఈ టెర్మినల్‌ను తెరిచి ఉంచండి. తదుపరి దశల కోసం కొత్త టెర్మినల్ తెరవండి.

## -2- డిపెండెన్సీలను ఇన్‌స్టాల్ చేసి ప్రాజెక్ట్‌ను బిల్డ్ చేయండి

```bash
cargo build
```

## -3- నమూనాను నడపండి

```bash
cargo run
```
