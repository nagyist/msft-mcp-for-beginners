# MCP stdio సర్వర్ పరిష్కారాలు

> **⚠️ ముఖ్యమైనది**: ఈ పరిష్కారాలు MCP స్పెసిఫికేషన్ 2025-06-18 ద్వారా సిఫార్సు చేయబడిన **stdio ట్రాన్స్‌పోర్ట్** ఉపయోగించడానికి నవీకరించబడ్డాయి. అసలు SSE (సర్వర్-సెంట్ ఈవెంట్స్) ట్రాన్స్‌పోర్ట్‌ను డిప్రికేటెడ్ చేయబడింది.

ప్రతి రన్‌టైమ్‌లో stdio ట్రాన్స్‌పోర్ట్ ఉపయోగించి MCP సర్వర్‌లను నిర్మించడానికి పూర్తి పరిష్కారాలు ఇక్కడ ఉన్నాయి:

- [TypeScript](../../../../../03-GettingStarted/05-stdio-server/solution/typescript) - పూర్తి stdio సర్వర్ అమలు
- [Python](../../../../../03-GettingStarted/05-stdio-server/solution/python) - asyncioతో Python stdio సర్వర్
- [.NET](../../../../../03-GettingStarted/05-stdio-server/solution/dotnet) - డిపెండెన్సీ ఇంజెక్షన్‌తో .NET stdio సర్వర్

ప్రతి పరిష్కారం చూపిస్తుంది:
- stdio ట్రాన్స్‌పోర్ట్ సెటప్ చేయడం
- సర్వర్ టూల్స్ నిర్వచించడం
- సరైన JSON-RPC సందేశ నిర్వహణ
- Claude వంటి MCP క్లయింట్లతో ఇంటిగ్రేషన్

అన్ని పరిష్కారాలు ప్రస్తుత MCP స్పెసిఫికేషన్‌ను అనుసరిస్తాయి మరియు ఉత్తమ పనితీరు మరియు భద్రత కోసం సిఫార్సు చేయబడిన stdio ట్రాన్స్‌పోర్ట్‌ను ఉపయోగిస్తాయి.
