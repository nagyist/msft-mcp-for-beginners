# ప్రాక్టికల్ అమలు

[![వాస్తవ టూల్స్ మరియు వర్క్‌ఫ్లోలతో MCP యాప్స్‌ను ఎలా నిర్మించాలి, పరీక్షించాలి మరియు డిప్లాయ్ చేయాలి](../../../translated_images/05.64bea204e25ca891e3dd8b8f960d2170b9a000d8364305f57db3ec4a2c049a9a.te.png)](https://youtu.be/vCN9-mKBDfQ)

_(ఈ పాఠం వీడియోను చూడడానికి పై చిత్రాన్ని క్లిక్ చేయండి)_

ప్రాక్టికల్ అమలు అనేది మోడల్ కాంటెక్స్ట్ ప్రోటోకాల్ (MCP) శక్తి స్పష్టంగా కనిపించే చోటు. MCP వెనుక సిద్ధాంతం మరియు ఆర్కిటెక్చర్‌ను అర్థం చేసుకోవడం ముఖ్యమైనప్పటికీ, ఈ భావనలను వాస్తవ ప్రపంచ సమస్యలను పరిష్కరించే పరిష్కారాలను నిర్మించడం, పరీక్షించడం మరియు డిప్లాయ్ చేయడంలో ఉపయోగించినప్పుడు నిజమైన విలువ వెలుగులోకి వస్తుంది. ఈ అధ్యాయం భావనాత్మక జ్ఞానం మరియు ప్రాక్టికల్ అభివృద్ధి మధ్య గ్యాప్‌ను భర్తీ చేస్తుంది, MCP ఆధారిత అప్లికేషన్లను జీవితం తీసుకురావడంలో మీకు మార్గనిర్దేశం చేస్తుంది.

మీరు తెలివైన సహాయకులను అభివృద్ధి చేస్తున్నారా, వ్యాపార వర్క్‌ఫ్లోలలో AIని సమ్మిళితం చేస్తున్నారా లేదా డేటా ప్రాసెసింగ్ కోసం కస్టమ్ టూల్స్‌ను నిర్మిస్తున్నారా, MCP ఒక సౌలభ్యమైన పునాది అందిస్తుంది. దాని భాష-నిరపేక్ష డిజైన్ మరియు ప్రముఖ ప్రోగ్రామింగ్ భాషల కోసం అధికారిక SDKలు విస్తృత శ్రేణి డెవలపర్లకు అందుబాటులో ఉంటాయి. ఈ SDKలను ఉపయోగించి, మీరు త్వరగా ప్రోటోటైప్ చేయవచ్చు, పునరావృతం చేయవచ్చు మరియు వివిధ ప్లాట్‌ఫారమ్‌లు మరియు వాతావరణాలలో మీ పరిష్కారాలను స్కేల్ చేయవచ్చు.

తదుపరి విభాగాలలో, మీరు MCPని C#, Java with Spring, TypeScript, JavaScript, మరియు Pythonలో ఎలా అమలు చేయాలో చూపించే ప్రాక్టికల్ ఉదాహరణలు, నమూనా కోడ్ మరియు డిప్లాయ్‌మెంట్ వ్యూహాలను కనుగొంటారు. మీరు MCP సర్వర్లను డీబగ్ చేయడం మరియు పరీక్షించడం, APIలను నిర్వహించడం మరియు Azure ఉపయోగించి క్లౌడ్‌కు పరిష్కారాలను డిప్లాయ్ చేయడం కూడా నేర్చుకుంటారు. ఈ ప్రాక్టికల్ వనరులు మీ అభ్యాసాన్ని వేగవంతం చేయడానికి మరియు మీరు ధైర్యంగా బలమైన, ప్రొడక్షన్-రెడీ MCP అప్లికేషన్లను నిర్మించడానికి రూపొందించబడ్డాయి.

## అవలోకనం

ఈ పాఠం అనేక ప్రోగ్రామింగ్ భాషలలో MCP అమలుకు సంబంధించిన ప్రాక్టికల్ అంశాలపై దృష్టి సారిస్తుంది. మేము C#, Java with Spring, TypeScript, JavaScript, మరియు Pythonలో MCP SDKలను ఉపయోగించి బలమైన అప్లికేషన్లను నిర్మించడం, MCP సర్వర్లను డీబగ్ చేయడం మరియు పరీక్షించడం, మరియు పునర్వినియోగ వనరులు, ప్రాంప్ట్‌లు మరియు టూల్స్ సృష్టించడం ఎలా చేయాలో పరిశీలిస్తాము.

## అభ్యాస లక్ష్యాలు

ఈ పాఠం ముగిసిన తర్వాత, మీరు చేయగలుగుతారు:

- వివిధ ప్రోగ్రామింగ్ భాషలలో అధికారిక SDKలను ఉపయోగించి MCP పరిష్కారాలను అమలు చేయడం
- MCP సర్వర్లను సిస్టమాటిక్‌గా డీబగ్ చేయడం మరియు పరీక్షించడం
- సర్వర్ ఫీచర్లను (వనరులు, ప్రాంప్ట్‌లు, మరియు టూల్స్) సృష్టించడం మరియు ఉపయోగించడం
- సంక్లిష్ట పనుల కోసం సమర్థవంతమైన MCP వర్క్‌ఫ్లోలను డిజైన్ చేయడం
- పనితీరు మరియు విశ్వసనీయత కోసం MCP అమలులను ఆప్టిమైజ్ చేయడం

## కోర్ సర్వర్ ఫీచర్లు

MCP సర్వర్లు ఈ ఫీచర్ల ఏదైనా కలయికను అమలు చేయవచ్చు:

### వనరులు

వనరులు వినియోగదారు లేదా AI మోడల్ ఉపయోగించడానికి కాంటెక్స్ట్ మరియు డేటాను అందిస్తాయి:

- డాక్యుమెంట్ రిపాజిటరీలు
- జ్ఞాన ఆధారాలు
- నిర్మిత డేటా మూలాలు
- ఫైల్ సిస్టమ్స్

### ప్రాంప్ట్‌లు

ప్రాంప్ట్‌లు వినియోగదారుల కోసం టెంప్లేటెడ్ సందేశాలు మరియు వర్క్‌ఫ్లోలు:

- ముందుగా నిర్వచించిన సంభాషణ టెంప్లేట్లు
- మార్గనిర్దేశక పరస్పర చర్య నమూనాలు
- ప్రత్యేక సంభాషణ నిర్మాణాలు

### టూల్స్

టూల్స్ AI మోడల్ అమలు చేయడానికి ఫంక్షన్లు:

- డేటా ప్రాసెసింగ్ యుటిలిటీల్స్
- బాహ్య API సమ్మిళితాలు
- గణనాత్మక సామర్థ్యాలు
- శోధన ఫంక్షనాలిటీ

## నమూనా అమలు: C# అమలు

అధికారిక C# SDK రిపాజిటరీలో MCP యొక్క వివిధ అంశాలను చూపించే అనేక నమూనా అమలులు ఉన్నాయి:

- **బేసిక్ MCP క్లయింట్**: MCP క్లయింట్ సృష్టించడం మరియు టూల్స్‌ను కాల్ చేయడం ఎలా అనేది చూపించే సాదా ఉదాహరణ
- **బేసిక్ MCP సర్వర్**: ప్రాథమిక టూల్ రిజిస్ట్రేషన్‌తో కనిష్ట సర్వర్ అమలు
- **అడ్వాన్స్డ్ MCP సర్వర్**: టూల్ రిజిస్ట్రేషన్, ఆథెంటికేషన్ మరియు లోపాల నిర్వహణతో పూర్తి ఫీచర్ల సర్వర్
- **ASP.NET ఇంటిగ్రేషన్**: ASP.NET కోర్‌తో ఇంటిగ్రేషన్ చూపించే ఉదాహరణలు
- **టూల్ అమలు నమూనాలు**: వివిధ క్లిష్టత స్థాయిలతో టూల్స్ అమలు చేయడానికి వివిధ నమూనాలు

MCP C# SDK ప్రివ్యూ లో ఉంది మరియు APIs మారవచ్చు. SDK అభివృద్ధి చెందుతున్న కొద్దీ మేము ఈ బ్లాగ్‌ను నిరంతరం నవీకరిస్తాము.

### ముఖ్య ఫీచర్లు

- [C# MCP Nuget ModelContextProtocol](https://www.nuget.org/packages/ModelContextProtocol)
- మీ [మొదటి MCP సర్వర్](https://devblogs.microsoft.com/dotnet/build-a-model-context-protocol-mcp-server-in-csharp/)ను నిర్మించడం.

పూర్తి C# అమలు నమూనాల కోసం, అధికారిక C# SDK నమూనాల రిపాజిటరీని సందర్శించండి: [https://github.com/modelcontextprotocol/csharp-sdk](https://github.com/modelcontextprotocol/csharp-sdk)

Java with Spring SDK సంస్థ స్థాయి ఫీచర్లతో బలమైన MCP అమలు ఎంపికలను అందిస్తుంది.

### ముఖ్య ఫీచర్లు

- Spring Framework ఇంటిగ్రేషన్
- బలమైన టైప్ సేఫ్టీ
- రియాక్టివ్ ప్రోగ్రామింగ్ మద్దతు
- సమగ్ర లోపాల నిర్వహణ

పూర్తి Java with Spring అమలు నమూనా కోసం, నమూనాల డైరెక్టరీలో [Java with Spring sample](samples/java/containerapp/README.md) చూడండి.

### ముఖ్య ఫీచర్లు

## API నిర్వహణ

Azure API నిర్వహణ MCP సర్వర్లను ఎలా సురక్షితం చేయవచ్చో మంచి సమాధానం. ఆలోచన ఏమిటంటే, మీ MCP సర్వర్ ముందు Azure API నిర్వహణ ఇన్స్టాన్స్‌ను ఉంచి, మీరు కావలసిన ఫీచర్లను నిర్వహించనివ్వడం:

## రిమోట్ MCP సర్వర్‌ను Azureకు డిప్లాయ్ చేయడం

ముందుగా చెప్పిన నమూనాను డిప్లాయ్ చేయగలమా చూద్దాం:

1. రిపోను క్లోన్ చేయండి

    ```bash
    git clone https://github.com/Azure-Samples/remote-mcp-apim-functions-python.git
    cd remote-mcp-apim-functions-python
    ```

1. `Microsoft.App` రిసోర్స్ ప్రొవైడర్‌ను రిజిస్టర్ చేయండి.

   - మీరు Azure CLI ఉపయోగిస్తుంటే, `az provider register --namespace Microsoft.App --wait` ను రన్ చేయండి.
   - మీరు Azure PowerShell ఉపయోగిస్తుంటే, `Register-AzResourceProvider -ProviderNamespace Microsoft.App` ను రన్ చేయండి. తర్వాత కొంత సమయం తర్వాత `(Get-AzResourceProvider -ProviderNamespace Microsoft.App).RegistrationState` ను రన్ చేసి రిజిస్ట్రేషన్ పూర్తయిందా చూడండి.

1. ఈ [azd](https://aka.ms/azd) కమాండ్‌ను రన్ చేసి API నిర్వహణ సేవ, ఫంక్షన్ యాప్(కోడ్‌తో) మరియు ఇతర అవసరమైన Azure వనరులను ప్రావిజన్ చేయండి

    ```shell
    azd up
    ```

    ఈ కమాండ్లు Azureపై అన్ని క్లౌడ్ వనరులను డిప్లాయ్ చేయాలి

### MCP ఇన్స్పెక్టర్‌తో మీ సర్వర్‌ను పరీక్షించడం

1. **కొత్త టెర్మినల్ విండోలో**, MCP ఇన్స్పెక్టర్‌ను ఇన్‌స్టాల్ చేసి రన్ చేయండి

    ```shell
    npx @modelcontextprotocol/inspector
    ```

    మీరు ఈ క్రింది లాంటి ఇంటర్‌ఫేస్‌ను చూడాలి:

    ![Connect to Node inspector](../../../translated_images/connect.141db0b2bd05f096fb1dd91273771fd8b2469d6507656c3b0c9df4b3c5473929.te.png)

1. MCP ఇన్స్పెక్టర్ వెబ్ యాప్‌ను URL నుండి CTRL క్లిక్ చేసి లోడ్ చేయండి (ఉదా: [http://127.0.0.1:6274/#resources](http://127.0.0.1:6274/#resources))
1. ట్రాన్స్‌పోర్ట్ టైప్‌ను `SSE` గా సెట్ చేయండి
1. `azd up` తర్వాత చూపించిన మీ రన్నింగ్ API నిర్వహణ SSE ఎండ్‌పాయింట్ URLను సెట్ చేసి **Connect** చేయండి:

    ```shell
    https://<apim-servicename-from-azd-output>.azure-api.net/mcp/sse
    ```

1. **List Tools**. ఒక టూల్‌పై క్లిక్ చేసి **Run Tool** చేయండి.

అన్ని దశలు సరిగా జరిగితే, మీరు ఇప్పుడు MCP సర్వర్‌కు కనెక్ట్ అయ్యారు మరియు టూల్‌ను కాల్ చేయగలిగారు.

## Azure కోసం MCP సర్వర్లు

[Remote-mcp-functions](https://github.com/Azure-Samples/remote-mcp-functions-dotnet): ఈ రిపాజిటరీలు Python, C# .NET లేదా Node/TypeScriptతో Azure Functions ఉపయోగించి కస్టమ్ రిమోట్ MCP (Model Context Protocol) సర్వర్లను నిర్మించడం మరియు డిప్లాయ్ చేయడం కోసం క్విక్‌స్టార్ట్ టెంప్లేట్‌లుగా ఉన్నాయి.

ఈ నమూనాలు డెవలపర్లకు పూర్తి పరిష్కారాన్ని అందిస్తాయి, ఇది:

- స్థానికంగా నిర్మించడం మరియు రన్ చేయడం: స్థానిక యంత్రంలో MCP సర్వర్‌ను అభివృద్ధి చేసి డీబగ్ చేయడం
- Azureకు డిప్లాయ్ చేయడం: సులభంగా క్లౌడ్‌కు azd up కమాండ్‌తో డిప్లాయ్ చేయడం
- క్లయింట్ల నుండి కనెక్ట్ కావడం: VS Code యొక్క Copilot ఏజెంట్ మోడ్ మరియు MCP ఇన్స్పెక్టర్ టూల్ సహా వివిధ క్లయింట్ల నుండి MCP సర్వర్‌కు కనెక్ట్ కావడం

### ముఖ్య ఫీచర్లు

- డిజైన్ ద్వారా భద్రత: MCP సర్వర్ కీలు మరియు HTTPS ఉపయోగించి సురక్షితం చేయబడింది
- ఆథెంటికేషన్ ఎంపికలు: బిల్ట్-ఇన్ ఆథ్ మరియు/లేదా API నిర్వహణ ఉపయోగించి OAuth మద్దతు
- నెట్‌వర్క్ ఐసోలేషన్: Azure వర్చువల్ నెట్‌వర్క్స్ (VNET) ఉపయోగించి నెట్‌వర్క్ ఐసోలేషన్ అనుమతిస్తుంది
- సర్వర్‌లెస్ ఆర్కిటెక్చర్: స్కేలబుల్, ఈవెంట్-డ్రివెన్ ఎగ్జిక్యూషన్ కోసం Azure Functions ఉపయోగిస్తుంది
- స్థానిక అభివృద్ధి: సమగ్ర స్థానిక అభివృద్ధి మరియు డీబగ్ మద్దతు
- సులభమైన డిప్లాయ్‌మెంట్: Azureకు సులభమైన డిప్లాయ్‌మెంట్ ప్రక్రియ

రిపాజిటరీలో అన్ని అవసరమైన కాన్ఫిగరేషన్ ఫైళ్లు, సోర్స్ కోడ్ మరియు ఇన్‌ఫ్రాస్ట్రక్చర్ నిర్వచనాలు ఉన్నాయి, తద్వారా మీరు ప్రొడక్షన్-రెడీ MCP సర్వర్ అమలుతో త్వరగా ప్రారంభించవచ్చు.

- [Azure Remote MCP Functions Python](https://github.com/Azure-Samples/remote-mcp-functions-python) - Pythonతో Azure Functions ఉపయోగించి MCP యొక్క నమూనా అమలు

- [Azure Remote MCP Functions .NET](https://github.com/Azure-Samples/remote-mcp-functions-dotnet) - C# .NETతో Azure Functions ఉపయోగించి MCP యొక్క నమూనా అమలు

- [Azure Remote MCP Functions Node/Typescript](https://github.com/Azure-Samples/remote-mcp-functions-typescript) - Node/TypeScriptతో Azure Functions ఉపయోగించి MCP యొక్క నమూనా అమలు.

## ముఖ్యమైన పాఠాలు

- MCP SDKలు భాష-ప్రత్యేక టూల్స్‌ను అందించి బలమైన MCP పరిష్కారాలను అమలు చేయడానికి సహాయపడతాయి
