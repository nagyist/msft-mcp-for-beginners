# మద్దతు

## సమస్యలను ఎలా నమోదు చేయాలి మరియు సహాయం పొందాలి  

ఈ ప్రాజెక్ట్ బగ్స్ మరియు ఫీచర్ అభ్యర్థనలను ట్రాక్ చేయడానికి GitHub Issues ను ఉపయోగిస్తుంది. డూప్లికేట్లను నివారించడానికి కొత్త సమస్యలను నమోదు చేయడానికి ముందు ఉన్న సమస్యలను దయచేసి శోధించండి. కొత్త సమస్యల కోసం, మీ బగ్ లేదా ఫీచర్ అభ్యర్థనను కొత్త Issue గా నమోదు చేయండి.

ఈ ప్రాజెక్ట్ ఉపయోగించడంపై సహాయం మరియు ప్రశ్నల కోసం, దయచేసి:
- నిపుణులు మరియు సహకార డెవలపర్లు చేరుకునే [Azure AI Foundry Discord](https://discord.com/invite/ByRwuEEgH4) ను తనిఖీ చేయండి
- ప్రోటోకాల్ సంబంధిత ప్రశ్నల కోసం [Model Context Protocol Community](https://modelcontextprotocol.io/community/) వనరులను ఉపయోగించండి

## Microsoft మద్దతు విధానం  

Model Context Protocol (MCP) for Beginners ప్రాజెక్ట్‌కు మద్దతు పై పేర్కొన్న వనరులకు పరిమితం.
