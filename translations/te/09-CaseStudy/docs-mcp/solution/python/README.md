# చైన్‌లిట్ & మైక్రోసాఫ్ట్ లెర్న్ డాక్స్ MCP తో స్టడీ ప్లాన్ జనరేటర్

## ముందస్తు అవసరాలు

- పైథాన్ 3.8 లేదా అంతకంటే పైగా
- పిప్ (Python ప్యాకేజ్ మేనేజర్)
- మైక్రోసాఫ్ట్ లెర్న్ డాక్స్ MCP సర్వర్‌కు కనెక్ట్ కావడానికి ఇంటర్నెట్ యాక్సెస్

## ఇన్‌స్టాలేషన్

1. ఈ రిపోజిటరీని క్లోన్ చేయండి లేదా ప్రాజెక్ట్ ఫైళ్లను డౌన్లోడ్ చేసుకోండి.
2. అవసరమైన డిపెండెన్సీలను ఇన్‌స్టాల్ చేయండి:

   ```bash
   pip install -r requirements.txt
   ```

## ఉపయోగం

### సన్నివేశం 1: డాక్స్ MCP కు సింపుల్ క్వెరీ
డాక్స్ MCP సర్వర్‌కు కనెక్ట్ అయ్యే, క్వెరీ పంపే, మరియు ఫలితాన్ని ప్రింట్ చేసే కమాండ్-లైన్ క్లయింట్.

1. స్క్రిప్ట్‌ను రన్ చేయండి:
   ```bash
   python scenario1.py
   ```
2. ప్రాంప్ట్ వద్ద మీ డాక్యుమెంటేషన్ ప్రశ్నను ఎంటర్ చేయండి.

### సన్నివేశం 2: స్టడీ ప్లాన్ జనరేటర్ (చైన్‌లిట్ వెబ్ యాప్)
వెబ్ ఆధారిత ఇంటర్‌ఫేస్ (చైన్‌లిట్ ఉపయోగించి) ఇది యూజర్లకు ఏదైనా సాంకేతిక అంశానికి వ్యక్తిగత, వారానికి వారీగా స్టడీ ప్లాన్ రూపొందించడానికి అనుమతిస్తుంది.

1. చైన్‌లిట్ యాప్‌ను ప్రారంభించండి:
   ```bash
   chainlit run scenario2.py
   ```
2. మీ టెర్మినల్‌లో ఇచ్చిన లోకల్ URL (ఉదా: http://localhost:8000) ను బ్రౌజర్‌లో ఓపెన్ చేయండి.
3. చాట్ విండోలో, మీ స్టడీ టాపిక్ మరియు మీరు చదవాలనుకునే వారాల సంఖ్యను ఎంటర్ చేయండి (ఉదా: "AI-900 సర్టిఫికేషన్, 8 వారాలు").
4. యాప్ వారానికి వారీగా స్టడీ ప్లాన్‌తో స్పందిస్తుంది, సంబంధిత మైక్రోసాఫ్ట్ లెర్న్ డాక్యుమెంటేషన్ లింకులతో సహా.

**అవసరమైన ఎన్విరాన్‌మెంట్ వేరియబుల్స్:**

సన్నివేశం 2 (Azure OpenAI తో చైన్‌లిట్ వెబ్ యాప్) ఉపయోగించడానికి, మీరు `python` డైరెక్టరీలో `.env` ఫైల్‌లో క్రింది ఎన్విరాన్‌మెంట్ వేరియబుల్స్ సెట్ చేయాలి:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

యాప్‌ను రన్ చేయడానికి ముందు ఈ విలువలను మీ Azure OpenAI వనరుల వివరాలతో పూరించండి.

> [!TIP]
> మీరు సులభంగా మీ స్వంత మోడల్స్‌ను [Azure AI Foundry](https://ai.azure.com/) ఉపయోగించి డిప్లాయ్ చేయవచ్చు.

### సన్నివేశం 3: VS కోడ్‌లో MCP సర్వర్‌తో ఇన్-ఎడిటర్ డాక్స్

డాక్యుమెంటేషన్ కోసం బ్రౌజర్ టాబ్స్ మార్చుకోవడం బదులు, మీరు Microsoft Learn Docs ను నేరుగా VS కోడ్‌లో MCP సర్వర్ ఉపయోగించి తీసుకురావచ్చు. ఇది మీకు వీలుగా చేస్తుంది:
- VS కోడ్‌లోనే డాక్స్ శోధించి చదవడం, కోడింగ్ వాతావరణం విడిచి వెళ్లకుండా.
- README లేదా కోర్సు ఫైళ్లలో డాక్యుమెంటేషన్ సూచనలు మరియు లింకులు నేరుగా చేర్చడం.
- GitHub Copilot మరియు MCP ను కలిసి ఉపయోగించి సులభమైన, AI ఆధారిత డాక్యుమెంటేషన్ వర్క్‌ఫ్లో.

**ఉదాహరణ ఉపయోగాలు:**
- కోర్సు లేదా ప్రాజెక్ట్ డాక్యుమెంటేషన్ రాయడంలో READMEకి త్వరగా సూచనల లింకులు జోడించడం.
- కోడ్ జనరేట్ చేయడానికి Copilot ఉపయోగించి, సంబంధిత డాక్స్ వెంటనే కనుగొనడానికి MCP ఉపయోగించడం.
- ఎడిటర్‌లో ఫోకస్ ఉండి ఉత్పాదకత పెంచుకోవడం.

> [!IMPORTANT]
> మీ వర్క్‌స్పేస్‌లో సరైన [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) కాన్ఫిగరేషన్ ఉందని నిర్ధారించుకోండి (స్థానం `.vscode/mcp.json`).

## సన్నివేశం 2 కోసం చైన్‌లిట్ ఎందుకు?

చైన్‌లిట్ అనేది సంభాషణాత్మక వెబ్ అప్లికేషన్లను నిర్మించడానికి ఆధునిక ఓపెన్-సోర్స్ ఫ్రేమ్‌వర్క్. ఇది Microsoft Learn Docs MCP సర్వర్ వంటి బ్యాక్‌ఎండ్ సర్వీసులకు కనెక్ట్ అయ్యే చాట్-ఆధారిత యూజర్ ఇంటర్‌ఫేస్‌లను సులభంగా సృష్టించడానికి సహాయపడుతుంది. ఈ ప్రాజెక్ట్ వ్యక్తిగతీకరించిన స్టడీ ప్లాన్లను రియల్ టైంలో సృష్టించడానికి చైన్‌లిట్‌ను ఉపయోగిస్తుంది. చైన్‌లిట్ ద్వారా, మీరు త్వరగా చాట్-ఆధారిత టూల్స్‌ను నిర్మించి, ఉత్పాదకత మరియు అభ్యాసాన్ని మెరుగుపరచవచ్చు.

## ఇది ఏమి చేస్తుంది

ఈ యాప్ యూజర్లకు ఒక టాపిక్ మరియు వ్యవధిని ఎంటర్ చేయడం ద్వారా వ్యక్తిగత స్టడీ ప్లాన్ సృష్టించడానికి అనుమతిస్తుంది. యాప్ మీ ఇన్‌పుట్‌ను పార్స్ చేసి, సంబంధిత కంటెంట్ కోసం Microsoft Learn Docs MCP సర్వర్‌ను క్వెరీ చేస్తుంది, ఫలితాలను వారానికి వారీగా ప్లాన్‌గా ఏర్పాటు చేస్తుంది. ప్రతి వారపు సిఫార్సులు చాట్‌లో ప్రదర్శించబడతాయి, దీని ద్వారా మీరు సులభంగా అనుసరించవచ్చు మరియు మీ పురోగతిని ట్రాక్ చేయవచ్చు. ఈ ఇంటిగ్రేషన్ ద్వారా మీరు ఎప్పుడూ తాజా, అత్యంత సంబంధిత అభ్యాస వనరులను పొందుతారు.

## నమూనా క్వెరీలు

యాప్ ఎలా స్పందిస్తుందో చూడటానికి ఈ క్వెరీలను చాట్ విండోలో ప్రయత్నించండి:

- `AI-900 సర్టిఫికేషన్, 8 వారాలు`
- `Azure Functions నేర్చుకోండి, 4 వారాలు`
- `Azure DevOps, 6 వారాలు`
- `Azureపై డేటా ఇంజనీరింగ్, 10 వారాలు`
- `Microsoft సెక్యూరిటీ ఫండమెంటల్స్, 5 వారాలు`
- `Power Platform, 7 వారాలు`
- `Azure AI సర్వీసులు, 12 వారాలు`
- `క్లౌడ్ ఆర్కిటెక్చర్, 9 వారాలు`

ఈ ఉదాహరణలు యాప్ వివిధ అభ్యాస లక్ష్యాలు మరియు కాలపరిమితుల కోసం ఎంత సౌలభ్యంగా ఉపయోగించవచ్చో చూపిస్తాయి.
