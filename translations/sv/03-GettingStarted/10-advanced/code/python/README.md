# Kör exempel

## Ställ in virtuell miljö

```sh
python -m venv venv
source ./venv/bin/activate
```

## Installera beroenden

```sh
pip install "mcp[cli]"
```

## Kör kod

```sh
python client.py
```

Du bör se texten:

```text
Available tools: ['add']
Result of add tool: meta=None content=[TextContent(type='text', text='8.0', annotations=None, meta=None)] structuredContent=None isError=False
```
