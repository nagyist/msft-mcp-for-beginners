# Zagon vzorca

## Namestitev odvisnosti

```bash
npm install
```

## Gradnja

```bash
npm run build
```

## Zagon strežnika

```bash
npm start
```

to bi moralo zagnati strežnik

### Zagon odjemalca

V ločenem terminalu zaženite:

```bash
node run client
```
