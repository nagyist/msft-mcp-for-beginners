# Exécuter l'exemple

## Configurer l'environnement virtuel

```sh
python -m venv venv
source ./venv/bin/activate
```

## Installer les dépendances

```sh
pip install "mcp[cli]"
```

## Exécuter le code

```sh
python client.py
```
