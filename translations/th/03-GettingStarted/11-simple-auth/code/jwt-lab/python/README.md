# รันตัวอย่าง

## ตั้งค่าสภาพแวดล้อม

```sh
python -m venv
source ./venv/bin/activate
```

## ติดตั้ง dependencies

```sh
pip install PyJWT
```

## รันโค้ด

```sh
python lab.py
```
