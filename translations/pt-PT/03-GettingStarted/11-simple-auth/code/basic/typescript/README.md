# Executar exemplo

## Instalar dependências

```bash
npm install
```

## Construir

```bash
npm run build
```

## Iniciar servidor

```bash
npm start
```

isto deve iniciar o servidor

### Iniciar cliente

Num terminal separado, execute:

```bash
node run client
```
