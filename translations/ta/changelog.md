# மாற்றங்கள்: MCP ஆரம்பக்கற்கான பாடத்திட்டம்

இந்த ஆவணம் ஆரம்பக்கற்கான மாடல் கண்டெக்ஸ்ட் பிரொடோக்கால் (MCP) பாடத்திட்டத்தில் செய்யப்பட்ட அனைத்து முக்கியமான மாற்றங்களின் பதிவாக செயல்படுகிறது. மாற்றங்கள் தேதி வரிசையில் மாற்றப்பட்டுள்ளன (புதிய மாற்றங்கள் முதலில்).

## ஜனவரி 28, 2026

### MCP спецификация 2025-11-25 இணக்கம் ஆய்வு

#### அடிப்படைக் கருத்துக்கள் மேம்பாடு (01-CoreConcepts/)
- **புதிய கிளையன்ட் முதன்மை - ரூட்ஸ்**: கோப்பு முறை எல்லைகளையும் அணுகல் அனுமதிகளையும் புரிந்துகொள்ள சர்வர்களுக்கு உதவும் முழுமையான ரூட்ஸ் கிளையன்ட் முதன்மை ஆவணப்படுத்தப்பட்டன
- **கருவி குறிப்பு**: சிறந்த கருவி இயக்க முடிவுகளுக்கான கருவி நடத்தை குறிப்பு (`readOnlyHint`, `destructiveHint`) ஆவணப்படுத்தப்பட்டது
- **சேம்பிளிங்-ல் கருவி அழைப்பு**: மாதிரி இயக்ககருணை சேம்பிளிங் கோரிக்கைகளின் போது `tools` மற்றும் `toolChoice` அளவுருக்கள் சேர்க்கப்பட்டன
- **URL வழி தூண்டுதல்**: சர்வர் தொடங்கும் வெளிப்புற வலை தொடர்புகளுக்கான URL அடிப்படையிலான தூண்டுதல் ஆவணம் சேர்க்கப்பட்டது
- **பணிகள் (பரிசோதனை)**: நிரந்தர செயல் போர்டர்கள் மற்றும் தாமதமான முடிவு மீட்புக்கான பரிசோதனை பணிகளின் புதிய பகுதி சேர்க்கப்பட்டது
- **ஐகான்கள் ஆதரவு**: கருவிகள், வளங்கள், வள மாதிரிகள் மற்றும் கேள்விகள் இப்போது கூடுதல் மெட்டாடேட்டாவுடன் ஐகான்களை கொண்டிருக்க முடியும்

#### ஆவணங்கள் புதுப்பிப்பு
- **README.md**: MCP спецификация 2025-11-25 பதிப்பு குறிப்பு மற்றும் தேதி அடிப்படையிலான பதிப்பு விளக்கம் சேர்க்கப்பட்டது
- **study_guide.md**: அடிப்படைக் கருத்துக்களில் பணிகள் மற்றும் கருவி குறிப்பு சேர்த்து பாடத்திட்டம் புதுப்பிக்கப்பட்டது; ஆவணம் நேரம் புதுப்பிக்கப்பட்டது

#### спецификация இணக்கம் உறுதிப்படுத்தல்
- **பிரொடோக்கால் பதிப்பு**: அனைத்து ஆவணங்களும் தற்போதைய MCP спецификация 2025-11-25 க்கு இணங்க உள்ளன
- **கட்டமைப்பு ஒத்திசைவு**: இரு அடுக்கு கட்டமைப்பு (தரவு அடுக்கு + போக்குவரத்து அடுக்கு) ஆவணப்படுத்தல் சரியானது என உறுதிசெய்தது
- **முதன்மை ஆவணங்கள்**: சர்வர் முதன்மைகள் (வளங்கள், கேள்விகள், கருவிகள்) மற்றும் கிளையன்ட் முதன்மைகள் (சேம்பிளிங், தூண்டுதல், பதிவு, ரூட்ஸ்) சரிபார்க்கப்பட்டன
- **போக்குவரத்து முறைமைகள்**: STDIO மற்றும் ஸ்ட்ரீமிங் HTTP போக்குவரத்து ஆவணப்படுத்தல் சரியானது என்பதை உறுதிசெய்தது
- **பாதுகாப்பு வழிகாட்டல்**: தற்போதைய MCP பாதுகாப்பு சிறந்த நடைமுறைகளுடன் ஒத்தசெயலில் உள்ளது

- **அரசாங்க அமைப்பு**: MCP நாடாளுமன்ற அமைப்புகளில் வேலை குழுக்கள் மற்றும் ஆர்வ குழுக்கள் அதிகாரப்பூர்வமாக அறிவிக்கப்பட்டன

இந்தச் சேர்ப்புகள் இன்னும் பலவாய்ந்த, பாதுகாப்பான மற்றும் நெகிழ்வான MCP சர்வர் அமலாக்கங்களை உருவாக்குவதற்கான நடைமுறை, கைமுறை வழிகாட்டல்களை வழங்குகின்றன; அடிப்படைக் கருத்துக்களை மேம்பட்ட உற்பத்தியான முறைமைகளுடன் இணைக்கின்றன.

- **கட்டமைக்கப்பட்ட கற்றல் இலக்குகள்**: தெளிவான, செயற்பாடான முடிவுகளுடன் கற்றல் இலக்குகள் மேம்படுத்தப்பட்டுள்ளன
- **தலைமுறை இணைப்புகள்**: தொடர்புடைய பாதுகாப்பு மற்றும் மைய கருத்து தலைப்புகளுக்கு இடையே இணைப்புகள் சேர்க்கப்பட்டன
- **தற்போதைய தகவல்கள்**: அனைத்து தேதி குறிப்பு மற்றும் விவரிப்பு இணைப்புகள் தற்போதைய தரநிலைகளுக்கு புதுப்பிக்கப்பட்டன
- **அமல்படுத்தல் வழிகாட்டி**: இரு பிரிவிலும் முழுவதும் வெளிப்படையான, செயற்பாடான அமல்படுத்தும் வழிகாட்டிகள் சேர்க்கப்பட்டன
