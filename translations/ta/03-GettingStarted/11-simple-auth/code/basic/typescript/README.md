# மாதிரி இயக்கவும்

## தேவையான பொருட்களை நிறுவவும்

```bash
npm install
```

## கட்டமைக்கவும்

```bash
npm run build
```

## சர்வரை தொடங்கவும்

```bash
npm start
```

இது சர்வரை தொடங்க வேண்டும்

### கிளையண்டை தொடங்கவும்

வேறு ஒரு டெர்மினலில், இதை இயக்கவும்:

```bash
node run client
```
