## தொடக்கம்  

[![Build Your First MCP Server](../../../translated_images/ta/04.0ea920069efd979a.webp)](https://youtu.be/sNDZO9N4m9Y)

_(இந்த பாடத்தின் வீடியோவை காண மேலே உள்ள படத்தை கிளிக் செய்யவும்)_

இந்த பகுதியில் பல பாடங்கள் உள்ளன:

- **1 உங்கள் முதல் சர்வர்**, இந்த முதல் பாடத்தில், உங்கள் முதல் சர்வரை உருவாக்க மற்றும் அதை இன்ஸ்பெக்டர் கருவியால் ஆய்வு செய்வது எப்படி என்று கற்றுக்கொள்ளலாம், இது உங்கள் சர்வரை சோதிக்க மற்றும் பிழை திருத்த உதவுகிறது, [பாடத்திற்கு](01-first-server/README.md)

- **2 கிளையன்ட்**, இந்த பாடத்தில், உங்கள் சர்வருடன் இணைக்கக்கூடிய கிளையன்ட்டை எழுதுவது எப்படி என்று கற்றுக்கொள்ளலாம், [பாடத்திற்கு](02-client/README.md)

- **3 LLM உடன் கிளையன்ட்**, கிளையன்ட்டை எழுதுவதற்கான இன்னும் சிறந்த வழி, அதில் ஒரு LLM சேர்த்து அது உங்கள் சர்வருடன் என்ன செய்ய வேண்டும் என்று "சரிவு" செய்ய உதவுகிறது, [பாடத்திற்கு](03-llm-client/README.md)

- **4 Visual Studio Code இல் GitHub Copilot முகவரியின் MCP சர்வரை பயன்படுத்துதல்**. இங்கே, நாம் Visual Studio Code இல் இருந்து எங்கள் MCP சர்வரை இயக்குகிறோம், [பாடத்திற்கு](04-vscode/README.md)

- **5 stdio போக்குவரத்துச் சேவையகம்** stdio போக்குவரத்து என்பது உள்ளுரியாக MCP சர்வர்-கிளையன்ட் தொடர்புக்கு பரிந்துரைக்கப்படும் நடைமுறை, பாதுகாப்பான துணைசெறிவை செயல்முறையுடன் வழங்குகிறது [பாடத்திற்கு](05-stdio-server/README.md)

- **6 MCP உடன் HTTP ஸ்ட்ரீமிங் (ஸ்ட்ரீமபிள் HTTP)**. சமகால HTTP ஸ்ட்ரீமிங் போக்குவரத்துக்குறித்துத் தெரிந்து கொள்ளவும் (மொத்தமாக MCP குறிப்பூட்டுதலைப் பின்பற்றும் பரிந்துரைக்கப்பட்ட முறையாக), முன்னேற்ற அறிவிப்புகள் மற்றும் ஸ்ட்ரீமபிள் HTTP மூலம் அளவிடக்கூடிய, நேரடியாக வேலை செய்யக்கூடிய MCP சர்வர்கள் மற்றும் கிளையன்ட்டுகளை எப்படி செயல்படுத்துவது என்பதையும் கற்றுக்கொள்ளவும். [பாடத்திற்கு](06-http-streaming/README.md)

- **7 VSCode இற்கான AI கருவி தொகுப்பை பயன்படுத்துதல்** உங்கள் MCP கிளையன்ட்டுகளை மற்றும் சர்வர்களை சோதிக்க மற்றும் பரிசோதிக்க [பாடத்திற்கு](07-aitk/README.md)

- **8 சோதனை**. இங்கே நாம் எமது சர்வர் மற்றும் கிளையன் பலவிதமாக சோதனை செய்ய எப்படி என்பதை கவனிப்போம், [பாடத்திற்கு](08-testing/README.md)

- **9கருவூலம்**. எமது MCP தீர்வுகளை தொடர்ந்து உட்புகுத்தும் விதிமுறைகளை பார்க்கும், [பாடத்திற்கு](09-deployment/README.md)

- **10 மேம்பட்ட சர்வர் பயன்பாடு**. மேம்பட்ட சர்வர் பயன்பாட்டைக் கொண்ட தலைப்பு, [பாடத்திற்கு](./10-advanced/README.md)

- **11 அங்கீகாரம்**. எளிய அங்கீகாரத்தை, அடிப்படையான அங்கீகாரம் முதல் JWT மற்றும் RBAC பயன்படுத்துவது வரை கையாளும். நீங்கள் இங்கே தொடங்கவும் பிறகு 5வது அத்தியாயத்தில் மேம்பட்ட தலைப்புகளைப் பார்க்கவும் மற்றும் 2வது அத்தியாயத்தில் பரிந்துரைக்கப்படும் பாதுகாப்பு பலப்படுத்துதல்களை மேற்கொள்ளவும் பரிந்துரைக்கப்படுகிறது, [பாடத்திற்கு](./11-simple-auth/README.md)

- **12 MCP ஹோஸ்ட்கள்**. பிரபலமான MCP ஹோஸ்ட் கிளையன்ட்டுகளை அமைத்து பயன்படுத்துதல், உதாரணமாக Claude Desktop, Cursor, Cline மற்றும் Windsurf. போக்குவரத்து வகைகள் மற்றும் கேள்வி தீர்வு பற்றி கற்று கொள்ளவும், [பாடத்திற்கு](./12-mcp-hosts/README.md)

- **13 MCP இன்ஸ்பெக்டர்**. MCP இன்ஸ்பெக்டர் கருவியைப் பயன்படுத்தி உங்கள் MCP சர்வர்களை பண்படுத்தவும் பரிசோதிக்கவும். கருவிகள், வளமுறைகள், மற்றும் நெறிமுறை செய்திகளை பிரச்னைத் தீர்க்க கற்றுக்கொள்ளவும், [பாடத்திற்கு](./13-mcp-inspector/README.md)

Model Context Protocol (MCP) என்பது LLMக்களுக்கு கண்காணிப்பை வழங்கும் விதத்தை ஒருங்கிணைக்கும் திறந்த நெறிமுறை. MCP ஐ AI பயன்பாடுகளுக்கு USB-C போர்ட் போன்றவையாக நினைக்கலாம் – இது AI மாதிரிகளைக் கொன்டைனர்கள் மற்றும் கருவிகளுடன் இணைக்கும் ஒருவித அளவுகோலான வழியை வழங்குகிறது.

## கற்றல் நோக்குகள்

இந்த பாடம் முடிவடையும்போது, நீங்கள்:

- C#, Java, Python, TypeScript மற்றும் JavaScript இல் MCP வளர்ப்பு சூழலை அமைப்பதைக் கற்றுக்கொள்ளவுள்ளீர்கள்
- தனிப்பயன் அம்சங்கள் (வளம், ஊக்குவிப்புகள் மற்றும் கருவிகள்) உடன் அடிப்படையான MCP சர்வர்களை கட்டமைத்து பரியளிக்க இயலும்
- MCP சர்வர்களுடன் இணைந்து இயங்கும் ஹோஸ்ட் பயன்பாடுகளை உருவாக்க இயலும்
- MCP அமல்படுத்தலினை சோதித்து பிழைத்திருத்த முடியும்
- பொதுவான அமைப்பு சவால்களையும் அவற்றின் தீர்வுகளையும் புரிந்துகொள்ள முடியும்
- உங்கள் MCP செயலாக்கங்களை பிரபலமான LLM சேவைகளுடன் இணைக்க முடியும்

## உங்கள் MCP சூழலை அமைத்தல்

MCP உடன் பணியாற்றத் தொடங்குமுன், உங்கள் வளர்ப்பு சூழலை தயார் செய்து, அடிப்படைக் காரியவள்களை புரிந்து கொள்வது அவசியம். இப்பகுதி MCP உடன் மென்மையான தொடக்கத்திற்கான ஆரம்ப அமைப்புக் கட்டளைகளை வழிகாட்டும்.

### முன் தேவைகள்

MCP வளர்ப்பில் தோவுபட்டுப்போகும் முன், நீங்கள் கீழ்காணும் அம்சங்களை வைத்திருப்பதை உறுதிப்படுத்துங்கள்:

- **மென்பொருள் வளர்ப்பு சூழல்**: தேர்ந்தெடுத்த மொழிக்கான (C#, Java, Python, TypeScript அல்லது JavaScript)
- **IDE/தொகுப்பி**: Visual Studio, Visual Studio Code, IntelliJ, Eclipse, PyCharm அல்லது எந்தவொரு நவீன குறியீட்டு தொகுப்பியோடு
- **பகுப்பு மேலாளர்கள்**: NuGet, Maven/Gradle, pip, அல்லது npm/yarn
- **API விசைகள்**: உங்கள் ஹோஸ்ட் பயன்பாடுகளில் பயன்படுத்தும் AI சேவைக்கானவைகள்

### அதிகாரப்பூர்வ SDKக்கள்

எதிர்கால அத்தியாயங்களில் Python, TypeScript, Java மற்றும் .NET பயன்படுத்தி தீர்வுகள் காணப்படுவதை நீங்கள் பார்க்கப்போகிறீர்கள். கீழே அதிகாரப்பூர்வமாக ஆதரிக்கப்படும் SDKக்கள் உள்ளன.

MCP பல மொழிகளுக்கு அதிகாரப்பூர்வ SDKக்களை வழங்குகிறது ([MCP Specification 2025-11-25](https://spec.modelcontextprotocol.io/specification/2025-11-25/)ன் இணைப்புடன்):

- [C# SDK](https://github.com/modelcontextprotocol/csharp-sdk) - Microsoft உடன் இணைந்து பராமரிக்கப்படுகிறது
- [Java SDK](https://github.com/modelcontextprotocol/java-sdk) - Spring AI உடன் இணைந்து பராமரிக்கப்படுகிறது
- [TypeScript SDK](https://github.com/modelcontextprotocol/typescript-sdk) - அதிகாரப்பூர்வ TypeScript செயல்படுத்தல்
- [Python SDK](https://github.com/modelcontextprotocol/python-sdk) - அதிகாரப்பூர்வ Python செயல்படுத்தல் (FastMCP)
- [Kotlin SDK](https://github.com/modelcontextprotocol/kotlin-sdk) - அதிகாரப்பூர்வ Kotlin செயல்படுத்தல்
- [Swift SDK](https://github.com/modelcontextprotocol/swift-sdk) - Loopwork AI உடன் இணைந்து பராமரிக்கப்படுகிறது
- [Rust SDK](https://github.com/modelcontextprotocol/rust-sdk) - அதிகாரப்பூர்வ Rust செயல்படுத்தல்
- [Go SDK](https://github.com/modelcontextprotocol/go-sdk) - அதிகாரப்பூர்வ Go செயல்படுத்தல்

## முக்கிய எடுத்துக்காட்டுகள்

- MCP வளர்ப்பு சூழலை அமைப்பது மொழி-சுட்டி SDKக்களுடன் எளிதாக உள்ளது
- MCP சர்வர்களை கட்டமைப்பது, கருவிகள் மற்றும் தெளிவான விளக்கங்களுடன் பதிவு செய்வதாகும்
- MCP கிளையன்ட்கள் சர்வர் மற்றும் மாதிரிகளுடன் இணைந்து விரிவாக்கப்பட்ட திறன்களைப் பயன்படுத்துகின்றன
- சோதனை மற்றும் பிழைத்திருத்துதல் MCP செயலாக்கங்களுக்குக் கூறுப்படி தேவை
- உட்புகுத்தல் தெரிவுகள் உள்ளூர் வளர்ச்சியிலிருந்து மேக அடிப்படையிலான தீர்வுகளுக்கு மாறுபடுகின்றன
