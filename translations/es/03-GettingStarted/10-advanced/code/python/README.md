# Ejecutar ejemplo

## Configurar entorno virtual

```sh
python -m venv venv
source ./venv/bin/activate
```

## Instalar dependencias

```sh
pip install "mcp[cli]"
```

## Ejecutar código

```sh
python client.py
```
