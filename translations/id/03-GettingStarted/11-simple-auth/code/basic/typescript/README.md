# Jalankan contoh

## Instal dependensi

```bash
npm install
```

## Bangun

```bash
npm run build
```

## Mulai server

```bash
npm start
```

ini seharusnya memulai server

### Mulai klien

Di terminal terpisah, jalankan:

```bash
node run client
```
