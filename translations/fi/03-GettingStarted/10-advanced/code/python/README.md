# Aja esimerkki

## Luo virtuaalinen ympäristö

```sh
python -m venv venv
source ./venv/bin/activate
```

## Asenna riippuvuudet

```sh
pip install "mcp[cli]"
```

## Suorita koodi

```sh
python client.py
```

Näet tekstin:

