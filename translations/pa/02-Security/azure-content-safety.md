# ਅਡਵਾਂਸ MCP ਸੁਰੱਖਿਆ ਐਜ਼ੂਰ ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਨਾਲ

> **OWASP MCP ਜੋਖਮ ਦਾ ਸਮਾਧਾਨ ਕੀਤਾ ਗਿਆ**: [MCP06 - ਪ੍ਰਾਸੰਗਿਕ ਪੇਲੋਡਾਂ ਰਾਹੀਂ ਪ੍ਰਾਂਪਟ ਇੰਜੈਕਸ਼ਨ](https://microsoft.github.io/mcp-azure-security-guide/mcp/mcp06-prompt-injection/)

ਐਜ਼ੂਰ ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਕਈ ਸ਼ਕਤੀਸ਼ালী ਸੰਦ ਪ੍ਰਦਾਨ ਕਰਦੀ ਹੈ ਜੋ ਤੁਹਾਡੇ MCP ਇੰਪਲੀਮੈਂਟੇਸ਼ਨਾਂ ਦੀ ਸੁਰੱਖਿਆ ਨੂੰ ਬਿਹਤਰ ਬਣਾਉਂਦੇ ਹਨ। ਹੱਥ-ਗੰਦੇ ਇੰਪਲੀਮੈਂਟੇਸ਼ਨ ਅਨੁਭਵ ਲਈ, ਵੇਖੋ [MCP ਸੁਰੱਖਿਆ ਸਮਿੱਟ ਵਰਕਸ਼ਾਪ (ਸ਼ੇਰਪਾ)](https://azure-samples.github.io/sherpa/) ਕੈਂਪ 3: I/O ਸੁਰੱਖਿਆ।

## ਪ੍ਰਾਂਪਟ ਸੀਲਡ

ਮਾਈਕ੍ਰੋਸਾਫਟ ਦੇ AI ਪ੍ਰਾਂਪਟ ਸੀਲਡ ਵਧੀਆ ਸੁਰੱਖਿਆ ਪ੍ਰਦਾਨ ਕਰਦੇ ਹਨ, ਜੋ ਸਿੱਧੇ ਅਤੇ ਅਸਿੱਧੇ ਪ੍ਰਾਂਪਟ ਇੰਜੈਕਸ਼ਨ ਹਮਲੇਆਂ ਤੋਂ ਬਚਾਵ ਕਰਦੇ ਹਨ:

1. **ਅਡਵਾਂਸ ਡਿਟੈਕਸ਼ਨ**: ਕੰਟੈਂਟ ਵਿੱਚ ਸ਼ਾਮਿਲ ਮਾਲੀਸ਼ੀਅਸ ਹੁਕਮਾਂ ਦੀ ਪਛਾਣ ਲਈ ਮਸ਼ੀਨ ਲਰਨਿੰਗ ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ।
2. **ਸਪੌਟਲਾਈਟਿੰਗ**: ਇਨਪੁੱਟ ਟੈਕਸਟ ਨੂੰ ਬਦਲ ਕੇ AI ਸਿਸਟਮਾਂ ਨੂੰ ਵੈਧ ਹੁਕਮਾਂ ਅਤੇ ਬਾਹਰੀ ਇਨਪੁੱਟ ਵਿਚਕਾਰ ਫਰਕ ਕਰਨ ਵਿੱਚ ਮਦਦ ਕਰਦਾ ਹੈ।
3. **ਡੈਲਮੀਟਰਜ਼ ਅਤੇ ਡੇਟਾ ਮਾਰਕਿੰਗ**: ਭਰੋਸੇਯੋਗ ਅਤੇ ਅਣਭਰੋਸੇਯੋਗ ਡਾਟਾ ਵਿਚਕਾਰ ਹੱਦਾਂ ਨਿਸ਼ਾਨਦਾਰ ਕਰਦਾ ਹੈ।
4. **ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਇੰਟਿਗ੍ਰੇਸ਼ਨ**: ਐਜ਼ੂਰ AI ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਨਾਲ ਕੰਮ ਕਰਦਾ ਹੈ ਤਾਂ ਜੋ ਜੇਲਬ੍ਰੇਕ ਕੋਸ਼ਿਸ਼ਾਂ ਅਤੇ ਹਾਨਿਕਾਰਕ ਸਮੱਗਰੀ ਦੀ ਪਛਾਣ ਕੀਤੀ ਜਾ ਸਕੇ।
5. **ਲਗਾਤਾਰ ਅੱਪਡੇਟਸ**: ਮਾਈਕ੍ਰੋਸਾਫਟ ਨਿੱਯਮਤ ਤੌਰ 'ਤੇ ਉੱਭਰ ਰਹੀਆਂ ਖਤਰਿਆਂ ਵਿਰੁੱਧ ਸੁਰੱਖਿਆ ਯੰਤਰਾਂ ਨੂੰ ਅਪਡੇਟ ਕਰਦਾ ਹੈ।

## MCP ਨਾਲ ਐਜ਼ੂਰ ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਲਾਗੂ ਕਰਨਾ

ਇਹ ਤਰੀਕਾ ਬਹੁ-ਪੱਧਰੀ ਸੁਰੱਖਿਆ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ:
- ਪ੍ਰਕਿਰਿਆ ਤੋਂ ਪਹਿਲਾਂ ਇਨਪੁੱਟਾਂ ਦੀ ਸਕੈਨਿੰਗ
- ਵਾਪਸੀ ਤੋਂ ਪਹਿਲਾਂ ਆਉਟਪੁੱਟਾਂ ਦੀ ਜਾਂਚ
- ਜਾਣੇ-ਪਹਚਾਣੇ ਹਾਨਿਕਾਰਕ ਪੈਟਰਨ ਲਈ ਬਲਾਕ ਲਿਸਟਾਂ ਦੀ ਵਰਤੋਂ
- ਐਜ਼ੂਰ ਦੇ ਲਗਾਤਾਰ ਅੱਪਡੇਟ ਕੀਤੇ ਜਾ ਰਹੇ ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਮਾਡਲਾਂ ਦੀ ਵਰਤੋਂ

## ਐਜ਼ੂਰ ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਸਰੋਤ

ਆਪਣੇ MCP ਸਰਵਰਾਂ ਨਾਲ ਐਜ਼ੂਰ ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਲਾਗੂ ਕਰਨ ਬਾਰੇ ਵਧੇਰੇ ਜਾਣਨ ਲਈ, ਇਹ ਅਧਿਕਾਰਿਕ ਸਰੋਤਾਂ ਦੀ ਸਲਾਹ ਲੋ:

1. [ਐਜ਼ੂਰ AI ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਦਸਤਾਵੇਜ਼ੀਕਰਨ](https://learn.microsoft.com/azure/ai-services/content-safety/) - ਐਜ਼ੂਰ ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਲਈ ਅਧਿਕਾਰਿਕ ਦਸਤਾਵੇਜ਼।
2. [ਪ੍ਰਾਂਪਟ ਸੀਲਡ ਦਸਤਾਵੇਜ਼ੀਕਰਨ](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/prompt-shield) - ਪ੍ਰਾਂਪਟ ਇੰਜੈਕਸ਼ਨ ਹਮਲਿਆਂ ਨੂੰ ਰੋਕਣ ਬਾਰੇ ਸਿੱਖੋ।
3. [ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ API ਰੈਫਰੰਸ](https://learn.microsoft.com/rest/api/contentsafety/) - ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਲਈ ਵਿਸਥਾਰਿਤ API ਰੈਫਰੰਸ।
4. [ਤੁਰੰਤ ਸ਼ੁਰੂਆਤ: C# ਨਾਲ ਐਜ਼ੂਰ ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-csharp) - C# ਦੀ ਵਰਤੋਂ ਨਾਲ ਤੁਰੰਤ ਲਾਗੂ ਕਰਨ ਲਈ ਮਾਰਗਦਰਸ਼ਨ।
5. [ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਕਲਾਇੰਟ ਲਾਇਬ੍ਰੇਰੀਜ਼](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-client-libraries-rest-api) - ਵੱਖ-ਵੱਖ ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾਵਾਂ ਲਈ ਕਲਾਇੰਟ ਲਾਇਬ੍ਰੇਰੀਜ਼।
6. [ਜੇਲਬ੍ਰੇਕ ਕੋਸ਼ਿਸ਼ਾਂ ਦੀ ਪਛਾਣ](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/jailbreak-detection) - ਜੇਲਬ੍ਰੇਕ ਕੋਸ਼ਿਸ਼ਾਂ ਨੂੰ ਪਛਾਣਣ ਅਤੇ ਰੋਕਣ ਬਾਰੇ ਖਾਸ ਮਾਰਗਦਰਸ਼ਨ।
7. [ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਲਈ ਬਿਹਤਰ ਅਮਲ](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/best-practices) - ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਨੂੰ ਪ੍ਰਭਾਵਸ਼ਾਲੀ ਢੰਗ ਨਾਲ ਅਮਲ ਕਰਨ ਲਈ ਬਿਹਤਰ ਅਭਿਆਸ।

ਵਧੇਰੇ ਵਿਸਤ੍ਰਿਤ ਲਾਗੂ ਕਰਨ ਲਈ, ਸਾਡੇ [ਐਜ਼ੂਰ ਸਮੱਗਰੀ ਸੁਰੱਖਿਆ ਲਾਗੂ ਕਰਨ ਦੀ ਗਾਈਡ](./azure-content-safety-implementation.md) ਨੂੰ ਵੇਖੋ।
