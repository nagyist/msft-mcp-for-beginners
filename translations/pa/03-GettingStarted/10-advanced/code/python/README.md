# ਨਮੂਨਾ ਚਲਾਓ

## ਵਰਚੁਅਲ ਵਾਤਾਵਰਨ ਸੈਟ ਕਰੋ

```sh
python -m venv venv
source ./venv/bin/activate
```

## ਨਿਰਭਰਤਾਵਾਂ ਇੰਸਟਾਲ ਕਰੋ

```sh
pip install "mcp[cli]"
```

## ਕੋਡ ਚਲਾਓ

```sh
python client.py
```
