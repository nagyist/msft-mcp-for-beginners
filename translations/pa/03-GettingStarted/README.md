## Getting Started  

[![Build Your First MCP Server](../../../translated_images/pa/04.0ea920069efd979a.webp)](https://youtu.be/sNDZO9N4m9Y)

_(ਇਸ ਪਾਠ ਦਾ ਵੀਡੀਓ ਦੇਖਣ ਲਈ ਉਪਰ ਦਿੱਤੀ ਤਸਵੀਰ 'ਤੇ ਕਲਿੱਕ ਕਰੋ)_

ਇਹ ਸੈਕਸ਼ਨ ਕਈ ਪਾਠਾਂ 'ਤੇ مشتمل ਹੈ:

- **1 ਤੁਹਾਡਾ ਪਹਿਲਾ ਸਰਵਰ**, ਇਸ ਪਹਿਲੇ ਪਾਠ ਵਿੱਚ, ਤੁਸੀਂ ਆਪਣੇ ਪਹਿਲੇ ਸਰਵਰ ਨੂੰ ਬਣਾਉਣਾ ਅਤੇ ਇੰਸਪੈਕਟਰ ਟੂਲ ਨਾਲ ਇਸ ਦੀ ਜਾਂਚ ਕਰਨਾ ਸਿੱਖੋਗੇ, ਜੋ ਕਿ ਸਰਵਰ ਦੀ ਟੈਸਟਿੰਗ ਅਤੇ ਡੀਬੱਗਿੰਗ ਦਾ ਇੱਕ ਕੀਮਤੀ ਤਰੀਕਾ ਹੈ, [ਪਾਠ ਵੱਲ](01-first-server/README.md)

- **2 ਕਲਾਇੰਟ**, ਇਸ ਪਾਠ ਵਿੱਚ, ਤੁਸੀਂ ਇੱਕ ਐਸਾ ਕਲਾਇੰਟ ਲਿਖਣਾ ਸਿੱਖੋਗੇ ਜੋ ਤੁਹਾਡੇ ਸਰਵਰ ਨਾਲ ਜੁੜ ਸਕੇ, [ਪਾਠ ਵੱਲ](02-client/README.md)

- **3 ਲਐਲਐਮ ਦੇ ਨਾਲ ਕਲਾਇੰਟ**, ਕਲਾਇੰਟ ਲਿਖਣ ਦਾ ਇੱਕ ਹੋਰ ਬਿਹਤਰ ਤਰੀਕਾ ਹੈ ਕਿ ਇਸ ਵਿੱਚ LLM ਸ਼ਾਮਲ ਕੀਤਾ ਜਾਵੇ ਜਿਵੇਂ ਕਿ ਇਹ ਤੁਹਾਡੇ ਸਰਵਰ ਨਾਲ "ਸੰਚਾਰ" ਕਰ ਸਕੇ ਕਿ ਕੀ ਕਰਨਾ ਹੈ, [ਪਾਠ ਵੱਲ](03-llm-client/README.md)

- **4 ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ ਕੋਡ ਵਿੱਚ ਸਰਵਰ ਗਿਟਹੱਬ ਕੋਪਾਇਲਟ ਏਜੰਟ ਮੋਡ ਦੀ ਵਰਤੋਂ**। ਇੱਥੇ ਅਸੀਂ ਆਪਣੇ MCP ਸਰਵਰ ਨੂੰ ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ ਕੋਡ ਦੇ ਅੰਦਰ ਚਲਾਉਣ ਬਾਰੇ ਵੇਖ ਰਹੇ ਹਾਂ, [ਪਾਠ ਵੱਲ](04-vscode/README.md)

- **5 stdio ਟਰਾਂਸਪੋਰਟ ਸਰਵਰ** stdio ਟਰਾਂਸਪੋਰਟ ਸਥਾਨਕ MCP ਸਰਵਰ ਤੋਂ ਕਲਾਇੰਟ ਸੰਚਾਰ ਲਈ ਸਿਫਾਰਸ਼ੀ ਮਿਆਰੀ ਹੈ, ਜੋ ਸੁਪ੍ਰੋਸੈਸ ਅਧਾਰਿਤ ਸੰਚਾਰ ਅਤੇ ਇਨਬਿਲਟ ਪ੍ਰੋਸੈਸ ਆਇਸੋਲੇਸ਼ਨ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ [ਪਾਠ ਵੱਲ](05-stdio-server/README.md)

- **6 MCP ਨਾਲ HTTP ਸਟ੍ਰੀਮਿੰਗ (ਸਟ੍ਰੀਮੇਬਲ HTTP)**। ਆਧੁਨਿਕ HTTP ਸਟ੍ਰੀਮਿੰਗ ਟਰਾਂਸਪੋਰਟ ਬਾਰੇ ਜਾਣੋ (ਜੋ ਕਿ ਦੂਰ-ਦਰਾਜ MCP ਸਰਵਰਾਂ ਲਈ ਸਿਫਾਰਸ਼ੀ ਹੈ [MCP Specification 2025-11-25](https://spec.modelcontextprotocol.io/specification/2025-11-25/basic/transports/#streamable-http) ਅਨੁਸਾਰ), ਪ੍ਰਗਟੀ ਸੂਚਨਾਵਾਂ, ਅਤੇ ਸਟ੍ਰੀਮੇਬਲ HTTP ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਸਕੇਲ ਕਰਨਯੋਗ, ਰੀਅਲ-ਟਾਈਮ MCP ਸਰਵਰ ਅਤੇ ਕਲਾਇੰਟ ਬਣਾਉਣ ਦਾ ਤਰੀਕਾ। [ਪਾਠ ਵੱਲ](06-http-streaming/README.md)

- **7 VSCode ਲਈ AI ਟੂਲਕਿਟ ਦੀ ਵਰਤੋਂ** ਜਿਸ ਨਾਲ ਤੁਸੀਂ ਆਪਣੇ MCP ਕਲਾਇੰਟ ਅਤੇ ਸਰਵਰ ਨੂੰ ਲੈ ਕੇ ਕੰਮ ਕਰ ਸਕਦੇ ਹੋ ਅਤੇ ਟੈਸਟ ਕਰ ਸਕਦੇ ਹੋ [ਪਾਠ ਵੱਲ](07-aitk/README.md)

- **8 ਟੈਸਟਿੰਗ**। ਇੱਥੇ ਅਸੀਂ ਖਾਸ ਤੌਰ ਤੇ ਦੇਖਾਂਗੇ ਕਿ ਕਿਵੇਂ ਭਿੰਨ-ਭਿੰਨ ਤਰੀਕਿਆਂ ਨਾਲ ਆਪਣੇ ਸਰਵਰ ਅਤੇ ਕਲਾਇੰਟ ਨੂੰ ਟੈਸਟ ਕਰਨਾ ਹੈ, [ਪਾਠ ਵੱਲ](08-testing/README.md)

- **9 ਡਿਪਲੌਇਮੈਂਟ**। ਇਸ ਅਧਿਆਇ ਵਿੱਚ ਤੁਸੀਂ ਆਪਣੇ MCP ਸਮਾਧਾਨਾਂ ਨੂੰ ਡਿਪਲੌਇ ਕਰਨ ਦੇ ਵੱਖ-ਵੱਖ ਤਰੀਕੇ ਵੇਖੋਗੇ, [ਪਾਠ ਵੱਲ](09-deployment/README.md)

- **10 ਅਡਵਾਂਸਡ ਸਰਵਰ ਇਸਤੇਮਾਲ**। ਇਸ ਅਧਿਆਇ ਵਿੱਚ ਅਡਵਾਂਸਡ ਸਰਵਰ ਇਸਤੇਮਾਲ ਬਾਰੇ ਜਾਣਕਾਰੀ ਹੋਵੇਗੀ, [ਪਾਠ ਵੱਲ](./10-advanced/README.md)

- **11 ਪ੍ਰਮਾਣਿਕਤਾ**। ਇਸ ਅਧਿਆਇ ਵਿੱਚ ਸਾਦਾ ਪ੍ਰਮਾਣਿਕਤਾ ਜੋੜਨ ਦੀ ਵਿਧੀ ਸਿੱਖਾਈ ਜਾਵੇਗੀ, ਬੇਸਿਕ ਆਥ ਤੋਂ JWT ਅਤੇ RBAC ਦੀ ਵਰਤੋਂ ਤੱਕ। ਤੁਹਾਨੂੰ ਇੱਥੇ ਸ਼ੁਰੂ ਕਰਨ ਦੀ ਸਿਫਾਰਸ਼ ਕੀਤੀ ਜਾਂਦੀ ਹੈ ਅਤੇ ਫਿਰ ਚੈਪਟਰ 5 ਦੇ ਅਡਵਾਂਸਡ ਟੌਪਿਕਸ ਨੂੰ ਦੇਖੋ ਅਤੇ ਚੈਪਟਰ 2 ਦੀ ਸਿਫਾਰਸ਼ਾਂ ਦੁਆਰਾ ਵਾਧੂ ਸੁਰੱਖਿਆ ਸਖਤੀ ਕਰੋ, [ਪਾਠ ਵੱਲ](./11-simple-auth/README.md)

- **12 MCP ਹੋਸਟਸ**। ਕੁਝ ਪ੍ਰਸਿੱਧ MCP ਹੋਸਟ ਕਲਾਇੰਟ ਜਿਵੇਂ ਕਿ Claude Desktop, Cursor, Cline ਅਤੇ Windsurf ਨੂੰ ਸੰਰਚਿਤ ਅਤੇ ਵਰਤੋਂ ਕਰੋ। ਟਰਾਂਸਪੋਰਟ ਕਿਸਮਾਂ ਅਤੇ ਟਰੱਬਲਸ਼ੂਟਿੰਗ ਸਿੱਖੋ, [ਪਾਠ ਵੱਲ](./12-mcp-hosts/README.md)

- **13 MCP ਇੰਸਪੈਕਟਰ**। MCP ਇੰਸਪੈਕਟਰ ਟੂਲ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਆਪਣੇ MCP ਸਰਵਰਾਂ ਦੀ ਇੰਟਰੈਕਟਿਵ ਡੀਬੱਗਿੰਗ ਅਤੇ ਟੈਸਟਿੰਗ ਕਰੋ। ਟਰੱਬਲਸ਼ੂਟਿੰਗ ਟੂਲ, ਸਰੋਤ ਸਾਧਨ ਅਤੇ ਪ੍ਰੋਟੋਕੋਲ ਸੁਨੇਹਿਆਂ ਬਾਰੇ ਸਿੱਖੋ, [ਪਾਠ ਵੱਲ](./13-mcp-inspector/README.md)

ਮਾਡਲ ਕਾਨਟੈਕਸਟ ਪ੍ਰੋਟੋਕੋਲ (MCP) ਇੱਕ ਖੁੱਲ੍ਹਾ ਪ੍ਰੋਟੋਕੋਲ ਹੈ ਜੋ ਕਿ ਇਹ ਮਿਆਰੀਕਰਨ ਕਰਦਾ ਹੈ ਕਿ ਐਪਲੀਕੇਸ਼ਨਾਂ LLMs ਨੂੰ ਕਿਵੇਂ ਸੰਦਰਭ ਪ੍ਰਦਾਨ ਕਰਦੀਆਂ ਹਨ। MCP ਨੂੰ ਇੱਕ USB-C ਪੋਰਟ ਵਾਂਗ ਸੋਚੋ - ਇਹ AI ਐਪਲੀਕੇਸ਼ਨਾਂ ਨੂੰ ਵੱਖ-ਵੱਖ ਡੇਟਾ ਸਰੋਤਾਂ ਅਤੇ ਟੂਲਾਂ ਨਾਲ ਜੋੜਨ ਦਾ ਇੱਕ ਮਿਆਰੀਕৃত ਤਰੀਕਾ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ।

## ਲਰਨਿੰਗ ਦੇ ਉਦੇਸ਼

ਇਸ ਪਾਠ ਦੇ ਅੰਤ ਤੱਕ, ਤੁਸੀਂ ਸਮਰੱਥ ਹੋਵੋਗੇ:

- C#, ਜਾਵਾ, ਪਾਇਥਨ, ਟਾਈਪਸਕ੍ਰਿਪਟ ਅਤੇ ਜਾਵਾਸਕ੍ਰਿਪਟ ਵਿੱਚ MCP ਲਈ ਵਿਕਾਸ ਵਾਤਾਵਰਣ ਸੈੱਟਅੱਪ ਕਰਨਾ
- ਮੁਖ MCP ਸਰਵਰਾਂ ਨੂੰ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ (ਸਰੋਤ, ਪ੍ਰਾਂਪਟ ਅਤੇ ਟੂਲਸ) ਨਾਲ ਬਣਾਉਣਾ ਅਤੇ ਡਿਪਲੌਇ ਕਰਨਾ
- MCP ਸਰਵਰਾਂ ਨਾਲ ਕਨੈਕਟ ਕਰਨ ਵਾਲੇ ਹੋਸਟ ਐਪਲੀਕੇਸ਼ਨਾਂ ਨੂੰ ਬਣਾਉਣਾ
- MCP ਇੰਪਲੀਮੈਂਟੇਸ਼ਨਾਂ ਦੀ ਟੈਸਟਿੰਗ ਅਤੇ ਡੀਬੱਗਿੰਗ ਕਰਨਾ
- ਆਮ ਸੈੱਟਅੱਪ ਚੁਣੌਤੀਆਂ ਅਤੇ ਉਨ੍ਹਾਂ ਦੇ ਹੱਲ ਸਮਝਣਾ
- ਆਪਣੇ MCP ਇੰਪਲੀਮੈਂਟੇਸ਼ਨਾਂ ਨੂੰ ਪ੍ਰਸਿੱਧ LLM ਸੇਵਾਵਾਂ ਨਾਲ ਜੁੜਨਾ

## ਆਪਣੇ MCP ਵਾਤਾਵਰਨ ਦੀ ਸੈਟਅੱਪਿੰਗ

MCP ਨਾਲ ਕੰਮ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ, ਇਹ ਜਰੂਰੀ ਹੈ ਕਿ ਤੁਸੀਂ ਆਪਣੇ ਵਿਕਾਸ ਵਾਤਾਵਰਣ ਨੂੰ ਤਿਆਰ ਕਰੋ ਅਤੇ ਮੂਲ ਕਾਰਜ ਪ੍ਰਵਾਹ ਨੂੰ ਸਮਝੋ। ਇਹ ਸੈਕਸ਼ਨ ਤੁਹਾਨੂੰ ਸ਼ੁਰੂਆਤੀ ਸੈਟਅੱਪ ਕਦਮਾਂ ਵਿੱਚ ਦਿਖਾਵੇਗਾ ਤਾਂ ਕਿ MCP ਨਾਲ ਕੰਮ ਸੌਖਾ ਹੋ ਜਾਵੇ।

### ਜ਼ਰੂਰੀਆਂ

MCP ਵਿਕਾਸ ਵਿੱਚ ਖੁਦ ਨੂੰ ਡੁਬੋਇਣ ਤੋਂ ਪਹਿਲਾਂ, ਇਹ ਯਕੀਨੀ ਬਣਾਓ ਕਿ ਤੁਹਾਡੇ ਕੋਲ ਹੈ:

- **ਵਿਕਾਸ ਵਾਤਾਵਰਣ**: ਤੁਹਾਡੇ ਚੁਣੇ ਭਾਸ਼ਾ ਲਈ (C#, ਜਾਵਾ, ਪਾਇਥਨ, ਟਾਈਪਸਕ੍ਰਿਪਟ, ਜਾਂ ਜਾਵਾਸਕ੍ਰਿਪਟ)
- **IDE/ਸੰਪਾਦਕ**: ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ, ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ ਕੋਡ, ਇੰਟੈਲੀਜੀ, ਇਕਲੀਪਸ, ਪਾਈਚਾਰਮ, ਜਾਂ ਕੋਈ ਵੀ ਆਧੁਨਿਕ ਕੋਡ ਸੰਪਾਦਕ
- **ਪੈਕੇਜ ਮੈਨੇਜਰ**: NuGet, Maven/Gradle, pip, ਜਾਂ npm/yarn
- **API ਕੁੰਜੀਆਂ**: ਜਿਹੜੀਆਂ ਕਿ ਤੁਸੀਂ ਆਪਣੇ ਹੋਸਟ ਐਪਲੀਕੇਸ਼ਨ ਵਿੱਚ ਵਰਤੋਂ ਕਰਨ ਵਾਲੀਆਂ AI ਸੇਵਾਵਾਂ ਲਈ ਲੱਗਦੀਆਂ ਹਨ


### ਅਧਿਕਾਰਤ SDKs

ਅਗਲੇ ਅਧਿਆਇਆਂ ਵਿੱਚ ਤੁਸੀਂ ਪਾਇਥਨ, ਟਾਈਪਸਕ੍ਰਿਪਟ, ਜਾਵਾ ਅਤੇ .NET ਵਰਗੇ ਹੱਲ ਵੇਖੋਗੇ। ਇਹ ਹਨ ਸਾਰੇ ਅਧਿਕਾਰਤ ਤੌਰ 'ਤੇ ਸਮਰਥਿਤ SDKs।

MCP ਵਰਗ-ਵੱਖ ਭਾਸ਼ਾਵਾਂ ਲਈ ਅਧਿਕਾਰਤ SDK ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ ([MCP Specification 2025-11-25](https://spec.modelcontextprotocol.io/specification/2025-11-25/) ਨਾਲ ਸੰਗਤੀ ਰੱਖਦੇ ਹੋਏ):
- [C# SDK](https://github.com/modelcontextprotocol/csharp-sdk) - ਮਾਈਕਰੋਸੌਫਟ ਦੇ ਸਹਿਯੋਗ ਨਾਲ ਸੰਭਾਲਿਆ ਜਾਂਦਾ ਹੈ
- [Java SDK](https://github.com/modelcontextprotocol/java-sdk) - Spring AI ਦੇ ਸਹਿਯੋਗ ਨਾਲ ਸੰਭਾਲਿਆ ਜਾਂਦਾ ਹੈ
- [TypeScript SDK](https://github.com/modelcontextprotocol/typescript-sdk) - ਅਧਿਕਾਰਤ ਟਾਈਪਸਕ੍ਰਿਪਟ ਨਿਰੀਆਣਾ
- [Python SDK](https://github.com/modelcontextprotocol/python-sdk) - ਅਧਿਕਾਰਤ ਪਾਇਥਨ ਨਿਰੀਆਣਾ (FastMCP)
- [Kotlin SDK](https://github.com/modelcontextprotocol/kotlin-sdk) - ਅਧਿਕਾਰਤ ਕੋਟਲਿਨ ਨਿਰੀਆਣਾ
- [Swift SDK](https://github.com/modelcontextprotocol/swift-sdk) - Loopwork AI ਦੇ ਸਹਿਯੋਗ ਨਾਲ ਸੰਭਾਲਿਆ ਜਾਂਦਾ ਹੈ
- [Rust SDK](https://github.com/modelcontextprotocol/rust-sdk) - ਅਧਿਕਾਰਤ ਰਸਟ ਨਿਰੀਆਣਾ
- [Go SDK](https://github.com/modelcontextprotocol/go-sdk) - ਅਧਿਕਾਰਤ ਗੋ ਨਿਰੀਆਣਾ

## ਮੁੱਖ ਨਤੀਜੇ

- MCP ਵਿਕਾਸ ਵਾਤਾਵਰਣ ਸੈੱਟਅੱਪ ਭਾਸ਼ਾ-ਵਿਸ਼ੇਸ਼ SDKs ਨਾਲ ਸਧਾਰਨ ਹੈ
- MCP ਸਰਵਰ ਬਣਾਉਣ ਵਿੱਚ ਸਾਫ਼ ਸਕੀਮਾਸ਼ੁਦਾ ਟੂਲ ਬਣਾਉਣਾ ਅਤੇ ਰਜਿਸਟਰ ਕਰਨਾ ਸ਼ਾਮਲ ਹੈ
- MCP ਕਲਾਇੰਟ ਸਰਵਰਾਂ ਅਤੇ ਮਾਡਲਾਂ ਨਾਲ ਜੁੜਦੇ ਹਨ ਤਾਂ ਜੋ ਵਧੇਰੇ ਸਮਰੱਥਾਵਾਂ ਲੈ ਸਕਣ
- ਟੈਸਟਿੰਗ ਅਤੇ ਡੀਬੱਗਿੰਗ ਭਰੋਸੇਯੋਗ MCP ਇੰਪਲੀਮੈਂਟੇਸ਼ਨਾਂ ਲਈ ਜਰੂਰੀ ਹਨ
- ਡਿਪਲੌਇਮੈਂਟ ਦੇ ਵਿਕਲਪ ਸਥਾਨਕ ਵਿਕਾਸ ਤੋਂ ਲੈ ਕੇ ਕਲਾਉਡ ਆਧਾਰਿਤ ਹੱਲਾਂ ਤਕ ਹਨ
