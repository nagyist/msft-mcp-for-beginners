# Käivita näidis

## Paigalda sõltuvused

```bash
npm install
```

## Ehita

```bash
npm run build
```

## Käivita server

```bash
npm start
```

see peaks serveri käivitama

### Käivita klient

Teises terminalis käivita:

```bash
node run client
```
