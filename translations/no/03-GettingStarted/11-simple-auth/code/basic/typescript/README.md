# Kjør eksempel

## Installer avhengigheter

```bash
npm install
```

## Bygg

```bash
npm run build
```

## Start server

```bash
npm start
```

dette bør starte serveren

### Start klient

I et separat terminalvindu, kjør:

```bash
node run client
```
