# Paleisti pavyzdį

## Įdiegti priklausomybes

```bash
npm install
```

## Sukurti

```bash
npm run build
```

## Paleisti serverį

```bash
npm start
```

tai turėtų paleisti serverį

### Paleisti klientą

Kitoje terminalo lange vykdykite:

```bash
node run client
```
