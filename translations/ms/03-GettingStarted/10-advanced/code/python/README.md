# Jalankan sampel

## Sediakan persekitaran maya

```sh
python -m venv venv
source ./venv/bin/activate
```

## Pasang kebergantungan

```sh
pip install "mcp[cli]"
```

## Jalankan kod

```sh
python client.py
```

Anda sepatutnya melihat teks:

```text
Available tools: ['add']

```
