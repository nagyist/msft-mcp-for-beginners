# Menjalankan sampel ini

## -1- Jalankan pelayan HTTP Streamable

Buka terminal dari root projek ini dan jalankan perintah berikut:

```bash
cd ../calculator-httpserver
cargo run
```

Biarkan terminal ini terbuka kerana ia akan menjalankan pelayan HTTP yang akan disambungkan oleh klien. Buka terminal baru untuk langkah seterusnya.

## -2- Pasang kebergantungan dan bina projek

```bash
cargo build
```

## -3- Jalankan sampel

```bash
cargo run
```
