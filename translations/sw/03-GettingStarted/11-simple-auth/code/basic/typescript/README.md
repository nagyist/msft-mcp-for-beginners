# Endesha mfano

## Sakinisha utegemezi

```bash
npm install
```

## Jenga

```bash
npm run build
```

## Anzisha seva

```bash
npm start
```

hii inapaswa kuanzisha seva

### Anzisha mteja

Katika terminal tofauti, endesha:

```bash
node run client
```
