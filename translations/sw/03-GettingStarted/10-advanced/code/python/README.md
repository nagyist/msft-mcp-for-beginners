# Endesha mfano

## Sanidi mazingira ya virusi

```sh
python -m venv venv
source ./venv/bin/activate
```

## Sakinisha mahitaji

```sh
pip install "mcp[cli]"
```

## Endesha msimbo

```sh
python client.py
```
