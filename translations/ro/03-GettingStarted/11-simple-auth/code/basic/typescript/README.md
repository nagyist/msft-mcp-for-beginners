# Rulează exemplul

## Instalează dependențele

```bash
npm install
```

## Construiește

```bash
npm run build
```

## Pornește serverul

```bash
npm start
```

acest lucru ar trebui să pornească serverul

### Pornește clientul

Într-un terminal separat, rulează:

```bash
node run client
```
