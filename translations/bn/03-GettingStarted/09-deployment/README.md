# MCP সার্ভার ডিপ্লয়মেন্ট

আপনার MCP সার্ভার ডিপ্লয় করলে অন্যরা আপনার স্থানীয় পরিবেশের বাইরেও এর সরঞ্জাম এবং সম্পদ ব্যবহার করতে পারে। আপনার স্কেলযোগ্যতা, নির্ভরযোগ্যতা, এবং ব্যবস্থাপনার সহজতার জন্য বিভিন্ন ডিপ্লয়মেন্ট কৌশল বিবেচনা করতে হয়। নিচে আপনি স্থানীয়, কন্টেইনারে এবং ক্লাউডে MCP সার্ভার ডিপ্লয় করার নির্দেশনা পাবেন।

## ওভারভিউ

এই পাঠে শেখানো হবে কিভাবে MCP সার্ভার অ্যাপ ডিপ্লয় করবেন।

## শেখার উদ্দেশ্যসমূহ

এই পাঠের শেষে, আপনি সক্ষম হবেন:

- বিভিন্ন ডিপ্লয়মেন্ট পদ্ধতি মূল্যায়ন করতে।
- আপনার অ্যাপ ডিপ্লয় করতে।

## স্থানীয় ডেভেলপমেন্ট এবং ডিপ্লয়মেন্ট

আপনার সার্ভার যদি ব্যবহারকারীর মেশিনে চালানোর জন্য হয়, তাহলে নিচের ধাপগুলি অনুসরণ করুন:

1. **সার্ভার ডাউনলোড করুন**। যদি আপনি সার্ভারটি না তৈরি করে থাকেন, তাহলে প্রথমে এটি আপনার মেশিনে ডাউনলোড করুন।
1. **সার্ভার প্রক্রিয়া শুরু করুন**: আপনার MCP সার্ভার অ্যাপ চালান।

SSE এর জন্য (stdio টাইপ সার্ভারের জন্য প্রয়োজন নেই)

1. **নেটওয়ার্কিং কনফিগার করুন**: নিশ্চিত করুন সার্ভারটি প্রত্যাশিত পোর্টে অ্যাক্সেসযোগ্য।
1. **ক্লায়েন্ট সংযোগ করুন**: `http://localhost:3000` এর মত লোকাল কানেকশন URL ব্যবহার করুন।

## ক্লাউড ডিপ্লয়মেন্ট

MCP সার্ভার বিভিন্ন ক্লাউড প্ল্যাটফর্মে ডিপ্লয় করা যেতে পারে:

- **সার্ভারলেস ফাংশন**: হালকা MCP সার্ভার সার্ভারলেস ফাংশন হিসেবে ডিপ্লয় করুন।
- **কন্টেইনার সার্ভিস**: Azure Container Apps, AWS ECS, বা Google Cloud Run এর মতো সার্ভিস ব্যবহার করুন।
- **কুবেরনেটস**: উচ্চ উপলভ্যতার জন্য কুবেরনেটস ক্লাস্টারে MCP সার্ভার ডিপ্লয় ও পরিচালনা করুন।

### উদাহরণ: Azure Container Apps

Azure Container Apps MCP সার্ভার ডিপ্লয়ের জন্য সহায়তা করে। এটি এখনো উন্নয়নের মধ্যে আছে এবং বর্তমানে SSE সার্ভার সমর্থন করে।

এটি করার উপায় হলো:

1. একটি রিপো ক্লোন করুন:

  ```sh
  git clone https://github.com/anthonychu/azure-container-apps-mcp-sample.git
  ```

1. এটি স্থানীয়ভাবে চালিয়ে পরীক্ষা করুন:

  ```sh
  uv venv
  uv sync

  # লিনাক্স/ম্যাকওএস
  export API_KEYS=<AN_API_KEY>
  # উইন্ডোজ
  set API_KEYS=<AN_API_KEY>

  uv run fastapi dev main.py
  ```

1. স্থানীয়ভাবে চেষ্টা করতে, একটি *mcp.json* ফাইল *.vscode* ডিরেক্টরিতে তৈরি করুন এবং নিম্নলিখিত বিষয়বস্তু যোগ করুন:

  ```json
  {
      "inputs": [
          {
              "type": "promptString",
              "id": "weather-api-key",
              "description": "Weather API Key",
              "password": true
          }
      ],
      "servers": {
          "weather-sse": {
              "type": "sse",
              "url": "http://localhost:8000/sse",
              "headers": {
                  "x-api-key": "${input:weather-api-key}"
              }
          }
      }
  }
  ```

  SSE সার্ভার চালু হলে, আপনি JSON ফাইলে প্লে আইকনে ক্লিক করতে পারেন, এখন সার্ভারের অস্ত্রগুলি GitHub Copilot দ্বারা শনাক্ত হওয়া উচিত, দেখতে পাবেন টুল আইকন।

1. ডিপ্লয় করতে, নিচের কমান্ডটি চালান:

  ```sh
  az containerapp up -g <RESOURCE_GROUP_NAME> -n weather-mcp --environment mcp -l westus --env-vars API_KEYS=<AN_API_KEY> --source .
  ```

এইভাবেই আপনি স্থানীয়ভাবে এবং Azure-এ এই ধাপগুলো অনুসরণ করে ডিপ্লয় করতে পারবেন।
