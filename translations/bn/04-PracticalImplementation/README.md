# বাস্তবিক প্রয়োগ

[![কিভাবে MCP অ্যাপ তৈরি, পরীক্ষা এবং বাস্তব সরঞ্জাম ও কর্মপ্রবাহ দিয়ে মোতায়েন করবেন](../../../translated_images/bn/05.64bea204e25ca891.webp)](https://youtu.be/vCN9-mKBDfQ)

_(এই পাঠের ভিডিও দেখতে উপরের ছবিতে ক্লিক করুন)_

বাস্তবিক প্রয়োগ হল যেখানে Model Context Protocol (MCP)-এর শক্তি স্পষ্ট হয়। MCP-এর তত্ত্ব এবং স্থাপত্য বোঝা গুরুত্বপূর্ণ হলেও, প্রকৃত মূল্য তখন আসে যখন আপনি এই ধারণাগুলো প্রয়োগ করে বাস্তব সমস্যা সমাধানের জন্য সমাধান তৈরি, পরীক্ষা এবং মোতায়েন করেন। এই অধ্যায়টি ধারণাগত জ্ঞান এবং হাতেকলমে উন্নয়নের মধ্যে সেতুবন্ধন সৃষ্টি করে, MCP ভিত্তিক অ্যাপ্লিকেশন জীবন্ত আকারে আনার প্রক্রিয়ায় আপনাকে পথনির্দেশনা দেয়।

আপনি যদি বুদ্ধিমান সহযোগী তৈরি করছিলেন, ব্যবসায়িক কর্মপ্রবাহে AI একত্রিত করতেন, অথবা ডেটা প্রক্রিয়াকরণের জন্য কাস্টম সরঞ্জাম নির্মাণ করতেন, MCP একটি নমনীয় ভিত্তি প্রদান করে। এর ভাষা-স্বাধীন ডিজাইন এবং জনপ্রিয় প্রোগ্রামিং ভাষার জন্য আনুষ্ঠানিক SDK গুলো এটিকে একটি বিস্তৃত ডেভেলপার দলে পৌঁছনযোগ্য করে তোলে। এই SDK গুলো ব্যবহার করে, আপনি দ্রুত প্রোটোটাইপ তৈরি, পুনরাবৃত্তি এবং বিভিন্ন প্ল্যাটফর্ম ও পরিবেশে আপনার সমাধানগুলো স্কেল করতে সক্ষম হবেন।

নিম্নলিখিত বিভাগগুলোতে আপনি বাস্তবিক উদাহরণ, নমুনা কোড, এবং মোতায়েন কৌশল পাবেন যা MCP কে C#, Java with Spring, TypeScript, JavaScript, এবং Python ভাষায় কিভাবে বাস্তবায়ন করবেন তা প্রদর্শন করে। এছাড়া MCP সার্ভার ডিবাগ এবং পরীক্ষা করা, API পরিচালনা, এবং Azure ব্যবহার করে ক্লাউডে সমাধান মোতায়েন করার পদ্ধতিও শিখবেন। এই হাতে-কলমে সম্পদগুলো আপনার শেখা দ্রুততর করবে এবং আপনাকে আত্মবিশ্বাসী হতে সাহায্য করবে শক্তিশালী, উৎপাদন-উপযোগী MCP অ্যাপ্লিকেশন তৈরি করতে।

## পর্যালোচনা

এই পাঠের ফোকাস MCP বাস্তবায়নের বাস্তবিক দিকগুলো নিয়ে বিভিন্ন প্রোগ্রামিং ভাষায়। আমরা শিখব কিভাবে MCP SDK গুলো ব্যবহার করে C#, Java with Spring, TypeScript, JavaScript, এবং Python তে শক্তিশালী অ্যাপ তৈরি করতে হয়, MCP সার্ভার ডিবাগ এবং পরীক্ষা করতে হয় এবং পুনঃব্যবহারযোগ্য রিসোর্স, প্রম্পট এবং টুলস তৈরি করতে হয়।

## শেখার লক্ষ্য

এই পাঠ শেষ পর্যন্ত আপনি সক্ষম হবেন:

- বিভিন্ন প্রোগ্রামিং ভাষায় আনুষ্ঠানিক SDK ব্যবহার করে MCP সমাধান বাস্তবায়ন করতে
- MCP সার্ভার সিস্টেম্যাটিকভাবে ডিবাগ এবং পরীক্ষা করতে
- সার্ভার ফিচার (রিসোর্স, প্রম্পট এবং টুলস) তৈরি ও ব্যবহার করতে
- জটিল কাজের জন্য কার্যকর MCP কর্মপ্রবাহ ডিজাইন করতে
- কর্মক্ষমতা এবং নির্ভরযোগ্যতার জন্য MCP বাস্তবায়নগুলো অপটিমাইজ করতে

## মূল সার্ভার ফিচার

MCP সার্ভারগুলি নিম্নলিখিত ফিচারগুলোর যেকোনো সমন্বয় প্রয়োগ করতে পারে:

### রিসোর্স

রিসোর্স ব্যবহারকারী বা AI মডেলের জন্য প্রাসঙ্গিক তথ্য এবং ডেটা প্রদান করে:

- ডকুমেন্ট রিপোজিটরি
- জ্ঞানভিত্তিক বেস
- সংগঠিত ডেটা সোর্স
- ফাইল সিস্টেম

### প্রম্পট

প্রম্পট হলো ব্যবহারকারীদের জন্য টেমপ্লেটকৃত বার্তা এবং কর্মপ্রবাহ:

- পূর্বনির্ধারিত কথোপকথন টেমপ্লেট
- গাইডড ইন্টারঅ্যাকশন প্যাটার্নস
- বিশেষায়িত ডায়ালগ কাঠামো

### টুলস

টুলস হলো AI মডেলকে কার্যকর করার জন্য ফাংশনসমূহ:

- ডেটা প্রক্রিয়াকরণ ইউটিলিটি
- বাহ্যিক API ইন্টিগ্রেশন
- গাণিতিক সক্ষমতা
- অনুসন্ধান ফাংশনালিটি

## নমুনা বাস্তবায়ন: C# বাস্তবায়ন

আনুষ্ঠানিক C# SDK রিপোজিটরিতে MCP’র বিভিন্ন দিক প্রদর্শনকারী কয়েকটি নমুনা বাস্তবায়ন রয়েছে:

- **মূল MCP ক্লায়েন্ট**: MCP ক্লায়েন্ট তৈরি এবং টুলস কল করার সহজ উদাহরণ
- **মূল MCP সার্ভার**: টুল নিবন্ধন সহ একটি মিনি মাল সার্ভার বাস্তবায়ন
- **উন্নত MCP সার্ভার**: সম্পূর্ণ ফিচারযুক্ত সার্ভার যার মধ্যে টুল নিবন্ধন, প্রমাণীকরণ এবং ত্রুটি হ্যান্ডলিং রয়েছে
- **ASP.NET ইন্টিগ্রেশন**: ASP.NET Core এর সাথে ইন্টিগ্রেশন প্রদর্শন করে এমন উদাহরণ
- **টুল বাস্তবায়ন প্যাটার্নস**: বিভিন্ন জটিলতার স্তরের টুল বাস্তবায়নের বিভিন্ন প্যাটার্ন

MCP C# SDK এখন প্রিভিউ পর্যায়ে এবং API গুলো পরিবর্তিত হতে পারে। SDK উন্নয়নের সাথে সাথে আমরা এই ব্লগ আপডেট করব।

### মূল ফিচারসমূহ

- [C# MCP Nuget ModelContextProtocol](https://www.nuget.org/packages/ModelContextProtocol)
- আপনার [প্রথম MCP সার্ভার তৈরি করা](https://devblogs.microsoft.com/dotnet/build-a-model-context-protocol-mcp-server-in-csharp/)।

সম্পূর্ণ C# বাস্তবায়ন নমুনার জন্য, আনুষ্ঠানিক [C# SDK নমুনা রিপোজিটরি](https://github.com/modelcontextprotocol/csharp-sdk) দেখুন

## API ব্যবস্থাপনা

Azure API Management একটি চমৎকার সমাধান MCP সার্ভারগুলি সুরক্ষিত করার জন্য। মূল ধারণা হল MCP সার্ভারের সামনে একটি Azure API Management ইনস্ট্যান্স স্থাপন করা এবং এটি সেইসব ফিচার সামলানোর অনুমতি দেয় যেগুলো আপনি চান, যেমন:

- রেট সীমাবদ্ধতা
- টোকেন ব্যবস্থাপনা
- মনিটরিং
- লোড ব্যালেন্সিং
- নিরাপত্তা

এখানে একটি Azure নমুনা আছে যা ঠিক এই কাজ করে, অর্থাৎ [MCP সার্ভার তৈরি করে এবং Azure API Management দিয়ে সুরক্ষিত করে](https://github.com/Azure-Samples/remote-mcp-apim-functions-python)।

- Azure API Management একটি গেটওয়ে হিসেবে কাজ করে এবং নীতিমালা ব্যবহার করে ট্রাফিক নিয়ন্ত্রণ ও পরিচালনা করে।
- Azure Monitor সমস্ত অনুরোধ লগ করে আরও বিশ্লেষণের জন্য।

## রিমোট MCP সার্ভার Azure-তে মোতায়েন করা

আমরা আগে যা উল্লেখ করেছি সেই নমুনাটি মোতায়েন করতে পারি কিনা দেখি:

1. রিপোজিটরি ক্লোন করুন

    ```bash
    git clone https://github.com/Azure-Samples/remote-mcp-apim-functions-python.git
    cd remote-mcp-apim-functions-python
    ```

1. `Microsoft.App` রিসোর্স প্রদানকারী নিবন্ধন করুন।

   - যদি আপনি Azure CLI ব্যবহার করেন, চালান `az provider register --namespace Microsoft.App --wait`।
   - যদি আপনি Azure PowerShell ব্যবহার করেন, চালান `Register-AzResourceProvider -ProviderNamespace Microsoft.App`। পরে কিছু সময় পর `(Get-AzResourceProvider -ProviderNamespace Microsoft.App).RegistrationState` চালিয়ে নিবন্ধন সম্পন্ন হয়েছে কিনা চেক করুন।

1. এই [azd](https://aka.ms/azd) কমান্ড চালান যা API Management সার্ভিস, ফাংশন অ্যাপ (কোড সহ) এবং অন্যান্য সব প্রয়োজনীয় Azure রিসোর্স provision করবে

    ```shell
    azd up
    ```

    এই কমান্ডগুলো Azure-তে সব ক্লাউড রিসোর্স মোতায়েন করবে

### MCP Inspector দিয়ে আপনার সার্ভার পরীক্ষা

1. একটি **নতুন টার্মিনাল উইন্ডোতে**, MCP Inspector ইনস্টল ও চালান

    ```shell
    npx @modelcontextprotocol/inspector
    ```

    আপনি নিম্নলিখিতর মত একটি ইন্টারফেস দেখতে পাবেন:

    ![Connect to Node inspector](../../../translated_images/bn/connect.141db0b2bd05f096.webp)

1. MCP Inspector ওয়েব অ্যাপটি লোড করার জন্য URL-এ CTRL ক্লিক করুন (যেমন [http://127.0.0.1:6274/#resources](http://127.0.0.1:6274/#resources))
1. ট্রান্সপোর্ট টাইপ `SSE` এ সেট করুন
1. `azd up` কমান্ডের পরে প্রদর্শিত আপনার চলমান API Management SSE এন্ডপয়েন্ট URL সেট করুন এবং **Connect** ক্লিক করুন:

    ```shell
    https://<apim-servicename-from-azd-output>.azure-api.net/mcp/sse
    ```

1. **List Tools**। একটি টুলে ক্লিক করে **Run Tool** করুন।

সবকিছু সঠিকভাবে হলে, আপনি এখন MCP সার্ভারের সাথে সংযুক্ত এবং একটি টুল কল করতে সক্ষম হয়েছেন।

## Azure এর জন্য MCP সার্ভার

[Remote-mcp-functions](https://github.com/Azure-Samples/remote-mcp-functions-dotnet): এই রিপোজিটরিগুলি Python, C# .NET বা Node/TypeScript ব্যবহার করে Azure Functions-এর মাধ্যমে কাস্টম রিমোট MCP (Model Context Protocol) সার্ভার তৈরি ও মোতায়েনের জন্য দ্রুত শুরু টেমপ্লেট।

নমুনাগুলো একটি পূর্ণাঙ্গ সমাধান প্রদান করে যা ডেভেলপারদের সক্ষম করে:

- স্থানীয়ভাবে তৈরি ও চালানো: একটি লোকাল মেশিনে MCP সার্ভার তৈরি ও ডিবাগ করা
- Azure এ মোতায়েন: সহজেই ক্লাউডে মোতায়েন করাঃ একটি সাধারণ azd up কমান্ডের মাধ্যমে
- ক্লায়েন্ট থেকে সংযোগ: বিভিন্ন ক্লায়েন্ট থেকে MCP সার্ভারের সাথে সংযোগ, যার মধ্যে আছে VS Code এর Copilot এজেন্ট মোড এবং MCP Inspector টুল

### মূল ফিচারসমূহ

- ডিজাইনে নিরাপত্তা: MCP সার্ভার কী এবং HTTPS ব্যবহার করে সুরক্ষিত
- প্রমাণীকরণ বিকল্প: OAuth সমর্থিত বিল্ট-ইন অথ অথবা API Management ব্যবহার করে
- নেটওয়ার্ক বিচ্ছিন্নতা: Azure Virtual Networks (VNET) ব্যবহার করে নেটওয়ার্ক বিচ্ছিন্নতার সুযোগ
- সার্ভারহীন আর্কিটেকচার: স্কেলেবল, ইভেন্ট-চালিত এক্সিকিউশনের জন্য Azure Functions ব্যবহার করে
- স্থানীয় উন্নয়ন: বিস্তৃত স্থানীয় উন্নয়ন ও ডিবাগিং সমর্থন
- সহজ মোতায়েন: Azure-এ মোতায়েন প্রক্রিয়া সরল

রিপোজিটরিতে প্রয়োজনীয় সমস্ত কনফিগারেশন ফাইল, সোর্স কোড এবং অবকাঠামো সংজ্ঞা অন্তর্ভুক্ত রয়েছে যাতে দ্রুত উৎপাদন-উপযোগী MCP সার্ভার বাস্তবায়নে শুরু করা যায়।

- [Azure Remote MCP Functions Python](https://github.com/Azure-Samples/remote-mcp-functions-python) - Python দিয়ে Azure Functions ব্যবহার করে MCP এর নমুনা বাস্তবায়ন

- [Azure Remote MCP Functions .NET](https://github.com/Azure-Samples/remote-mcp-functions-dotnet) - C# .NET দিয়ে Azure Functions ব্যবহার করে MCP এর নমুনা বাস্তবায়ন

- [Azure Remote MCP Functions Node/Typescript](https://github.com/Azure-Samples/remote-mcp-functions-typescript) - Node/TypeScript দিয়ে Azure Functions ব্যবহার করে MCP এর নমুনা বাস্তবায়ন

## প্রধান সিদ্ধান্তসমূহ

- MCP SDK গুলো নির্দিষ্ট ভাষার জন্য শক্তিশালী MCP সমাধান বাস্তবায়নের সরঞ্জাম প্রদান করে
- ডিবাগিং এবং পরীক্ষণ প্রক্রিয়া নির্ভরযোগ্য MCP অ্যাপ্লিকেশনের জন্য অত্যন্ত গুরুত্বপূর্ণ
- পুনঃব্যবহারযোগ্য প্রম্পট টেমপ্লেট AI ইনটারঅ্যাকশনগুলোর নিরবিচ্ছিন্নতা নিশ্চিত করে
- সুপরিকল্পিত কর্মপ্রবাহ জটিল কাজগুলো বিভিন্ন সরঞ্জাম ব্যবহার করে সমন্বয় করতে পারে
- MCP সমাধান বাস্তবায়নের জন্য নিরাপত্তা, কর্মক্ষমতা এবং ত্রুটি পরিচালনার বিষয় বিবেচনা করা আবশ্যক

## অনুশীলন

আপনার ক্ষেত্রের একটি বাস্তবিক সমস্যার জন্য একটি MCP কর্মপ্রবাহ ডিজাইন করুন:

1. এমন ৩-৪টি টুল চিহ্নিত করুন যা এই সমস্যা সমাধানে সহায়ক হবে
2. একটি কর্মপ্রবাহ ডায়াগ্রাম তৈরি করুন যেখানে দেখানো হবে এই টুলগুলো কিভাবে সংযুক্ত
3. আপনার পছন্দের ভাষায় একটি টুলের মৌলিক সংস্করণ বাস্তবায়ন করুন
4. একটি প্রম্পট টেমপ্লেট তৈরি করুন যা মডেলকে আপনার টুলটি কার্যকরভাবে ব্যবহার করতে সাহায্য করবে
