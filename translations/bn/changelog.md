# চেইঞ্জলগ: MCP ফর বিগিনার্স কারিকুলাম

এই ডকুমেন্টটি মডেল কনটেক্সট প্রোটোকল (MCP) ফর বিগিনার্স কারিকুলামে করা সকল গুরুত্বপূর্ণ পরিবর্তনের একটি রেকর্ড হিসেবে কাজ করে। পরিবর্তনগুলি উল্টো কালানুক্রমিক ক্রমে ডকুমেন্ট করা হয়েছে (নতুন পরিবর্তন প্রথমে)।

## জানুয়ারি ২৮, ২০২৬

### MCP স্পেসিফিকেশন ২০২৫-১১-২৫ সম্মতি পর্যালোচনা

#### কোর কনসেপ্টস উন্নয়ন (01-CoreConcepts/)
- **নতুন ক্লায়েন্ট প্রিমিটিভ - Roots**: সার্ভারগুলিকে ফাইলসিস্টেম সীমানা এবং অ্যাক্সেস অনুমতি বুঝতে সক্ষম করে এমন বিস্তৃত ডকুমেন্টেশন যোগ করা হয়েছে
- **টুল অ্যানোটেশনস**: টুল আচরণগত অ্যানোটেশন (`readOnlyHint`, `destructiveHint`) সম্পর্কিত ডকুমেন্টেশন সংযোজন, যাতে টুল কার্যকর করার সিদ্ধান্ত উন্নত হয়
- **স্যাম্পলিংয়ে টুল কলিং**: স্যাম্পলিং ডকুমেন্টেশনে `tools` এবং `toolChoice` প্যারামিটার যুক্ত করা হয়েছে, যা মডেল-চালিত টুল আহ্বান সক্ষম করে
- **URL মোড ইলিসিটেশন**: সার্ভার-উদ্ভূত বাইরের ওয়েব ইন্টারঅ্যাকশনের জন্য URL-ভিত্তিক ইলিসিটেশনের ডকুমেন্টেশন সংযোজন
- **টাস্কস (প্রায়োগিক)**: টাস্কস ফিচার সম্পর্কে নতুন অংশ সংযোজন, যা টেকসই এক্সিকিউশন র‍্যাপার এবং ফলাফল বিলম্ব retrieval ডকুমেন্ট করে
- **আইকন সমর্থন**: টুলস, রিসোর্স, রিসোর্স টেমপ্লেট, এবং প্রম্পটগুলোতে অতিরিক্ত মেটাডেটা হিসেবে আইকন অন্তর্ভুক্তির উল্লেখ

#### ডকুমেন্টেশন আপডেটস
- **README.md**: MCP স্পেসিফিকেশন 2025-11-25 সংস্করণ রেফারেন্স এবং ডেট-বেসড ভার্সনিং ব্যাখ্যা যোগ করা হয়েছে
- **study_guide.md**: কারিকুলাম মানচিত্রে টাস্কস এবং টুল অ্যানোটেশনস কোর কনসেপ্টস সেকশনে সংযোজন; ডকুমেন্ট সিগনার আপডেট করা হয়েছে

#### স্পেসিফিকেশন সম্মতি যাচাই
- **প্রোটোকল ভার্সন**: সকল ডকুমেন্টেশন MCP স্পেসিফিকেশন 2025-11-25 কে রেফারেন্স করছে নিশ্চিত করা হয়েছে
- **আর্কিটেকচার সামঞ্জস্য**: দুই-স্তরি আর্কিটেকচার (ডাটা লেয়ার + ট্রান্সপোর্ট লেয়ার) ডকুমেন্টেশনের সঠিকতা নিশ্চিত করা হয়েছে
- **প্রিমিটিভস ডকুমেন্টেশন**: সার্ভার প্রিমিটিভস (রিসোর্স, প্রম্পট, টুলস) এবং ক্লায়েন্ট প্রিমিটিভস (স্যাম্পলিং, ইলিসিটেশন, লগিং, রুটস) যাচাই করা হয়েছে
- **ট্রান্সপোর্ট মেকানিজম**: STDIO ও স্ট্রিমেবল HTTP ট্রান্সপোর্ট ডকুমেন্টেশনের সঠিকতা নিশ্চিত করা হয়েছে
- **সিকিউরিটি গাইডেন্স**: MCP সিকিউরিটি সর্বোত্তম চর্চাসমূহ ডকুমেন্টেশনের সামঞ্জস্য নিশ্চিত

#### MCP 2025-11-25 মূল ফিচারস ডকুমেন্ট করা হয়েছে
- **OpenID Connect ডিস্কভারী**: OIDC দ্বারা অথ সার্ভার ডিস্কভারী
- **OAuth ক্লায়েন্ট ID মেটাডেটা ডকুমেন্টস**: ক্লায়েন্ট রেজিস্ট্রেশন প্রক্রিয়া সুপারিশ
- **JSON স্কিমা ২০২০-১২**: MCP স্কিমা ডেফিনিশনের ডিফল্ট ডায়ালেক্ট
- **SDK টিয়ারিং সিস্টেম**: SDK ফিচার সমর্থন ও রক্ষণাবেক্ষণের নিয়মকানুন
- **গভর্নেন্স স্ট্রাকচার**: MCP গভর্নেন্সের ওয়ার্কিং গ্রুপ ও ইন্টারেস্ট গ্রুপের আনুষ্ঠানিকীকরণ

### কারিকুলাম-ওয়াইড MCP স্পেসিফিকেশন ২০২৫-১১-২৫ সামঞ্জস্য

#### মডিউল ০৩ - শুরু করা
- **SDK ডকুমেন্টেশন**: অফিসিয়াল SDK তালিকায় Go SDK যোগ; সকল SDK রেফারেন্স MCP স্পেসিফিকেশন ২০২৫-১১-২৫ অনুযায়ী আপডেট
- **ট্রান্সপোর্ট ব্যাখ্যা**: STDIO ও HTTP স্ট্রীমিং ট্রান্সপোর্ট বর্ণনায় স্পেসিফিকেশন রেফারেন্স যুক্ত

#### মডিউল ০৪ - প্র্যাকটিক্যাল ইমপ্লিমেন্টেশন
- **SDK আপডেট**: Go SDK যুক্ত; SDK তালিকায় স্পেসিফিকেশন ভার্সন রেফারেন্স সংযোজন
- **অথরাইজেশন স্পেসিফিকেশন**: MCP অথরাইজেশন স্পেসিফিকেশন লিংক নতুন ২০২৫-১১-২৫ ভার্সনে আপডেট

#### মডিউল ০৫ - অ্যাডভান্সড টপিকস
- **নতুন ফিচারস**: MCP স্পেসিফিকেশন ২০২৫-১১-২৫ এর নতুন ফিচারস (টাস্কস, টুল অ্যানোটেশনস, URL মোড ইলিসিটেশন, রুটস) সম্পর্কে নোট যোগ
- **সিকিউরিটি রিসোর্স**: OWASP MCP Top 10 এবং Sherpa ওয়ার্কশপ লিংক অতিরিক্ত রেফারেন্স হিসেবে যোগ

#### মডিউল ০৬ - কমিউনিটি অবদান
- **SDK তালিকা**: Swift ও Rust SDK যোগ; স্পেসিফিকেশন লিংক ২০২৫-১১-২৫ ভার্সন এ আপডেট
- **স্পেসিফিকেশন রেফারেন্স**: MCP স্পেসিফিকেশনের সরাসরি URL লিংক আপডেট

#### মডিউল ০৭ - প্রাথমিক গ্রহণযোগ্যতা থেকে শিক্ষণ
- **রিসোর্স আপডেট**: MCP স্পেসিফিকেশন ২০২৫-১১-২৫ লিংক এবং OWASP MCP Top 10 অতিরিক্ত রিসোর্সে যোগ

#### মডিউল ০৮ - সেরা অভ্যাস
- **স্পেসিফিকেশন ভার্সন**: MCP স্পেসিফিকেশন রেফারেন্স আপডেট ২০২৫-১১-২৫ এ
- **সিকিউরিটি রিসোর্স**: OWASP MCP Top 10 ও Sherpa ওয়ার্কশপ অতিরিক্ত রেফারেন্সে যুক্ত

#### মডিউল ১০ - AI ওয়ার্কফ্লো সহজীকরণ
- **ব্যাজ আপডেট**: MCP ভার্সন ব্যাজ SDK ভার্সন (1.9.3) থেকে স্পেসিফিকেশন ভার্সন (২০২৫-১১-২৫) এ পরিবর্তন
- **রিসোর্স লিংক**: MCP স্পেসিফিকেশন লিংক আপডেট এবং OWASP MCP Top 10 যোগ

#### মডিউল ১১ - MCP সার্ভার হ্যান্ডস-অন ল্যাবস
- **স্পেসিফিকেশন রেফারেন্স**: MCP স্পেসিফিকেশন লিংক ২০২৫-১১-২৫ ভার্সনে আপডেট
- **সিকিউরিটি রিসোর্স**: অফিসিয়াল রিসোর্সে OWASP MCP Top 10 যোগ

## ডিসেম্বর ১৮, ২০২৫

### সিকিউরিটি ডকুমেন্টেশন আপডেট - MCP স্পেসিফিকেশন ২০২৫-১১-২৫

#### MCP সিকিউরিটি সেরা চর্চাসমূহ (02-Security/mcp-best-practices.md) - স্পেসিফিকেশন ভার্সন আপডেট
- **প্রোটোকল ভার্সন আপডেট**: সর্বশেষ MCP স্পেসিফিকেশন ২০২৫-১১-২৫ (নভেম্বর ২৫, ২০২৫ রিলিজ) উল্লেখ করা হয়েছে
  - সকল স্পেসিফিকেশন ভার্সন রেফারেন্স ২০২৫-০৬-১৮ থেকে ২০২৫-১১-২৫ এ আপডেট করা হয়েছে
  - ডকুমেন্ট তারিখ রেফারেন্স আগস্ট ১৮, ২০২৫ থেকে ডিসেম্বর ১৮, ২০২৫ এ পরিবর্তিত
  - নিশ্চিত করা হয়েছে সকল স্পেসিফিকেশন URL বর্তমান ডকুমেন্টেশনে নির্দেশ করছে
- **বিষয়বস্তু যাচাই**: সর্বশেষ মান অনুযায়ী সিকিউরিটি সেরা চর্চার ব্যাপক যাচাই
  - **মাইক্রোসফট সিকিউরিটি সলিউশনস**: প্রম্পট শিল্ডস ("জেলব্রেক রিস্ক ডিটেকশন" পূর্বে), Azure Content Safety, Microsoft Entra ID, Azure Key Vault এর বর্তমান টার্মিনোলজি ও লিংক যাচাই
  - **OAuth 2.1 সিকিউরিটি**: সর্বশেষ OAuth সিকিউরিটি সেরা চর্চার সাথে সামঞ্জস্য নিশ্চিত
  - **OWASP স্ট্যান্ডার্ডস**: LLMs এর জন্য OWASP Top 10 রেফারেন্স এখনো প্রাসঙ্গিক যাচাই
  - **Azure সার্ভিসেস**: সকল মাইক্রোসফট এজুর ডকুমেন্টেশন লিংক ও সেরা চর্চার যাচাই
- **স্ট্যান্ডার্ডস সামঞ্জস্য**: সকল উল্লেখিত সিকিউরিটি স্ট্যান্ডার্ডস বর্তমান নিশ্চিত
  - NIST এআই ঝুঁকি ব্যবস্থাপনা ফ্রেমওয়ার্ক
  - ISO 27001:2022
  - OAuth 2.1 সিকিউরিটি সেরা অভ্যাস
  - Azure সিকিউরিটি ও কমপ্লায়েন্স ফ্রেমওয়ার্কস
- **ইমপ্লিমেন্টেশন রিসোর্স**: সকল ইমপ্লিমেন্টেশন গাইড লিংক ও রিসোর্স যাচাই
  - Azure API ম্যানেজমেন্ট অথেনটিকেশন প্যাটার্নস
  - Microsoft Entra ID ইন্টিগ্রেশন গাইড
  - Azure Key Vault সিক্রেট ম্যানেজমেন্ট
  - DevSecOps পাইপলাইন ও মনিটরিং সলিউশনস

### ডকুমেন্টেশন কোয়ালিটি আশ্বাস
- **স্পেসিফিকেশন সম্মতি**: সমস্ত বাধ্যতামূলক MCP সিকিউরিটি প্রয়োজনীয়তা (MUST/MUST NOT) সর্বশেষ স্পেসিফিকেশনের সাথে সামঞ্জস্য নিশ্চিত
- **রিসোর্স কারেন্সি**: মাইক্রোসফট ডকুমেন্টেশন, সিকিউরিটি স্ট্যান্ডার্ডস, ইমপ্লিমেন্টেশন গাইডের সকল বহিরাগত লিংক যাচাই
- **সেরা চর্চা কাভারেজ**: অথেনটিকেশন, অথরাইজেশন, AI-নির্দিষ্ট হুমকি, সাপ্লাই চেইন সিকিউরিটি এবং এন্টারপ্রাইজ প্যাটার্নসের ব্যাপক কভারেজ নিশ্চিত

## অক্টোবর ৬, ২০২৫

### শুরু করার অংশ সম্প্রসারণ – উন্নত সার্ভার ব্যবহার ও সহজ অথেনটিকেশন

#### উন্নত সার্ভার ব্যবহার (03-GettingStarted/10-advanced)
- **নতুন অধ্যায় যোগ**: উন্নত MCP সার্ভার ব্যবহারের ব্যাপক গাইড, নিয়মিত এবং নিম্ন-স্তরের সার্ভার আর্কিটেকচার উভয় ব্যাখ্যা সহ।
  - **নিয়মিত বনাম নিম্ন-স্তর সার্ভার**: উভয়ের বিস্তারিত তুলনা এবং পাইথন ও টাইপস্ক্রিপ্ট কোড উদাহরণ
  - **হ্যান্ডলার-ভিত্তিক ডিজাইন**: স্কেলেবল, নমনীয় সার্ভার ইমপ্লিমেন্টেশনের জন্য টুল/রিসোর্স/প্রম্পট ম্যানেজমেন্টের ব্যাখ্যা
  - **প্র্যাকটিক্যাল প্যাটার্নস**: বাস্তব বিশ্বের পরিস্থিতি যেখানে নিম্ন-স্তর সার্ভার প্যাটার্ন সুবিধাজনক উন্নত ফিচার ও আর্কিটেকচারের জন্য

#### সহজ অথেনটিকেশন (03-GettingStarted/11-simple-auth)
- **নতুন অধ্যায় যোগ**: MCP সার্ভারে সহজ অথেনটিকেশন বাস্তবায়নের স্টেপ-বাই-স্টেপ গাইড
  - **অথ কনসেপ্টস**: অথেনটিকেশন বনাম অথরাইজেশন, এবং ক্রেডেনশিয়াল হ্যান্ডলিং পরিষ্কার ব্যাখ্যা
  - **বেসিক অথ বাস্তবায়ন**: পাইথন (স্টারলেটে) ও টাইপস্ক্রিপ্ট (এক্সপ্রেস)-এ মিডলওয়্যার-ভিত্তিক অথেনটিকেশন প্যাটার্নসহ কোড স্যাম্পলস
  - **উন্নত সিকিউরিটিতে অগ্রগতি**: সহজ অথ দিয়ে শুরু করে OAuth 2.1 এবং RBAC-এ অগ্রসর হওয়ার নির্দেশনা, উন্নত সিকিউরিটি মডিউলের রেফারেন্স সহ

এই সংযোজনসমূহ প্রাকটিক্যাল, হ্যান্ডস-অন গাইডেন্স প্রদান করে যা আরও মজবুত, সুরক্ষিত, এবং নমনীয় MCP সার্ভার ইমপ্লিমেন্টেশন তৈরি করার জন্য মৌলিক ধারণা থেকে উন্নত প্রোডাকশন প্যাটার্নস এর মধ্যে সেতুবন্ধন সৃষ্টি করে।

### ডকুমেন্টেশন ব্যাপক আপডেট - MCP 2025-06-18 স্ট্যান্ডার্ডস  

#### MCP সিকিউরিটি সেরা অনুশীলন (02-Security/) - সম্পূর্ণ আধুনিকীকরণ  
- **MCP-SECURITY-BEST-PRACTICES-2025.md**: MCP Specification 2025-06-18 এর সাথে সঙ্গতিপূর্ণ সম্পূর্ণ পুনঃলিখন  
  - **আবশ্যিক চাহিদাসমূহ**: অফিসিয়াল স্পেসিফিকেশন থেকে স্পষ্ট MUST/MUST NOT রিকোয়ারমেন্টস যোগ করা হয়েছে স্পষ্ট ভিজ্যুয়াল সূচকের সঙ্গে  
  - **১২টি মূল সিকিউরিটি অনুশীলন**: ১৫ আইটেমের তালিকা থেকে ব্যাপক সিকিউরিটি ডোমেনে পুনর্গঠন  
    - টোকেন সিকিউরিটি ও অথেন্টিকেশন বহিরাগত পরিচয় প্রদানকারীর সঙ্গে ইন্টিগ্রেশন  
    - সেশন ম্যানেজমেন্ট ও ট্রান্সপোর্ট সিকিউরিটি ক্রিপ্টোগ্রাফিক চাহিদাসমূহ সহ  
    - AI-নির্দিষ্ট হুমকি সুরক্ষা Microsoft Prompt Shields ইন্টিগ্রেশনের সঙ্গে  
    - এক্সেস কন্ট্রোল ও পারমিশনস ন্যূনতম অনুমতির নীতির সঙ্গে  
    - কন্টেন্ট সেফটি ও মনিটরিং Azure Content Safety ইন্টিগ্রেশনের সঙ্গে  
    - সাপ্লাই চেইন সিকিউরিটি ব্যাপক কম্পোনেন্ট যাচাইকরণের সঙ্গে  
    - OAuth সিকিউরিটি ও Confused Deputy প্রতিরোধ PKCE বাস্তবায়নের সঙ্গে  
    - ইনসিডেন্ট রেসপন্স ও রিকভারি অটোমেটেড সক্ষমতার সঙ্গে  
    - কমপ্লায়েন্স ও গভর্নেন্স নিয়ন্ত্রণের সাথে নিবিড় তাল মিলানো  
    - উন্নত সিকিউরিটি কন্ট্রোলস শূন্য বিশ্বাস স্থাপত্য সহ  
    - Microsoft সিকিউরিটি ইকোসিস্টেম ইন্টিগ্রেশন ব্যাপক সমাধানসহ  
    - ধারাবাহিক সিকিউরিটি বিবর্তন অভিযোজনযোগ্য অনুশীলনের মাধ্যমে  
  - **Microsoft সিকিউরিটি সলিউশনস**: Prompt Shields, Azure Content Safety, Entra ID, ও GitHub Advanced Security-এর জন্য উন্নত গাইডেন্স  
  - **বাস্তবায়ন রিসোর্স**: অফিসিয়াল MCP ডকুমেন্টেশন, Microsoft সিকিউরিটি সলিউশন, সিকিউরিটি স্ট্যান্ডার্ডস ও বাস্তবায়ন গাইডস অনুসারে শ্রেণীবদ্ধ সম্পদ লিঙ্কসমূহ  

#### উন্নত সিকিউরিটি কন্ট্রোলস (02-Security/) - এন্টারপ্রাইজ বাস্তবায়ন

    - উন্নত অথেন্টিকেশন ও অথরাইজেশন Microsoft Entra ID ইন্টিগ্রেশনের সঙ্গে  
    - টোকেন সিকিউরিটি ও অ্যান্টি-পাসথ্রু কন্ট্রোল ব্যাপক যাচাইকরণসহ

    - AI-নির্দিষ্ট সিকিউরিটি কন্ট্রোলস প্রম্পট ইনজেকশন ও টুল পয়জনিং প্রতিরোধের সঙ্গে

    - মনিটরিং ও ডিটেকশন কন্ট্রোলগুলি SIEM ইন্টিগ্রেশন সহ  
    - ইনসিডেন্ট রেসপন্স ও রিকভারি অটোমেটেড সক্ষমতা সহ  
  - **বাস্তবায়ন উদাহরণ**: বিস্তারিত YAML কনফিগারেশন ব্লক এবং কোড উদাহরণ যোগ করা হয়েছে  
  - **Microsoft সলিউশন ইন্টিগ্রেশন**: Azure সিকিউরিটি সার্ভিস, GitHub Advanced Security, এন্টারপ্রাইজ আইডেন্টিটি ম্যানেজমেন্ট বিস্তৃতরূপে অন্তর্ভুক্ত

  - ক্লায়েন্টগুলি প্রোটোকল সংযোগকারী হিসেবে বর্ণিত যা এক-থেকে-এক সার্ভার সম্পর্ক বজায় রাখে
  - সার্ভারগুলো স্থানীয় বনাম দূরবর্তী মোতায়েন পরিস্থিতির সাথে উন্নত
- **প্রাথমিক পুনর্গঠন**: সার্ভার এবং ক্লায়েন্ট প্রিমিটিভগুলির সম্পূর্ণ পরিমার্জনা
  - সার্ভার প্রিমিটিভ: রিসোর্স (ডেটা সোর্স), প্রম্পট (টেমপ্লেট), টুলস (নিষ্পাদনযোগ্য ফাংশন) বিস্তারিত ব্যাখ্যা ও উদাহরণের সাথে
  - ক্লায়েন্ট প্রিমিটিভ: স্যাম্পলিং (LLM সম্পূর্ণকরণ), ইলিসিটেশন (ব্যবহারকারী ইনপুট), লগিং (ডিবাগিং/মনিটরিং)
  - বর্তমান ডিসকভারি (`*/list`), রিট্রিভাল (`*/get`), এবং এক্সিকিউশনের (`*/call`) পদ্ধতি প্যাটার্ন দিয়ে আপডেট করা হয়েছে
- **প্রোটোকল আর্কিটেকচার**: দুই-স্তরীয় আর্কিটেকচার মডেল প্রবর্তন
  - ডেটা স্তর: JSON-RPC 2.0 ভিত্তি সঙ্গে লাইফসাইকেল ম্যানেজমেন্ট ও প্রিমিটিভ
  - পরিবহন স্তর: STDIO (স্থানীয়) এবং স্ট্রিমেবল HTTP সহ SSE (দূরবর্তী) পরিবহন মেকানিজম
- **সিকিউরিটি ফ্রেমওয়ার্ক**: ব্যাপক সুরক্ষা নীতিমালা যার মধ্যে স্পষ্ট ব্যবহারকারী সম্মতি, ডেটা গোপনীয়তা সুরক্ষা, টুল এক্সিকিউশন নিরাপত্তা, এবং পরিবহন স্তর নিরাপত্তা অন্তর্ভুক্ত
- **যোগাযোগ প্যাটার্ন**: প্রোটোকল বার্তাগুলিকে ইনিশিয়ালাইজেশন, ডিসকভারি, এক্সিকিউশন, এবং বিজ্ঞপ্তি প্রবাহ দেখাতে আপডেট করা হয়েছে
- **কোড উদাহরণ**: বর্তমান MCP SDK প্যাটার্ন অনুযায়ী বহু-ভাষার উদাহরণ (.NET, Java, Python, JavaScript) রিফ্রেশ করা হয়েছে

#### সিকিউরিটি (02-Security/) - ব্যাপক সিকিউরিটি ওভারহল  
- **মানদণ্ডের সামঞ্জস্যতা**: MCP স্পেসিফিকেশন 2025-06-18 সিকিউরিটি চাহিদার পূর্ণ সামঞ্জস্যতা
- **প্রমাণীকরণের বিকাশ**: কাস্টম OAuth সার্ভার থেকে বাহ্যিক পরিচয় প্রদানকারী প্রতিনিধি (Microsoft Entra ID) এর বিবর্তন নথিভুক্ত
- **AI-विशেষ হুমকি বিশ্লেষণ**: আধুনিক AI আক্রমণ ভেক্টরগুলোর উন্নত কভারেজ
  - বিশদ প্রম্পট ইনজেকশন আক্রমণের দৃশ্যপট বাস্তব উদাহরণসহ
  - টুল পয়জনিং মেকানিজম এবং "রাগ পুল" আক্রমণ প্যাটার্ন
  - প্রসঙ্গ উইন্ডো পয়জনিং এবং মডেল বিভ্রান্তি আক্রমণ
- **Microsoft AI সিকিউরিটি সমাধান**: Microsoft সিকিউরিটি ইকোসিস্টেমের ব্যাপক কভারেজ
  - AI প্রম্পট শিল্ডস উন্নত অনুসন্ধান, স্পটলাইটিং, এবং ডিলিমিটার কৌশলসহ
  - Azure কন্টেন্ট সেফটি ইন্টিগ্রেশন প্যাটার্নস
  - GitHub এডভান্সড সিকিউরিটি সাপ্লাই চেইন সুরক্ষার জন্য
- **উন্নত হুমকি প্রশমকরণ**: বিস্তারিত সিকিউরিটি নিয়ন্ত্রণগুলি
  - MCP-নির্দিষ্ট আক্রমণ দৃশ্যপট এবং গাণিতিক সেশন আইডি প্রয়োজনীয়তার সঙ্গে সেশন হাইজ্যাকিং জন্য
  - MCP প্রক্সি পরিস্থিতিতে বিভ্রান্ত ডেপুটি সমস্যা এবং স্পষ্ট সম্মতি প্রয়োজনীয়তা
  - টোকেন পাসথ্রু দুর্বলতা এবং বাধ্যতামূলক যাচাইকরণ নিয়ন্ত্রণ
- **সাপ্লাই চেইন সিকিউরিটি**: ফাউন্ডেশন মডেল, এমবেডিংস সার্ভিস, প্রসঙ্গ প্রদানকারী এবং তৃতীয় পক্ষের API সহ বিস্তৃত AI সাপ্লাই চেইন শাখা
- **ফাউন্ডেশন সিকিউরিটি**: এন্টারপ্রাইজ সিকিউরিটি প্যাটার্নসের সঙ্গে উন্নত সংযোজন, যার মধ্যে জিরো ট্রাস্ট আর্কিটেকচার এবং Microsoft সিকিউরিটি ইকোসিস্টেম অন্তর্ভুক্ত
