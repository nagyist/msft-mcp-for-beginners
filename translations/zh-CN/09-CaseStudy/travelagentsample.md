# 案例研究：Azure AI 旅行代理 – 参考实现

## 概述

[Azure AI 旅行代理](https://github.com/Azure-Samples/azure-ai-travel-agents) 是微软开发的一个综合性参考解决方案，展示了如何使用模型上下文协议（MCP）、Azure OpenAI 和 Azure AI 搜索构建多智能体、AI 驱动的旅行规划应用。该项目展示了编排多个 AI 代理、集成企业数据以及为现实场景提供安全、可扩展平台的最佳实践。

## 主要功能
- **多智能体编排：** 利用 MCP 协调专门的代理（如航班、酒店和行程代理），协作完成复杂的旅行规划任务。
- **企业数据集成：** 连接 Azure AI 搜索和其他企业数据源，提供最新、相关的旅行推荐信息。
- **安全、可扩展架构：** 利用 Azure 服务实现身份验证、授权以及可扩展部署，遵循企业安全最佳实践。
- **可扩展工具：** 实现可复用的 MCP 工具和提示模板，支持快速适应新领域或业务需求。
- **用户体验：** 提供基于对话的界面，让用户与旅行代理交互，由 Azure OpenAI 和 MCP 提供支持。

## 架构
![Architecture](https://raw.githubusercontent.com/Azure-Samples/azure-ai-travel-agents/main/docs/ai-travel-agents-architecture-diagram.png)

### 架构图说明

Azure AI 旅行代理解决方案架构注重模块化、可扩展性以及多 AI 代理和企业数据源的安全集成。主要组件及数据流如下：

- **用户界面：** 用户通过对话式 UI（如网页聊天或 Teams 机器人）与系统交互，发送查询并接收旅行推荐。
- **MCP 服务器：** 作为中央协调器，接收用户输入，管理上下文，并通过模型上下文协议协调专门代理（如 FlightAgent、HotelAgent、ItineraryAgent）的操作。
- **AI 代理：** 每个代理负责特定领域（航班、酒店、行程），实现为 MCP 工具。代理使用提示模板和逻辑处理请求并生成响应。
- **Azure OpenAI 服务：** 提供先进的自然语言理解和生成，帮助代理理解用户意图并生成对话式响应。
- **Azure AI 搜索及企业数据：** 代理查询 Azure AI 搜索和其他企业数据源，以获取最新的航班、酒店和旅行选项信息。
- **身份验证与安全：** 集成 Microsoft Entra ID 以实现安全身份验证，并对所有资源应用最小权限访问控制。
- **部署：** 设计为部署在 Azure 容器应用，确保可扩展性、监控与运营效率。

该架构实现了多 AI 代理的无缝编排、安全集成企业数据以及构建领域特定 AI 解决方案的稳健、可扩展平台。

## 架构图分步说明
想象你正在计划一次大型旅行，有一支专业助理团队帮你处理每个细节。Azure AI 旅行代理系统的工作方式类似，每个部分（就像团队成员）都有特定任务。下面介绍各部分如何协同工作：

### 用户界面（UI）：
把它看作你的旅行代理前台。你（用户）在这里提问或提出请求，比如“帮我找一趟飞往巴黎的航班”。这可以是网站上的聊天窗口或消息应用。

### MCP 服务器（协调者）：
MCP 服务器就像前台的经理，听你提出的请求，决定由哪个专家（代理）处理。它跟踪你的对话，确保一切顺利进行。

### AI 代理（专家助理）：
每个代理都是某一领域的专家——某个专门负责航班，另一个专注酒店，还有专门安排行程。你提出旅行需求后，MCP 服务器会将请求转发给相应代理。代理利用它们的知识和工具，为你寻找最佳方案。

### Azure OpenAI 服务（语言专家）：
它就像一位语言专家，不管你怎么表达，都能准确理解你的意思。帮助代理理解请求，并用自然流畅的对话语言回应你。

### Azure AI 搜索与企业数据（信息库）：
想象一个庞大且最新的图书馆，里面有所有最新旅行信息——航班时刻、酒店空房等。代理从这里检索最准确的答案给你。

### 身份验证与安全（保安）：
就像保安检查谁能进入特定区域，这部分确保只有授权的用户和代理可以访问敏感信息，保护你的数据安全和隐私。

### 在 Azure 容器应用上的部署（大楼）：
这些助理和工具都协作在一个安全且可扩展的大楼（云端）里。这意味着系统能同时处理大量用户请求，并且随时可用。

## 系统如何协作工作：

你在前台（UI）提出问题。
经理（MCP 服务器）确定由哪个专家（代理）帮助你。
专家使用语言专家（OpenAI）理解你的请求，利用信息库（AI 搜索）寻找最佳答案。
保安（身份验证）确保整个过程安全。
这一切都在可靠且可扩展的大楼（Azure 容器应用）内进行，保证你的体验流畅且安全。
这种团队合作使系统能快速、安全地帮你规划旅行，仿佛现代办公室中一群优秀旅行代理共同协作！

## 技术实现
- **MCP 服务器：** 托管核心编排逻辑，暴露代理工具，管理多步骤旅行规划的上下文。
- **代理：** 每个代理（如 FlightAgent、HotelAgent）实现为 MCP 工具，拥有自己的提示模板和逻辑。
- **Azure 集成：** 使用 Azure OpenAI 实现自然语言理解，使用 Azure AI 搜索实现数据检索。
- **安全：** 集成 Microsoft Entra ID 进行身份验证，并为所有资源应用最小权限访问控制。
- **部署：** 支持部署到 Azure 容器应用，实现可扩展性和运营效率。
