# Beispiel ausführen

## Abhängigkeiten installieren

```bash
npm install
```

## Build erstellen

```bash
npm run build
```

## Server starten

```bash
npm start
```

Dies sollte den Server starten.

### Client starten

In einem separaten Terminal ausführen:

```bash
node run client
```
