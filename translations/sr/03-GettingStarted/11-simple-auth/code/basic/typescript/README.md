# Покрени пример

## Инсталирај зависности

```bash
npm install
```

## Изгради

```bash
npm run build
```

## Покрени сервер

```bash
npm start
```

ово би требало да покрене сервер

### Покрени клијент

У другом терминалу, покрените:

```bash
node run client
```
