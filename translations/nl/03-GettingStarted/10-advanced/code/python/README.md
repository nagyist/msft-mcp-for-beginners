# Voorbeeld uitvoeren

## Virtuele omgeving instellen

```sh
python -m venv venv
source ./venv/bin/activate
```

## Afhankelijkheden installeren

```sh
pip install "mcp[cli]"
```

## Code uitvoeren

```sh
python client.py
```
