# नमूना चलाएं

## निर्भरता स्थापित करें

```bash
npm install
```

## निर्माण करें

```bash
npm run build
```

## सर्वर शुरू करें

```bash
npm start
```

यह सर्वर शुरू कर देना चाहिए

### क्लाइंट शुरू करें

एक अलग टर्मिनल में, चलाएं:

```bash
node run client
```
