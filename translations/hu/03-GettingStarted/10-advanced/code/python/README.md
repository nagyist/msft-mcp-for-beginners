# Mintapéldák futtatása

## Virtuális környezet beállítása

```sh
python -m venv venv
source ./venv/bin/activate
```

## Függőségek telepítése

```sh
pip install "mcp[cli]"
```

## Kód futtatása

```sh
python client.py
```

A következő szöveget kell látnod:

