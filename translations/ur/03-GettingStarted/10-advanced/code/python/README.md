# نمونہ چلائیں

## ورچوئل ماحول ترتیب دیں

```sh
python -m venv venv
source ./venv/bin/activate
```

## ضروریات انسٹال کریں

```sh
pip install "mcp[cli]"
```

## کوڈ چلائیں

```sh
python client.py
```
