# Esegui esempio

## Installa dipendenze

```bash
npm install
```

## Compila

```bash
npm run build
```

## Avvia server

```bash
npm start
```

questo dovrebbe avviare il server

### Avvia client

In un terminale separato, esegui:

```bash
node run client
```
