# Αρχείο αλλαγών: Πρόγραμμα Σπουδών MCP για Αρχάριους

Αυτό το έγγραφο χρησιμεύει ως αρχείο όλων των σημαντικών αλλαγών που έγιναν στο πρόγραμμα σπουδών Model Context Protocol (MCP) για Αρχάριους. Οι αλλαγές καταγράφονται με αντίστροφη χρονολογική σειρά (οι πιο πρόσφατες αλλαγές πρώτες).

- **Προστέθηκε Νέο Κεφάλαιο**: Βήμα-βήμα οδηγός υλοποίησης απλής αυθεντικοποίησης σε MCP servers.

#### Βασικές Έννοιες (01-CoreConcepts/) - Πλήρης Εκσυγχρονισμός
- **Ενημέρωση Έκδοσης Πρωτοκόλλου**: Αναφορά στην τρέχουσα Προδιαγραφή MCP 2025-06-18 με έκδοση βασισμένη σε ημερομηνία (μορφή YYYY-MM-DD)
- **Εξευγενισμός Αρχιτεκτονικής**: Βελτιωμένες περιγραφές Hosts, Clients και Servers για να αντικατοπτρίζουν τα τρέχοντα πρότυπα αρχιτεκτονικής MCP
  - Οι Hosts ορίζονται πλέον με σαφήνεια ως εφαρμογές AI που συντονίζουν πολλαπλές συνδέσεις πελατών MCP  
  - Οι πελάτες περιγράφονται ως συνδέτες πρωτοκόλλου που διατηρούν σχέσεις server ένα προς ένα  
  - Οι servers έχουν βελτιωθεί με σενάρια τοπικής έναντι απομακρυσμένης εγκατάστασης  
- **Πρωτογενής Αναδιάρθρωση**: Ολοκληρωμένη αναθεώρηση των server και client primitives  
  - Server Primitives: Πόροι (πηγές δεδομένων), Προτροπές (πρότυπα), Εργαλεία (εκτελέσιμες λειτουργίες) με λεπτομερείς εξηγήσεις και παραδείγματα  
  - Client Primitives: Δειγματοληψία (LLM completions), Εlicitation (εισαγωγή χρήστη), Καταγραφή (debugging/monitoring)  
  - Ενημερωμένα με τρέχοντα πρότυπα μεθόδων ανακάλυψης (`*/list`), ανάκτησης (`*/get`), και εκτέλεσης (`*/call`)  
- **Αρχιτεκτονική Πρωτοκόλλου**: Εισαγωγή μοντέλου αρχιτεκτονικής δύο επιπέδων  
  - Επίπεδο Δεδομένων: Βάση JSON-RPC 2.0 με διαχείριση κύκλου ζωής και primitives  
  - Επίπεδο Μεταφοράς: STDIO (τοπικό) και Streamable HTTP με SSE (απομακρυσμένο) μηχανισμούς μεταφοράς  
- **Πλαίσιο Ασφαλείας**: Ολοκληρωμένες αρχές ασφαλείας που περιλαμβάνουν ρητή συγκατάθεση χρήστη, προστασία απορρήτου δεδομένων, ασφάλεια εκτέλεσης εργαλείων και ασφάλεια επιπέδου μεταφοράς  
- **Πρότυπα Επικοινωνίας**: Ενημερωμένα μηνύματα πρωτοκόλλου για εμφάνιση ροών αρχικοποίησης, ανακάλυψης, εκτέλεσης και ειδοποίησης  
- **Παραδείγματα Κώδικα**: Ανανεωμένα πολύγλωσσα παραδείγματα (.NET, Java, Python, JavaScript) που αντικατοπτρίζουν τα τρέχοντα πρότυπα MCP SDK
