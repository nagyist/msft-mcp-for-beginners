# नमुना चालवा

## आभासी वातावरण सेट करा

```sh
python -m venv venv
source ./venv/bin/activate
```

## अवलंबन स्थापित करा

```sh
pip install "mcp[cli]"
```

## कोड चालवा

```sh
python client.py
```
