# Spuštění ukázky

## Nastavení virtuálního prostředí

```sh
python -m venv venv
source ./venv/bin/activate
```

## Instalace závislostí

```sh
pip install "mcp[cli]"
```

## Spuštění kódu

```sh
python client.py
```

Měli byste vidět text:

