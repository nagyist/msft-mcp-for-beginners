# Spuštění ukázky

## Instalace závislostí

```bash
npm install
```

## Sestavení

```bash
npm run build
```

## Spuštění serveru

```bash
npm start
```

tím by se měl spustit server

### Spuštění klienta

V jiném terminálu spusťte:

```bash
node run client
```
