# Örnek Çalıştırma

## Bağımlılıkları Yükle

```bash
npm install
```

## Derleme

```bash
npm run build
```

## Sunucuyu Başlat

```bash
npm start
```

Bu işlem sunucuyu başlatmalıdır.

### İstemciyi Başlat

Ayrı bir terminalde şu komutu çalıştırın:

```bash
node run client
```
