# കേസ് സ്റ്റഡി: Azure AI ട്രാവൽ ഏജന്റുമാർ – റഫറൻസ് ഇംപ്ലിമെന്റേഷൻ

## അവലോകനം

[Azure AI ട്രാവൽ ഏജന്റുമാർ](https://github.com/Azure-Samples/azure-ai-travel-agents) മൈക്രോസോഫ്റ്റ് വികസിപ്പിച്ച ഒരു സമഗ്ര റഫറൻസ് സൊലൂഷനാണ്, ഇത് മോഡൽ കോൺടെക്സ്റ്റ് പ്രോട്ടോകോൾ (MCP), Azure OpenAI, Azure AI Search എന്നിവ ഉപയോഗിച്ച് മൾട്ടി-ഏജന്റ്, AI-ചാലിത ട്രാവൽ പ്ലാനിംഗ് അപ്ലിക്കേഷൻ നിർമ്മിക്കുന്നതിനെ അവതരിപ്പിക്കുന്നു. ഈ പ്രോജക്ട് ഒന്നിലധികം AI ഏജന്റുമാരെ സമന്വയിപ്പിക്കുക, എന്റർപ്രൈസ് ഡേറ്റ ചേര്ക്ക്, സുരക്ഷിതവും വിപുലീകരിക്കാവുന്നതുമായ പ്ലാറ്റ്ഫോം വിജയകരമായി എത്തിക്കുക എന്നതിന് മികച്ച രീതികൾ പ്രദർശിപ്പിക്കുന്നു.

## പ്രധാന സവിശേഷതകൾ
- **മൾട്ടി-ഏജന്റ് ഓർക്കസ്ട്രേഷൻ:** MCP ഉപയോഗിച്ച് പ്രത്യേകജ്ഞരായ ഏജന്റുമാരെ (ഉദാ. വിമാന, ഹോട്ടൽ, യാത്രാമാർഗ്ഗ ഏജന്റുമാർ) കോർഡിനേറ്റ് ചെയ്ത് ഓഫിസീൽ ട്രാവൽ പ്ലാനിംഗ് ജോലികൾ പൂർത്തിയാക്കുന്നു.
- **എന്റർപ്രൈസ് ഡേറ്റ ഇന്റഗ്രേഷൻ:** Azure AI Search ഉൾപ്പെടെ മറ്റ് എന്റർപ്രൈസ് ഡേറ്റ സ്രോതസ്സുകളുമായി ബന്ധിപ്പിച്ച് അപ്‌ടൂ-ഡേറ്റ് ഉള്ള ബന്ധപ്പെട്ട വിവരങ്ങൾ നൽകുന്നു.
- **സുരക്ഷിതവും സ്കെയിലബിളുമായ ആർക്കിടെക്ചർ:** എന്റർപ്രൈസ് സുരക്ഷാ മികച്ച രീതികൾ പാലിച്ച് Azure സർവീസുകൾ രേഖപ്പെടുത്തിയ പരിശോധന, അനുവാദം, സ്കെയിലിങ് ഉൾപ്പെടെ ഉപയോഗിക്കുന്നു.
- **വ്യവസ്ഥാപിത ടൂളിംഗ്:** പുനരുപയോഗയോഗ്യമായ MCP ടൂളുകളും പ്രോംപ്റ്റ് ടെംപ്ലേറ്റുകളും നടപ്പിലാക്കി ബിസിനസ് ആവശ്യങ്ങൾക്കോ പുതിയ ഡൊമൈനുകൾക്കോ വേഗത്തിൽ അനുസരിക്കാൻ കഴിയും.
- **ഉപയോഗം അനുഭവം:** Azure OpenAI, MCP ഉപയോഗിച്ച് എളുപ്പത്തിൽ ട്രാവൽ ഏജന്റുമാരുമായി സംവദിക്കാൻ ഒരു സംഭാഷണാത്മക ഇന്റർഫേസ് നൽകുന്നു.

## ആർക്കിടെക്ചർ
![Architecture](https://raw.githubusercontent.com/Azure-Samples/azure-ai-travel-agents/main/docs/ai-travel-agents-architecture-diagram.png)

### ആർക്കിടെക്ചർ ചിത്രത്തിന്റെ വിവരണം

Azure AI ട്രാവൽ ഏജന്റുമാർ പരിഹാരം മൾട്ടിപ്പിൾ AI ഏജന്റുമാരും എന്റർപ്രൈസ് ഡേറ്റ സ്രോതസ്സുകളും സ്‌ഥിരമായി, സുരക്ഷിതമായി സംയോജിപ്പിക്കാനും, സ്കെയിലബിൾ ആർക്കിടെക്ചർ എന്നിവയ്‌ക്കായി രൂപകല്‍പ്പന ചെയ്തതാണ്. പ്രധാന ഘടകങ്ങളും ഡേറ്റ ഫ്ലോയും താഴെപ്പറയുന്നതാണ്:

- **ഉപയോക്തൃ ഇന്റ്റർഫേസ്:** വെബ് ചാറ്റ്, Teams ബോട്ട് പോലുള്ള സംഭാഷണ UI വഴി ഉപയോക്താക്കൾ системе ആശയവിനിമയം നടത്തുന്നു, യാത്ര ശിപാർശകൾ ലഭിക്കുന്നു.
- **MCP സെർവർ:** സെൻട്രൽ ഓർക്കസ്ട്രേറ്ററായാണ്, ഉപയോക്തൃ ഇൻപുട്ട് സ്വീകരിച്ച് കോൺടെക്സ്റ്റ് കൈകാര്യം ചെയ്തു പ്രത്യേക ഏജന്റുമാരെ (FlightAgent, HotelAgent, ItineraryAgent) മോഡൽ കോൺടെക്സ്റ്റ് പ്രോട്ടോക്കോൾ വഴി നിയന്ത്രിക്കുന്നു.
- **AI ഏജന്റുമാർ:** ഓരോ ഏജന്റുമാരും (വിമാനങ്ങൾ, ഹോട്ടലുകൾ, യാത്രാമാർഗ്ഗം) പ്രത്യേക ഡൊമൈനുകളിൽ വിദഗ്ധനായ MCP ടൂൾ രൂപത്തിൽ നടപ്പിലാക്കപ്പെട്ടിരിക്കുന്നു. പ്രോംപ്റ്റ് ടെംപ്ലേറ്റുകളും ലജിക്കും ഉപയോഗിച്ച് ആവശ്യങ്ങൾ കൈകാര്യം ചെയ്യുന്നു.
- **Azure OpenAI സർവീസ്:** ഉപയോക്തൃ ഉദ്ധേശം മനസ്സിലാക്കി സംഭാഷണാത്മക മറുപടികൾ സൃഷ്ടിക്കുന്നതിന് പുരോഗമന പ്രാകൃതഭാഷാ മനസ്സിലാക്കലും സൃഷ്ടിയുമാണ് നൽകുന്നത്.
- **Azure AI Search & എന്റർപ്രൈസ് ഡേറ്റ:** എജന്റുമാർ വിമാനങ്ങൾ, ഹോട്ടലുകൾ, യാത്രാ ഓപ്ഷനുകൾ സംബന്ധിച്ച ഏറ്റവും പുതിയ വിവരങ്ങൾ തിരയാൻ Azure AI Search ഉൾപ്പെടെയുള്ള ഡേറ്റ സ്രോതസ്സുകൾ ഉപയോഗിക്കുന്നു.
- **പഠന-പരിശോധന & സുരക്ഷ:** Microsoft Entra ID ഉപയോഗിച്ചുള്ള സുരക്ഷിത അംഗീകാരം, കുറഞ്ഞ അവകാശprivilege ഉള‍ള ആക്‌സസ് നിയന്ത്രണങ്ങൾ എല്ലാതിലും നിർവഹിക്കുന്നു.
- **ഡിപ്പ്ലോയ്‌മെന്റ്:** Azure കൺറ്റെയ്нераാപ്പുകളിൽാക്കാൻ രൂപകൽപ്പന ചെയ്‌തിരിക്കുന്നത്, സ്കെയിലബിILITY, നിരീക്ഷണം, മെച്ചപ്പെട്ട പ്രവർത്തനക്ഷമത എന്നിവ ഉറപ്പാക്കുന്നു.

ഈ ആർക്കിടെക്ചർ മൾട്ടി AI ഏജന്റുമാരുടെ ബന്ധുമുപരി സുരക്ഷിതവും വിപുലമായ എന്റർപ്രൈസ് ഡാറ്റയും സംയോജിപ്പിക്കുന്നതും, ഡൊമെയ്ൻ-സവിശേഷ AI പരിഹാരങ്ങൾ നിർമ്മിക്കുന്നതിനുള്ള ഒരു ശക്തവായ പ്ളാറ്റ്ഫോം ഒരുക്കുന്നതുമാണ്.

## ആർക്കിടെക്ചർ ചിത്രത്തിന്റെ ഘട്ടം ഘട്ടം വിശദീകരണം
ഒരു വലിയ യാത്രാ പദ്ധതിയിടാൻ നിങ്ങളെ സഹായിക്കുന്ന വിദഗ്ധ സഹായി സംഘമുള്ളതായി تصورിക്കുക. Azure AI ട്രാവൽ ഏജന്റുമാർ സിസ്റ്റം ഇതുപോലെ പ്രവർത്തിക്കുന്നു, ഓരോ ഘടകവും (അല്ലെങ്കിൽ ടീം അംഗങ്ങൾ) ഒരു പ്രത്യേക ജോലി നിർവഹിക്കുന്നു. ഇതാ എങ്ങനെ എല്ലാം ഒത്തുചേർന്ന് പ്രവർത്തിക്കുന്നു:

### ഉപയോക്തൃ ഇന്റർഫേസ് (UI):
ഇത് നിങ്ങളുടെ ട്രാവൽ ഏജന്റ് മുന്നിലെ വർക്ക്സ്പേസ് പോലെയാണ്. നിങ്ങള്‍ (“ഉപയോക്താവ്”) ഇവിടെ “എനിക്ക് പാരിസിലേക്കുള്ള ഒരു ഫ്ലൈറ്റ് കണ്ടെത്തൂ” എന്ന് ചോദിക്കാം. ഇത് വെബ് ചാറ്റ് വിൻഡോ അല്ലെങ്കിൽ സന്ദേശ വ്യവസ്ഥാപനം ആപ്പ് ആയിരിക്കാം.

### MCP സെർവർ (സംഘടനാധkari):
MCP സെർവർ നിങ്ങളുടെ അഭ്യർത്ഥന കേട്ട് ഏതാണ് പ്രത്യേകജ്ഞൻ സങ്കീർണ്ണമായ ഭാഗം കൈകാര്യംിക്കേണ്ടത് എന്ന് തീരുമാനിക്കുന്നത് പോലെയാണ്. സംഭാഷണ കണക്കുകൾ സൂക്ഷിച്ച് എല്ലാം തരുൺകാലീനമായി നടക്കുന്നുമാണ് ഉറപ്പാക്കുന്നത്.

### AI ഏജന്റുമാർ (വിദഗ്ധ സഹായികൾ):
ഓരോ ഏജന്റും ഒരു പ്രത്യേക മേഖലയിലെ വിദഗ്ധൻ ആണ് – ഒരു൯ വിമാനങ്ങൾ, മറ്റൊരു൯ ഹോട്ടലുകൾ, മറ്റൊരു൯ യാത്രാമാർഗ്ഗ രൂപകല്പന. നിങ്ങൾ ഒരു യാത്ര ചോദിച്ചാൽ MCP സെർവർ ആവശ്യാനുസരണം ഏജന്റുമാരിലേക്ക് നിങ്ങളുടെ അഭ്യർത്ഥന അയക്കും. അവർ അവരുടെ അറിവും ടൂളുകളും ഉപയോഗിച്ച് മികച്ച ഓപ്ഷനുകൾ കണ്ടെത്തും.

### Azure OpenAI സർവീസ് (ഭാഷ വിദഗ്ധൻ):
നിങ്ങൾ എന്താണെന്ന് എങ്ങനെയായാലും തെറ്റ് കൂടാതെ മനസ്സിലാക്കുന്ന ഭാഷാ വിദഗ്ധനാണ് ഇത്. ഇത് ഏജന്റുമാരെ സഹായിച്ച് ഉപയോക്തൃ അഭ്യർത്ഥനകൾ വ്യാഖ്യാനിക്കുകയും സ്വാഭാവിക സംഭാഷണ ഭാഷയിൽ മറുപടി നൽകുകയും ചെയ്യുന്നു.

### Azure AI Search & എന്റർപ്രൈസ് ഡേറ്റ (വിവരശാല):
മികച്ച, അപ്‌ഡേറ്റായ ടെർക്കങ്ങളും വിമാന സമയ പട്ടികകളും ഹോട്ടൽ ലഭ്യതകളും ഉൾപ്പെടുന്ന ലോകരാഷ്ട്രത്തെ പോലെ ഒരു വലിയ ലൈബ്രറിയെ നിങ്ങൾക്ക് സങ്കല്പിക്കാം. ഏജന്റുമാർ ഈ ലൈബ്രറിയിൽ തിരയുമ്പോൾ ഏറ്റവും കൃത്യമായ മറുപടികളും ലഭിക്കുന്നു.

### പഠന-പരിശോധന & സുരക്ഷ (സുരക്ഷാ താരംഗം):
ഒരു സുരക്ഷാർക്ഷകൻ പ്രത്യേക സ്ഥലങ്ങളിൽ പ്രവേശനം പരിശോധിക്കുന്നത് പോലെയാണ് ഇത് ഉറപ്പ് വരുത്തുന്നത്. അവൻ മാത്രമേ അംഗീകൃത വ്യക്തികളും ഏജന്റുമാരും സമ്പർക്കിക്കാവൂ. ഇത് നിങ്ങളുടെ ഡാറ്റ സുരക്ഷിതവും സ്വകാര്യവുമാക്കുന്നു.

### Azure കൺറ്റെയ്‌നർ ആപ്പുകളിൽ ഡിപ്പ്ലോയ്മെന്റ് (ഭവനം):
ഈ സഹായികളും ടൂളുകളും ഒരുമിച്ചു വെയ്ക്കപ്പെടുന്ന സുരക്ഷിതവും സ്കെയിലബിളുമായ കെട്ടിടം (ക്ലൗഡ്) ആണിത്. ഇതിലൂടെ സിസ്റ്റം ഒരുങ്ങിയത് ഒരേ സമയം നിരവധി ഉപയോക്താക്കളെ കൈകാര്യം ചെയ്യാനും നിങ്ങളുടെ എന്നു വേണമെന്നും ലഭ്യമാകാനും കഴിയും.

## എല്ലാം ഒരുമിച്ച് എങ്ങനെ പ്രവർത്തിക്കുന്നു:

നിങ്ങൾ മുന്നിലെ ഡെസ്കിൽ ചോദ്യം ചൊല്ലുകയാണ് (UI).
മാനേജർ (MCP സെർവർ) സഹായിക്കുന്ന ഏജന്റ്മാരെ തിരഞ്ഞെടുക്കുന്നു.
വിദഗ്ധൻ ഭാഷാ വിദഗ്ധൻ (OpenAI) ഉപയോഗിച്ച് നിങ്ങളുടെ അഭ്യർത്ഥന മനസ്സിലാക്കുകയും ലൈബ്രറി (AI Search) ഉപയോഗിച്ച് ഏറ്റവും ഉചിതമായ ഉത്തരം കണ്ടെത്തുകയും ചെയ്യുന്നു.
സുരക്ഷാ താരംഗം (Authentication) എല്ലാം സുരക്ഷിതമാണെന്ന് ഉറപ്പ് വരുത്തുന്നു.
ഇവ ശരിയായ, സ്കെയിലബിള്‍ കെട്ടിടം (Azure Container Apps) ഉള്ളിൽ നടക്കുന്നു, അതുകൊണ്ട് നിങ്ങളുടെ അനുഭവം നിഴലും സുരക്ഷിതവുമാണ്.
ഈ സംഘംപ്രവർത്തനം ദുരിതമില്ലാതെ നിവൃത്തിയോടെ നിങ്ങളുടെ യാത്രാ പദ്ധതി ചെയ്യാൻ സഹായിക്കുന്നു, ഇവിടെയുള്ള വിദഗ്ധ ട്രാവൽ ഏജന്റുമാരുടെ സംഘം ഒത്തു ചേർന്ന് പ്രവർത്തിക്കുന്ന ഒരു ആധുനിക ഓഫിസിന് സമാനമായി!

## സാങ്കേതിക നടപ്പാക്കൽ
- **MCP സെർവർ:** കോർ ഓർക്കസ്ട്രേഷൻ ലജിക്ക് ഹോസ്റ്റ് ചെയ്യുന്നു, ഏജന്റ്മാരുടെ ടൂളുകൾ പ്രദർശിപ്പിക്കുന്നു, മൾട്ടി-സ്റ്റെപ്പ് ട്രാവൽ പ്ലാനിംഗ് വർക്ക്‌ഫ്ലോകൾക്കായി കോൺടെക്സ്റ്റ് കൈകാര്യം ചെയ്യുന്നു.
- **ഏജന്റ്മാർ:** ഓരോ ഏജന്റും (ഉദാ. FlightAgent, HotelAgent) സ്വന്തം പ്രോംപ്റ്റ് ടെംപ്ലേറ്റുകളും ലജിക്കും ഉള്ള MCP ടൂൾ ആയി നടപ്പിലാക്കുന്നു.
- **Azure ഇന്റഗ്രേഷൻ:** പ്രാകൃത ഭാഷ മനസ്സിലാക്കലിനും ഡേറ്റ ആക്‌സസിനും Azure OpenAI, Azure AI Search ഉപയോഗിക്കുന്നു.
- **സുരക്ഷ:** Microsoft Entra ID ഉപയോഗിച്ച് അംഗീകാരം കൈകാര്യം ചെയ്ത് എല്ലാ വിഭവങ്ങൾക്കും കുറഞ്ഞ അവകാശ അക്‌സസ് നിയന്ത്രണങ്ങൾ പ്രയോഗിക്കുന്നു.
- **ഡിപ്പ്ലോയ്മെന്റ്:** സ്കെയിലിംഗ് മെച്ചപ്പെടുത്തുന്നതിനായി Azure Container Appsിൽ ഡിപ്പ്ലോയ്മെന്റ് പിന്തുണയ്ക്കുന്നു.

## ഫലങ്ങളുംപ്രഭാവവും
- MCP ഉപയോഗിച്ച് യാഥാർത്ഥ്യ-ഉത്പാദന നിലപാടിലുള്ള നിരവധി AI ഏജന്റ്മാരെ എങ്ങനെ കോർഡിനേറ്റ് ചെയ്യാമെന്നു കാണിക്കുന്നു.
- ഏജന്റ് കോർഡിനേഷൻ, ഡാറ്റ ഇന്റഗ്രേഷൻ, സുരക്ഷിത ഡിപ്പ്ലോയ്മെന്റ് എന്നിവയ്ക്കായി പുനരുപയോഗയോഗ്യമായ പാറ്റേണുകൾ നൽകി സൊലൂഷൻ വികസനം ലഘൂകരിക്കുന്നു.
- MCP, Azure സർവീസുകൾ ഉപയോഗിച്ച് ഡൊമെയ്ൻ-പ്രత్యേക, AI-സമ്പന്ന ആപ്ലിക്കേഷനുകൾ നിർമ്മിക്കാനുള്ള ബ്ലൂപ്രിന്റ് ആയി സേവനം നൽകുന്നു.
