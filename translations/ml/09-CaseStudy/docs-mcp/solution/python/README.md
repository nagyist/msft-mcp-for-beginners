# Chainlit & Microsoft Learn Docs MCP ഉപയോഗിച്ച് സ്റ്റഡി പ്ലാൻ ജനറേറ്റർ

## മുൻകൂട്ടി ആവശ്യമായവ

- Python 3.8 അല്ലെങ്കിൽ അതിനുമുകളിൽ
- pip (Python പാക്കേജ് മാനേജർ)
- Microsoft Learn Docs MCP സെർവറുമായി ബന്ധപ്പെടാൻ ഇന്റർനെറ്റ് ആക്‌സസ്

## ഇൻസ്റ്റലേഷൻ

1. ഈ റിപോസിറ്ററി ക്ലോൺ ചെയ്യുക അല്ലെങ്കിൽ പ്രോജക്ട് ഫയലുകൾ ഡൗൺലോഡ് ചെയ്യുക.
2. ആവശ്യമായ ഡിപ്പെൻഡൻസികൾ ഇൻസ്റ്റാൾ ചെയ്യുക:

   ```bash
   pip install -r requirements.txt
   ```

## ഉപയോഗം

### സീനാരിയോ 1: Docs MCP-യിലേക്ക് ലളിതമായ ക്വറി
Docs MCP സെർവറുമായി കണക്ട് ചെയ്ത് കമാൻഡ്-ലൈൻ ക്ലയന്റ് ഒരു ക്വറി അയച്ച് ഫലം പ്രിന്റ് ചെയ്യുന്നു.

1. സ്ക്രിപ്റ്റ് റൺ ചെയ്യുക:
   ```bash
   python scenario1.py
   ```
2. പ്രോംപ്റ്റിൽ നിങ്ങളുടെ ഡോക്യുമെന്റേഷൻ ചോദ്യം നൽകുക.

### സീനാരിയോ 2: സ്റ്റഡി പ്ലാൻ ജനറേറ്റർ (Chainlit വെബ് ആപ്പ്)
ഉപയോക്താക്കൾക്ക് ഏതെങ്കിലും സാങ്കേതിക വിഷയത്തിന് വ്യക്തിഗതമായി ആഴ്ചവാരമായി സ്റ്റഡി പ്ലാൻ സൃഷ്ടിക്കാൻ Chainlit ഉപയോഗിച്ച് വെബ് അടിസ്ഥാനത്തിലുള്ള ഇന്റർഫേസ്.

1. Chainlit ആപ്പ് ആരംഭിക്കുക:
   ```bash
   chainlit run scenario2.py
   ```
2. നിങ്ങളുടെ ടെർമിനലിൽ നൽകിയ ലോക്കൽ URL (ഉദാ: http://localhost:8000) ബ്രൗസറിൽ തുറക്കുക.
3. ചാറ്റ് വിൻഡോയിൽ നിങ്ങളുടെ പഠന വിഷയം, പഠിക്കാൻ ആഗ്രഹിക്കുന്ന ആഴ്ചകളുടെ എണ്ണം നൽകുക (ഉദാ: "AI-900 സർട്ടിഫിക്കേഷൻ, 8 ആഴ്ച").
4. ആപ്പ് ആഴ്ചവാരമായി സ്റ്റഡി പ്ലാൻ നൽകും, അതിൽ ബന്ധപ്പെട്ട Microsoft Learn ഡോക്യുമെന്റേഷൻ ലിങ്കുകളും ഉൾപ്പെടും.

**ആവശ്യമായ എൻവയോൺമെന്റ് വേരിയബിളുകൾ:**

സീനാരിയോ 2 (Azure OpenAI ഉപയോഗിച്ച് Chainlit വെബ് ആപ്പ്) ഉപയോഗിക്കാൻ, `python` ഡയറക്ടറിയിലുള്ള `.env` ഫയലിൽ താഴെ പറയുന്ന എൻവയോൺമെന്റ് വേരിയബിളുകൾ സജ്ജമാക്കണം:

```
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=
AZURE_OPENAI_API_KEY=
AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_API_VERSION=
```

ആപ്പ് റൺ ചെയ്യുന്നതിന് മുമ്പ് നിങ്ങളുടെ Azure OpenAI റിസോഴ്‌സ് വിശദാംശങ്ങൾ ഈ മൂല്യങ്ങളിൽ പൂരിപ്പിക്കുക.

> [!TIP]
> [Azure AI Foundry](https://ai.azure.com/) ഉപയോഗിച്ച് നിങ്ങളുടെ സ്വന്തം മോഡലുകൾ എളുപ്പത്തിൽ ഡിപ്ലോയ് ചെയ്യാം.

### സീനാരിയോ 3: VS Code-ൽ MCP സെർവറുമായി ഇൻ-എഡിറ്റർ ഡോക്സ്

ബ്രൗസർ ടാബുകൾ മാറി ഡോക്യുമെന്റേഷൻ തിരയുന്നതിനുപകരം, Microsoft Learn Docs നേരിട്ട് VS Code-ലേക്ക് MCP സെർവറിലൂടെ കൊണ്ടുവരാം. ഇതിലൂടെ നിങ്ങൾക്ക്:
- കോഡിംഗ് പരിസരത്ത് നിന്ന് പുറത്തു പോകരുതാതെ VS Code-ൽ തന്നെ ഡോക്സ് തിരയാനും വായിക്കാനും കഴിയും.
- README അല്ലെങ്കിൽ കോഴ്‌സ് ഫയലുകളിൽ നേരിട്ട് റഫറൻസ് ഡോക്യുമെന്റേഷൻ ലിങ്കുകൾ ചേർക്കാം.
- GitHub Copilot-ഉം MCP-ഉം ചേർന്ന് എഐ-പവർഡ് ഡോക്യുമെന്റേഷൻ പ്രവാഹം സൃഷ്ടിക്കാം.

**ഉദാഹരണ ഉപയോഗങ്ങൾ:**
- കോഴ്‌സ് അല്ലെങ്കിൽ പ്രോജക്ട് ഡോക്യുമെന്റേഷൻ എഴുതുമ്പോൾ README-യിൽ റഫറൻസ് ലിങ്കുകൾ വേഗത്തിൽ ചേർക്കുക.
- കോഡ് ജനറേറ്റ് ചെയ്യാൻ Copilot ഉപയോഗിച്ച് MCP ഉപയോഗിച്ച് ബന്ധപ്പെട്ട ഡോക്സ് ഉടൻ കണ്ടെത്തുക.
- എഡിറ്ററിൽ ശ്രദ്ധ കേന്ദ്രീകരിച്ച് ഉൽപാദകത്വം വർദ്ധിപ്പിക്കുക.

> [!IMPORTANT]
> നിങ്ങളുടെ വർക്ക്‌സ്പേസിൽ സാധുവായ [`mcp.json`](../../../../../../09-CaseStudy/docs-mcp/solution/scenario3/mcp.json) കോൺഫിഗറേഷൻ ഉണ്ടെന്ന് ഉറപ്പാക്കുക (സ്ഥാനം `.vscode/mcp.json`).

## സീനാരിയോ 2-നായി Chainlit തിരഞ്ഞെടുക്കേണ്ടത് എന്തുകൊണ്ട്?

Chainlit ഒരു ആധുനിക ഓപ്പൺ-സോഴ്‌സ് ഫ്രെയിംവർക്ക് ആണ്, സംഭാഷണ വെബ് ആപ്ലിക്കേഷനുകൾ നിർമ്മിക്കാൻ. ഇത് Microsoft Learn Docs MCP സെർവറുപോലുള്ള ബാക്ക്‌എൻഡ് സർവീസുകളുമായി കണക്ട് ചെയ്യുന്ന ചാറ്റ് അടിസ്ഥാനത്തിലുള്ള യൂസർ ഇന്റർഫേസുകൾ എളുപ്പത്തിൽ സൃഷ്ടിക്കാൻ സഹായിക്കുന്നു. ഈ പ്രോജക്ട് വ്യക്തിഗത സ്റ്റഡി പ്ലാനുകൾ റിയൽ ടൈമിൽ സൃഷ്ടിക്കാൻ Chainlit ഉപയോഗിക്കുന്നു. Chainlit ഉപയോഗിച്ച്, ഉൽപാദകത്വവും പഠനവും മെച്ചപ്പെടുത്തുന്ന ചാറ്റ് അടിസ്ഥാനത്തിലുള്ള ടൂളുകൾ വേഗത്തിൽ നിർമ്മിച്ച് ഡിപ്ലോയ് ചെയ്യാം.

## ഇത് ചെയ്യുന്നത് എന്ത്?

ഈ ആപ്പ് ഉപയോക്താക്കൾക്ക് ഒരു വിഷയം, ദൈർഘ്യം നൽകുന്നതിലൂടെ വ്യക്തിഗത സ്റ്റഡി പ്ലാൻ സൃഷ്ടിക്കാൻ അനുവദിക്കുന്നു. ആപ്പ് നിങ്ങളുടെ ഇൻപുട്ട് പാഴ്‌സ് ചെയ്ത്, Microsoft Learn Docs MCP സെർവറിൽ ബന്ധപ്പെട്ട ഉള്ളടക്കം ക്വറി ചെയ്ത്, ഫലങ്ങൾ ആഴ്ചവാരമായി ക്രമീകരിച്ച പ്ലാനായി ഒരുക്കുന്നു. ഓരോ ആഴ്ചയുടെയും ശുപാർശകൾ ചാറ്റിൽ പ്രദർശിപ്പിക്കപ്പെടുന്നു, ഇത് പിന്തുടരാനും പുരോഗതി ട്രാക്ക് ചെയ്യാനും എളുപ്പമാക്കുന്നു. ഇന്റഗ്രേഷൻ നിങ്ങൾക്ക് എപ്പോഴും ഏറ്റവും പുതിയ, ഏറ്റവും പ്രസക്തമായ പഠന വിഭവങ്ങൾ ലഭ്യമാക്കുന്നു.
