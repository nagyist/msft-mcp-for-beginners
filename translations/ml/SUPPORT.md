# പിന്തുണ

## പ്രശ്നങ്ങൾ ഫയൽ ചെയ്യാനും സഹായം ലഭിക്കാനും  

ഈ പ്രോജക്ട് ബഗുകളും ഫീച്ചർ അഭ്യർത്ഥനകളും ട്രാക്ക് ചെയ്യാൻ GitHub Issues ഉപയോഗിക്കുന്നു. പുനരാവൃതികൾ ഒഴിവാക്കാൻ പുതിയ പ്രശ്നങ്ങൾ ഫയൽ ചെയ്യുന്നതിന് മുമ്പ് നിലവിലുള്ള പ്രശ്നങ്ങൾ തിരയുക. പുതിയ പ്രശ്നങ്ങൾക്കായി, നിങ്ങളുടെ ബഗ് അല്ലെങ്കിൽ ഫീച്ചർ അഭ്യർത്ഥന പുതിയ ഒരു Issue ആയി ഫയൽ ചെയ്യുക.

ഈ പ്രോജക്ട് ഉപയോഗിക്കുന്നതിനെക്കുറിച്ചുള്ള സഹായത്തിനും ചോദ്യങ്ങൾക്കുമായി, ദയവായി:
- വിദഗ്ധരും സഹപ്രവർത്തകരും കൂടിയുള്ള [Azure AI Foundry Discord](https://discord.com/invite/ByRwuEEgH4) പരിശോധിക്കുക
- പ്രോട്ടോക്കോൾ-ബന്ധപ്പെട്ട ചോദ്യങ്ങൾക്ക് [Model Context Protocol Community](https://modelcontextprotocol.io/community/) വിഭവങ്ങൾ ഉപയോഗിക്കുക

## Microsoft പിന്തുണ നയം  

Model Context Protocol (MCP) for Beginners പ്രോജക്ടിനുള്ള പിന്തുണ മുകളിൽ പട്ടികപ്പെടുത്തിയ വിഭവങ്ങളിലേക്കാണ് പരിമിതമായിരിക്കുന്നത്.
