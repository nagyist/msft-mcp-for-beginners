# ഈ സാമ്പിൾ പ്രവർത്തിപ്പിക്കൽ

## -1- Streamable HTTP സെർവർ പ്രവർത്തിപ്പിക്കുക

ഈ പ്രോജക്റ്റിന്റെ റൂട്ടിൽ നിന്ന് ഒരു ടെർമിനൽ തുറന്ന് താഴെ കൊടുത്തിരിക്കുന്ന കമാൻഡുകൾ പ്രവർത്തിപ്പിക്കുക:

```bash
cd ../calculator-httpserver
cargo run
```

ക്ലയന്റ് കണക്ട് ചെയ്യാൻ ഉപയോഗിക്കുന്ന HTTP സെർവർ പ്രവർത്തിപ്പിക്കുന്നതിനാൽ ഈ ടെർമിനൽ തുറന്നിരിക്കണം. അടുത്ത ഘട്ടങ്ങൾക്ക് പുതിയ ഒരു ടെർമിനൽ തുറക്കുക.

## -2- ആശ്രിതങ്ങൾ ഇൻസ്റ്റാൾ ചെയ്ത് പ്രോജക്റ്റ് ബിൽഡ് ചെയ്യുക

```bash
cargo build
```

## -3- സാമ്പിൾ പ്രവർത്തിപ്പിക്കുക

```bash
cargo run
```
