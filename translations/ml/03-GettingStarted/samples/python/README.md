# MCP കാൽക്കുലേറ്റർ സർവർ (Python)



അടിസ്ഥാന കാൽക്കുലേറ്റർ പ്രവർത്തനം നൽകുന്ന Python-ൽ ലളിതമായ ഒരു മോഡൽ കോൺടെക്സ്റ്റ് പ്രോട്ടോക്കോൾ (MCP) സർവർ നടപ്പാക്കൽ.


## ഇൻസ്റ്റലേഷൻ

ആവശ്യമായ ഡിപ്പൻഡൻസികൾ ഇൻസ്റ്റാൾ ചെയ്യുക:

```bash
pip install -r requirements.txt
```

അല്ലെങ്കിൽ MCP Python SDK നേരിട്ട് ഇൻസ്റ്റാൾ ചെയ്യുക:

```bash
pip install mcp>=1.18.0
```

## ഉപയോഗം

### സർവർ പ്രവർത്തിപ്പിക്കൽ

സർവർ MCP ക്ലയന്റുകൾ (ഉദാ: Claude Desktop) ഉപയോഗിക്കാൻ രൂപകൽപ്പന ചെയ്തതാണ്. സർവർ ആരംഭിക്കാൻ:

```bash
python mcp_calculator_server.py
```

**കുറിപ്പ്**: ടർമിനലിൽ നേരിട്ട് പ്രവർത്തിപ്പിക്കുമ്പോൾ JSON-RPC സാധുതാ പിശകുകൾ കാണാം. ഇത് സാധാരണ പെരുമാറ്റമാണ് - സർവർ ശരിയായ ഫോർമാറ്റിലുള്ള MCP ക്ലയന്റ് സന്ദേശങ്ങൾക്കായി കാത്തിരിക്കുന്നു.

### ഫംഗ്ഷനുകൾ പരിശോധന

കാൽക്കുലേറ്റർ ഫംഗ്ഷനുകൾ ശരിയായി പ്രവർത്തിക്കുന്നുണ്ടോ എന്ന് പരിശോധിക്കാൻ:

```bash
python test_calculator.py
```

## പ്രശ്നപരിഹാരം

### ഇറക്കുമതി പിശകുകൾ

`ModuleNotFoundError: No module named 'mcp'` എന്ന പിശക് കാണുമ്പോൾ MCP Python SDK ഇൻസ്റ്റാൾ ചെയ്യുക:

```bash
pip install mcp>=1.18.0
```

### നേരിട്ട് പ്രവർത്തിപ്പിക്കുമ്പോൾ JSON-RPC പിശകുകൾ

സർവർ നേരിട്ട് പ്രവർത്തിപ്പിക്കുമ്പോൾ "Invalid JSON: EOF while parsing a value" പോലുള്ള പിശകുകൾ പ്രതീക്ഷിക്കാവുന്നതാണ്. സർവർക്ക് MCP ക്ലയന്റ് സന്ദേശങ്ങൾ വേണം, നേരിട്ട് ടർമിനൽ ഇൻപുട്ട് അല്ല.
