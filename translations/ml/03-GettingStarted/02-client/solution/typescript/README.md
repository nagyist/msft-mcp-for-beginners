# ഈ സാമ്പിൾ പ്രവർത്തിപ്പിക്കൽ

നിങ്ങൾക്ക് `uv` ഇൻസ്റ്റാൾ ചെയ്യാൻ ശുപാർശ ചെയ്യുന്നു, പക്ഷേ അത് നിർബന്ധമല്ല, [നിർദ്ദേശങ്ങൾ](https://docs.astral.sh/uv/#highlights) കാണുക

## -1- ആശ്രിതങ്ങൾ ഇൻസ്റ്റാൾ ചെയ്യുക

```bash
npm install
```

## -3- സെർവർ പ്രവർത്തിപ്പിക്കുക


```bash
npm run build
```

## -4- ക്ലയന്റ് പ്രവർത്തിപ്പിക്കുക

```sh
npm run client
```

നിങ്ങൾക്ക് താഴെപറയുന്നതുപോലുള്ള ഫലം കാണാം:

```text
Prompt:  {
  type: 'text',
  text: 'Please review this code:\n\nconsole.log("hello");'
}
Resource template:  file
Tool result:  { content: [ { type: 'text', text: '9' } ] }
```
