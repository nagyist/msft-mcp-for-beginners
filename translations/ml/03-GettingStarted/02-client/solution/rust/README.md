# ഈ സാമ്പിൾ പ്രവർത്തിപ്പിക്കൽ

> [!NOTE]
> ഈ റസ്റ്റ് ക്ലയന്റ് ഈ പ്രോജക്റ്റിന്റെ സഹോദര ഡയറക്ടറിയായ [server](../../../../../../03-GettingStarted/02-client/solution/server) ഡയറക്ടറിയിൽ സാമ്പിൾ MCP സെർവർ പ്രവർത്തിപ്പിക്കാൻ dotnet 8.0 SDK ഇൻസ്റ്റാൾ ചെയ്തിട്ടുണ്ടെന്ന് കരുതുന്നു.

## -1- ആശ്രിതങ്ങൾ ഇൻസ്റ്റാൾ ചെയ്ത് പ്രോജക്റ്റ് നിർമ്മിക്കുക

```bash
cargo build
```

## -2- സാമ്പിൾ പ്രവർത്തിപ്പിക്കുക

```bash
cargo run
```
