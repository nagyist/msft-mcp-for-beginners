# സാമ്പിൾ പ്രവർത്തിപ്പിക്കൽ

ഇവിടെ നിങ്ങൾക്ക് ഇതിനകം പ്രവർത്തനക്ഷമമായ ഒരു സെർവർ കോഡ് ഉണ്ടെന്ന് فرضിക്കുന്നു. ദയവായി മുൻ അധ്യായങ്ങളിൽ നിന്നുള്ള ഒരു സെർവർ കണ്ടെത്തുക.

## mcp.json സജ്ജമാക്കുക

റഫറൻസിനായി നിങ്ങൾ ഉപയോഗിക്കുന്ന ഒരു ഫയൽ ഇതാണ്, [mcp.json](../../../../../03-GettingStarted/04-vscode/solution/mcp.json).

നിങ്ങളുടെ സെർവറിന്റെ പൂർണ്ണ പാതയും പ്രവർത്തിപ്പിക്കാൻ ആവശ്യമായ പൂർണ്ണ കമാൻഡും ഉൾപ്പെടെ സെർവർ എൻട്രി ആവശ്യാനുസരണം മാറ്റുക.

മുകളിൽ പരാമർശിച്ച ഉദാഹരണ ഫയലിൽ സെർവർ എൻട്രി ഇങ്ങനെ കാണപ്പെടുന്നു:

<details>
<summary>node.js</summary>
```json
"hello-mcp": {
    "command": "node",
    "args": [
        "build/index.js"
    ]
}
```
</details>

<details>
<summary>.NET</summary>

GitHub റിപോസിറ്ററി റൂട്ട് നൽകേണ്ടി വരാം, അത് `git rev-parse --show-toplevel` കമാൻഡ് ഉപയോഗിച്ച് കണ്ടെത്താം.

```jsonc
{
  "inputs": [
    {
      "type": "promptString",
      "id": "repository-root",
      "description": "The absolute path to the repository root"
    }
  ],
  "servers": {
    "calculator-mcp-dotnet": {
      "type": "stdio",
      "command": "dotnet",
      "args": [
        "run",
        "--project",
        "${input:repository-root}/03-GettingStarted/02-client/solution/server/server.csproj"
      ]
    }
  }
}
```

</details>

ഇത് `node build/index.js` എന്ന കമാൻഡ് പ്രവർത്തിപ്പിക്കുന്നതിനെ സമാനമാണ്.

- നിങ്ങളുടെ സെർവർ ഫയൽ എവിടെയാണ് എന്നതനുസരിച്ച് അല്ലെങ്കിൽ തിരഞ്ഞെടുക്കപ്പെട്ട റൺടൈം, സെർവർ സ്ഥലം അനുസരിച്ച് സെർവർ എൻട്രി മാറ്റുക.

## സെർവറിൽ ഫീച്ചറുകൾ ഉപയോഗിക്കുക

- *mcp.json* ഫയൽ *./vscode* ഫോൾഡറിൽ ചേർത്ത ശേഷം `play` ഐക്കൺ ക്ലിക്ക് ചെയ്യുക,

    GitHub Copilot-ൽ ചാറ്റ് ഫീൽഡിന്റെ മുകളിൽ സ്ഥിതിചെയ്യുന്ന ടൂളിംഗ് ഐക്കൺ ലഭ്യമായ ടൂളുകളുടെ എണ്ണം വർദ്ധിപ്പിക്കുന്നതായി മാറുന്നത് ശ്രദ്ധിക്കുക.

## ഒരു ടൂൾ പ്രവർത്തിപ്പിക്കുക

- നിങ്ങളുടെ ചാറ്റ് വിൻഡോയിൽ നിങ്ങളുടെ ടൂളിന്റെ വിവരണത്തിന് അനുയോജ്യമായ ഒരു പ്രോംപ്റ്റ് ടൈപ്പ് ചെയ്യുക. ഉദാഹരണത്തിന് `add` ടൂൾ പ്രവർത്തിപ്പിക്കാൻ "add 3 to 20" പോലൊരു വാചകം ടൈപ്പ് ചെയ്യുക.

    ചാറ്റ് ടെക്സ്റ്റ് ബോക്സിന്റെ മുകളിൽ ടൂൾ തിരഞ്ഞെടുക്കാൻ നിങ്ങൾക്ക് കാണിക്കുന്ന ഒരു ടൂൾ പ്രദർശിപ്പിക്കുന്നതായി കാണണം, താഴെ കാണുന്ന ചിത്രം പോലെ:

    ![VS Code indicating it wanting to run a tool](../../../../../translated_images/ml/vscode-agent.d5a0e0b897331060.webp)

    ടൂൾ തിരഞ്ഞെടുക്കുമ്പോൾ, മുൻപ് പറഞ്ഞതുപോലെ നിങ്ങളുടെ പ്രോംപ്റ്റ് "23" എന്ന സംഖ്യാ ഫലം നൽകും.
