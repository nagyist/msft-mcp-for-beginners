# ಮೂಲ ಕ್ಯಾಲ್ಕುಲೇಟರ್ MCP ಸೇವೆ

ಈ ಸೇವೆ ಮಾದರಿ ಸಾಂದರ್ಭಿಕ ಪ್ರೋಟೋಕಾಲ್ (MCP) ಮೂಲಕ ಮೂಲ ಕ್ಯಾಲ್ಕುಲೇಟರ್ ಕಾರ್ಯಾಚರಣೆಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ. ಇದು MCP ಅನುಷ್ಠಾನಗಳನ್ನು ಕಲಿಯುತ್ತಿರುವ ಆರಂಭಿಕರಿಗೆ ಸರಳ ಉದಾಹರಣೆಯಾಗಿ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ.

ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ, ನೋಡಿ [C# SDK](https://github.com/modelcontextprotocol/csharp-sdk)

## ವೈಶಿಷ್ಟ್ಯಗಳು

ಈ ಕ್ಯಾಲ್ಕುಲೇಟರ್ ಸೇವೆ ಕೆಳಗಿನ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ನೀಡುತ್ತದೆ:

1. **ಮೂಲ ಅಂಕಗಣಿತ ಕಾರ್ಯಾಚರಣೆಗಳು**:
   - ಎರಡು ಸಂಖ್ಯೆಗಳ ಸೇರ್ಪಡೆ
   - ಒಂದು ಸಂಖ್ಯೆಯಿಂದ ಮತ್ತೊಂದು ಸಂಖ್ಯೆಯ ವ್ಯತ್ಯಾಸ
   - ಎರಡು ಸಂಖ್ಯೆಗಳ ಗುಣಾಕಾರ
   - ಒಂದು ಸಂಖ್ಯೆಯನ್ನು ಮತ್ತೊಂದು ಸಂಖ್ಯೆಯಿಂದ ಭಾಗಿಸುವುದು (ಶೂನ್ಯ ಭಾಗಿಸುವಿಕೆ ಪರಿಶೀಲನೆಯೊಂದಿಗೆ)

## `stdio` ಪ್ರಕಾರ ಬಳಕೆ

## ಸಂರಚನೆ

1. **MCP ಸರ್ವರ್‌ಗಳನ್ನು ಸಂರಚಿಸಿ**:
   - ನಿಮ್ಮ ವರ್ಕ್‌ಸ್ಪೇಸ್ ಅನ್ನು VS ಕೋಡ್‌ನಲ್ಲಿ ತೆರೆಯಿರಿ.
   - MCP ಸರ್ವರ್‌ಗಳನ್ನು ಸಂರಚಿಸಲು ನಿಮ್ಮ ವರ್ಕ್‌ಸ್ಪೇಸ್ ಫೋಲ್ಡರ್‌ನಲ್ಲಿ `.vscode/mcp.json` ಫೈಲ್ ರಚಿಸಿ. ಉದಾಹರಣೆಯ ಸಂರಚನೆ:

     ```jsonc
     {
       "inputs": [
         {
           "type": "promptString",
           "id": "repository-root",
           "description": "The absolute path to the repository root"
         }
       ],
       "servers": {
         "calculator-mcp-dotnet": {
           "type": "stdio",
           "command": "dotnet",
           "args": [
             "run",
             "--project",
             "${input:repository-root}/03-GettingStarted/samples/csharp/src/calculator.csproj"
           ]
         }
       }
     }
     ```

   - ನಿಮಗೆ GitHub ರೆಪೊಸಿಟರಿ ರೂಟ್ ನಮೂದಿಸಲು ಕೇಳಲಾಗುತ್ತದೆ, ಇದನ್ನು `git rev-parse --show-toplevel` ಕಮಾಂಡ್‌ನಿಂದ ಪಡೆಯಬಹುದು.

## ಸೇವೆ ಬಳಕೆ

MCP ಪ್ರೋಟೋಕಾಲ್ ಮೂಲಕ ಈ ಸೇವೆ ಕೆಳಗಿನ API ಎಂಡ್ಪಾಯಿಂಟ್‌ಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ:

- `add(a, b)`: ಎರಡು ಸಂಖ್ಯೆಗಳ ಸೇರ್ಪಡೆ
- `subtract(a, b)`: ಎರಡನೇ ಸಂಖ್ಯೆಯನ್ನು ಮೊದಲನೆಯದರಿಂದ ಕಡಿಮೆಮಾಡಿ
- `multiply(a, b)`: ಎರಡು ಸಂಖ್ಯೆಗಳ ಗುಣಾಕಾರ
- `divide(a, b)`: ಮೊದಲ ಸಂಖ್ಯೆಯನ್ನು ಎರಡನೇ ಸಂಖ್ಯೆಯಿಂದ ಭಾಗಿಸಿ (ಶೂನ್ಯ ಪರಿಶೀಲನೆಯೊಂದಿಗೆ)
- isPrime(n): ಸಂಖ್ಯೆ ಪ್ರೈಮ್ ಆಗಿದೆಯೇ ಎಂದು ಪರಿಶೀಲಿಸಿ

## VS ಕೋಡ್‌ನಲ್ಲಿ Github Copilot ಚಾಟ್ ಮೂಲಕ ಪರೀಕ್ಷೆ

1. MCP ಪ್ರೋಟೋಕಾಲ್ ಬಳಸಿ ಸೇವೆಗೆ ವಿನಂತಿ ಮಾಡಲು ಪ್ರಯತ್ನಿಸಿ. ಉದಾಹರಣೆಗೆ, ನೀವು ಕೇಳಬಹುದು:
   - "5 ಮತ್ತು 3 ಸೇರಿಸಿ"
   - "4 ರಿಂದ 10 ಕಡಿಮೆಮಾಡಿ"
   - "6 ಮತ್ತು 7 ಗುಣಿಸಿ"
   - "8 ಅನ್ನು 2 ರಿಂದ ಭಾಗಿಸಿ"
   - "37854 ಪ್ರೈಮ್ ಆಗಿದೆಯೇ?"
   - "4242 ಮೊದಲು ಮತ್ತು ನಂತರ 3 ಪ್ರೈಮ್ ಸಂಖ್ಯೆಗಳು ಯಾವುವು?"
2. ಇದು ಟೂಲ್ಸ್ ಬಳಸುತ್ತಿರುವುದನ್ನು ಖಚಿತಪಡಿಸಲು ಪ್ರಾಂಪ್ಟ್‌ಗೆ #MyCalculator ಸೇರಿಸಿ. ಉದಾಹರಣೆಗೆ:
   - "5 ಮತ್ತು 3 ಸೇರಿಸಿ #MyCalculator"
   - "4 ರಿಂದ 10 ಕಡಿಮೆಮಾಡಿ #MyCalculator"

## ಕಂಟೈನರೈಜ್ಡ್ ಆವೃತ್ತಿ

ಹಿಂದಿನ ಪರಿಹಾರವು .NET SDK ಸ್ಥಾಪಿತವಾಗಿದ್ದಾಗ ಮತ್ತು ಎಲ್ಲಾ ಅವಲಂಬನೆಗಳು ಸಿದ್ಧವಾಗಿದ್ದಾಗ ಉತ್ತಮವಾಗಿದೆ. ಆದರೆ, ನೀವು ಪರಿಹಾರವನ್ನು ಹಂಚಿಕೊಳ್ಳಲು ಅಥವಾ ಬೇರೆ ಪರಿಸರದಲ್ಲಿ ಚಾಲನೆ ಮಾಡಲು ಬಯಸಿದರೆ, ನೀವು ಕಂಟೈನರೈಜ್ಡ್ ಆವೃತ್ತಿಯನ್ನು ಬಳಸಬಹುದು.

1. ಡೋಕರ್ ಅನ್ನು ಪ್ರಾರಂಭಿಸಿ ಮತ್ತು ಅದು ಚಾಲನೆಯಲ್ಲಿರುವುದನ್ನು ಖಚಿತಪಡಿಸಿ.
1. ಟರ್ಮಿನಲ್‌ನಿಂದ `03-GettingStarted\samples\csharp\src` ಫೋಲ್ಡರ್‌ಗೆ ನವಿಗೇಟ್ ಮಾಡಿ
1. ಕ್ಯಾಲ್ಕುಲೇಟರ್ ಸೇವೆಯ ಡೋಕರ್ ಇಮೇಜ್ ನಿರ್ಮಿಸಲು ಕೆಳಗಿನ ಕಮಾಂಡ್ ಅನ್ನು ನಿರ್ವಹಿಸಿ (`<YOUR-DOCKER-USERNAME>` ಅನ್ನು ನಿಮ್ಮ ಡೋಕರ್ ಹಬ್ ಬಳಕೆದಾರಹೆಸರಿನಿಂದ ಬದಲಾಯಿಸಿ):
   ```bash
   docker build -t <YOUR-DOCKER-USERNAME>/mcp-calculator .
   ``` 
1. ಇಮೇಜ್ ನಿರ್ಮಿಸಿದ ನಂತರ, ಅದನ್ನು ಡೋಕರ್ ಹಬ್‌ಗೆ ಅಪ್‌ಲೋಡ್ ಮಾಡೋಣ. ಕೆಳಗಿನ ಕಮಾಂಡ್ ಅನ್ನು ರನ್ ಮಾಡಿ:
   ```bash
    docker push <YOUR-DOCKER-USERNAME>/mcp-calculator
  ```

## Use the Dockerized Version

1. In the `.vscode/mcp.json` file, replace the server configuration by the following:
   ```json
    "mcp-calc": {
      "command": "docker",
      "args": [
        "run",
        "--rm",
        "-i",
        "<YOUR-DOCKER-USERNAME>/mcp-calc"
      ],
      "envFile": "",
      "env": {}
    }
   ```
   ಸಂರಚನೆಯನ್ನು ನೋಡಿದಾಗ, ಕಮಾಂಡ್ `docker` ಆಗಿದ್ದು, args ಗಳು `run --rm -i <YOUR-DOCKER-USERNAME>/mcp-calc` ಆಗಿವೆ. `--rm` ಫ್ಲ್ಯಾಗ್ ಕಂಟೈನರ್ ನಿಲ್ಲಿಸಿದ ನಂತರ ಅದನ್ನು ತೆಗೆದುಹಾಕುತ್ತದೆ, ಮತ್ತು `-i` ಫ್ಲ್ಯಾಗ್ ಕಂಟೈನರ್‌ನ ಸ್ಟ್ಯಾಂಡರ್ಡ್ ಇನ್‌ಪುಟ್‌ನೊಂದಿಗೆ ಸಂವಹನ ಮಾಡಲು ಅನುಮತಿಸುತ್ತದೆ. ಕೊನೆಯ ಆರ್ಗ್ಯುಮೆಂಟ್ ನಾವು ನಿರ್ಮಿಸಿ ಡೋಕರ್ ಹಬ್‌ಗೆ ಪುಷ್ ಮಾಡಿದ ಇಮೇಜಿನ ಹೆಸರು.

## ಡೋಕರ್ ಆವೃತ್ತಿಯನ್ನು ಪರೀಕ್ಷಿಸಿ

`"mcp-calc": {` ಮೇಲಿನ ಸಣ್ಣ ಸ್ಟಾರ್ಟ್ ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ MCP ಸರ್ವರ್ ಅನ್ನು ಪ್ರಾರಂಭಿಸಿ, ಮತ್ತು ಹಿಂದಿನಂತೆ ನೀವು ಕ್ಯಾಲ್ಕುಲೇಟರ್ ಸೇವೆಗೆ ಕೆಲವು ಗಣಿತ ಕಾರ್ಯಗಳನ್ನು ಕೇಳಬಹುದು.
