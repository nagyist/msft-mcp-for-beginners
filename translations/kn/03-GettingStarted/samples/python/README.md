# MCP ಕ್ಯಾಲ್ಕ್ಯುಲೇಟರ್ ಸರ್ವರ್ (Python)



ಸರಳವಾದ ಮಾದರಿ ಸಾಂದರ್ಭಿಕ ಪ್ರೋಟೋಕಾಲ್ (MCP) ಸರ್ವರ್ ಅನುಷ್ಠಾನ Python ನಲ್ಲಿ, ಇದು ಮೂಲ ಕ್ಯಾಲ್ಕ್ಯುಲೇಟರ್ ಕಾರ್ಯಕ್ಷಮತೆಯನ್ನು ಒದಗಿಸುತ್ತದೆ.


## ಸ್ಥಾಪನೆ

ಅಗತ್ಯವಿರುವ ಅವಲಂಬನೆಗಳನ್ನು ಸ್ಥಾಪಿಸಿ:

```bash
pip install -r requirements.txt
```

ಅಥವಾ MCP Python SDK ನೇರವಾಗಿ ಸ್ಥಾಪಿಸಿ:

```bash
pip install mcp>=1.18.0
```

## ಬಳಕೆ

### ಸರ್ವರ್ ಚಾಲನೆ

ಸರ್ವರ್ ಅನ್ನು MCP ಗ್ರಾಹಕರು (Claude Desktop ಹೋಲುವವರು) ಬಳಸಲು ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ. ಸರ್ವರ್ ಪ್ರಾರಂಭಿಸಲು:

```bash
python mcp_calculator_server.py
```

**ಗಮನಿಸಿ**: ಟರ್ಮಿನಲ್‌ನಲ್ಲಿ ನೇರವಾಗಿ ಚಾಲನೆ ಮಾಡಿದಾಗ, ನೀವು JSON-RPC ಮಾನ್ಯತೆ ದೋಷಗಳನ್ನು ಕಾಣಬಹುದು. ಇದು ಸಾಮಾನ್ಯ ವರ್ತನೆ - ಸರ್ವರ್ ಸರಿಯಾದ ಸ್ವರೂಪದ MCP ಗ್ರಾಹಕ ಸಂದೇಶಗಳನ್ನು ಕಾಯುತ್ತಿದೆ.

### ಕಾರ್ಯಗಳನ್ನು ಪರೀಕ್ಷಿಸುವುದು

ಕ್ಯಾಲ್ಕ್ಯುಲೇಟರ್ ಕಾರ್ಯಗಳು ಸರಿಯಾಗಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತಿರುವುದನ್ನು ಪರೀಕ್ಷಿಸಲು:

```bash
python test_calculator.py
```

## ಸಮಸ್ಯೆ ಪರಿಹಾರ

### ಆಮದು ದೋಷಗಳು

`ModuleNotFoundError: No module named 'mcp'` ಎಂಬ ದೋಷ ಕಾಣಿಸಿದರೆ, MCP Python SDK ಅನ್ನು ಸ್ಥಾಪಿಸಿ:

```bash
pip install mcp>=1.18.0
```

### ನೇರವಾಗಿ ಚಾಲನೆ ಮಾಡಿದಾಗ JSON-RPC ದೋಷಗಳು

"Invalid JSON: EOF while parsing a value" ಹೋಲುವ ದೋಷಗಳು ನೇರವಾಗಿ ಸರ್ವರ್ ಚಾಲನೆ ಮಾಡಿದಾಗ ನಿರೀಕ್ಷಿತ. ಸರ್ವರ್‌ಗೆ ನೇರ ಟರ್ಮಿನಲ್ ಇನ್ಪುಟ್ ಅಲ್ಲ, MCP ಗ್ರಾಹಕ ಸಂದೇಶಗಳು ಬೇಕಾಗಿವೆ.
