# ಈ ಮಾದರಿಯನ್ನು ಚಾಲನೆ ಮಾಡುವುದು

> [!NOTE]
> ಈ ರಸ್ಟ್ ಕ್ಲೈಂಟ್ ಈ ಪ್ರಾಜೆಕ್ಟ್‌ಗೆ ಸಹೋದರ ಡೈರೆಕ್ಟರಿಯಾದ [server](../../../../../../03-GettingStarted/02-client/solution/server) ಡೈರೆಕ್ಟರಿಯಲ್ಲಿ ಮಾದರಿ MCP ಸರ್ವರ್ ಅನ್ನು ಚಾಲನೆ ಮಾಡಲು ನೀವು ಡಾಟ್‌ನೆಟ್ 8.0 SDK ಅನ್ನು ಸ್ಥಾಪಿಸಿರುವಿರಿ ಎಂದು ಊಹಿಸುತ್ತದೆ.

## -1- ಅವಲಂಬನೆಗಳನ್ನು ಸ್ಥಾಪಿಸಿ ಮತ್ತು ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು ನಿರ್ಮಿಸಿ

```bash
cargo build
```

## -2- ಮಾದರಿಯನ್ನು ಚಾಲನೆ ಮಾಡಿ

```bash
cargo run
```
