# ಮೂಲ ಕ್ಯಾಲ್ಕುಲೇಟರ್ MCP ಸೇವೆ

ಈ ಸೇವೆ ಸ್ಪ್ರಿಂಗ್ ಬೂಟ್‌ನೊಂದಿಗೆ ವೆಬ್‌ಫ್ಲಕ್ಸ್ ಸಾರಿಗೆ ಬಳಸಿ ಮಾದರಿ ಸಾಂದರ್ಭಿಕ ಪ್ರೋಟೋಕಾಲ್ (MCP) ಮೂಲಕ ಮೂಲ ಕ್ಯಾಲ್ಕುಲೇಟರ್ ಕಾರ್ಯಾಚರಣೆಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ. ಇದು MCP ಅನುಷ್ಠಾನಗಳನ್ನು ಕಲಿಯುತ್ತಿರುವ ಆರಂಭಿಕರಿಗೆ ಸರಳ ಉದಾಹರಣೆಯಾಗಿ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ.

ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ, [MCP ಸರ್ವರ್ ಬೂಟ್ ಸ್ಟಾರ್ಟರ್](https://docs.spring.io/spring-ai/reference/api/mcp/mcp-server-boot-starter-docs.html) ಉಲ್ಲೇಖ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ನೋಡಿ.


## ಸೇವೆಯನ್ನು ಬಳಸುವುದು

ಸೇವೆಯು MCP ಪ್ರೋಟೋಕಾಲ್ ಮೂಲಕ ಕೆಳಗಿನ API ಎಂಡ್ಪಾಯಿಂಟ್‌ಗಳನ್ನು ಬಹಿರಂಗಪಡಿಸುತ್ತದೆ:

- `add(a, b)`: ಎರಡು ಸಂಖ್ಯೆಗಳನ್ನೂ ಸೇರಿಸಿ
- `subtract(a, b)`: ಎರಡನೇ ಸಂಖ್ಯೆಯನ್ನು ಮೊದಲನೆಯದರಿಂದ ಕಡಿಮೆಮಾಡಿ
- `multiply(a, b)`: ಎರಡು ಸಂಖ್ಯೆಗಳ ಗುಣಾಕಾರ ಮಾಡಿ
- `divide(a, b)`: ಮೊದಲ ಸಂಖ್ಯೆಯನ್ನು ಎರಡನೇ ಸಂಖ್ಯೆಯಿಂದ ಭಾಗಿಸಿ (ಶೂನ್ಯ ಪರಿಶೀಲನೆಯೊಂದಿಗೆ)
- `power(base, exponent)`: ಸಂಖ್ಯೆಯ ಘಾತವನ್ನು ಲೆಕ್ಕಿಸಿ
- `squareRoot(number)`: ವರ್ಗಮೂಲ ಲೆಕ್ಕಿಸಿ (ನಕಾರಾತ್ಮಕ ಸಂಖ್ಯೆ ಪರಿಶೀಲನೆಯೊಂದಿಗೆ)
- `modulus(a, b)`: ಭಾಗಾಕ್ಷರ ಲೆಕ್ಕಿಸಿ
- `absolute(number)`: ಪರಮಾಣು ಮೌಲ್ಯ ಲೆಕ್ಕಿಸಿ

## ಅವಲಂಬನೆಗಳು

ಪ್ರಾಜೆಕ್ಟ್ ಕೆಳಗಿನ ಪ್ರಮುಖ ಅವಲಂಬನೆಗಳನ್ನು ಅಗತ್ಯವಿದೆ:

```xml
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-starter-mcp-server-webflux</artifactId>
</dependency>
```

## ಪ್ರಾಜೆಕ್ಟ್ ನಿರ್ಮಾಣ

ಮೇವನ್ ಬಳಸಿ ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು ನಿರ್ಮಿಸಿ:
```bash
./mvnw clean install -DskipTests
```

## ಸರ್ವರ್ ಚಾಲನೆ

### ಜಾವಾ ಬಳಸಿ

```bash
java -jar target/calculator-server-0.0.1-SNAPSHOT.jar
```

### MCP ಇನ್ಸ್‌ಪೆಕ್ಟರ್ ಬಳಸಿ

MCP ಇನ್ಸ್‌ಪೆಕ್ಟರ್ MCP ಸೇವೆಗಳೊಂದಿಗೆ ಸಂವಹನ ಮಾಡಲು ಸಹಾಯಕ ಸಾಧನವಾಗಿದೆ. ಈ ಕ್ಯಾಲ್ಕುಲೇಟರ್ ಸೇವೆಯೊಂದಿಗೆ ಇದನ್ನು ಬಳಸಲು:

1. **MCP ಇನ್ಸ್‌ಪೆಕ್ಟರ್ ಅನ್ನು ಸ್ಥಾಪಿಸಿ ಮತ್ತು ಹೊಸ ಟರ್ಮಿನಲ್ ವಿಂಡೋದಲ್ಲಿ ಚಾಲನೆ ಮಾಡಿ**:
   ```bash
   npx @modelcontextprotocol/inspector
   ```

2. **ಅ್ಯಪ್ ತೋರಿಸುವ URL ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ ವೆಬ್ UI ಗೆ ಪ್ರವೇಶಿಸಿ** (ಸಾಮಾನ್ಯವಾಗಿ http://localhost:6274)

3. **ಸಂಪರ್ಕವನ್ನು ಸಂರಚಿಸಿ**:
   - ಸಾರಿಗೆ ಪ್ರಕಾರವನ್ನು "SSE" ಆಗಿ ಸೆಟ್ ಮಾಡಿ
   - ನಿಮ್ಮ ಚಾಲನೆಯಲ್ಲಿರುವ ಸರ್ವರ್‌ನ SSE ಎಂಡ್ಪಾಯಿಂಟ್ URL ಅನ್ನು ಸೆಟ್ ಮಾಡಿ: `http://localhost:8080/sse`
   - "Connect" ಕ್ಲಿಕ್ ಮಾಡಿ

4. **ಸಾಧನಗಳನ್ನು ಬಳಸಿ**:
   - ಲಭ್ಯವಿರುವ ಕ್ಯಾಲ್ಕುಲೇಟರ್ ಕಾರ್ಯಾಚರಣೆಗಳನ್ನು ನೋಡಲು "List Tools" ಕ್ಲಿಕ್ ಮಾಡಿ
   - ಒಂದು ಸಾಧನವನ್ನು ಆಯ್ಕೆಮಾಡಿ ಮತ್ತು ಕಾರ್ಯಾಚರಣೆ ನಡೆಸಲು "Run Tool" ಕ್ಲಿಕ್ ಮಾಡಿ
