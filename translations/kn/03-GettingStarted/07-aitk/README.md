# Visual Studio Code ಗಾಗಿ AI Toolkit ವಿಸ್ತರಣೆದಿಂದ ಸರ್ವರ್ ಅನ್ನು ಉಪಯೋಗಿಸುವುದು

ನೀವು AI ಏಜೆಂಟ್ ಅನ್ನು ನಿರ್ಮಿಸುತ್ತಿರುವಾಗ, ಅದು ಕೇವಲ ಬುದ್ಧಿವಂತ ಉತ್ತರಗಳನ್ನು ರಚಿಸುವುದಲ್ಲ; ನಿಮ್ಮ ಏಜೆಂಟ್‌ಗೆ ಕ್ರಮ ಕೈಗೊಳ್ಳುವ ಸಾಮರ್ಥ್ಯವನ್ನು ನೀಡುವುದು ಕೂಡ ಮುಖ್ಯ. ಅದಕ್ಕಾಗಿ Model Context Protocol (MCP) ಇದೆ. MCP ಏಜೆಂಟ್‌ಗಳಿಗೆ ಹೊರಗಿನ ಸಾಧನಗಳು ಮತ್ತು ಸೇವೆಗಳಿಗೆ ಸुसಂಗತ ರೀತಿಯಲ್ಲಿ ಪ್ರವೇಶಿಸಲು ಸುಲಭವಾಗಿಸುತ್ತದೆ. ಇದನ್ನು ನಿಮ್ಮ ಏಜೆಂಟ್ ಅನ್ನು ನಿಜವಾಗಿಯೂ ಬಳಸಬಹುದಾದ ಉಪಕರಣಪೆಟ್ಟಿಗೆಯಲ್ಲಿ ಸಂಪರ್ಕಿಸುವಂತೆ ಭಾವಿಸಿ.

ನೀವು ನಿಮ್ಮ ಏಜೆಂಟ್ ಅನ್ನು ಕ್ಯಾಲ್ಕುಲೇಟರ್ MCP ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕಿಸಿದರೆ, ಅಚಾನಕ್ ನಿಮ್ಮ ಏಜೆಂಟ್ “47 ಗುಣ 89 ಎಷ್ಟು?” ಎಂಬ ಪ್ರಾಂಪ್ಟ್ ಪಡೆಯುವುದರಿಂದ ಗಣಿತ ಕಾರ್ಯಗಳನ್ನು ನಿರ್ವಹಿಸಬಹುದು—ಲಾಜಿಕ್ ಅನ್ನು ಹಾರ್ಡ್‌ಕೋಡ್ ಮಾಡಬೇಕಾಗಿಲ್ಲ ಅಥವಾ ಕಸ್ಟಮ್ APIಗಳನ್ನು ನಿರ್ಮಿಸುವ ಅಗತ್ಯವಿಲ್ಲ.

## ಅವಲೋಕನ

ಈ ಪಾಠವು Visual Studio Code ನಲ್ಲಿ [AI Toolkit](https://aka.ms/AIToolkit) ವಿಸ್ತರಣೆಯೊಂದಿಗೆ ಕ್ಯಾಲ್ಕುಲೇಟರ್ MCP ಸರ್ವರ್ ಅನ್ನು ಏಜೆಂಟ್‌ಗೆ ಹೇಗೆ ಸಂಪರ್ಕಿಸುವುದು ಎಂಬುದನ್ನು ಒಳಗೊಂಡಿದೆ, ಇದು ನಿಮ್ಮ ಏಜೆಂಟ್‌ಗೆ ಸಹಜ ಭಾಷೆಯ ಮೂಲಕ ಸೇರಿಸುವಿಕೆ, ಕಡಿತ, ಗುಣಾಕಾರ ಮತ್ತು ಭಾಗಾಕಾರದಂತಹ ಗಣಿತ ಕಾರ್ಯಗಳನ್ನು ನಿರ್ವಹಿಸಲು ಸಾಧ್ಯವಾಗಿಸುತ್ತದೆ.

AI Toolkit Visual Studio Code ಗಾಗಿ ಶಕ್ತಿಶಾಲಿ ವಿಸ್ತರಣೆ ಆಗಿದ್ದು, ಏಜೆಂಟ್ ಅಭಿವೃದ್ಧಿಯನ್ನು ಸರಳಗೊಳಿಸುತ್ತದೆ. AI ಎಂಜಿನಿಯರ್‌ಗಳು ಜನರೇಟಿವ್ AI ಮಾದರಿಗಳನ್ನು ಸ್ಥಳೀಯವಾಗಿ ಅಥವಾ ಕ್ಲೌಡ್‌ನಲ್ಲಿ ಅಭಿವೃದ್ಧಿಪಡಿಸಿ ಪರೀಕ್ಷಿಸುವ ಮೂಲಕ ಸುಲಭವಾಗಿ AI ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸಬಹುದು. ಈ ವಿಸ್ತರಣೆ ಇಂದಿನ ಬಹುತೇಕ ಪ್ರಮುಖ ಜನರೇಟಿವ್ ಮಾದರಿಗಳನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.

*ಗಮನಿಸಿ*: AI Toolkit ಪ್ರಸ್ತುತ Python ಮತ್ತು TypeScript ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.

## ಕಲಿಕೆಯ ಉದ್ದೇಶಗಳು

ಈ ಪಾಠದ ಕೊನೆಯಲ್ಲಿ, ನೀವು ಈ ಕೆಳಗಿನವುಗಳನ್ನು ಮಾಡಲು ಸಾಧ್ಯವಾಗುತ್ತದೆ:

- AI Toolkit ಮೂಲಕ MCP ಸರ್ವರ್ ಅನ್ನು ಉಪಯೋಗಿಸುವುದು.
- MCP ಸರ್ವರ್ ನೀಡುವ ಸಾಧನಗಳನ್ನು ಕಂಡುಹಿಡಿದು ಬಳಸಲು ಏಜೆಂಟ್ ಸಂರಚನೆಯನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡುವುದು.
- ಸಹಜ ಭಾಷೆಯ ಮೂಲಕ MCP ಸಾಧನಗಳನ್ನು ಬಳಸುವುದು.

## ವಿಧಾನ

ಇದನ್ನು ನಾವು ಉನ್ನತ ಮಟ್ಟದಲ್ಲಿ ಹೇಗೆ ಕೈಗೊಳ್ಳಬೇಕು ಎಂಬುದು ಇಲ್ಲಿದೆ:

- ಏಜೆಂಟ್ ಅನ್ನು ರಚಿಸಿ ಮತ್ತು ಅದರ ಸಿಸ್ಟಮ್ ಪ್ರಾಂಪ್ಟ್ ಅನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಿ.
- ಕ್ಯಾಲ್ಕುಲೇಟರ್ ಸಾಧನಗಳೊಂದಿಗೆ MCP ಸರ್ವರ್ ರಚಿಸಿ.
- ಏಜೆಂಟ್ ಬಿಲ್ಡರ್ ಅನ್ನು MCP ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕಿಸಿ.
- ಸಹಜ ಭಾಷೆಯ ಮೂಲಕ ಏಜೆಂಟ್‌ನ ಸಾಧನ ಆಮಂತ್ರಣವನ್ನು ಪರೀಕ್ಷಿಸಿ.

ಚೆನ್ನಾಗಿದೆ, ಈಗ ನಾವು ಪ್ರಕ್ರಿಯೆಯನ್ನು ಅರ್ಥಮಾಡಿಕೊಂಡಿದ್ದೇವೆ, MCP ಮೂಲಕ ಹೊರಗಿನ ಸಾಧನಗಳನ್ನು ಉಪಯೋಗಿಸಲು AI ಏಜೆಂಟ್ ಅನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡೋಣ, ಅದರ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಹೆಚ್ಚಿಸೋಣ!

### -2- ಏಜೆಂಟ್‌ಗೆ ಸಿಸ್ಟಮ್ ಪ್ರಾಂಪ್ಟ್ ರಚಿಸಿ

ಏಜೆಂಟ್ ಸಿದ್ಧವಾದ ಮೇಲೆ, ಅದರ ವ್ಯಕ್ತಿತ್ವ ಮತ್ತು ಉದ್ದೇಶವನ್ನು ವ್ಯಾಖ್ಯಾನಿಸುವ ಸಮಯ. ಈ ವಿಭಾಗದಲ್ಲಿ, ನೀವು **Generate system prompt** ವೈಶಿಷ್ಟ್ಯವನ್ನು ಉಪಯೋಗಿಸಿ ಏಜೆಂಟ್‌ನ ನಿರೀಕ್ಷಿತ ವರ್ತನೆ (ಈ ಪ್ರಕರಣದಲ್ಲಿ ಕ್ಯಾಲ್ಕುಲೇಟರ್ ಏಜೆಂಟ್) ಅನ್ನು ವರ್ಣಿಸಿ, ಮಾದರಿ ಸಿಸ್ಟಮ್ ಪ್ರಾಂಪ್ಟ್ ಅನ್ನು ಬರೆಯುವಂತೆ ಮಾಡುತ್ತೀರಿ.

![Visual Studio Code ಗಾಗಿ AI Toolkit ನ "Calculator Agent" ಇಂಟರ್ಫೇಸ್ ಸ್ಕ್ರೀನ್‌ಶಾಟ್, "Generate a prompt" ಎಂಬ ಶೀರ್ಷಿಕೆಯೊಂದಿಗೆ ಒಂದು ಮೋಡಲ್ ವಿಂಡೋ ತೆರೆದಿದೆ. ಮೋಡಲ್‌ನಲ್ಲಿ ಪ್ರಾಂಪ್ಟ್ ಟೆಂಪ್ಲೇಟನ್ನು ಮೂಲಭೂತ ವಿವರಗಳನ್ನು ಹಂಚಿಕೊಂಡು ರಚಿಸಬಹುದು ಎಂದು ವಿವರಿಸಲಾಗಿದೆ ಮತ್ತು ಒಂದು ಪಠ್ಯ ಬಾಕ್ಸ್‌ನಲ್ಲಿ ಉದಾಹರಣೆಯ ಸಿಸ್ಟಮ್ ಪ್ರಾಂಪ್ಟ್ ಇದೆ: "ನೀವು ಸಹಾಯಕ ಮತ್ತು ಪರಿಣಾಮಕಾರಿ ಗಣಿತ ಸಹಾಯಕ. ಮೂಲಭೂತ ಅಂಕಗಣಿತ ಸಮಸ್ಯೆ ನೀಡಿದಾಗ, ನೀವು ಸರಿಯಾದ ಫಲಿತಾಂಶವನ್ನು ನೀಡುತ್ತೀರಿ." ಕೆಳಗೆ "Close" ಮತ್ತು "Generate" ಬಟನ್‌ಗಳು ಇವೆ. ಹಿನ್ನೆಲೆಯಲ್ಲಿ ಏಜೆಂಟ್ ಸಂರಚನೆಯ ಭಾಗಗಳು, ಆಯ್ಕೆಮಾಡಲಾದ ಮಾದರಿ "OpenAI GPT-4o (via GitHub)" ಮತ್ತು ಸಿಸ್ಟಮ್ ಮತ್ತು ಬಳಕೆದಾರ ಪ್ರಾಂಪ್ಟ್ ಕ್ಷೇತ್ರಗಳು ಕಾಣಿಸುತ್ತವೆ.](../../../../translated_images/aitk-generate-prompt.ba9e69d3d2bbe2a26444d0c78775540b14196061eee32c2054e9ee68c4f51f3a.kn.png)

1. **Prompts** ವಿಭಾಗದಲ್ಲಿ **Generate system prompt** ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ. ಇದು ಪ್ರಾಂಪ್ಟ್ ಬಿಲ್ಡರ್ ಅನ್ನು ತೆರೆಯುತ್ತದೆ, ಇದು AI ಬಳಸಿ ಏಜೆಂಟ್‌ಗೆ ಸಿಸ್ಟಮ್ ಪ್ರಾಂಪ್ಟ್ ರಚಿಸುತ್ತದೆ.
1. **Generate a prompt** ವಿಂಡೋದಲ್ಲಿ ಈ ಪಠ್ಯವನ್ನು ನಮೂದಿಸಿ: `You are a helpful and efficient math assistant. When given a problem involving basic arithmetic, you respond with the correct result.`
1. **Generate** ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ. ಕೆಳಬಲಭಾಗದಲ್ಲಿ ಒಂದು ಸೂಚನೆ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತದೆ, ಸಿಸ್ಟಮ್ ಪ್ರಾಂಪ್ಟ್ ರಚನೆಯ ಪ್ರಗತಿಯನ್ನು ಸೂಚಿಸುತ್ತದೆ. ಪ್ರಾಂಪ್ಟ್ ರಚನೆ ಪೂರ್ಣಗೊಂಡ ನಂತರ, ಅದು **Agent (Prompt) Builder** ನ **System prompt** ಕ್ಷೇತ್ರದಲ್ಲಿ ಕಾಣಿಸುತ್ತದೆ.
1. **System prompt** ಪರಿಶೀಲಿಸಿ ಮತ್ತು ಅಗತ್ಯವಿದ್ದರೆ ತಿದ್ದುಪಡಿ ಮಾಡಿ.

### -3- MCP ಸರ್ವರ್ ರಚಿಸಿ

ನೀವು ಈಗ ಏಜೆಂಟ್‌ನ ಸಿಸ್ಟಮ್ ಪ್ರಾಂಪ್ಟ್ ಅನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಿದ್ದೀರಿ—ಅದರ ವರ್ತನೆ ಮತ್ತು ಪ್ರತಿಕ್ರಿಯೆಗಳನ್ನು ಮಾರ್ಗದರ್ಶನ ಮಾಡುತ್ತದೆ—ಈಗ ಏಜೆಂಟ್‌ಗೆ ಪ್ರಾಯೋಗಿಕ ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಒದಗಿಸುವ ಸಮಯ. ಈ ವಿಭಾಗದಲ್ಲಿ, ನೀವು ಸೇರಿಸುವಿಕೆ, ಕಡಿತ, ಗುಣಾಕಾರ ಮತ್ತು ಭಾಗಾಕಾರ ಗಣನೆಗಳನ್ನು ನಿರ್ವಹಿಸುವ ಸಾಧನಗಳೊಂದಿಗೆ ಕ್ಯಾಲ್ಕುಲೇಟರ್ MCP ಸರ್ವರ್ ರಚಿಸುವಿರಿ. ಈ ಸರ್ವರ್ ನಿಮ್ಮ ಏಜೆಂಟ್‌ಗೆ ಸಹಜ ಭಾಷೆಯ ಪ್ರಾಂಪ್ಟ್‌ಗಳಿಗೆ ತಕ್ಷಣ ಗಣಿತ ಕಾರ್ಯಗಳನ್ನು ನಿರ್ವಹಿಸಲು ಅವಕಾಶ ನೀಡುತ್ತದೆ.

!["Visual Studio Code ಗಾಗಿ AI Toolkit ವಿಸ್ತರಣೆಯಲ್ಲಿನ Calculator Agent ಇಂಟರ್ಫೇಸ್‌ನ ಕೆಳಭಾಗದ ಭಾಗದ ಸ್ಕ್ರೀನ್‌ಶಾಟ್. ಇದು “Tools” ಮತ್ತು “Structure output” ಎಂಬ ವಿಸ್ತರಿಸಬಹುದಾದ ಮೆನುಗಳನ್ನು ತೋರಿಸುತ್ತದೆ, ಜೊತೆಗೆ “Choose output format” ಎಂಬ ಡ್ರಾಪ್‌ಡೌನ್ ಮೆನು “text” ಗೆ ಸೆಟ್ ಆಗಿದೆ. ಬಲಕ್ಕೆ, “+ MCP Server” ಎಂಬ ಬಟನ್ ಇದೆ, ಇದು Model Context Protocol ಸರ್ವರ್ ಸೇರಿಸಲು. Tools ವಿಭಾಗದ ಮೇಲೆ ಚಿತ್ರ ಐಕಾನ್ ಪ್ಲೇಸ್‌ಹೋಲ್ಡರ್ ಇದೆ.](../../../../translated_images/aitk-add-mcp-server.9742cfddfe808353c0caf9cc0a7ed3e80e13abf4d2ebde315c81c3cb02a2a449.kn.png)

AI Toolkit ನಿಮ್ಮ ಸ್ವಂತ MCP ಸರ್ವರ್ ರಚಿಸಲು ಸುಲಭಗೊಳಿಸುವ ಟೆಂಪ್ಲೇಟುಗಳನ್ನು ಹೊಂದಿದೆ. ನಾವು ಕ್ಯಾಲ್ಕುಲೇಟರ್ MCP ಸರ್ವರ್ ರಚಿಸಲು Python ಟೆಂಪ್ಲೇಟನ್ನು ಉಪಯೋಗಿಸುವೆವು.

*ಗಮನಿಸಿ*: AI Toolkit ಪ್ರಸ್ತುತ Python ಮತ್ತು TypeScript ಅನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.

1. **Agent (Prompt) Builder** ನ **Tools** ವಿಭಾಗದಲ್ಲಿ **+ MCP Server** ಬಟನ್ ಕ್ಲಿಕ್ ಮಾಡಿ. ವಿಸ್ತರಣೆ **Command Palette** ಮೂಲಕ ಸೆಟಪ್ ವಿಸಾರ್ಡ್ ಪ್ರಾರಂಭಿಸುತ್ತದೆ.
1. **+ Add Server** ಆಯ್ಕೆಮಾಡಿ.
1. **Create a New MCP Server** ಆಯ್ಕೆಮಾಡಿ.
1. ಟೆಂಪ್ಲೇಟಾಗಿ **python-weather** ಆಯ್ಕೆಮಾಡಿ.
1. MCP ಸರ್ವರ್ ಟೆಂಪ್ಲೇಟು ಉಳಿಸಲು **Default folder** ಆಯ್ಕೆಮಾಡಿ.
1. ಸರ್ವರ್‌ಗೆ ಈ ಹೆಸರು ನೀಡಿ: **Calculator**
1. ಹೊಸ Visual Studio Code ವಿಂಡೋ ತೆರೆಯುತ್ತದೆ. **Yes, I trust the authors** ಆಯ್ಕೆಮಾಡಿ.
1. ಟರ್ಮಿನಲ್ ಬಳಸಿ ವರ್ಚುವಲ್ ಎನ್ವಿರಾನ್‌ಮೆಂಟ್ ರಚಿಸಿ: `python -m venv .venv`
1. ಟರ್ಮಿನಲ್ ಬಳಸಿ ವರ್ಚುವಲ್ ಎನ್ವಿರಾನ್‌ಮೆಂಟ್ ಸಕ್ರಿಯಗೊಳಿಸಿ:
    1. ವಿಂಡೋಸ್ - `.venv\Scripts\activate`
    1. macOS/Linux - `source .venv/bin/activate`
1. ಟರ್ಮಿನಲ್ ಬಳಸಿ ಅವಲಂಬನೆಗಳನ್ನು ಸ್ಥಾಪಿಸಿ: `pip install -e .[dev]`
1. **Activity Bar** ನ **Explorer** ವೀಕ್ಷಣೆಯಲ್ಲಿ **src** ಡೈರೆಕ್ಟರಿಯನ್ನು ವಿಸ್ತರಿಸಿ ಮತ್ತು **server.py** ಫೈಲ್ ತೆರೆಯಿರಿ.
1. **server.py** ಫೈಲ್‌ನಲ್ಲಿನ ಕೋಡ್ ಅನ್ನು ಕೆಳಗಿನದಾಗಿ ಬದಲಿಸಿ ಮತ್ತು ಉಳಿಸಿ:

    ```python
    """
    Sample MCP Calculator Server implementation in Python.

    
    This module demonstrates how to create a simple MCP server with calculator tools
    that can perform basic arithmetic operations (add, subtract, multiply, divide).
    """
    
    from mcp.server.fastmcp import FastMCP
    
    server = FastMCP("calculator")
    
    @server.tool()
    def add(a: float, b: float) -> float:
        """Add two numbers together and return the result."""
        return a + b
    
    @server.tool()
    def subtract(a: float, b: float) -> float:
        """Subtract b from a and return the result."""
        return a - b
    
    @server.tool()
    def multiply(a: float, b: float) -> float:
        """Multiply two numbers together and return the result."""
        return a * b
    
    @server.tool()
    def divide(a: float, b: float) -> float:
        """
        Divide a by b and return the result.
        
        Raises:
            ValueError: If b is zero
        """
        if b == 0:
            raise ValueError("Cannot divide by zero")
        return a / b
    ```

### -4- ಕ್ಯಾಲ್ಕುಲೇಟರ್ MCP ಸರ್ವರ್‌ನೊಂದಿಗೆ ಏಜೆಂಟ್ ಅನ್ನು ಚಾಲನೆ ಮಾಡಿ

ನಿಮ್ಮ ಏಜೆಂಟ್‌ಗೆ ಈಗ ಸಾಧನಗಳಿವೆ, ಅವುಗಳನ್ನು ಬಳಸುವ ಸಮಯ ಬಂದಿದೆ! ಈ ವಿಭಾಗದಲ್ಲಿ, ನೀವು ಏಜೆಂಟ್‌ಗೆ ಪ್ರಾಂಪ್ಟ್‌ಗಳನ್ನು ಸಲ್ಲಿಸಿ, ಏಜೆಂಟ್ ಕ್ಯಾಲ್ಕುಲೇಟರ್ MCP ಸರ್ವರ್‌ನಿಂದ ಸರಿಯಾದ ಸಾಧನವನ್ನು ಬಳಸುತ್ತಿದೆಯೇ ಎಂದು ಪರೀಕ್ಷಿಸುವಿರಿ.

![Visual Studio Code ಗಾಗಿ AI Toolkit ವಿಸ್ತರಣೆಯಲ್ಲಿನ Calculator Agent ಇಂಟರ್ಫೇಸ್ ಸ್ಕ್ರೀನ್‌ಶಾಟ್. ಎಡ ಪ್ಯಾನೆಲ್‌ನಲ್ಲಿ “Tools” ಅಡಿಯಲ್ಲಿ local-server-calculator_server ಎಂಬ MCP ಸರ್ವರ್ ಸೇರಿಸಲಾಗಿದೆ, ನಾಲ್ಕು ಲಭ್ಯವಿರುವ ಸಾಧನಗಳನ್ನು ತೋರಿಸುತ್ತದೆ: add, subtract, multiply, ಮತ್ತು divide. ನಾಲ್ಕು ಸಾಧನಗಳು ಸಕ್ರಿಯವಾಗಿವೆ ಎಂದು ಬ್ಯಾಡ್ಜ್ ತೋರಿಸುತ್ತದೆ. ಕೆಳಗೆ “Structure output” ವಿಭಾಗ ಕುಗ್ಗಿಸಲಾಗಿದೆ ಮತ್ತು ನೀಲಿ “Run” ಬಟನ್ ಇದೆ. ಬಲ ಪ್ಯಾನೆಲ್‌ನಲ್ಲಿ “Model Response” ಅಡಿಯಲ್ಲಿ ಏಜೆಂಟ್ multiply ಮತ್ತು subtract ಸಾಧನಗಳನ್ನು ಕ್ರಮವಾಗಿ {"a": 3, "b": 25} ಮತ್ತು {"a": 75, "b": 20} ಇನ್ಪುಟ್‌ಗಳೊಂದಿಗೆ ಆಮಂತ್ರಿಸುತ್ತದೆ. ಅಂತಿಮ “Tool Response” 75.0 ಎಂದು ತೋರಿಸಲಾಗಿದೆ. ಕೆಳಗೆ “View Code” ಬಟನ್ ಇದೆ.](../../../../translated_images/aitk-agent-response-with-tools.e7c781869dc8041a25f9903ed4f7e8e0c7e13d7d94f6786a6c51b1e172f56866.kn.png)

ನೀವು ನಿಮ್ಮ ಸ್ಥಳೀಯ ಡೆವ್ ಯಂತ್ರದಲ್ಲಿ MCP ಸರ್ವರ್ ಅನ್ನು **Agent Builder** ಮೂಲಕ MCP ಕ್ಲೈಂಟ್ ಆಗಿ ಚಾಲನೆ ಮಾಡುತ್ತೀರಿ.

1. MCP ಸರ್ವರ್ ಡಿಬಗ್ ಮಾಡಲು `F5` ಒತ್ತಿ. **Agent (Prompt) Builder** ಹೊಸ ಎಡಿಟರ್ ಟ್ಯಾಬ್‌ನಲ್ಲಿ ತೆರೆಯುತ್ತದೆ. ಸರ್ವರ್ ಸ್ಥಿತಿ ಟರ್ಮಿನಲ್‌ನಲ್ಲಿ ಕಾಣಿಸುತ್ತದೆ.
1. **Agent (Prompt) Builder** ನ **User prompt** ಕ್ಷೇತ್ರದಲ್ಲಿ ಈ ಪ್ರಾಂಪ್ಟ್ ನಮೂದಿಸಿ: `I bought 3 items priced at $25 each, and then used a $20 discount. How much did I pay?`

1. ಈ ಕೆಳಗಿನವುಗಳು ಸಂಭವಿಸಬೇಕು:
    - ಏಜೆಂಟ್ ಗಣನೆಗೆ ಸಹಾಯ ಮಾಡಲು **multiply** ಮತ್ತು **subtract** ಸಾಧನಗಳನ್ನು ಆಯ್ಕೆಮಾಡುತ್ತದೆ.

    - **subtract** ಸಾಧನಕ್ಕೆ `a` ಮತ್ತು `b` ಮೌಲ್ಯಗಳನ್ನು ನಿಯೋಜಿಸಲಾಗುತ್ತದೆ.
    - ಪ್ರತಿ ಸಾಧನದಿಂದ ಪ್ರತಿಕ್ರಿಯೆ **Tool Response** ನಲ್ಲಿ ನೀಡಲಾಗುತ್ತದೆ.
    - ಮಾದರಿಯಿಂದ ಅಂತಿಮ ಔಟ್‌ಪುಟ್ **Model Response** ನಲ್ಲಿ ನೀಡಲಾಗುತ್ತದೆ.
1. ಏಜೆಂಟ್ ಅನ್ನು ಇನ್ನಷ್ಟು ಪರೀಕ್ಷಿಸಲು ಹೆಚ್ಚುವರಿ ಪ್ರಾಂಪ್ಟ್‌ಗಳನ್ನು ಸಲ್ಲಿಸಿ. ನೀವು **User prompt** ಕ್ಷೇತ್ರದಲ್ಲಿ ಪ್ರಾಂಪ್ಟ್ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿ ಬದಲಾಯಿಸಬಹುದು.
1. ಪರೀಕ್ಷೆ ಮುಗಿದ ಮೇಲೆ, ಟರ್ಮಿನಲ್‌ನಲ್ಲಿ **CTRL/CMD+C** ಒತ್ತಿ ಸರ್ವರ್ ನಿಲ್ಲಿಸಿ.

ನಿಮ್ಮ **server.py** ಫೈಲ್‌ಗೆ ಹೆಚ್ಚುವರಿ ಸಾಧನ ಎಂಟ್ರಿಯನ್ನು ಸೇರಿಸಲು ಪ್ರಯತ್ನಿಸಿ (ಉದಾ: ಸಂಖ್ಯೆಯ ವರ್ಗಮೂಲವನ್ನು ಹಿಂತಿರುಗಿಸುವುದು). ನಿಮ್ಮ ಹೊಸ ಸಾಧನ (ಅಥವಾ ಇತ್ತೀಚೆಗೆ ಇದ್ದ ಸಾಧನಗಳು) ಬಳಸಬೇಕಾದ ಪ್ರಾಂಪ್ಟ್‌ಗಳನ್ನು ಸಲ್ಲಿಸಿ. ಹೊಸ ಸಾಧನಗಳನ್ನು ಲೋಡ್ ಮಾಡಲು ಸರ್ವರ್ ಅನ್ನು ಮರುಪ್ರಾರಂಭಿಸುವುದನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ.
