# ಮಾದರಿ ಚಾಲನೆ

## ಅವಲಂಬನೆಗಳನ್ನು ಸ್ಥಾಪಿಸಿ

```bash
npm install
```

## ನಿರ್ಮಿಸಿ

```bash
npm run build
```

## ಸರ್ವರ್ ಪ್ರಾರಂಭಿಸಿ

```bash
npm start
```

ಇದು ಸರ್ವರ್ ಅನ್ನು ಪ್ರಾರಂಭಿಸಬೇಕು

### ಕ್ಲೈಂಟ್ ಪ್ರಾರಂಭಿಸಿ

ಬೇರೆ ಟರ್ಮಿನಲ್‌ನಲ್ಲಿ, ಚಾಲನೆ ಮಾಡಿ:

```bash
node run client
```
