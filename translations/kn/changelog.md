# ಚೇಂಜ್ಲಾಗ್: ಆರಂಭಕರಿಗಾಗಿ MCP ಪಠ್ಯಕ್ರಮ

ಈ ಡಾಕ್ಯುಮೆಂಟ್ ಆಫ್ ಪ್ರಾಮುಖ್ಯ ಬದಲಾವಣೆಗಳನ್ನು ದಾಖಲಿಸುವ ದಾಖಲೆ ಆಗಿದ್ದು, ಮಾದರಿ ಸಂಧರ್ಭ ಪ್ರೋಟೋಕಾಲ್ (MCP) ಆರಂಭಕರಿಗಾಗಿ ಪಠ್ಯಕ್ರಮದಲ್ಲಿ ಮಾಡಿದ ಎಲ್ಲ ಪ್ರಮುಖ ಬದಲಾವಣೆಗಳನ್ನುย้อนಕ್ರಮದಲ್ಲಿ ದಾಖಲಿಸಲಾಗಿದೆ (ಹೊಸ ಬದಲಾವಣೆಗಳು ಮೊದಲಿಗೆ).

## ಫೆಬ್ರವರಿ 5, 2026

  - ಎಲ್ಲಾ ಪ್ರಮುಖ ಹೋಸ್ಟ್‌ಗಳಿಗಾಗಿ JSON ಸಂರಚನಾ ಟೆಂಪ್ಲेट್‌ಗಳು  
  - ಸಂಚಾರ ಪ್ರಕಾರಗಳ তুলನೆ ಟೇಬಲ್ (stdio, SSE/HTTP, ವೆಬ್‌ಸಾಕೆಟ್)  
  - ಸಾಮಾನ್ಯ ಸಂಪರ್ಕ ಸಮಸ್ಯೆಗಳ ನಿವಾರಣೆ  
  - ಹೋಸ್ಟ್ ಸಂರಚನೆಗಾಗಿ ಭದ್ರತಾ ಉತ್ತಮ ಅನುಸಂಧಾನಗಳು

- **13-mcp-inspector/README.md**: MCP ಇನ್ಸ್‌ಪೆಕ್ಟರ್ಿಗಾಗಿ ಹೊಸ ಡಿಬಾಗಿಂಗ್ ಮಾರ್ಗದर्शಿ  
  - ಸ್ಥಾಪನೆ ವಿಧಾನಗಳು (npx, npm ಗ್ಲೋಬಲ್, ಮೂಲದಿಂದ)  
  - stdio ಮತ್ತು HTTP/SSE ಮೂಲಕ ಸರ್ವರ್ಗಳಿಗೆ ಸಂಪರ್ಕ  
  - ಟೆಸ್ಟಿಂಗ್ ಸಾಧನಗಳು, ಸಂಪನ್ಮೂಲಗಳು ಮತ್ತು ಪ್ರಾಂಪ್ಟ್ ವರ್ಕ್ಫ್ಲೋಗಳೇ  
  - VS ಕೋಡ್ ಇಂಟಿಗ್ರೇಶನ್ MCP ಇನ್ಸ್‌ಪೆಕ್ಟರ್ ಜೊತೆಗೆ  
  - ಸಾಮಾನ್ಯ ಡಿಬಾಗಿಂಗ್ ಸಂದರ್ಭಗಳು ಹಾಗೂ ಪರಿಹಾರಗಳು

**ಮಾಡ್ಯೂಲ್ 04 - ಪ್ರಾಯೋಗಿಕ ಅನುಷ್ಠಾನ**

  - ಕ್ಲೈಂಟ್-ಸೈಡ್ ಪೇಜಿನೇಷನ್ ನಿರ್ವಹಣೆ

  - ಕಾರ್ಯಕ್ಷಮತೆ ಸುಧಾರಣಾ ಶಿಫಾರಸುಗಳು

**ಮಾಡ್ಯೂಲ್ 05 - ಮುಂದಿನ ವಿಷಯಗಳು**

## ಜನವರಿ 28, 2026

### MCP ಮಾಹಿತಿಯ 2025-11-25 ಅನುಕೂಲ ಮಾನ್ಯತೆ ಪರಿಶೀಲನೆ

#### ಮೂಲ ಕಲಿಕೆಗಳ ವೃದ್ಧಿ (01-CoreConcepts/)  
- **ಹೊಸ ಕ್ಲೈಂಟ್ ಪ್ರಿಮಿಟಿವ್ - ರೂಟ್ಸ್**: ಫೈಲ್‌ಸಿಸ್ಟಮ್ ಗಡಲಿಗಳನ್ನು ಮತ್ತು ಪ್ರವೇಶ ಅನುಮತಿಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವ ಸೆರ್ವರ್ಗಳಿಗಾಗಿ ಪೂರ್ಣ ದಾಖಲೆ ಸೇರಿಸಲಾಗಿದೆ  
- **ಟೂಲ್ ಅನೋಟೇಶನ್ಸ್**: ಉತ್ತಮ ಟೂಲ್ ಕಾರ್ಯಾಚರಣೆಗೆ `readOnlyHint`, `destructiveHint` ಅನೋಟೇಶನ್‌ಗಳ ದಾಖಲೆ ಸೇರಿಸಲಾಗಿದೆ  
- **ಸ್ಯಾಂಪ್ಲಿಂಗ್‌ನಲ್ಲಿ ಟೂಲ್ ಕರೆ**: ಸ್ಯಾಂಪ್ಲಿಂಗ್ ವಿನಂತಿಗಳಲ್ಲಿ ಮಾದರಿ ಚಾಲಿತ ಟೂಲ್ ಕರೆಗಳಿಗಾಗಿ `tools` ಮತ್ತು `toolChoice` ಪ್ಯಾರಾಮಿ ಟರ್‌ಗಳ ದಾಖಲೆ تازهಗೊಳಿಸಲಾಗಿದೆ  
- **URL ಮೋಡ್ ಪ್ರೇರಣೆ**: ಸರ್ವರ್ ಪ್ರೇರಿತ ಬಾಹ್ಯ ವೆಬ್ ಸಂವಾದದ URL ಆಧಾರಿತ ಪ್ರೇರಣೆಯ ಕುರಿತು ದಾಖಲೆ ಸೇರಿಸಲಾಗಿದೆ  
- **ಕಾರ್ಯಗಳಲ್ಲಿ (ಪ್ರಾಯೋಗಿಕ)**: ದೀರ್ಘಕಾಲಿಕ ಕಾರ್ಯಾಚರಣೆ ಸರ್ಪರಕ್ಷಕರು ಮತ್ತು ವಿಳಂಬ ಫಲಿತಾಂಶ ಪಡೆಯುವ ವ್ಯವಸ್ಥೆಯ ಕುರಿತು ಹೊಸ ವಿಭಾಗ ಸೇರಿಸಲಾಗಿದೆ  
- **ಐಕೋನ್ ಬೆಂಬಲ**: ತಂತ್ರಗಳು, ಸಂಪನ್ಮೂಲಗಳು, ಸಂಪನ್ಮೂಲ ಟೆಂಪ್ಲೇಟುಗಳು ಮತ್ತು ಪ್ರಾಂಪ್ಟ್‌ಗಳು ಈಗ ಐಕಾನ್‌ಗಳನ್ನು ಹೆಚ್ಚುವರಿ ಮೆಟಾಡೇಟಾ ಆಗಿ ಹೊಂದಬಹುದಾಗಿದೆ ಎಂಬುದು ಗಮನಿಸಿದ್ದು

#### ನಿರ್ದಿಷ್ಟ ಪಡೆಯುವಿಕೆ ಪರಿಶೀಲನೆ  
- **ಪ್ರೋಟೋಕಾಲ್ ಆವೃತ್ತಿ**: ಎಲ್ಲಾ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ದೇವಸ್ಥಿತ MCP ಮಾದರಿ 2025-11-25 ಅನ್ನು ಉಲ್ಲೇಖಿಸುತ್ತದೆ ಎಂದು ದೃಢಪಡಿಸಲಾಗಿದೆ  
- **ವಾಸ್ತುಶಿಲ್ಪ ಹೊಂದಾಣಿಕೆ**: ಎರಡು-ಮಟ್ಟಿನ ವಾಸ್ತುಶಿಲ್ಪ (ಡೇಟಾ ಲೇಯರ್ + ಸಂಚಾರ ಲೇಯರ್) ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಸರಿಯಾಗಿರುವುದು ದೃಢಪಟ್ಟಿದೆ  
- **ಪ್ರಿಮಿಟಿವ್‌ಗಳ ದಾಖಲೆ**: ಸರ್ವರ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳು (ಸಂಪನ್ಮೂಲಗಳು, ಪ್ರಾಂಪ್ಟ್‌ಗಳು, ಉಪಕರಣಗಳು) ಮತ್ತು ಕ್ಲೈಂಟ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳು (ಸ್ಯಾಂಪ್ಲಿಂಗ್, ಪ್ರೇರಣೆ, ಲಾಗಿಂಗ್, ರೂಟ್ಸ್) ಪರಿಶೀಲಿಸಲಾಗಿದೆ  
- **ಸಂಚಾರ ಸಾಧನಗಳು**: STDIO ಮತ್ತು ಸ್ಟ್ರೀಮೇಬಲ್ HTTP ಸಂಚಾರಕ್ಕಾಗಿ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಸರಿಹೊಂದಿಸಿದೆ  
- **ಭದ್ರತಾ ಮಾರ್ಗದರ್ಶನ**: MCP ಭದ್ರತಾ ಉತ್ತಮ ಶಿಕ್ಷೆಗಳೊಂದಿಗೆ ಹೊಂದಾಣಿಕೆ ದೃಢಪಟ್ಟಿದೆ

- **OpenID ಸಂಪರ್ಕ ಪತ್ತೆ**: OIDC ಮೂಲಕ ಪರಿಶೀಲನೆ ಸರ್ವರ್ ಪತ್ತೆಯಾಗಿದೆ  
- **OAuth ಕ್ಲೈಂಟ್ ಐಡಿ ಮೆಟಾಡೇಟಾ ಡಾಕ್ಯುಮೆಂಟ್‌ಗಳು**: ಕ್ಲೈಂಟ್ ನೋಂದಣಿಗಾಗಿ ಶಿಫಾರಸು  
- **JSON ಸ್ಕೀಮಾ 2020-12**: MCP ಸ್ಕೀಮಾ ವ್ಯಾಖ್ಯಾನದ ಡೀಫಾಲ್ಟ್ ಡಯಾಲೆಕ್ಟ್  
- **SDK ತರಬೆತಿ ವ್ಯವಸ್ಥೆ**: SDK ವೈಶಿಷ್ಟ್ಯಗಳ ಬೆಂಬಲ ಮತ್ತು ನಿರ್ವಹಣೆಗೆ ಅಧಿಕೃತ ಅವಶ್ಯಕತೆಗಳು  
- **ಶಾಸನ ವ್ಯವಸ್ಥೆ**: MCP ಆಡಳಿತದಲ್ಲಿ ಕಾರ್ಯಗತ ಬಳಕೆಗಾರ ಗುಂಪುಗಳು ಮತ್ತು ಆಸಕ್ತಿಗಾಗಿ ಗುಂಪುಗಳು

### ಭದ್ರತಾ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಪ್ರಮುಖ تازهಗೊಳಿಸುವಿಕೆ (02-Security/)

#### MCP ಭದ್ರತಾ ಶೃಂಗಸಭ ಕಾರ್ಯಾಗಾರ (ಶರ್ಪಾ) ಇಂಟಿಗ್ರೇಶನ್  
- **ಹೊಸ ಹ್ಯಾಂಡ್ಸ್-ಆನ್ ತರಬೇತಿ ಸಂಪನ್ಮೂಲ**: ಎಲ್ಲಾ ಭದ್ರತಾ ಡಾಕ್ಯುಮೆಂಟೇಶನ್‌ಗಳಲ್ಲಿ [MCP Security Summit Workshop (Sherpa)](https://azure-samples.github.io/sherpa/) ಸಮಗ್ರ ಸಂಯೋಜನೆ  
- **ಅಭಿಯಾನ ಮಾರ್ಗ ವಲಯ**: ಬೇಸ್ ಕ್ಯಾಂಪ್‌ನಿಂದ ಶೃಂಗಸಭದವರೆಗೆ ಪೂರ್ಣ ಶಿಬಿರದಿಂದ ಶಿಬಿರ ಪ್ರಗತಿ ದಾಖಲಿಸಲಾಗಿದೆ  
- **OWASP ಹೊಂದಾಣಿಕೆ**: ಎಲ್ಲಾ ಭದ್ರತಾ ಮಾರ್ಗದರ್ಶನೆಗಳು ಈಗ OWASP MCP ಅಜೂರ್ ಭದ್ರತಾ ಮಾರ್ಗದರ್ಶಿ ಅಪಾಯಗಳೊಂದಿಗೆ ಹೊಂದಿಕೊಂಡಿವೆ

#### OWASP MCP ಟಾಪ್ 10 ಸಂಯೋಜನೆ  
- **ಹೊಸ ವಿಭಾಗ**: MCP ಟಾಪ್ 10 ಸುರಕ್ಷತಾ ಅಪಾಯಗಳ ಕ್ರಮ ಪಟ್ಟಿಗೆ ಅಜೂರ್ ತಡೆಯೊಳಗೊಬ್ಬದರೊಂದಿಗೆ ಮುಖ್ಯ ಭದ್ರತಾ README ಸೇರಿಸಲಾಗಿದೆ  
- **ಅಪಾಯ ಆಧಾರಿತ ಡಾಕ್ಯುಮೆಂಟೇಶನ್**: mcp-security-controls-2025.md ಡಾಕ್ಯುಮೆಂಟ್‌ನಲ್ಲೂ ಎಲ್ಲಾ ಭದ್ರತಾ ಡೊಮೇನ್ಗೆ OWASP MCP ಅಪಾಯ ಉಲ್ಲೇಖಗಳು ಸೇರಿಸಲಾಗಿದೆ  
- **ಸೂತ್ರ ವಾಸ್ತುಶಿಲ್ಪ**: OWASP MCP ಅಜೂರ್ ಭದ್ರತಾ ಮಾರ್ಗದರ್ಶಿ ಸೂತ್ರ ವಾಸ್ತುಶಿಲ್ಪ ಮತ್ತು ಅನುಷ್ಠಾನ ಮಾದರಿಗಳೊಂದಿಗೆ ಲಿಂಕ್ ಮಾಡಲಾಗಿದೆ

#### تازهಗೊಳಿಸಿರುವ ಭದ್ರತಾ ಫೈಲುಗಳು

#### MCP ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು (02-Security/mcp-best-practices.md) - ಸ್ಪೆಕ್ ಆವೃತ್ತಿ تازهಗೊಳಿಸುವಿಕೆ

- **ಮಾನದಂಡ ಹೊಂದಾಣಿಕೆ**: ಎಲ್ಲಾ ಉಲ್ಲೇಖಿತ ಭದ್ರತಾ ನಿಯಮಗಳು ಇತ್ತೀಚಿನವು ಎಂದು ದೃಢಪಡಿಸಲಾಗಿದೆ

- **ಅನುಷ್ಠಾನ ಸಂಪನ್ಮೂಲಗಳು**: ಎಲ್ಲಾ ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳ ಲಿಂಕುಗಳು ಮತ್ತು ಸಂಪನ್ಮೂಲಗಳು ಪರಿಶೀಲಿಸಲಾಗಿದೆ

  - ಡೆವ್‌ಸೆಕ್ಓಪ್ಸ್ ಪೈಪ್ಲೈನ್‌ಗಳು ಮತ್ತು ಮೇಲ್ವಿಚಾರಣಾ ಪರಿಹಾರಗಳು

### ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಗುಣಮಟ್ಟ ಖಚಿತತೆ  
- **ಸ್ಪೆಕ್ ಅನುಕೂಲ ಪರಿಶೀಲನೆ**: ಎಲ್ಲಾ MCP ಭದ್ರತಾ ಅವಶ್ಯಕತೆಗಳು (ಅನಿವಾರ್ಯ/ವಿರೋಧ) ಇತ್ತೀಚಿನ ಸ್ಪೆಕ್‌ಗಳಿಗೆ ಹೊಂದಿವೆ ಎಂದು ಖಚಿತಪಡಿಸಲಾಗಿದೆ  
- **ಸಂಪನ್ಮೂಲ ದಿನಾಂಕ ಸಂರಕ್ಷಣೆ**: ಮೈಕ್ರೋಸಾಫ್ಟ್ ಡಾಕ್ಯುಮೆಂಟೇಶನ್, ಭದ್ರತಾ ಮಾನದಂಡಗಳು ಮತ್ತು ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳ ಹೊರಗಿನ ಲಿಂಕುಗಳನ್ನು ಪರಿಶೀಲಿಸಲಾಗಿದೆ  
- **ಉತ್ತಮ ಅಭ್ಯಾಸಗಳ ವ್ಯಾಪ್ತಿ**: ಪ್ರಾಮಾಣಿಕತೆ, ಅಧಿಕೃತೀಕರಣ, AI-ನಿರ್ದಿಷ್ಟ ಅಪಾಯಗಳು, ಸರಬರಾಜು ಸರಪಳಿ ಭದ್ರತೆ ಮತ್ತು ಎಂಟರ್ಪ್ರೈಸ್ ಮಾದರಿಗಳ ಸಮಗ್ರ ಒಳಗಿರಿಸುವಿಕೆ ದೃಢಪಡಿಸಲಾಗಿದೆ

## ಅಕ್ಟೋಬರ್ 6, 2025

### ಪ್ರಾರಂಭಿಸುವಿಕೆ ವಿಭಾಗ ವಿಸ್ತರಣೆ – ಮುಂದಿನ ಮಟ್ಟದ ಸರ್ವರ್ ಬಳಕೆ ಮತ್ತು ಸರಳ ಪ್ರಾಮಾಣೀಕರಣ

#### ಮುಂದಿನ ಮಟ್ಟದ ಸರ್ವರ್ ಬಳಕೆ (03-GettingStarted/10-advanced)  
- **ಹೊಸ ಅಧ್ಯಾಯ ಸೇರಿಸಲಾಗಿದೆ**: ಸರಾಸರಿ ಮತ್ತು ಕಡಿಮೆ ಮಟ್ಟದ ಸರ್ವರ್ ವಾಸ್ತುಶಿಲ್ಪಗಳನ್ನು ಕವರ್ಕೊಳ್ಳುವ ಮುಂದಿನ ಮಟ್ಟದ MCP ಸರ್ವರ್ ಬಳಕೆಯ ಸಮಗ್ರ ಮಾರ್ಗದರ್ಶಿ ಪರಿಚಯಿಸಲಾಗಿದೆ  
  - **ಸಾಮಾನ್ಯ ವಿರುದ್ಧ ಕಡಿಮೆ ಮಟ್ಟದ ಸರ್ವರ್**: ಪೈಥಾನ್ ಮತ್ತು ಟೈಪ್‌ಸ್ಕ್ರಿಪ್ಟ್ ಸಹಿತ ಎರಡರಿಗಾಗಿ ವಿವರವಾದ ಹೋಲಿಕೆಯು ಮತ್ತು ಕೋಡ್ ಉದಾಹರಣೆಗಳು  
  - **ಹ್ಯಾಂಡ್ಲರ್ ಆಧಾರಿತ ವಿನ್ಯಾಸ**: ವಿಶාල, ನಿಬಂಧನದ ಸರ್ವರ್ ಅನುಷ್ಠಾನಗಳಿಗೆ ಟೂಲ್/ಸಂಪನ್ಮೂಲ/ಪ್ರಾಂಪ್ಟ್ ನಿರ್ವಹಣೆಯ ವಿವರಣೆ  
  - **ಪ್ರಾಯೋಗಿಕ ಮಾದರಿಗಳು**: ನಿಜದ ಜಗತ್ತಿನ ಪರಿಸ್ಥಿತಿಗಳಲ್ಲಿ ಕಡಿಮೆ ಮಟ್ಟದ ಸರ್ವರ್ ಮಾದರಿಗಳು ಅಭಿವೃದ್ಧಿಯ ವೈಶಿಷ್ಟ್ಯತೆಗಳಿಗೆ ಹೇಗೆ ಸಹಾಯ ಮಾಡುತ್ತವೆ ಎಂಬ ವಿವರಣೆ

#### ಸರಳ ಪ್ರಾಮಾಣೀಕರಣ (03-GettingStarted/11-simple-auth)  
- **ಹೊಸ ಅಧ್ಯಾಯ ಸೇರಿಸಲಾಗಿದೆ**: MCP ಸರ್ವರ್‌ಗಳಲ್ಲಿ ಸರಳ ಪ್ರಾಮಾಣೀಕರಣ ಅನುಷ್ಠಾನದ ಹಂತಬಂಧವಾದ ಮಾರ್ಗದರ್ಶಿ  
  - **ಅಧಿಕೃತ ತತ್ವಗಳು**: ಪ್ರಾಮಾಣೀಕರಣ ಮತ್ತು ಅಧಿಕೃತೀಕರಣದ ಸ್ಪಷ್ಟ ವಿವರಣೆ ಮತ್ತು ಪ್ರಮಾಣೀಕರಣ ನಿರ್ವಹಣೆ  
  - **ಅಡಳಿತ ಪ್ರಾಮಾಣೀಕರಣ ಅನುಷ್ಠಾನ**: ಪೈಥಾನ್ (Starlette) ಮತ್ತು ಟೈಪ್‌ಸ್ಕ್ರಿಪ್ಟ್ (Express) ನಲ್ಲಿ ಮಧ್ಯವರ್ತಿ ಆಧಾರಿತ ಪ್ರಾಮಾಣೀಕರಣ ಮಾದರಿಗಳು, ಕೋಡ್ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ  
  - **ಮುಂದಿನ ಭದ್ರತಾ ಮಾರ್ಗಗಳಲ್ಲಿ ಪ್ರಗತಿಶೀಲತೆ**: ಸರಳ ಪ್ರಾಮಾಣೀಕರಣದಿಂದ ಪ್ರಾರಂಭಿಸಿ OAuth 2.1 ಮತ್ತು RBAC ಗೆ ಮುನ್ನಡೆಸುವ ಮಾರ್ಗದರ್ಶನ, ಮುಂದಿನ ಬದಿಗೆ ಭದ್ರತಾ ಮಾಡ್ಯೂಲ್‌ಗಳ ಉಲ್ಲೇಖಗಳೊಂದಿಗೆ

ಈ ಸೇರಿಸುಗಳು ತನ್ನ ಮೂಲ ತತ್ವಗಳೊಂದಿಗಿನ ಹಲವರ ಕಾಲ್ಗೊಂಡು, ಹೆಚ್ಚು ಬಲವಾದ, ಸುರಕ್ಷಿತ ಮತ್ತು ನಿಬಂಧನಯುಕ್ತ MCP ಸರ್ವರ್ ಅನುಷ್ಠಾನಗಳನ್ನು ನಿರ್ಮಿಸಲು ಪ್ರಾಯೋಗಿಕ, ಕೈಕಲಿಕೆ ಮಾರ್ಗದರ್ಶನ ನೀಡುತ್ತವೆ.

  - **ಮೂಲತತ್ವಗಳು**: ಸಂಪೂರ್ಣ ಪ್ರಸಂಗ ಹಂಚಿಕೆ, ಕ್ರಿಯಾ ನಿರ್ಣಯ ಜಾಗೃತಿ, ಮತ್ತು ಪ್ರಸಂಗ ವಿಂಡೋ ನಿರ್ವಹಣೆ  
  - **MCP ಪ್ರೋಟೋಕಾಲ್ ಹೊಂದಿಕೆ**: MCP ವಿನ್ಯಾಸವು ಪ್ರಸಂಗ ಅಭಿಯંત્રಣ ಸವಾಲುಗಳನ್ನು ಹೇಗೆ ಉದ್ದೇಶಿಸುತ್ತದೆ  
    - ಪ್ರಸಂಗ ವಿಂಡೋ ಮಿತಿಗಳು ಮತ್ತು ಪ್ರಗತಿಶೀಲ ಲೋಡಿಂಗ್ ತಂತ್ರಗಳು  
    - ಪ್ರಾಸಂಗಿಕತೆ ನಿರ್ಧಾರ ಮತ್ತು ಡೈನಾಮಿಕ್ ಪ್ರಸಂಗ ಪರಿಶೋಧನೆ  
    - ಬಹು-ಮೋಡಲ್ ಪ್ರಸಂಗ ನಿರ್ವಹಣೆ ಮತ್ತು ಭದ್ರತೆ ವಿಚಾರಣೆ  
  - **ಅನುಷ್ಠಾನ ವಿಧಾನಗಳು**: ಏಕ-ಥ್ರೆಡ್ ವಿರುದ್ಧ ಬಹು-ಏಜೆಂಟ್ ವಾಸ್ತುಶಿಲ್ಪಗಳು  
    - ಪ್ರಸಂಗ ಚಂಕಿಂಗ್ ಮತ್ತು ಪ್ರಾಥಮ್ಯತೆಯ ತಂತ್ರಗಳು  
    - ಪ್ರಗತಿಶೀಲ ಪ್ರಸಂಗ ಲೋಡಿಂಗ್ ಮತ್ತು ಒತ್ತಡಕಟ್ ತಂತ್ರಗಳು  
    - સ્તರದ ಪ್ರಸಂಗ ವಿಧಾನಗಳು ಮತ್ತು ಪರಿಶೋಧನೆ ಸುಧಾರಣೆ  
  - **ಮಾಪನ ಫ್ರೇಮ್ವರ್ಕ್**: ಪ್ರಸಂಗ ಪರಿಣಾಮಕಾರಿತ್ವ ಮೌಲ್ಯಮಾಪನಕ್ಕೆ ಹೊರಗೊಳ್ಳುತ್ತಿರುವ ಮೆಟ್ರಿಕ್‌ಗಳು  
    - ಇನ್ಪುಟ್ ಕಾರ್ಯಕ್ಷಮತೆ, ಕಾರ್ಯಕ್ಷಮತೆ, ಗುಣಮಟ್ಟ, ಬಳಕೆದಾರ ಅನುಭವ ವಿಚಾರಣೆ  
    - ಪ್ರಸಂಗ ಸುಧಾರಣೆಗೆ ಪ್ರಯೋಗಾತ್ಮಕ ವಿಧಾನಗಳು  
    - ವಿಫಲತೆ ವಿಶ್ಲೇಷಣೆ ಮತ್ತು ಸುಧಾರಣಾ ವಿಧಾನಶಾಸ್ತ್ರ  

#### ಪಠ್ಯಕ್ರಮ ನಾವಿಗೇಶನ್ ನವೀಕರಣಗಳು (README.md)  
- **ಸುಧಾರಿತ ಮಾಯಾಜಾಲ ರಚನೆ**: ಹೊಸ ಅಗ್ರಗಣ್ಯ ವಿಷಯಗಳನ್ನು ಸೇರಿಸುವ ರೀತಿಯಲ್ಲಿ ಪಠ್ಯಕ್ರಮ ಟೇಬಲ್ ನವೀಕರಿಸಲಾಗಿದೆ  
  - ಪ್ರಸಂಗ ಅಭಿಯಂತ್ರಣ (5.14) ಮತ್ತು ಕಸ್ಟಮ್ ಸಂಚಾರಣ (5.15) ಎಂಟ್ರಿಗಳು ಸೇರಿಸಲಾಗಿದೆ  
  - ಎಲ್ಲಾ ಮಾಡ್ಯೂಲ್‌ಗಳಲ್ಲಿ ಸ್ಥಿರ ಸ್ವರೂಪ ಮತ್ತು ನಾವಿಗೇಶನ್ ಲಿಂಕ್‌ಗಳು  
  - ಈಗಿರುವ ವಿಷಯ ವ್ಯಾಪ್ತಿಗೆ ಅನುಗುಣವಾಗಿ ವಿವರಣೆಗಳನ್ನು ನವೀಕರಿಸಲಾಗಿದೆ

#### MCP ಭದ್ರತೆ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು (02-Security/) - ಸಂಪೂರ್ಣ ಆಧುನೀಕರಣ  
- **MCP-SECURITY-BEST-PRACTICES-2025.md**: MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-06-18 ಜೊತೆಗೆ ಸಂಪೂರ್ಣ ಪುನರ್ ಬರವಣಿಗೆ  
  - **ತೀವ್ರ ಅಗತ್ಯತೆಗಳು**: ಅಧಿಕೃತ ನಿರ್ದಿಷ್ಟತೆಗಳಿಂದ ಸ್ಪಷ್ಟ MUST/MUST NOT ಅಗತ್ಯತೆಗಳು ಜೋಡಣೆ ಮತ್ತು ದೃಶ್ಯ ಸೂಚಕರು  
  - **12 ಮುಖ್ಯ ಭದ್ರತೆ ಅಭ್ಯಾಸಗಳು**: 15-ಐಟಂ ಪಟ್ಟಿ ಇಳಿಕೆ ಮಾಡಿ ಸಮಗ್ರ ಭದ್ರತೆ ಕ್ಷೇತ್ರಗಳಿಗೆ ನಿರ್ಮಾಣ  
    - ಟೋಕನ್ ಭದ್ರತೆ & ದೃಢೀಕರಣ, ಬಾಹ್ಯ ಗುರುತು ಪೂರೈಕೆದಾರ ಏಕೀಕರಣ  
    - ಸೆಷನ್ ನಿರ್ವಹಣೆ & ಸಂಚರಣೆ ಭದ್ರತೆ, ಕ್ರಿಪ್ಟೋಗ್ರಾಫಿಕ್ ಅವಶ್ಯಕತೆಗಳೊಂದಿಗೆ  
    - AI-ವಿಶೇಷ ಥ್ರೆಟ್ ರಕ್ಷಣೆ Microsoft Prompt Shields ಏಕೀಕರಣ  
    - ಪ್ರವೇಶ ನಿಯಂತ್ರಣ & ಅನುಮತಿಗಳು, ಕನಿಷ್ಟ ವಿಶೇಷಣ ತತ್ವ  
    - ವಿಷಯ ಸುರಕ್ಷತೆ & ಮೇಲ್ವಿಚಾರಣೆ, ಆಜೂರ್ ವಿಷಯ ಭದ್ರತೆ ಏಕೀಕರಣ  
    - ಸರಬರಾಜು ಸರಪಳಿ ಭದ್ರತೆ, ಸಮಗ್ರ ಘಟಕ ಪರಿಶೀಲನೆ  
    - OAuth ಭದ್ರತೆ & ಗೊಂದಲದ ಡೆಪ್ಯೂ ರಕ್ಷಣೆ PKCE ಅನುಷ್ಠಾನ  
    - ಘಟನೆ ಪ್ರತಿಕ್ರಿಯೆ & ಮರುಸ್ಥಾಪನೆ ಸ್ವಯಂಕ್ರಿಯಾತ್ಮಕ ಸಾಮರ್ಥ್ಯಗಳು  
    - ಅನುಕೂಲತೆ & ಆಡಳಿತ ನಿಯಮಗಳ ಅನುಗುಣತೆ  
    - ಉನ್ನತ ಭದ್ರತೆ ನಿಯಂತ್ರಣೆಗಳು, ಶೂನ್ಯ ನಂಬಿಕೆ ವಾಸ್ತುಶಿಲ್ಪ  
    - ಮೈಸ್ಕ್ರೋಸಾಫ್ಟ್ ಭದ್ರತೆ ಪರಿಸರ ಏಕೀಕರಣ ಸಮಗ್ರ ಮಾರ್ಗಸೂಚಿಗಳು  
    - ನಿರಂತರ ಭದ್ರತಾ ಅಭಿವೃದ್ಧಿ ಹೊಂದಾಣಿಕೆ  
  - **Microsoft ಭದ್ರತಾ ಪರಿಹಾರಗಳು**: Prompt Shields, ಆಜೂರ್ ವಿಷಯ ಭದ್ರತೆ, Entra ID, GitHub అడ్వಾನ್ಸ್‌ಡ್ ಭದ್ರತೆ ಏಕೀಕರಣ ಸುಧಾರಣೆಗಳು  
  - **ಅನುಷ್ಠಾನ ಸಂಪನ್ಮೂಲಗಳು**: ಅಧಿಕೃತ MCP ಡಾಕ್ಯುಮೆಂಟೇಶನ್, Microsoft ಭದ್ರತಾ ಪರಿಹಾರಗಳು, ಭದ್ರತಾ ಮಾನದಂಡಗಳು, ಅನುಷ್ಠಾನ ಮಾದರಿ ಸಂಗ್ರಹಗಳ ವರ್ಗೀಕರಣ  

#### ಉನ್ನತ ಭದ್ರತೆ ನಿಯಂತ್ರಣೆಗಳು (02-Security/) - ಎಂಟರ್ಪ್ರೈಸ್ ಅನುಷ್ಠಾನ  
- **MCP-SECURITY-CONTROLS-2025.md**: ಎಂಟರ್ಪ್ರೈಸ್-ಗ್ರೇಡ್ ಭದ್ರತೆ ಫ್ರೇಮ್ವರ್ಕ್ ಮೂಲಕ ಸಂಪೂರ್ಣ ಪುನರ್‌ಲೆಖನ  
  - **9 ಸಮಗ್ರ ಭದ್ರತೆ ಕ್ಷೇತ್ರಗಳು**: ಮೂಲ ನಿಯಂತ್ರಣಗಳಿಗಿಂತ ವಿಸ್ತಾರಗೊಳಿಸಿದ ಎಂಟರ್ಪ್ರೈಸ್ ಫ್ರೇಮ್ವರ್ಕ್  
    - ಎಡ್ವಾನ್ಸ್ಡ್ ದೃಢೀಕರಣ & ಪ್ರಾಧಿಕಾರ, Microsoft Entra ID ಏಕೀಕರಣ  
    - ಟೋಕನ್ ಭದ್ರತೆ & ಆಂಟಿ-ಪಾಸ್‌ಥ್ರೂ ನಿಯಂತ್ರಣಗಳು ಸಕಾಲಿಕ ಪರಿಶೀಲನೆ ಸೇರಿ  
    - ಸೆಷನ್ ಭದ್ರತೆ ನಿಯಂತ್ರಣೆಗಳು, ಹೈಜೆಕಿಂಗ್ ತಡೆ  
    - AI-ವಿಶೇಷ ಭದ್ರತೆ ನಿಯಂತ್ರಣೆಗಳು, ಪ್ರಾಂಪ್ಟ್ ಇಂಜಕ್ಷನ್ ಮತ್ತು ಟೂಲ್ ವಿಷಕಾರಕತೆಯ ತಡೆ  
    - ಗೊಂದಲದ ಡೆಪ್ಯೂ ಆಕ್ರಮಣ ತಡೆ, OAuth ಪ್ರಾಕ್ಸಿ ಭದ್ರತೆ  
    - ಉಪಕರಣ ಕಾರ್ಯಾಚರಣೆ ಭದ್ರತೆ, ಸ್ಯಾಂಡ್‌ಬಾಕ್ಸಿಂಗ್ ಮತ್ತು ಐಸೊಲೇಷನ್  
    - ಸರಬರಾಜು ಸರಪಳಿ ಭದ್ರತೆ ನಿಯಂತ್ರಣೆಗಳು, ಅವಲಂಬನೆ ಪರಿಶೀಲನೆ  
    - ಮೇಲ್ವಿಚಾರಣೆ & ಪತ್ತೆ ನಿಯಂತ್ರಣೆಗಳು SIEM ಏಕೀಕರಣದೊಂದಿಗೆ  
    - ಘಟನಾ ಪ್ರತಿಕ್ರಿಯೆ & ಮರುಸ್ಥಾಪನೆ ಸ್ವಯಂಕ್ರಿಯಾತ್ಮಕ ಸಾಮರ್ಥ್ಯಗಳು  
  - **ಅನುಷ್ಠಾನ ಉದಾಹರಣೆಗಳು**: ವಿವರವಾದ YAML ಸಂರಚನಾ ಘಟಕಗಳು ಮತ್ತು ಕೋಡ್ ಉದಾಹರಣೆಗಳು ಸೇರಿಸಲಾಗಿದೆ  
  - **Microsoft ಪರಿಹಾರಗಳ ಏಕೀಕರಣ**: ಆಜೂರ್ ಭದ್ರತಾ ಸೇವೆಗಳು, GitHub ಅಡ್ವಾನ್ಸ್‌ಡ್ ಭದ್ರತೆ, ಎಂಟರ್ಪ್ರೈಸ್ ಗುರುತು ನಿರ್ವಾಹಣೆಯ ಸಮಗ್ರ ಆವರಣೆ  

#### ಅಗ್ರಗಣ್ಯ ವಿಷಯ ಭದ್ರತೆ (05-AdvancedTopics/mcp-security/) - ಉತ್ಪಾದನಾ-ತಯಾರ ಅನುಷ್ಠಾನ

  - **ಎಂಟರ್ಪ್ರೈಸ್ ಭದ್ರತೆ ಏಕೀಕರಣ**: ಆಜೂರ್ ಅಪ್ಲಿಕೇಶನ್ ಇನ್‌ಸೈಟ್ಸ್ ಮೇಲ್ವಿಚಾರಣೆ, ಧಮಕಿ ಪತ್ತೆ ಪೈಪ್‌ಲೈನ್‌ಗಳು, ಸರಬರಾಜು ಸರಪಳಿ ಭದ್ರತೆ  
  - **ಅನುಷ್ಠಾನ ಪರಿಶೀಲನಾ ಪಟ್ಟಿಕೆ**: ಅಗತ್ಯ ಮತ್ತು ಶಿಫಾರಸು ಮಾಡಿದ ಭದ್ರತಾ ನಿಯಂತ್ರಣಗಳ ಸ್ಪಷ್ಟ ಅಗತ್ಯ ಹಾಗೂ Microsoft ಭದ್ರತೆ ಪರಿಸರ ಲಾಭಗಳು  

### ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಗುಣಮಟ್ಟ & ಮಾನದಂಡ ಹೊಂದಿಕೆ  
- **ನಿರ್ದಿಷ್ಟತೆ ಉಲ್ಲೇಖಗಳು**: ಎಲ್ಲಾ ಉಲ್ಲೇಖಗಳನ್ನು MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-06-18 ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ  
- **Microsoft ಭದ್ರತೆ ಪರಿಸರ**: ಎಲ್ಲಾ ಭದ್ರತಾ ಡಾಕ್ಯುಮೆಂಟೇಶನ್‌ನಲ್ಲಿ ಏಕೀಕರಣ ಮಾರ್ಗಸೂಚಿಗಳ ಸುಧಾರಣೆ  
- **ಕಾರ್ಯಾತ್ಮಕ ಅನುಷ್ಠಾನ**: .NET, ಜಾವಾ, ಪೈಥಾನ್‌ನಲ್ಲಿ ವಿಸ್ತೃತ ಕೋಡ್ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ ಎಂಟರ್ಪ್ರೈಸ್ ಮಾದರಿಗಳು  
- **ಸಂಪನ್ಮೂಲ ಸಂಘಟನೆ**: ಅಧಿಕೃತ ಡಾಕ್ಯುಮೆಂಟೇಶನ್, ಭದ್ರತಾ ಮಾನದಂಡಗಳು ಮತ್ತು ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳು ಸಮಗ್ರ ವರ್ಗೀಕರಣ  
- **ದೃಶ್ಯ ಸೂಚಕರು**: ಅಗತ್ಯ ಮತ್ತು ಶಿಫಾರಸು ಮಾಡಿದ ಅಭ್ಯಾಸಗಳ ಸ್ಪಷ್ಟ ಗುರುತುಗಳೊಂದಿಗೆ  

#### ಕೋರ್ ಸಂಶೋಧನೆಗಳು (01-CoreConcepts/) - ಸಂಪೂರ್ಣ ಆಧುನೀಕರಣ  
- **ಪ್ರೋಟೋಕಾಲ್ ಸಂಸ್ಕರಣೆ ನವೀಕರಣ**: MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-06-18 ಹಾಗೂ ದಿನಾಂಕ ಆಧಾರಿತ ಸಂಸ್ಕರಣೆಯ (YYYY-MM-DD ಪ್ರರೂಪ) ಉಲ್ಲೇಖ  
- **ವಾಸ್ತುಶಿಲ್ಪ ಪರಿಷ್ಕರಣೆ**: ಪ್ರಸ್ತುತ MCP ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳನ್ನು ಪ್ರತಿಬಿಂಬಿಸುವಂತೆ ಹೋಸ್ಟ್‌ಗಳು, ಕ್ಲೈಂಟ್‌ಗಳು ಮತ್ತು ಸರ್ವರ್‌ಗಳ ವಿವರಣೆ ವಿಸ್ತರಣೆ
  - AI ಅನ್ವಯprogrammaಗಳಾಗಿ ಸ್ಪಷ್ಟವಾಗಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾದ ಹೋಸ್ಟ್ಗಳು ಈಗ ಬಹು MCP ಕ್ಲೈಂಟ್ ಸಂಪರ್ಕಗಳನ್ನು ಸಂಯೋಜಿಸುತ್ತವೆ
  - ಕ್ಲೈಂಟ್‌ಗಳನ್ನು ಒಂದು-ಒಂದು ಸರ್ವರ್ ಸಂಪರ್ಕಗಳನ್ನು ಕಾಪಾಡುವ ಪ್ರೋಟೋ ಕಾಲ್ ಸಂಪರ್ಕಕಾರಿ ಎಂದು ವಿವರಿಸಲಾಗಿದೆ
  - ಸರ್ವರ್‌ಗಳಿಗೆ ಸ್ಥಳೀಯ ಮತ್ತು ದೂರದ ನಿಯೋಜನೆ ಪರಿಕಲ್ಪನೆಗಳೊಂದಿಗೆ ವೃದ್ಧಿಸಲಾಗಿದೆ
- **ಮೊದಲಿನ ಪುನರ್ ಸಂಯೋಜನೆ**: ಸರ್ವರ್ ಮತ್ತು ಕ್ಲೈಂಟ್ ಮೂಲಭೂತ ವಿಭಿನ್ನಗಳ ಸಂಪೂರ್ಣ ಪುನರ್ನಿರ್ಮಾಣ
  - ಸರ್ವರ್ ಮೂಲಭೂತಗಳು: ಸಂಪನ್ಮೂಲಗಳು (ಡೇಟಾ ಮೂಲಗಳು), ಪ್ರಾಂಪ್ಟ್‌ಗಳು (ಟೆಂಪ್ಲೇಟುಗಳು), ಸಾಧನಗಳು (ಕಾರ್ಯನಿರ್ವಹಣೆಯ ಕಾರ್ಯಗಳು) ವಿವರಿತ ವಿವರಣೆಗಳೊಂದಿಗೆ ಮತ್ತು ಉದಾಹರಣೆಗಳೊಂದಿಗೆ
  - ಕ್ಲೈಂಟ್ ಮೂಲಭೂತಗಳು: ಸ್ಯಾಂಪ್ಲಿಂಗ್ (LLM ಪೂರ್ಣಗೊಳ್ಳುವಿಕೆ), ಪ್ರೇರಣೆ (ಬಳಕೆದಾರ ಇನ್ಪುಟ್), ಲಾಗಿಂಗ್ (ಡೀಬಗಿಂಗ್/ಮಾನಿಟರಿಂಗ್)
  - ಪ್ರಸ್ತುತ ಪತ್ತೆಗೊಳಿಸುವಿಕೆ (`*/list`), ಹಿಂತೆಗೆದುಕೊಳ್ಳುವಿಕೆ (`*/get`), ಮತ್ತು ಕಾರ್ಯನಿರ್ವಹಣಾ (`*/call`) ವಿಧಾನ ಮಾದರಿಗಳೊಂದಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
- **ಪ್ರೋಟೋಕಾಲ್ ವಾಸ್ತುಶಿಲ್ಪ**: ಎರಡು-ಮಟ್ಟದ ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಯನ್ನು ಪರಿಚಯಿಸಲಾಗಿದೆ
  - ಡೇಟಾ ಲೇಯರ್: JSON-RPC 2.0 ಆಧಾರಿತ ಜೀವನಚಕ್ರ ನಿರ್ವಹಣೆ ಮತ್ತು ಮೂಲಭೂತಗಳೊಂದಿಗೆ
  - ಸಾರಿಗೆ ಲೇಯರ್: STDIO (ಸ್ಥಳೀಯ) ಮತ್ತು Streamable HTTP ಎಸ್‌ಎಸ್‌ಇ (ದೂರದ) ಸಾರಿಗೆ ಯಂತ್ರಗಳು
- **ಭದ್ರತಾ ರೂಪರೇಖೆ**: ಸ್ಪಷ್ಟ ಬಳಕೆದಾರ ಅನುಮತಿ, ದತ್ತಾಂಶ ಗೌಪ್ಯತೆ ರಕ್ಷಣೆ, ಸಾಧನ ಕಾರ್ಯನಿರ್ವಹಣೆಯ ಸುರಕ್ಷತೆ, ಮತ್ತು ಸಾರಿಗೆ ಲೇಯರ್ ಭದ್ರತೆ ಸೇರಿದಂತೆ ಸಂಪೂರ್ಣ ಭದ್ರತಾ ಸಿದ್ಧಾಂತಗಳು
- **ಸಂವಹನ ಮಾದರಿಗಳು**: ಪ್ರೋಟೋಕಾಲ್ ಸಂದೇಶಗಳನ್ನು ಪ್ರಾರಂಭ, ಪತ್ತೆ, ಕಾರ್ಯನಿರ್ವಹಣೆ, ಮತ್ತು ಅಧಿಸೂಚನೆ ಪ್ರವಾಹಗಳನ್ನು ತೋರಿಸಲು ನವೀಕರಿಸಲಾಗಿದೆ
- **ಕೋಡ್ ಉದಾಹರಣೆಗಳು**: ಪ್ರಸ್ತುತ MCP SDK ಮಾದರಿಗಳನ್ನು ಪ್ರತಿಬಿಂಬಿಸುವ ಬಹುಭಾಷಾ ಉದಾಹರಣೆಗಳನ್ನು ನವೀಕರಿಸಲಾಗಿದೆ (.NET, ಜಾವಾ, ಪೈಥಾನ್, ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್)

#### ಭದ್ರತೆ (02-Security/) - ಸಂಪೂರ್ಣ ಭದ್ರತಾ ಪುನರ್ ನಿರ್ಮಾಣ  
- **ಮಾನುಕೋಲಿಸು ಇನ್ನೊಂದು ಷರತ್ತು**: MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-06-18 ಭದ್ರತಾ ಅಗತ್ಯತೆಗಳೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಸಂರ್‌ಮೇಳ
- **ಪ್ರಮಾಣೀಕರಣ ಅಭಿವೃದ್ಧಿ**: ಕಸ್ಟಮ್ OAuth ಸರ್ವರ್‌ಗಳಿಂದ ಬಾಹ್ಯ ಐಡೆಂಟಿಟಿ ಪ್ರೊವೈಡರ್ ನಿಯೋಜನೆ (Microsoft Entra ID) ದಶಾಂಶ ದಾಖಲಾಗಿದೆ
- **AI-ನಿರ್ದಿಷ್ಟ ಬೆದರಿಕೆ ವಿಶ್ಲೇಷಣೆ**: ಆಧುನಿಕ AI ದಾಳಿ ಮಾರ್ಗಗಳ ಸುಧಾರಿತ ವ್ಯಾಪ್ತಿ
  - ವಾಸ್ತವಿಕ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ ವಿವರಿತ ಪ್ರಾಂಪ್ಟ್ ಇಂಜೆಕ್ಷನ್ ದಾಳಿ ಸಂದರ್ಭಗಳು
  - ಸಾಧನ ವಿಷವಿಷಬಂಧ ಮತ್ತು "ರಗ್ ಪುಲ್" ದಾಳಿ ಮಾದರಿಗಳು
  - ಸನ್ನಿವೇಶ ವಿಂಡೋ ವಿಷವಿಷಬಂಧ ಮತ್ತು ಮಾದರಿ ಗೊಂದಲ ದಾಳಿಗಳು
- **Microsoft AI ಭದ್ರತಾ ಪರಿಹಾರಗಳು**: Microsoft ಭದ್ರತಾ ಪರಿಸರದ ಸಂಪೂರ್ಣ ವ್ಯಾಪ್ತಿ
  - ಅಗ್ರಗಣ್ಯ ಪತ್ತೆ, ಸ್ಪೋಟಲೈಟ್, ಮತ್ತು ಡಿಲಿಮಿಟರ್ ತಂತ್ರಗಳಿಂದ AI ಪ್ರಾಂಪ್ಟ್ ಶೀಲ್ಡ್ಗಳು
  - ಅಜ್ಯೂರ್ ವಿಷಯ ಭದ್ರತೆ ಸಂಯೋಜನೆ ಮಾದರಿಗಳು
  - ಸರಬರಾಜು ಸರಣಿ ರಕ್ಷಣೆಗೆ GitHub ಅಗ್ರಭದ್ರತೆ
- **ಅಗ್ರ ಭದ್ರತಾ ನಿಭಂಧನೆಗಳು**: 
  - MCP-ನಿರ್ದಿಷ್ಟ ದಾಳಿ ಸಂದರ್ಭಗಳು ಮತ್ತು ಕ್ರಿಪ್ಟೋಗ್ರಾಫಿಕ್ ಸೆಶನ್ ಐಡಿ ಅಗತ್ಯಗಳೊಂದಿಗೆ ಸೆಶನ್ ಹೈಜೆಕಿಂಗ್ ನಿಖರ ನಿಯಂತ್ರಣಗಳು
  - MCP ಪ್ರಾಕ್ಸಿ ಸಂದರ್ಭಗಳಲ್ಲಿ ಗೊಂದಲಿದ್ದ ಜನಪ್ರತಿನಿಧಿ ಸಮಸ್ಯೆಗಳಿಗೆ ಸ್ಪಷ್ಟ ಅನುಮತಿ ಅಗತ್ಯಗಳೊಂದಿಗೆ ನಿಭಂಧನೆಗಳು
  - ಟೋಕನ್ ಪಾಸ್ಥ್ರೂ ದೋಷರಹಿತ ವಿಲಕ್ಷಣಗಳಿಗೆ ಬಾಧ್ಯತರ ಪರಿಶೀಲನಾ ನಿಯಂತ್ರಣಗಳು
- **ಸರಬರಾಜು ಸರಣಿ ಭದ್ರತೆ**: ಮೂಲ ಮಾದರಿಗಳು, ಎम्बೆಡಿಂಗ್ ಸೇವೆಗಳು, ಸನ್ನಿವೇಶ ಪ್ರೊವೈಡರ್‌ಗಳು, ಮತ್ತು ತೃತೀಯ-ಪಕ್ಷ APIಗಳು ಸೇರಿದಂತೆ AI ಸರಬರಾಜು ಸರಣಿ ವ್ಯಾಪ್ತಿಗೆ ವಿಸ್ತಾರ
- **ಮೂಲಭೂತ ಭದ್ರತೆ**: ಶೂನ್ಯ ನಂಬಿಕೆ ವಾಸ್ತುಶಿಲ್ಪ ಮತ್ತು Microsoft ಭದ್ರತಾ ಪರಿಸರದೊಂದಿಗೆ ಎಂಟರ್ಪ್ರೈಸ್ ಭದ್ರತಾ ಮಾದರಿಗಳ ಸುಧಾರಿತ ಏಕೀಕರಣ
- **ಸಂಪನ್ಮೂಲ ಸಂಘಟನೆ**: ಪ್ರಕಾರಗಳೊಂದಿಗೆ ವರ್ಗೀಕೃತ ಸಂಪೂರ್ಣ ಸಂಪನ್ಮೂಲ ಲಿಂಕ್‌ಗಳು (ಅಧಿಕೃತ ದಸ್ತಾವೇಜುಗಳು, ಮಾನದಂಡಗಳು, ಸಂಶೋಧನೆ, Microsoft ಪರಿಹಾರಗಳು, ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳು)

### ದಸ್ತಾವೇಜು ಗುಣಮಟ್ಟ ಸುಧಾರಣೆಗಳ  
- **ಗುರುತಿಸಲಾದ ಕಲಿಕೆ ಗುರಿಗಳು**: ನಿರ್ದಿಷ್ಟ, ಕೈಗೆಟುಕುವ ಫಲಿತಾಂಶಗಳೊಂದಿಗೆ ಕಲಿಕೆ ಗುರಿಗಳನ್ನು ಸುಧಾರಿಸಲಾಗಿದೆ
- **ಕ್ರಾಸ್-ರೆಫರೆನ್ಸ್‌ಗಳು**: ಸಂಬಂಧಿತ ಭದ್ರತಾ ಮತ್ತು ಮೂಲಭೂತ ಪರಿಕಲ್ಪನೆ ವಿಷಯಗಳ ನಡುವೆ ಲಿಂಕ್‌ಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ
- **ಪ್ರಸ್ತುತ ಮಾಹಿತಿಗಳು**: ಎಲ್ಲಾ ದಿನಾಂಕ ಉಲ್ಲೇಖಗಳು ಮತ್ತು ನಿರ್ದಿಷ್ಟತೆ ಲಿಂಕ್‌ಗಳನ್ನು ಪ್ರಸ್ತುತ ಮಾನದಂಡಗಳಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
- **ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶನ**: ಇಬ್ಬರು ವಿಭಾಗಗಳಲ್ಲಿಯೂ ನಿರ್ದಿಷ್ಟ, ಕೈಗೆಟುಕುವ ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ

- MCP ನಿರ್ದಿಷ್ಟತೆ ಲಿಂಕ್‌ಗಳನ್ನು ಸರಿಯಾದ URLಗಳಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
- ಪಠ್ಯಕ್ರಮ ರಚನೆಯಲ್ಲಿ ಸನ್ನಿವೇಶ ಎಂಜಿನಿಯರಿಂಗ್ ವಿಭಾಗ (5.14) ಸೇರಿಸಲಾಗಿದೆ

### ಅಧ್ಯಯನ ಮಾರ್ಗದರ್ಶಿ ನವೀಕರಣಗಳು  
- ಪ್ರಸ್ತುತ ಸಂಗ್ರಹಣೆಯ ನಿರ್ಮಾಣಕ್ಕೆ ಅನುಗುಣವಾಗಿ ಅಧ್ಯಯನ ಮಾರ್ಗದರ್ಶಿಯನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ಮರುರಚಿಸಲಾಗಿದೆ
- MCP ಕ್ಲೈಂಟ್‌ಗಳು ಮತ್ತು ಸಾಧನಗಳ ಹೊಸ ವಿಭಾಗಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ ಮತ್ತು ಜನಪ್ರಿಯ MCP ಸರ್ವರ್‌ಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ
- ದೃಶ್ಯ ಪಠ್ಯಕ್ರಮ ನಕ್ಷೆಯನ್ನು ಎಲ್ಲಾ ವಿಷಯಗಳಿಗೆ ಸರಿಯಾಗಿ ಪ್ರತಿಬಿಂಬಿಸಲು ನವೀಕರಿಸಲಾಗಿದೆ
- ಎಲ್ಲ ವಿಶೇಷ ವಿಷಯಗಳನ್ನು ಒಳಗೊಂಡಂತೆ ಉನ್ನತ ವಿಷಯಗಳ ವಿವರಣೆಗಳನ್ನು ಹೆಚ್ಚಿಸಲಾಗಿದೆ
- ಕೇಸ್ ಸ್ಟಡೀಸ್ ವಿಭಾಗವನ್ನು ನಿಖರ ಉದಾಹರಣೆಗಳಿಗೆ ತಕ್ಕಂತೆ ನವೀಕರಿಸಲಾಗಿದೆ
- ಈ ಸಂಪೂರ್ಣ ಚೇಂಜ್ ಲಾಗ್ ಸೇರಿಸಲಾಗಿದೆ

### ಸಮುದಾಯ ಕೊಡುಗೆಗಳು (06-CommunityContributions/)  
- ಚಿತ್ರ ತಯಾರಿಕೆಯ MCP ಸರ್ವರ್‌ಗಳ ವಿವರವಾದ ಮಾಹಿತಿ ಸೇರಿಸಲಾಗಿದೆ
- VSCode ನಲ್ಲಿ ಕ್ಲೋದ್ ಬಳಕೆಯ ಸಂಪೂರ್ಣ ವಿಭಾಗ ಸೇರಿಸಲಾಗಿದೆ
- ಕ್ಲ LINE ಟರ್ಮಿನಲ್ ಕ್ಲೈಂಟ್ ಸೆಟ್‌ಅಪ್ ಮತ್ತು ಬಳಕೆ ಸೂಚನೆಗಳು ಸೇರಿಸಲಾಗಿದೆ
- ಎಲ್ಲಾ ಜನಪ್ರಿಯ ಕ್ಲೈಂಟ್ ಆಯ್ಕೆಗಳು ಸಹಿತ MCP ಕ್ಲೈಂಟ್ ವಿಭಾಗವನ್ನು ನವೀಕರಿಸಲಾಗಿದೆ
- ಕೊಡುಗೆ ಉದಾಹರಣೆಗಳನ್ನು ಹೆಚ್ಚು ನಿಖರ ಕೋಡ್ ಮಾದರಿಗಳೊಂದಿಗೆ ಹೆಚ್ಚಿಸಲಾಗಿದೆ

### ಉನ್ನತ ವಿಷಯಗಳು (05-AdvancedTopics/)  
- ಅನುಕ್ರಮಣಿಕ ಮೌಲ್ಯಮಾನಗಳೊಂದಿಗೆ ಎಲ್ಲಾ ವಿಶೇಷ ವಿಷಯ ಫೋಲ್ಡರ್‌ಗಳನ್ನು ವ್ಯವಸ್ಥಿತಗೊಳಿಸಲಾಗಿದೆ
- ಸನ್ನಿವೇಶ ಎಂಜಿನಿಯರಿಂಗ್ ವಸ್ತುಗಳು ಮತ್ತು ಉದಾಹರಣೆಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ
- ಫೌಂಡ್ರಿ ಏಜೆಂಟ್ ಏಕೀಕರಣ ದಸ್ತಾವೇಜು ಸೇರಿಸಲಾಗಿದೆ
- ಎಂಟ್ರಾ ID ಭದ್ರತಾ ಏಕೀಕರಣ ದಸ್ತಾವೇಜನ್ನು ಸುಧಾರಿಸಲಾಗಿದೆ
