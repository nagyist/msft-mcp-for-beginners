# ಬದಲಾವಣೆ ಚರಿತ್ರೆ: MCP for Beginners ಪಠ್ಯಕ್ರಮ

ಈ ದಾಖಲೆ Model Context Protocol (MCP) for Beginners ಪಠ್ಯಕ್ರಮದಲ್ಲಿ ಮಾಡಲಾದ ಎಲ್ಲಾ ಪ್ರಮುಖ ಬದಲಾವಣೆಗಳ ದಾಖಲೆ ಆಗಿದೆ. ಬದಲಾವಣೆಗಳನ್ನು ವಿಪರೀತ ಕಾಲಕ್ರಮದಲ್ಲಿ ದಾಖಲಿಸಲಾಗಿದೆ (ಹೊಸ ಬದಲಾವಣೆಗಳು ಮೊದಲು).

## ಅಕ್ಟೋಬರ್ 6, 2025

### ಪ್ರಾರಂಭಿಸುವ ವಿಭಾಗ ವಿಸ್ತರಣೆ – ಉನ್ನತ ಸರ್ವರ್ ಬಳಕೆ ಮತ್ತು ಸರಳ ಪ್ರಮಾಣೀಕರಣ

#### ಉನ್ನತ ಸರ್ವರ್ ಬಳಕೆ (03-GettingStarted/10-advanced)
- **ಹೊಸ ಅಧ್ಯಾಯ ಸೇರಿಸಲಾಗಿದೆ**: ಉನ್ನತ MCP ಸರ್ವರ್ ಬಳಕೆಯ ಸಮಗ್ರ ಮಾರ್ಗದರ್ಶಿ ಪರಿಚಯಿಸಲಾಗಿದೆ, ಸಾಮಾನ್ಯ ಮತ್ತು ಕಡಿಮೆ ಮಟ್ಟದ ಸರ್ವರ್ ವಾಸ್ತುಶಿಲ್ಪಗಳನ್ನು ಒಳಗೊಂಡಿದೆ.
  - **ಸಾಮಾನ್ಯ vs. ಕಡಿಮೆ ಮಟ್ಟದ ಸರ್ವರ್**: ಎರಡೂ ವಿಧಾನಗಳಿಗಾಗಿ Python ಮತ್ತು TypeScript ನಲ್ಲಿ ವಿವರವಾದ ಹೋಲಿಕೆ ಮತ್ತು ಕೋಡ್ ಉದಾಹರಣೆಗಳು.
  - **ಹ್ಯಾಂಡ್ಲರ್ ಆಧಾರಿತ ವಿನ್ಯಾಸ**: ವಿಸ್ತಾರಗೊಳ್ಳುವ, ಲವಚಿಕ ಸರ್ವರ್ ಅನುಷ್ಠಾನಗಳಿಗೆ ಹ್ಯಾಂಡ್ಲರ್ ಆಧಾರಿತ ಉಪಕರಣ/ಸಂಪನ್ಮೂಲ/ಪ್ರಾಂಪ್ಟ್ ನಿರ್ವಹಣೆಯ ವಿವರಣೆ.
  - **ಪ್ರಾಯೋಗಿಕ ಮಾದರಿಗಳು**: ಉನ್ನತ ವೈಶಿಷ್ಟ್ಯಗಳು ಮತ್ತು ವಾಸ್ತುಶಿಲ್ಪಕ್ಕಾಗಿ ಕಡಿಮೆ ಮಟ್ಟದ ಸರ್ವರ್ ಮಾದರಿಗಳು ಉಪಯುಕ್ತವಾಗುವ ನೈಜ ಜಗತ್ತಿನ ಸಂದರ್ಭಗಳು.

#### ಸರಳ ಪ್ರಮಾಣೀಕರಣ (03-GettingStarted/11-simple-auth)
- **ಹೊಸ ಅಧ್ಯಾಯ ಸೇರಿಸಲಾಗಿದೆ**: MCP ಸರ್ವರ್‌ಗಳಲ್ಲಿ ಸರಳ ಪ್ರಮಾಣೀಕರಣವನ್ನು ಅನುಷ್ಠಾನಗೊಳಿಸುವ ಹಂತ ಹಂತ ಮಾರ್ಗದರ್ಶಿ.
  - **ಪ್ರಮಾಣೀಕರಣ ಸಂಪ್ರದಾಯಗಳು**: ಪ್ರಮಾಣೀಕರಣ ಮತ್ತು ಪ್ರಾಧಿಕಾರಣದ ಸ್ಪಷ್ಟ ವಿವರಣೆ ಮತ್ತು ಪ್ರಮಾಣಪತ್ರ ನಿರ್ವಹಣೆ.
  - **ಮೂಲಭೂತ ಪ್ರಮಾಣೀಕರಣ ಅನುಷ್ಠಾನ**: Python (Starlette) ಮತ್ತು TypeScript (Express) ನಲ್ಲಿ ಮಧ್ಯವರ್ತಿ ಆಧಾರಿತ ಪ್ರಮಾಣೀಕರಣ ಮಾದರಿಗಳು, ಕೋಡ್ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ.
  - **ಉನ್ನತ ಭದ್ರತೆ ಕಡೆಗೆ ಪ್ರಗತಿ**: ಸರಳ ಪ್ರಮಾಣೀಕರಣದಿಂದ ಪ್ರಾರಂಭಿಸಿ OAuth 2.1 ಮತ್ತು RBAC ಗೆ ಮುಂದುವರಿಯುವ ಮಾರ್ಗದರ್ಶನ, ಉನ್ನತ ಭದ್ರತಾ ಘಟಕಗಳಿಗೆ ಉಲ್ಲೇಖಗಳೊಂದಿಗೆ.

ಈ ಸೇರ್ಪಡೆಗಳು ಹೆಚ್ಚು ಬಲಿಷ್ಠ, ಭದ್ರ ಮತ್ತು ಲವಚಿಕ MCP ಸರ್ವರ್ ಅನುಷ್ಠಾನಗಳನ್ನು ನಿರ್ಮಿಸಲು ಪ್ರಾಯೋಗಿಕ, ಕೈಯಲ್ಲಿ ಮಾರ್ಗದರ್ಶನವನ್ನು ಒದಗಿಸುತ್ತವೆ, ಮೂಲಭೂತ ಸಂಪ್ರದಾಯಗಳನ್ನು ಉನ್ನತ ಉತ್ಪಾದನಾ ಮಾದರಿಗಳೊಂದಿಗೆ ಸೇರ್ಪಡೆ ಮಾಡುತ್ತವೆ.

## ಸೆಪ್ಟೆಂಬರ್ 29, 2025

### MCP ಸರ್ವರ್ ಡೇಟಾಬೇಸ್ ಏಕೀಕರಣ ಪ್ರಯೋಗಾಲಯಗಳು - ಸಮಗ್ರ ಕೈಯಲ್ಲಿ ಕಲಿಕೆ ಮಾರ್ಗ

#### 11-MCPServerHandsOnLabs - ಹೊಸ ಸಂಪೂರ್ಣ ಡೇಟಾಬೇಸ್ ಏಕೀಕರಣ ಪಠ್ಯಕ್ರಮ
- **ಸಂಪೂರ್ಣ 13-ಪ್ರಯೋಗಾಲಯ ಕಲಿಕೆ ಮಾರ್ಗ**: PostgreSQL ಡೇಟಾಬೇಸ್ ಏಕೀಕರಣದೊಂದಿಗೆ ಉತ್ಪಾದನಾ-ಸಿದ್ಧ MCP ಸರ್ವರ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವ ಸಮಗ್ರ ಕೈಯಲ್ಲಿ ಪಠ್ಯಕ್ರಮ ಸೇರಿಸಲಾಗಿದೆ
  - **ನೈಜ ಜಗತ್ತಿನ ಅನುಷ್ಠಾನ**: ಎಂಟರ್‌ಪ್ರೈಸ್-ಗ್ರೇಡ್ ಮಾದರಿಗಳನ್ನು ತೋರಿಸುವ Zava Retail ವಿಶ್ಲೇಷಣೆ ಬಳಕೆ ಪ್ರಕರಣ
  - **ಸಂರಚಿತ ಕಲಿಕೆ ಪ್ರಗತಿ**:
    - **ಪ್ರಯೋಗಾಲಯಗಳು 00-03: ಮೂಲಭೂತಗಳು** - ಪರಿಚಯ, ಕೋರ್ ವಾಸ್ತುಶಿಲ್ಪ, ಭದ್ರತೆ ಮತ್ತು ಬಹು-ಭಾಡಿಗೆ, ಪರಿಸರ ಸೆಟ್‌ಅಪ್
    - **ಪ್ರಯೋಗಾಲಯಗಳು 04-06: MCP ಸರ್ವರ್ ನಿರ್ಮಾಣ** - ಡೇಟಾಬೇಸ್ ವಿನ್ಯಾಸ ಮತ್ತು ಸ್ಕೀಮಾ, MCP ಸರ್ವರ್ ಅನುಷ್ಠಾನ, ಉಪಕರಣ ಅಭಿವೃದ್ಧಿ  
    - **ಪ್ರಯೋಗಾಲಯಗಳು 07-09: ಉನ್ನತ ವೈಶಿಷ್ಟ್ಯಗಳು** - ಅರ್ಥಪೂರ್ಣ ಹುಡುಕಾಟ ಏಕೀಕರಣ, ಪರೀಕ್ಷೆ ಮತ್ತು ಡಿಬಗಿಂಗ್, VS ಕೋಡ್ ಏಕೀಕರಣ
    - **ಪ್ರಯೋಗಾಲಯಗಳು 10-12: ಉತ್ಪಾದನೆ ಮತ್ತು ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು** - ನಿಯೋಜನೆ ತಂತ್ರಗಳು, ಮೇಲ್ವಿಚಾರಣೆ ಮತ್ತು ಗಮನಾರ್ಹತೆ, ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು ಮತ್ತು ಆಪ್ಟಿಮೈಜೆಷನ್
  - **ಎಂಟರ್‌ಪ್ರೈಸ್ ತಂತ್ರಜ್ಞಾನಗಳು**: FastMCP ಫ್ರೇಮ್ವರ್ಕ್, pgvector ಜೊತೆಗೆ PostgreSQL, Azure OpenAI ಎಂಬೆಡ್ಡಿಂಗ್ಸ್, Azure ಕಂಟೈನರ್ ಅಪ್ಸ್, ಅಪ್ಲಿಕೇಶನ್ ಇನ್ಸೈಟ್ಸ್
  - **ಉನ್ನತ ವೈಶಿಷ್ಟ್ಯಗಳು**: ಸಾಲು ಮಟ್ಟದ ಭದ್ರತೆ (RLS), ಅರ್ಥಪೂರ್ಣ ಹುಡುಕಾಟ, ಬಹು-ಭಾಡಿಗೆ ಡೇಟಾ ಪ್ರವೇಶ, ವೆಕ್ಟರ್ ಎಂಬೆಡ್ಡಿಂಗ್ಸ್, ನೈಜ-ಸಮಯ ಮೇಲ್ವಿಚಾರಣೆ

#### ಪದಸಂಪದ ಮಾನಕೀಕರಣ - ಘಟಕದಿಂದ ಪ್ರಯೋಗಾಲಯ ಪರಿವರ್ತನೆ
- **ಸಮಗ್ರ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ನವೀಕರಣ**: 11-MCPServerHandsOnLabs ನಲ್ಲಿ ಎಲ್ಲಾ README ಫೈಲ್‌ಗಳನ್ನು "ಘಟಕ" ಬದಲು "ಪ್ರಯೋಗಾಲಯ" ಪದವನ್ನು ಬಳಸುವಂತೆ ಕ್ರಮಬದ್ಧವಾಗಿ ನವೀಕರಿಸಲಾಗಿದೆ
  - **ವಿಭಾಗ ಶೀರ್ಷಿಕೆಗಳು**: ಎಲ್ಲಾ 13 ಪ್ರಯೋಗಾಲಯಗಳಲ್ಲಿ "ಈ ಘಟಕದಲ್ಲಿ ಏನು ಒಳಗೊಂಡಿದೆ" ಅನ್ನು "ಈ ಪ್ರಯೋಗಾಲಯದಲ್ಲಿ ಏನು ಒಳಗೊಂಡಿದೆ" ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
  - **ವಿಷಯ ವಿವರಣೆ**: "ಈ ಘಟಕ ಒದಗಿಸುತ್ತದೆ..." ಅನ್ನು "ಈ ಪ್ರಯೋಗಾಲಯ ಒದಗಿಸುತ್ತದೆ..." ಗೆ ಬದಲಿಸಲಾಗಿದೆ
  - **ಕಲಿಕೆ ಗುರಿಗಳು**: "ಈ ಘಟಕದ ಕೊನೆಯಲ್ಲಿ..." ಅನ್ನು "ಈ ಪ್ರಯೋಗಾಲಯದ ಕೊನೆಯಲ್ಲಿ..." ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
  - **ನಾವಿಗೇಶನ್ ಲಿಂಕ್‌ಗಳು**: ಎಲ್ಲಾ "ಘಟಕ XX:" ಉಲ್ಲೇಖಗಳನ್ನು "ಪ್ರಯೋಗಾಲಯ XX:" ಗೆ ಪರಿವರ್ತಿಸಲಾಗಿದೆ
  - **ಪೂರ್ಣತೆ ಟ್ರ್ಯಾಕಿಂಗ್**: "ಈ ಘಟಕವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ..." ಅನ್ನು "ಈ ಪ್ರಯೋಗಾಲಯವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ನಂತರ..." ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
  - **ತಾಂತ್ರಿಕ ಉಲ್ಲೇಖಗಳನ್ನು ಉಳಿಸಲಾಗಿದೆ**: ಸಂರಚನಾ ಫೈಲ್‌ಗಳಲ್ಲಿ Python ಘಟಕ ಉಲ್ಲೇಖಗಳನ್ನು (ಉದಾ., `"module": "mcp_server.main"`) ಉಳಿಸಲಾಗಿದೆ

#### ಅಧ್ಯಯನ ಮಾರ್ಗದರ್ಶಿ ಸುಧಾರಣೆ (study_guide.md)
- **ದೃಶ್ಯ ಪಠ್ಯಕ್ರಮ ನಕ್ಷೆ**: ಹೊಸ "11. ಡೇಟಾಬೇಸ್ ಏಕೀಕರಣ ಪ್ರಯೋಗಾಲಯಗಳು" ವಿಭಾಗವನ್ನು ಸಮಗ್ರ ಪ್ರಯೋಗಾಲಯ ರಚನೆ ದೃಶ್ಯೀಕರಣದೊಂದಿಗೆ ಸೇರಿಸಲಾಗಿದೆ
- **ರಿಪೊಸಿಟರಿ ರಚನೆ**: ಹತ್ತು ವಿಭಾಗಗಳಿಂದ ಹನ್ನೊಂದು ಮುಖ್ಯ ವಿಭಾಗಗಳಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ, 11-MCPServerHandsOnLabs ವಿವರಗಳೊಂದಿಗೆ
- **ಕಲಿಕೆ ಮಾರ್ಗದರ್ಶನ**: 00-11 ವಿಭಾಗಗಳನ್ನು ಒಳಗೊಂಡಂತೆ ನಾವಿಗೇಶನ್ ಸೂಚನೆಗಳನ್ನು ಸುಧಾರಿಸಲಾಗಿದೆ
- **ತಂತ್ರಜ್ಞಾನ ವ್ಯಾಪ್ತಿ**: FastMCP, PostgreSQL, Azure ಸೇವೆಗಳ ಏಕೀಕರಣ ವಿವರಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ
- **ಕಲಿಕೆ ಫಲಿತಾಂಶಗಳು**: ಉತ್ಪಾದನಾ-ಸಿದ್ಧ ಸರ್ವರ್ ಅಭಿವೃದ್ಧಿ, ಡೇಟಾಬೇಸ್ ಏಕೀಕರಣ ಮಾದರಿಗಳು ಮತ್ತು ಎಂಟರ್‌ಪ್ರೈಸ್ ಭದ್ರತೆ ಮೇಲೆ ಒತ್ತಡ

#### ಮುಖ್ಯ README ರಚನೆ ಸುಧಾರಣೆ
- **ಪ್ರಯೋಗಾಲಯ ಆಧಾರಿತ ಪದಸಂಪದ**: 11-MCPServerHandsOnLabs ನಲ್ಲಿ ಮುಖ್ಯ README.md ಅನ್ನು "ಪ್ರಯೋಗಾಲಯ" ರಚನೆಯನ್ನು ಸತತವಾಗಿ ಬಳಸುವಂತೆ ನವೀಕರಿಸಲಾಗಿದೆ

### ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಗುಣಮಟ್ಟ ಮತ್ತು ಸತತತೆ ಸುಧಾರಣೆಗಳು

- **ಕಲಿಕೆ ಪ್ರಗತಿ**: ಮೂಲಭೂತ ಸಂಪ್ರದಾಯಗಳಿಂದ ಉತ್ಪಾದನಾ ನಿಯೋಜನೆಗೆ ಸ್ಪಷ್ಟ, ಸಂರಚಿತ ಮಾರ್ಗ

  - **AI ಸೆಕ್ಯುರಿಟಿ ಏಕೀಕರಣ**: Microsoft Prompt Shields ಮತ್ತು Azure Content Safety ಅನುಷ್ಠಾನವು ವಿವರವಾದ ಪೈಥಾನ್ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ  
  - **ಅಡ್ವಾನ್ಸ್ಡ್ ಥ್ರೆಟ್ ನಿವಾರಣೆ**: ಸಮಗ್ರ ಅನುಷ್ಠಾನ ಉದಾಹರಣೆಗಳು  
    - PKCE ಮತ್ತು ಬಳಕೆದಾರ ಅನುಮತಿ ಪರಿಶೀಲನೆಯೊಂದಿಗೆ Confused Deputy ದಾಳಿ ತಡೆ  
    - ಪ್ರೇಕ್ಷಕ ಪರಿಶೀಲನೆ ಮತ್ತು ಸುರಕ್ಷಿತ ಟೋಕನ್ ನಿರ್ವಹಣೆಯೊಂದಿಗೆ ಟೋಕನ್ ಪಾಸ್ತ್ರೂ ತಡೆ  
    - ಕ್ರಿಪ್ಟೋಗ್ರಾಫಿಕ್ ಬಾಂಧನ ಮತ್ತು ವರ್ತನೆ ವಿಶ್ಲೇಷಣೆಯೊಂದಿಗೆ ಸೆಷನ್ ಹೈಜ್ಯಾಕಿಂಗ್ ತಡೆ  
  - **ಎಂಟರ್‌ಪ್ರೈಸ್ ಸೆಕ್ಯುರಿಟಿ ಏಕೀಕರಣ**: Azure Application Insights ಮಾನಿಟರಿಂಗ್, ಥ್ರೆಟ್ ಡಿಟೆಕ್ಷನ್ ಪೈಪ್ಲೈನ್ಗಳು ಮತ್ತು ಸರಬರಾಜು ಸರಣಿ ಸೆಕ್ಯುರಿಟಿ  
  - **ಅನುಷ್ಠಾನ ಚೆಕ್‌ಲಿಸ್ಟ್**: ಸ್ಪಷ್ಟ ಅಗತ್ಯ ಮತ್ತು ಶಿಫಾರಸು ಮಾಡಿದ ಸೆಕ್ಯುರಿಟಿ ನಿಯಂತ್ರಣಗಳು Microsoft ಸೆಕ್ಯುರಿಟಿ ಪರಿಸರದ ಲಾಭಗಳೊಂದಿಗೆ  

### ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಗುಣಮಟ್ಟ ಮತ್ತು ಮಾನದಂಡ ಹೊಂದಾಣಿಕೆ  
- **ನಿರ್ದಿಷ್ಟತೆ ಉಲ್ಲೇಖಗಳು**: ಎಲ್ಲಾ ಉಲ್ಲೇಖಗಳನ್ನು ಪ್ರಸ್ತುತ MCP ನಿರ್ದಿಷ್ಟತೆ 2025-06-18 ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ  
- **Microsoft ಸೆಕ್ಯುರಿಟಿ ಪರಿಸರ**: ಎಲ್ಲಾ ಸೆಕ್ಯುರಿಟಿ ಡಾಕ್ಯುಮೆಂಟೇಶನ್‌ನಲ್ಲಿ ಏಕೀಕರಣ ಮಾರ್ಗದರ್ಶನವನ್ನು ವೃದ್ಧಿಪಡಿಸಲಾಗಿದೆ  
- **ಪ್ರಾಯೋಗಿಕ ಅನುಷ್ಠಾನ**: ಎಂಟರ್‌ಪ್ರೈಸ್ ಮಾದರಿಗಳೊಂದಿಗೆ .NET, ಜಾವಾ ಮತ್ತು ಪೈಥಾನ್ ನಲ್ಲಿ ವಿವರವಾದ ಕೋಡ್ ಉದಾಹರಣೆಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ  
- **ಸಂಪನ್ಮೂಲ ಸಂಘಟನೆ**: ಅಧಿಕೃತ ಡಾಕ್ಯುಮೆಂಟೇಶನ್, ಸೆಕ್ಯುರಿಟಿ ಮಾನದಂಡಗಳು ಮತ್ತು ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳ ಸಮಗ್ರ ವರ್ಗೀಕರಣ  
- **ದೃಶ್ಯ ಸೂಚಕಗಳು**: ಅಗತ್ಯವಿರುವ ಅವಶ್ಯಕತೆಗಳು ಮತ್ತು ಶಿಫಾರಸು ಮಾಡಿದ ಅಭ್ಯಾಸಗಳ ಸ್ಪಷ್ಟ ಗುರುತು  

#### ಕೋರ್ ಕನ್ಸೆಪ್ಟ್ಸ್ (01-CoreConcepts/) - ಸಂಪೂರ್ಣ ಆಧುನೀಕರಣ  
- **ಪ್ರೋಟೋಕಾಲ್ ಆವೃತ್ತಿ ನವೀಕರಣ**: ಪ್ರಸ್ತುತ MCP ನಿರ್ದಿಷ್ಟತೆ 2025-06-18 ಗೆ ದಿನಾಂಕ ಆಧಾರಿತ ಆವೃತ್ತಿ (YYYY-MM-DD ಫಾರ್ಮ್ಯಾಟ್) ಉಲ್ಲೇಖವನ್ನು ನವೀಕರಿಸಲಾಗಿದೆ  
- **ವಾಸ್ತುಶಿಲ್ಪ ಪರಿಷ್ಕರಣೆ**: ಪ್ರಸ್ತುತ MCP ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳನ್ನು ಪ್ರತಿಬಿಂಬಿಸುವಂತೆ ಹೋಸ್ಟ್‌ಗಳು, ಕ್ಲೈಂಟ್‌ಗಳು ಮತ್ತು ಸರ್ವರ್‌ಗಳ ವಿವರಣೆಗಳನ್ನು ವೃದ್ಧಿಪಡಿಸಲಾಗಿದೆ  
  - ಹೋಸ್ಟ್‌ಗಳು ಈಗ ಬಹು MCP ಕ್ಲೈಂಟ್ ಸಂಪರ್ಕಗಳನ್ನು ಸಂಯೋಜಿಸುವ AI ಅಪ್ಲಿಕೇಶನ್‌ಗಳಾಗಿ ಸ್ಪಷ್ಟವಾಗಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ  
  - ಕ್ಲೈಂಟ್‌ಗಳು ಪ್ರೋಟೋಕಾಲ್ ಸಂಪರ್ಕಕಾರರಾಗಿ ಒಂದರಿಂದ ಒಂದರ ಸರ್ವರ್ ಸಂಬಂಧಗಳನ್ನು ನಿರ್ವಹಿಸುವಂತೆ ವರ್ಣಿಸಲಾಗಿದೆ  
  - ಸರ್ವರ್‌ಗಳು ಸ್ಥಳೀಯ ಮತ್ತು ದೂರಸ್ಥ ನಿಯೋಜನೆ ದೃಶ್ಯಾವಳಿಗಳೊಂದಿಗೆ ವೃದ್ಧಿಪಡಿಸಲಾಗಿದೆ  
- **ಪ್ರಿಮಿಟಿವ್ ಪುನರ್‌ರಚನೆ**: ಸರ್ವರ್ ಮತ್ತು ಕ್ಲೈಂಟ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳ ಸಂಪೂರ್ಣ ಪುನರ್‌ರಚನೆ  
  - ಸರ್ವರ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳು: ಸಂಪನ್ಮೂಲಗಳು (ಡೇಟಾ ಮೂಲಗಳು), ಪ್ರಾಂಪ್ಟ್‌ಗಳು (ಟೆಂಪ್ಲೇಟುಗಳು), ಉಪಕರಣಗಳು (ಕಾರ್ಯನಿರ್ವಹಣಾ ಫಂಕ್ಷನ್‌ಗಳು) ವಿವರವಾದ ವಿವರಣೆ ಮತ್ತು ಉದಾಹರಣೆಗಳೊಂದಿಗೆ  
  - ಕ್ಲೈಂಟ್ ಪ್ರಿಮಿಟಿವ್‌ಗಳು: ಸ್ಯಾಂಪ್ಲಿಂಗ್ (LLM ಪೂರ್ಣಗೊಳಿಸುವಿಕೆ), ಎಲಿಸಿಟೇಶನ್ (ಬಳಕೆದಾರ ಇನ್‌ಪುಟ್), ಲಾಗಿಂಗ್ (ಡಿಬಗ್/ಮಾನದಂಡ)  
  - ಪ್ರಸ್ತುತ ಡಿಸ್ಕವರಿ (`*/list`), ರಿಟ್ರೀವಲ್ (`*/get`), ಮತ್ತು ಕಾರ್ಯನಿರ್ವಹಣೆ (`*/call`) ವಿಧಾನ ಮಾದರಿಗಳೊಂದಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ  
- **ಪ್ರೋಟೋಕಾಲ್ ವಾಸ್ತುಶಿಲ್ಪ**: ಎರಡು-ಮಟ್ಟದ ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಯನ್ನು ಪರಿಚಯಿಸಲಾಗಿದೆ  
  - ಡೇಟಾ ಲೇಯರ್: JSON-RPC 2.0 ಆಧಾರ ಮತ್ತು ಜೀವನಚಕ್ರ ನಿರ್ವಹಣೆ ಮತ್ತು ಪ್ರಿಮಿಟಿವ್‌ಗಳೊಂದಿಗೆ  
  - ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ಲೇಯರ್: STDIO (ಸ್ಥಳೀಯ) ಮತ್ತು ಸ್ಟ್ರೀಮಬಲ್ HTTP SSE (ದೂರಸ್ಥ) ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ಯಂತ್ರಗಳು  
- **ಸೆಕ್ಯುರಿಟಿ ಫ್ರೇಮ್ವರ್ಕ್**: ಸ್ಪಷ್ಟ ಬಳಕೆದಾರ ಅನುಮತಿ, ಡೇಟಾ ಗೌಪ್ಯತೆ ರಕ್ಷಣೆ, ಉಪಕರಣ ಕಾರ್ಯನಿರ್ವಹಣಾ ಸುರಕ್ಷತೆ ಮತ್ತು ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ಲೇಯರ್ ಸೆಕ್ಯುರಿಟಿ ಸೇರಿದಂತೆ ಸಮಗ್ರ ಸೆಕ್ಯುರಿಟಿ ತತ್ವಗಳು  
- **ಸಂವಹನ ಮಾದರಿಗಳು**: ಪ್ರೋಟೋಕಾಲ್ ಸಂದೇಶಗಳನ್ನು ಪ್ರಾರಂಭ, ಡಿಸ್ಕವರಿ, ಕಾರ್ಯನಿರ್ವಹಣೆ ಮತ್ತು ಸೂಚನೆ ಹರಿವುಗಳನ್ನು ತೋರಿಸಲು ನವೀಕರಿಸಲಾಗಿದೆ  
- **ಕೋಡ್ ಉದಾಹರಣೆಗಳು**: ಪ್ರಸ್ತುತ MCP SDK ಮಾದರಿಗಳನ್ನು ಪ್ರತಿಬಿಂಬಿಸುವ ಬಹುಭಾಷಾ (.NET, ಜಾವಾ, ಪೈಥಾನ್, ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್) ಉದಾಹರಣೆಗಳನ್ನು ನವೀಕರಿಸಲಾಗಿದೆ  

#### ಸೆಕ್ಯುರಿಟಿ (02-Security/) - ಸಮಗ್ರ ಸೆಕ್ಯುರಿಟಿ ಪುನರ್‌ರಚನೆ  
- **ಮಾನದಂಡ ಹೊಂದಾಣಿಕೆ**: MCP ನಿರ್ದಿಷ್ಟತೆ 2025-06-18 ಸೆಕ್ಯುರಿಟಿ ಅವಶ್ಯಕತೆಗಳೊಂದಿಗೆ ಸಂಪೂರ್ಣ ಹೊಂದಾಣಿಕೆ  
- **ಪ್ರಮಾಣೀಕರಣ ಅಭಿವೃದ್ಧಿ**: ಕಸ್ಟಮ್ OAuth ಸರ್ವರ್‌ಗಳಿಂದ ಬಾಹ್ಯ ಗುರುತಿನ ಒದಗಿಸುವವರ ನಿಯೋಜನೆ (Microsoft Entra ID) ಗೆ ಅಭಿವೃದ್ಧಿ ದಾಖಲಿಸಲಾಗಿದೆ  
- **AI-ನಿರ್ದಿಷ್ಟ ಥ್ರೆಟ್ ವಿಶ್ಲೇಷಣೆ**: ಆಧುನಿಕ AI ದಾಳಿ ಮಾರ್ಗಗಳ ವಿಸ್ತೃತ ಆವರಣೆ  
  - ವಾಸ್ತವಿಕ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ ವಿವರವಾದ ಪ್ರಾಂಪ್ಟ್ ಇಂಜೆಕ್ಷನ್ ದಾಳಿ ದೃಶ್ಯಗಳು  
  - ಉಪಕರಣ ವಿಷಕಾರಕತೆ ಯಂತ್ರಗಳು ಮತ್ತು "ರಗ್ ಪುಲ್" ದಾಳಿ ಮಾದರಿಗಳು  
  - ಸಾಂದರ್ಭಿಕ ವಿಂಡೋ ವಿಷಕಾರಕತೆ ಮತ್ತು ಮಾದರಿ ಗೊಂದಲ ದಾಳಿಗಳು  
- **Microsoft AI ಸೆಕ್ಯುರಿಟಿ ಪರಿಹಾರಗಳು**: Microsoft ಸೆಕ್ಯುರಿಟಿ ಪರಿಸರದ ಸಮಗ್ರ ಆವರಣೆ  
  - AI ಪ್ರಾಂಪ್ಟ್ ಶೀಲ್ಡ್‌ಗಳು ಉನ್ನತ ಪತ್ತೆ, ಸ್ಪಾಟ್‌ಲೈಟಿಂಗ್ ಮತ್ತು ಡೆಲಿಮಿಟರ್ ತಂತ್ರಗಳೊಂದಿಗೆ  
  - Azure Content Safety ಏಕೀಕರಣ ಮಾದರಿಗಳು  
  - ಸರಬರಾಜು ಸರಣಿ ರಕ್ಷಣೆಗೆ GitHub ಅಡ್ವಾನ್ಸ್ಡ್ ಸೆಕ್ಯುರಿಟಿ  
- **ಅಡ್ವಾನ್ಸ್ಡ್ ಥ್ರೆಟ್ ನಿವಾರಣೆ**: ವಿವರವಾದ ಸೆಕ್ಯುರಿಟಿ ನಿಯಂತ್ರಣಗಳು  
  - MCP-ನಿರ್ದಿಷ್ಟ ದಾಳಿ ದೃಶ್ಯಗಳೊಂದಿಗೆ ಸೆಷನ್ ಹೈಜ್ಯಾಕಿಂಗ್ ಮತ್ತು ಕ್ರಿಪ್ಟೋಗ್ರಾಫಿಕ್ ಸೆಷನ್ ID ಅವಶ್ಯಕತೆಗಳು  
  - MCP ಪ್ರಾಕ್ಸಿ ದೃಶ್ಯಗಳಲ್ಲಿ Confused Deputy ಸಮಸ್ಯೆಗಳು ಸ್ಪಷ್ಟ ಅನುಮತಿ ಅವಶ್ಯಕತೆಗಳೊಂದಿಗೆ  
  - ಟೋಕನ್ ಪಾಸ್ತ್ರೂ ದುರ್ಬಲತೆಗಳು ಅಗತ್ಯ ಪರಿಶೀಲನೆ ನಿಯಂತ್ರಣಗಳೊಂದಿಗೆ  
- **ಸರಬರಾಜು ಸರಣಿ ಸೆಕ್ಯುರಿಟಿ**: ಫೌಂಡೇಶನ್ ಮಾದರಿಗಳು, ಎम्बೆಡ್ಡಿಂಗ್ ಸೇವೆಗಳು, ಸಾಂದರ್ಭಿಕ ಒದಗಿಸುವವರು ಮತ್ತು ಮೂರನೇ ಪಕ್ಷ API ಗಳನ್ನು ಒಳಗೊಂಡ AI ಸರಬರಾಜು ಸರಣಿ ವಿಸ್ತರಣೆ  
- **ಫೌಂಡೇಶನ್ ಸೆಕ್ಯುರಿಟಿ**: ಶೂನ್ಯ ನಂಬಿಕೆ ವಾಸ್ತುಶಿಲ್ಪ ಮತ್ತು Microsoft ಸೆಕ್ಯುರಿಟಿ ಪರಿಸರ ಸೇರಿದಂತೆ ಎಂಟರ್‌ಪ್ರೈಸ್ ಸೆಕ್ಯುರಿಟಿ ಮಾದರಿಗಳೊಂದಿಗೆ ವೃದ್ಧಿಪಡಿಸಿದ ಏಕೀಕರಣ  
- **ಸಂಪನ್ಮೂಲ ಸಂಘಟನೆ**: ಪ್ರಕಾರಗಳ ಪ್ರಕಾರ ಸಮಗ್ರ ಸಂಪನ್ಮೂಲ ಲಿಂಕ್‌ಗಳನ್ನು ವರ್ಗೀಕರಿಸಲಾಗಿದೆ (ಅಧಿಕೃತ ಡಾಕ್ಸ್, ಮಾನದಂಡಗಳು, ಸಂಶೋಧನೆ, Microsoft ಪರಿಹಾರಗಳು, ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳು)  

### ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಗುಣಮಟ್ಟ ಸುಧಾರಣೆಗಳು  
- **ಸಂರಚಿತ ಅಧ್ಯಯನ ಗುರಿಗಳು**: ನಿರ್ದಿಷ್ಟ, ಕಾರ್ಯನಿರ್ವಹಣೀಯ ಫಲಿತಾಂಶಗಳೊಂದಿಗೆ ಅಧ್ಯಯನ ಗುರಿಗಳನ್ನು ವೃದ್ಧಿಪಡಿಸಲಾಗಿದೆ  
- **ಕ್ರಾಸ್-ರೆಫರೆನ್ಸಸ್**: ಸಂಬಂಧಿತ ಸೆಕ್ಯುರಿಟಿ ಮತ್ತು ಕೋರ್ ಕನ್ಸೆಪ್ಟ್ ವಿಷಯಗಳ ನಡುವೆ ಲಿಂಕ್‌ಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ  
- **ಪ್ರಸ್ತುತ ಮಾಹಿತಿ**: ಎಲ್ಲಾ ದಿನಾಂಕ ಉಲ್ಲೇಖಗಳು ಮತ್ತು ನಿರ್ದಿಷ್ಟತೆ ಲಿಂಕ್‌ಗಳನ್ನು ಪ್ರಸ್ತುತ ಮಾನದಂಡಗಳಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ  
- **ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶನ**: ಎರಡೂ ವಿಭಾಗಗಳಲ್ಲಿ ನಿರ್ದಿಷ್ಟ, ಕಾರ್ಯನಿರ್ವಹಣೀಯ ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶನಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ
