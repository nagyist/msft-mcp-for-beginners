# MCP ಜೊತೆಗೆ ಅಜುರ್ ವಿಷಯ ಸುರಕ್ಷತೆಗೆ ಅನುಷ್ಠಾನಗೊಳಿಸುವಿಕೆ

> **OWASP MCP જોખમ સરણેલું**: [MCP06 - ಸ್ಫೂರ್ತಿದಾಯಕಅರ್ಪಣೆಯ_Contextual Payloads ಮೂಲಕ ಪ್ರಾಂಪ್ಟ್ ಇಂಜೆಕ್ಷನ್](https://microsoft.github.io/mcp-azure-security-guide/mcp/mcp06-prompt-injection/)

ಪ್ರಾಂಪ್ಟ್ ಇಂಜೆಕ್ಷನ್, ಸಲಕರಣೆ ವಿಷಾಕರಣ ಮತ್ತು ಇತರ AI-ನಿರ್ದಿಷ್ಟ ಭದ್ರತಾ ದುರ್ಬಲತೆಗಳಿಗೆ ವಿರುದ್ಧವಾಗಿ MCP ಭದ್ರತೆಯನ್ನು ಬಲಪಡಿಸಲು, ಅಜುರ್ ವಿಷಯ ಸುರಕ್ಷತೆಯನ್ನು ಸಮಾವೇಶ ಮಾಡುವುದು ಅತ್ಯಂತ ಶಿಫಾರಸುಮಾಡಲಾಗಿದೆ. ಈ ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಕೆ [MCP ಭದ್ರತಾ ಶಿಬಿರ ಕಾರ್ಯಾಗಾರ (ಶೆರ್ಪ)](https://azure-samples.github.io/sherpa/) ಶಿಬಿರ 3: ಐ/ಓ ಭದ್ರತೆಗಾಗಿ ಹೊಂದಿಕೊಳ್ಳುತ್ತದೆ.

## MCP ಸರ್ವರ್ ಜೊತೆಗೆ ಸಮಾವೇಶ

ನಿಮ್ಮ MCP ಸರ್ವರ್ ಜೊತೆಗೆ ಅಜುರ್ ವಿಷಯ ಸುರಕ್ಷತೆಯನ್ನು ಹೊಂದಿಸಲು, ನಿಮ್ಮ ವಿನಂತಿ ಪ್ರಕ್ರಿಯೆ ಪೈಪ್ಲೈನ್‌ನಲ್ಲಿ ಮಧ್ಯವರ್ತಿ ರೂಪದಲ್ಲಿ ವಿಷಯ ಸುರಕ್ಷತಾ ಫಿಲ್ಟರ್ ಅನ್ನು ಸೇರಿಸಿ:

1. ಸರ್ವರ್ ಪ್ರಾರಂಭಿಕೆಯ ಸಮಯದಲ್ಲಿ ಫಿಲ್ಟರ್ ಪ್ರಾರಂಭಿಸಿ  
2. ಪ್ರಕ್ರಿಯೆಗೊಳ್ಳುವ ಮೊದಲು ಎಲ್ಲಾ ಬಂದಿರುವ ಸಲಕರಣೆ ವಿನಂತಿಗಳನ್ನು ಪರಿಶೀಲಿಸಿ  
3. ಗ್ರಾಹಕರಿಗೆ ಹಿಂತಿರುಗಿಸುವ ಮೊದಲು ಎಲ್ಲಾ ಹೊರಹೊಮ್ಮುವ ಪ್ರತಿಕ್ರಿಯೆಗಳನ್ನು ಪರಿಶೀಲಿಸಿ  
4. ಸುರಕ್ಷತಾ ಉಲ್ಲಂಘನೆಗಳ ಲಾಗಿಂಗ್ ಮತ್ತು ಎಚ್ಚರಿಕೆ  
5. ವಿಫಲವಾದ ವಿಷಯ ಸುರಕ್ಷತಾ ಪರಿಶೀಲನೆಗಳಿಗೆ ಸೂಕ್ತ ತ್ರುಟಿಹANDLE ಮಾಡುವುದು  

ಇದರಿಂದ ಸಾಮಾಜಿಕ ರಕ್ಷಣೆಯಾಗಿ ಒದಗುತ್ತದೆ:  
- ಪ್ರಾಂಪ್ಟ್ ಇಂಜೆಕ್ಷನ್ ದಾಳಿಗಳು  
- ಸಲಕರಣೆ ವಿಷಾಕರಣ ಪ್ರಯತ್ನಗಳು  
- ಕೆಟ್ಟ ಉದ್ದೇಶಿತ ಇನ್ಪುಟ್‌ಗಳ ಮೂಲಕ ಡೇಟಾ ಹೊರಹಾಕು ತಡೆ  
- ಹಾನಿಕಾರಕ ವಿಷಯಗಳ ಉತ್ಪಾದನೆ  

## ಅಜುರ್ ವಿಷಯ ಸುರಕ್ಷತಾ ಸಮಾವೇಶಕ್ಕೆ ಅತ್ಯುತ್ತಮ ಅಭ್ಯಾಸಗಳು

1. **ಕಸ್ಟಮ್ ಬ್ಲಾಕ್‌ಲಿಸ್ಟ್ಗಳು**: MCP ಇಂಜೆಕ್ಷನ್ ಮಾದರಿಗಳಿಗೆ ವಿಶೇಷವಾಗಿ ತಯಾರಿಸಿದ ಕಸ್ಟಮ್ ಬ್ಲಾಕ್‌ಲಿಸ್ಟ್‌ಗಳ ರಚನೆ  
2. **ತೀವ್ರತೆ ಹೊಂದಿಕೆ**: ನಿಮ್ಮ ನಿರ್ದಿಷ್ಟ ಉಪಯೋಗ ಮತ್ತು ಅನುಭವ ಪ್ರಮಾಣಾನುಸಾರ ತೀವ್ರತೆ ಮಿತಿಗಳನ್ನು ಹೊಂದಿಸಿ  
3. **ಸಮಗ್ರ ವ್ಯಾಪ್ತಿಯು**: ಎಲ್ಲಾ ಇನ್ಪುಟ್‌ಗಳು ಮತ್ತು ಔಟ್‌ಪುಟ್‌ಗಳಿಗೆ ವಿಷಯ ಸುರಕ್ಷತಾ ಪರಿಶೀಲನೆಗಳ ಅನ್ವಯ  
4. **ಕಾರ್ಯಕ್ಷಮತೆ ಪರಿಷ್ಕರಣೆ**: ಮರುಪಾವತಗೊಳ್ಳುವ ವಿಷಯ ಸುರಕ್ಷತಾ ಪರಿಶೀಲನೆಗಳಿಗೆ ಕ್ಯಾಶಿಂಗ್ ಪರಿಗಣನೆ  
5. **ಫಾಲ್ಬ್ಯಾಕ್ ಯಂತ್ರಗಳು**: ವಿಷಯ ಸುರಕ್ಷತಾ ಸೇವೆಗಳು ಲಭ್ಯವಿಲ್ಲದಿದ್ದಾಗ ಸ್ಪಷ್ಟವಾದ ಫಾಲ್ಬ್ಯಾಕ್ ಕಾರ್ಯ ಶೈಲಿಗಳನ್ನು ನಿರ್ಧರಿಸಿ  
6. **ಬಳಕೆದಾರ ಪ್ರತಿಕ್ರಿಯೆ**: ಸುರಕ್ಷತಾ ಸಮಸ್ಯೆಗಳ ಕಾರಣದಿಂದ ವಿಷಯವನ್ನು ತಡೆಯುವಾಗ ಬಳಕೆದಾರರಿಗೆ ಸ್ಪಷ್ಟ ಪ್ರತಿಕ್ರಿಯೆ ನೀಡಿರಿ  
7. **ನಿರ_continuous ಸುಧಾರಣೆ**: ಹುಟ್ಟುತ್ತಿರುವ ಬೆದರಿಕೆಗಳ ಆಧಾರದ ಮೇಲೆ ಬ್ಲಾಕ್‌ಲಿಸ್ಟ್ ಮತ್ತು ಮಾದರಿಗಳನ್ನು ನಿಯಮಿತವಾಗಿ ನವೀಕರಿಸಿ
