# ಅಟುತ್ಯ MCP ಭದ್ರತೆ ಜೊತೆಗೆ ಅಜ್ಯೂರ್ ವಿಷಯ ಭದ್ರತೆ

> **OWASP MCP ಅಪಾಯ ಎದುರಿಸಲಾಗಿದೆ**: [MCP06 - ಪರಿಸರ ಪೇಲೋಡ್‌ಗಳ ಮೂಲಕ ಪ್ರಾಂಪ್ಟ್ ಇಂಜೆಕ್ಷನ್](https://microsoft.github.io/mcp-azure-security-guide/mcp/mcp06-prompt-injection/)

ಅಜ್ಯೂರ್ ವಿಷಯ ಭದ್ರತೆ ಕೆಲವು ಶಕ್ತಿಶಾಲಿ ಯಂತ್ರಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ, ಅದು ನಿಮ್ಮ MCP ಅನುಷ್ಟಾನಗಳನ್ನು ಭದ್ರತೆಗೊಳಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ. ಹಸ್ತಚಾಲಿತ ಅನುಷ್ಟಾನದ ಅನುಭವಕ್ಕಾಗಿ, ನೋಡಿ [MCP ಭದ್ರತೆ ಸಮಿತಿ ಕಾರ್ಯಾಗಾರ (ಶೆರ್ಪಾ)](https://azure-samples.github.io/sherpa/) ಶಿಬಿರ 3: I/O ಭದ್ರತೆ.

## ಪ್ರಾಂಪ್ಟ್ ಕವಚಗಳು

ಮೈಕ್ರೋಸಾಫ್ಟ್‌ನ ಎಐ ಪ್ರಾಂಪ್ಟ್ ಕವಚಗಳು ನೇರ ಮತ್ತು ಪರೋಕ್ಷ ಪ್ರಾಂಪ್ಟ್ ಇಂಜೆಕ್ಷನ್ ದಾಳಿಗಳ ವಿರುದ್ಧ ಬಲವಾದ ರಕ್ಷಣೆ ನೀಡುತ್ತವೆ:

1. **ಅತ್ಯಾಧುನಿಕ ಪತ್ತೆ**: ವಿಷಯದಲ್ಲಿ ಸಮರ್ಪಿತ ದುಷ್ಟ ಸೂಚನೆಗಳನ್ನು ಗುರುತಿಸಲು ಯಂತ್ರ ಅಭ್ಯಾಸವನ್ನು ಬಳಸುತ್ತದೆ.
2. **ಸ್ಪಾಟ್‌ಲೈಟಿಂಗ್**: ಎಂಟ್ರಿ ಪಠ್ಯವನ್ನು ಪರಿವರ್ತನೆ ಮಾಡಿ ಎಐ ವ್ಯವಸ್ಥೆಗಳು ಮಾನ್ಯ ಸೂಚನೆಗಳು ಮತ್ತು ಹೊರಗಿನ ಎಂಟ್ರಿಗಳನ್ನು ವ್ಯತ್ಯಾಸ ಮಾಡಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ.
3. **ಗುರುತು ಚಿಹ್ನೆಗಳು ಮತ್ತು ಡೇಟಾ ಗುರುತುಮಾಡುವಿಕೆ**: ನಂಬಿಗಸ್ಥ ಮತ್ತು ಅಸಂಬಂಧಿತ ಡೇಟಾ ನಡುವೆ ಗಡಿ ಗುರುತಿಸುತ್ತದೆ.
4. **ವಿಷಯ ಭದ್ರತೆ ಏಕೀಕರಣ**: ಅಜ್ಯೂರ್ ಎಐ ವಿಷಯ ಭದ್ರತೆಯೊಂದಿಗೆ ಕೆಲಸ ಮಾಡಿ ಜೈಲ್ಬ್ರೆಕ್ ಪ್ರಯತ್ನಗಳು ಮತ್ತು ಹಾನಿಕರ ವಿಷಯವನ್ನು ಪತ್ತೆಮಾಡುತ್ತದೆ.
5. **ನಿರಂತರ ನವೀಕರಣಗಳು**: ಮೈಕ್ರೋಸಾಫ್ಟ್ ಉದಯೋನ್ಮುಖ ಅಪಾಯಗಳ ವಿರುದ್ಧ ರಕ್ಷಣೆ ಯಂತ್ರಗಳನ್ನು ನಿಯಮಿತವಾಗಿ ನವೀಕರಿಸುತ್ತದೆ.

## MCP ಜೊತೆಗೆ ಅಜ್ಯೂರ್ ವಿಷಯ ಭದ್ರತೆ ಅನುಷ್ಟಾನ

ಈ ವಿಧಾನ ಬಹು-ಮಟ್ಟದ ರಕ್ಷಣೆ ಒದಗಿಸುತ್ತದೆ:
- ಪ್ರಕ್ರಿಯೆಗೊಳಿಸುವ ಮೊದಲು ಇನ್ಪುಟ್‌ಗಳನ್ನು ಪರಿಶೀಲಿಸುವುದು
- ಮರುಪಡೆಯುವುದಕ್ಕೂ ಮುನ್ನ ಔಟ್‌ಪುಟ್‌ಗಳನ್ನು ಪರಿಶೀಲಿಸುವುದು
- ತಿಳಿದುಕೊಂಡ ಹಾನಿಕರ ಮಾದರಿಗಳಿಗಾಗಿ ತಡೆಪಟ್ಟಿಗಳ ಬಳಕೆ
- ಅಜ್ಯೂರ್‌ನ ನಿರಂತರ ನವೀಕರಿಸಲಾದ ವಿಷಯ ಭದ್ರತೆ ಮಾದರಿಗಳನ್ನು ಪ್ರಯೋಜನ ಪಡೆಯುವುದು

## ಅಜ್ಯೂರ್ ವಿಷಯ ಭದ್ರತೆ ಸಂಪನ್ಮೂಲಗಳು

ನಿಮ್ಮ MCP ಸರ್ವರ್‌ಗಳೊಂದಿಗೆ ಅಜ್ಯೂರ್ ವಿಷಯ ಭದ್ರತೆಯ ಅನುಷ್ಟಾನ ಕುರಿತು ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ ಈ ಅಧಿಕೃತ ಸಂಪನ್ಮೂಲಗಳನ್ನು ಪರಿಶೀಲಿಸಿ:

1. [ಅಜ್ಯೂರ್ ಎಐ ವಿಷಯ ಭದ್ರತೆ ಡಾಕ್ಯುಮೆಂಟೇಶನ್](https://learn.microsoft.com/azure/ai-services/content-safety/) - ಅಜ್ಯೂರ್ ವಿಷಯ ಭದ್ರತೆಯ ಅಧಿಕೃತ ಡಾಕ್ಯುಮೆಂಟೇಶನ್.
2. [ಪ್ರಾಂಪ್ಟ್ ಕವಚ ಡಾಕ್ಯುಮೆಂಟೇಶನ್](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/prompt-shield) - ಪ್ರಾಂಪ್ಟ್ ಇಂಜೆಕ್ಷನ್ ದಾಳಿಗಳನ್ನು ತಡೆಯುವ ವಿಧಾನವನ್ನು ತಿಳಿದುಕೊಳ್ಳಿ.
3. [ವಿಷಯ ಭದ್ರತೆ API ಉಲ್ಲೇಖ](https://learn.microsoft.com/rest/api/contentsafety/) - ವಿಷಯ ಭದ್ರತೆ ಅನುಷ್ಟಾನಕ್ಕೆ ವಿವರವಾದ API ಉಲ್ಲೇಖ.
4. [ಕ್ವಿಕ್‌ಸ್ಟಾರ್ಟ್: C# ಜೊತೆಗೆ ಅಜ್ಯೂರ್ ವಿಷಯ ಭದ್ರತೆ](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-csharp) - C# ಬಳಸಿ ತ್ವರಿತ ಅನುಷ್ಟಾನ ಮಾರ್ಗದರ್ಶಿ.
5. [ವಿಷಯ ಭದ್ರತೆ ಕ್ಲೈಂಟ್ ಗ್ರಂಥಾಲಯಗಳು](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-client-libraries-rest-api) - ವಿವಿಧ ಪ್ರೋಗ್ರಾಮಿಂಗ್ ಭಾಷೆಗಳಿಗಾಗಿ ಕ್ಲೈಂಟ್ ಗ್ರಂಥಾಲಯಗಳು.
6. [ಜೈಲ್ಬ್ರೆಕ್ ಪ್ರಯತ್ನಗಳನ್ನು ಪತ್ತೆಮಾಡುವುದು](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/jailbreak-detection) - ಜೈಲ್ಬ್ರೆಕ್ ಪ್ರಯತ್ನಗಳನ್ನು ಪತ್ತೆಮಾಡಲು ಮತ್ತು ತಡೆಯಲು ನಿರ್ದಿಷ್ಟ ಮಾರ್ಗಸೂಚಿಗಳು.
7. [ವಿಷಯ ಭದ್ರತೆ ಅತ್ಯುತ್ತಮ ಅಭ್ಯಾಸಗಳು](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/best-practices) - ವಿಷಯ ಭದ್ರತೆಯನ್ನು ಪರಿಣಾಮಕಾರಿಯಾಗಿ ಅನುಷ್ಟಾನ ಮಾಡಲು ಅತ್ಯುತ್ತಮ ಅಭ್ಯಾಸಗಳು.

ಹೆಚ್ಚಿನ ವಿವರಗಳಿಗಾಗಿ, ನಮ್ಮ [ಅಜ್ಯೂರ್ ವಿಷಯ ಭದ್ರತೆ ಅನುಷ್ಟಾನ ಮಾರ್ಗದರ್ಶಿ](./azure-content-safety-implementation.md) ಅನ್ನು ನೋಡಿ.
