# ಮಾದರಿ ಸಂದರ್ಭ ಪ್ರೋಟೋಕಾಲ್ (MCP) ಪೈಥಾನ್ ಅನುಷ್ಠಾನ

ಈ ರೆಪೊಸಿಟರಿ ಮಾದರಿ ಸಂದರ್ಭ ಪ್ರೋಟೋಕಾಲ್ (MCP) ನ ಪೈಥಾನ್ ಅನುಷ್ಠಾನವನ್ನು ಒಳಗೊಂಡಿದೆ, ಇದು MCP ಮಾನದಂಡವನ್ನು ಬಳಸಿಕೊಂಡು ಸಂವಹನ ಮಾಡುವ ಸರ್ವರ್ ಮತ್ತು ಕ್ಲೈಂಟ್ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಹೇಗೆ ರಚಿಸುವುದನ್ನು ತೋರಿಸುತ್ತದೆ.

## ಅವಲೋಕನ

MCP ಅನುಷ್ಠಾನವು ಎರಡು ಮುಖ್ಯ ಘಟಕಗಳನ್ನು ಹೊಂದಿದೆ:

1. **MCP ಸರ್ವರ್ (`server.py`)** - ಒಂದು ಸರ್ವರ್ ಇದು:
   - **ಉಪಕರಣಗಳು**: ದೂರದಿಂದ ಕರೆಮಾಡಬಹುದಾದ ಕಾರ್ಯಗಳು
   - **ಸಂಪನ್ಮೂಲಗಳು**: ಪಡೆಯಬಹುದಾದ ಡೇಟಾ
   - **ಪ್ರಾಂಪ್ಟ್‌ಗಳು**: ಭಾಷಾ ಮಾದರಿಗಳಿಗಾಗಿ ಪ್ರಾಂಪ್ಟ್‌ಗಳನ್ನು ರಚಿಸುವ ಟೆಂಪ್ಲೇಟುಗಳು

2. **MCP ಕ್ಲೈಂಟ್ (`client.py`)** - ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕ ಹೊಂದಿ ಅದರ ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ಬಳಸುವ ಕ್ಲೈಂಟ್ ಅಪ್ಲಿಕೇಶನ್

## ವೈಶಿಷ್ಟ್ಯಗಳು

ಈ ಅನುಷ್ಠಾನವು ಕೆಲವು ಪ್ರಮುಖ MCP ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ತೋರಿಸುತ್ತದೆ:

### ಉಪಕರಣಗಳು
- `completion` - AI ಮಾದರಿಗಳಿಂದ ಪಠ್ಯ ಪೂರ್ಣಗೊಳಿಸುವಿಕೆಗಳನ್ನು ರಚಿಸುತ್ತದೆ (ಅನೂಕೃತ)
- `add` - ಎರಡು ಸಂಖ್ಯೆಗಳ ಮೊತ್ತವನ್ನು ಸೇರಿಸುವ ಸರಳ ಕ್ಯಾಲ್ಕ್ಯುಲೇಟರ್

### ಸಂಪನ್ಮೂಲಗಳು
- `models://` - ಲಭ್ಯವಿರುವ AI ಮಾದರಿಗಳ ಬಗ್ಗೆ ಮಾಹಿತಿ ನೀಡುತ್ತದೆ
- `greeting://{name}` - ನೀಡಲಾದ ಹೆಸರಿಗಾಗಿ ವೈಯಕ್ತಿಕೃತ ಸ್ವಾಗತವನ್ನು ನೀಡುತ್ತದೆ

### ಪ್ರಾಂಪ್ಟ್‌ಗಳು
- `review_code` - ಕೋಡ್ ಪರಿಶೀಲನೆಗಾಗಿ ಪ್ರಾಂಪ್ಟ್ ರಚಿಸುತ್ತದೆ

## ಸ್ಥಾಪನೆ

ಈ MCP ಅನುಷ್ಠಾನವನ್ನು ಬಳಸಲು, ಅಗತ್ಯವಿರುವ ಪ್ಯಾಕೇಜ್‌ಗಳನ್ನು ಸ್ಥಾಪಿಸಿ:

```powershell
pip install mcp-server mcp-client
```

## ಸರ್ವರ್ ಮತ್ತು ಕ್ಲೈಂಟ್ ಚಾಲನೆ

### ಸರ್ವರ್ ಪ್ರಾರಂಭಿಸುವುದು

ಒಂದು ಟರ್ಮಿನಲ್ ವಿಂಡೋದಲ್ಲಿ ಸರ್ವರ್ ಅನ್ನು ಚಾಲನೆ ಮಾಡಿ:

```powershell
python server.py
```

MCP CLI ಬಳಸಿ ಅಭಿವೃದ್ಧಿ ಮೋಡ್‌ನಲ್ಲಿ ಸರ್ವರ್ ಅನ್ನು ಕೂಡ ಚಾಲನೆ ಮಾಡಬಹುದು:

```powershell
mcp dev server.py
```

ಅಥವಾ Claude ಡೆಸ್ಕ್‌ಟಾಪ್‌ನಲ್ಲಿ ಸ್ಥಾಪಿಸಬಹುದು (ಲಭ್ಯವಿದ್ದರೆ):

```powershell
mcp install server.py
```

### ಕ್ಲೈಂಟ್ ಚಾಲನೆ

ಮತ್ತೊಂದು ಟರ್ಮಿನಲ್ ವಿಂಡೋದಲ್ಲಿ ಕ್ಲೈಂಟ್ ಅನ್ನು ಚಾಲನೆ ಮಾಡಿ:

```powershell
python client.py
```

ಇದು ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕ ಹೊಂದಿ ಲಭ್ಯವಿರುವ ಎಲ್ಲಾ ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ.

### ಕ್ಲೈಂಟ್ ಬಳಕೆ

ಕ್ಲೈಂಟ್ (`client.py`) ಎಲ್ಲಾ MCP ಸಾಮರ್ಥ್ಯಗಳನ್ನು ತೋರಿಸುತ್ತದೆ:

```powershell
python client.py
```

ಇದು ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕ ಹೊಂದಿ ಉಪಕರಣಗಳು, ಸಂಪನ್ಮೂಲಗಳು ಮತ್ತು ಪ್ರಾಂಪ್ಟ್‌ಗಳ ಸೇರಿದಂತೆ ಎಲ್ಲಾ ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ಪ್ರಯೋಗಿಸುತ್ತದೆ. ಔಟ್‌ಪುಟ್‌ನಲ್ಲಿ ತೋರಿಸಲಾಗುವುದು:

1. ಕ್ಯಾಲ್ಕ್ಯುಲೇಟರ್ ಉಪಕರಣ ಫಲಿತಾಂಶ (5 + 7 = 12)
2. "ಜೀವನದ ಅರ್ಥವೇನು?" ಎಂಬ ಪ್ರಶ್ನೆಗೆ ಪೂರ್ಣಗೊಳಿಸುವಿಕೆ ಉಪಕರಣದ ಪ್ರತಿಕ್ರಿಯೆ
3. ಲಭ್ಯವಿರುವ AI ಮಾದರಿಗಳ ಪಟ್ಟಿ
4. "MCP ಎಕ್ಸ್‌ಪ್ಲೋರರ್" ಗೆ ವೈಯಕ್ತಿಕೃತ ಸ್ವಾಗತ
5. ಕೋಡ್ ಪರಿಶೀಲನೆ ಪ್ರಾಂಪ್ಟ್ ಟೆಂಪ್ಲೇಟು

## ಅನುಷ್ಠಾನ ವಿವರಗಳು

ಸರ್ವರ್ `FastMCP` API ಬಳಸಿ ಅನುಷ್ಠಾನಗೊಳಿಸಲಾಗಿದೆ, ಇದು MCP ಸೇವೆಗಳನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಲು ಉನ್ನತ ಮಟ್ಟದ ಅವಲೋಕನಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ. ಉಪಕರಣಗಳನ್ನು ಹೇಗೆ ವ್ಯಾಖ್ಯಾನಿಸುವುದೆಂಬ ಸರಳ ಉದಾಹರಣೆ ಇಲ್ಲಿದೆ:

```python
@mcp.tool()
def add(a: int, b: int) -> int:
    """Add two numbers together
    
    Args:
        a: First number
        b: Second number
    
    Returns:
        The sum of the two numbers
    """
    logger.info(f"Adding {a} and {b}")
    return a + b
```

ಕ್ಲೈಂಟ್ MCP ಕ್ಲೈಂಟ್ ಲೈಬ್ರರಿ ಬಳಸಿ ಸರ್ವರ್‌ಗೆ ಸಂಪರ್ಕ ಹೊಂದಿ ಕರೆಮಾಡುತ್ತದೆ:

```python
async with stdio_client(server_params) as (reader, writer):
    async with ClientSession(reader, writer) as session:
        await session.initialize()
        result = await session.call_tool("add", arguments={"a": 5, "b": 7})
```
